## 📐 故事五部分结构

```
┌─────┐   ┌─────────┐   ┌─────┐   ┌─────┐   ┌─────┐
│激励  │ → │进展纠葛  │ → │ 危机 │ → │ 高潮 │ → │ 结局 │
│事件  │   │         │   │     │   │     │   │     │
└─────┘   └─────────┘   └─────┘   └─────┘   └─────┘
```

## 一、激励事件 (Inciting Incident)

### 1. 定义
**激励事件是故事讲述的第一个重大事件**，是一切后续情节的首要导因，也是打破主人公生活平衡的决定性事件。

### 2. 发生方式
| 类型 | 说明 | 例子 |
|------|------|------|
| **随机（巧合）** | 天降横祸/天赐洪福 | 意外获得宝藏、突发灾难 |
| **有因（决定）** | 来自主人公的抉择 | 主动接受任务、做出承诺 |

### 3. 定位
- 主情节激励事件：**前1/4时段内**（电影前30分钟内）
- 主情节的激励事件是一个**"大钩子"**
- 每一个次情节都有自己的激励事件

### 4. 功能检查清单
- [ ] 能否打破主人公生活中各处力量的平衡？
- [ ] 能否激起主人公恢复平衡的欲望？
- [ ] 能否激发出自觉欲望和不自觉欲望？
- [ ] 能否将主人公送上一条满足欲望的求索之路？

### 5. 构思技巧
思考两个极端问题：
- **最好的是什么？如何变成最坏的？如何再次逆转到主人公的救赎？**
- **最坏的是什么？如何变成最好的？最后如何再次导致主人公的毁灭？**

> 💡 激励事件的冲击给我们创造了到达**生活极限**的机会。

---

## 二、进展纠葛 (Progressive Complications)

### 1. 纠葛 vs 进展纠葛
| 概念 | 定义 |
|------|------|
| **纠葛** | 为人物的生活制造磨难 |
| **进展纠葛** | 人物面对越来越强大的对抗力量，产生越来越多的冲突，朝着最后的行动向前推进 |

### 2. 冲突层面（三层冲突）

```
┌─────────────────────────────────────┐
│           个人-外界冲突              │  社会机构、环境、他人
├─────────────────────────────────────┤
│             个人冲突                 │  与亲密关系的冲突
├─────────────────────────────────────┤
│             内心冲突                 │  欲望、恐惧、矛盾
└─────────────────────────────────────┘
```

| 类型 | 冲突层面 |
|------|----------|
| **纠葛型** | 只发生于一个层面 |
| **复杂型** | 发生在所有三个层面（最佳） |

### 3. 冲突原则
> **若无冲突，故事中的一切都不可能向前发展。**

冲突力量构成：
- 内心冲突
- 个人冲突  
- 大千世界的冲突
- 终极问题（爱、价值、公平……）

## 三、故事的宏观结构——幕 (Acts)

### 三幕结构时间分配

```
         通常占25%，高潮发生                    最后一幕时间最短
         在20-30分钟之间                              ↓
              ↓
激励事件 → ┌────────┐      ┌────────┐      ┌────────┐
          │ 第一幕  │      │ 第二幕  │      │ 第三幕  │
主情节 ●──│────────│──────│────────│──────│────────│
          1分钟  30分钟   70分钟  100分钟  118分钟 120分钟
```

### 四个重大场景
一个三幕故事要求**四个重大场景**：
1. **激励事件**（故事开篇）
2. **第一幕高潮**
3. **第二幕高潮**
4. **第三幕高潮**（故事高潮）

### 次情节的幕结构
- 次情节A：可能只有一幕
- 次情节B：可能有两幕
- 次情节C：可能有三幕

## 四、场景设计 (Scene Design)

### 场景的本质
> **一个场景即是一个微缩的故事**，即在一个连续的时空中通过**冲突**而表现出来的、改变人物生活中**负载着价值的情境**的一个动作。

### 场景四要素
1. **转折点** - 惊奇、增强好奇心、见识和新方向
2. **伏笔/分晓** - 铺设伏笔、分晓揭示
3. **情感动力** - 快感与痛感的交替
4. **选择** - 两难之境的抉择

### 🔧 场景分析五步程序

| 步骤 | 内容 | 检查点 |
|------|------|--------|
| **第一步** | 确定冲突 | 谁驱动？他想要什么？对抗力量是什么？ |
| **第二步** | 标注开篇价值 | 价值是正面还是负面？ |
| **第三步** | 将场景分解为节拍 | 节拍=人物行为中动作/反应的交流 |
| **第四步** | 标注结尾价值 | 与开篇比较，价值是否发生变化？ |
| **第五步** | 概述节拍并标注转折点 | 找到期望与结果不一致的那一瞬间 |

### 节拍的定义
> **节拍**指的是人物行为中动作/反应的一种交流。

- 审查场景的第一个动作：用进行时短语表达，如"正在……"
- 这个短语**不仅能指称动作，还能表达人物感情感情**
- 当行为发生明显变化时，才会出现一个新的节拍

## 五、情感设计 (Emotional Design)

### 核心情感：快感 vs 痛感

| 快感 | 痛感 |
|------|------|
| 欢乐、爱情、幸福、狂喜、愉悦、销魂、刺激、至福 | 痛苦、害怕、焦虑、恐惧、悲伤、屈辱、病魔、凄凉、紧张、悔恨 |

### 情感递减定理
> **我们对某事体验越多，它所产生的效果就会越少。**

因此故事必须创造出**正面情感和负面情感之间的动态交替**。

### 情感 vs 感情 vs 基调

| 概念 | 定义 |
|------|------|
| **情感** | 短期体验，迅速达到高峰，然后燃烧殆尽 |
| **感情** | 长期的、具有渗透性的背景情绪，施加决定性影响 |
| **基调** | 观众获得的**总体感觉**，由光、色、动作、剪辑、配置、对白、音乐带来的心理反应 |

### 移情三要素
1. 首先，我们必须**移情于人物**
2. 其次，我们必须知道人物**想要什么**并希望他得到
3. 第三，我们必须明白人物生活中**利害攸关的价值**

## 六、选择的性质 (The Nature of Choice)

### 真正的选择是两难之择

| 情境 | 说明 |
|------|------|
| **情境一** | 不可调和的两善取其一的选择（两个都想要，但只能选一个） |
| **情境二** | 两恶取其轻的选择（两个都不想要，但必须选一个） |

> 💡 选择绝不是疑惑，必须是**两难**，不是在是非善恶之间，而必须是在具有**同等份量或价值**的两种正面欲望或两种负面欲望之间。

### 三方框架

```
        (人物A) ──────── 正面/中性/负面 ──────── (人物B)
                              +/-
                               │
                               │
                               │
                           (人物C)

选择是三方的，可以在两者中选其一：
A (+/-) + B (+/-) → C (+/-)
A (+/-) + C (+/-) → B (+/-)  
B (+/-) + C (+/-) → A (+/-)
```

## 七、危机 (Crisis)

### 危机的定义
> **危机是指决定。** 人物每次张嘴说"这个"而不说"那个"时，都是在作出自发的决定。**大写的危机是一个终极决定。**

### 危机 = 危险 + 机会
- 主人公的求索之路引导他通过纠葛进展，直到他采取了能实现欲望的几乎所有行动
- 只剩一个，他发现自己已经到达故事主线的终点
- 他的下一个行动将是他的**最后行动**

### "危险机会"的时刻
这一"危险机会"的时刻是故事中**最紧张的点**，大家都想知道结果将会如何。

### 危机必须是真正的两难之境
- 在不可调和的两善之间的选择
- 或对两恶之轻的选择
- 或同时面临两种情况的选择

> 💡 它能将过组人工置身于生活中**最大的压力**之下。

### 危机揭示人物
> **主人公在此如何选择恶役使我们对他的深层性格——其人性的终极表现——有一个最深刻的认识。**

---

## 八、高潮 (Climax)

### 高潮的意义
> **高潮必须充满意义。**

从正面到负面或者从负面到正面，或有反讽或无反讽的**价值剧变**——朝向绝对而又不可逆转的具有最大负荷的价值的摇摆。

**这一变化可以打动观众的心。**

### 高潮的效果
高潮一旦呼之欲出，故事便已经进入一种**从尾到头**而不是从头到尾的意义重大的改写状态。

> 💡 创造力的流动常常是**从结果到原因**。

### 结局的关键
> **所有故事结局的关键是给予观众他们想要的东西——情感的满足——一个满足预期的高潮。**

### 高潮中的危机
主人公选择采取的那一行动成为故事中**最重大的事件**，将导致一个正面的、负面的或者具有反讽意义的正面/负面的**故事高潮**。

### 高潮的定位
- 高潮的位置取决于**高潮动作的长度**
- 一般而言，危机和高潮都是在**最后时刻**才发生的
- 而且发生在**同一场景**

### 危机的设计
> **危机决定和高潮动作通常是在讲述过程的末尾发生在同一景点中一段连续的时间内。**

危机决定必须是一个**有意而为的静态时刻**。

---

## 九、结局 (Resolution)

### 结局在最高潮之后

### 结局的三个用途

| 用途 | 说明 |
|------|------|
| **用途一** | 提供一个机会来使次情节在主情节的高潮之前或之中达到高潮，需要一个属于它自己的场景 |
| **用途二** | 展示高潮效果的影响所及——如人物的变化和社会的影响等 |
| **用途三** | 表示对观众的尊重。如果高潮已经打动观众，结局的关键是创造一种**"壮观场面和真理"融为一体**的效果 |

## 🔧 实操检查清单

### 故事整体检查
- [ ] 激励事件是否在前1/4时段内？
- [ ] 激励事件是否打破主人公生活平衡？
- [ ] 进展纠葛是否有三层冲突（内心/个人/外界）？
- [ ] 危机是否是真正的两难选择？
- [ ] 高潮是否有价值剧变？
- [ ] 结局是否给予观众情感满足？

### 场景检查
- [ ] 场景有明确的冲突吗？
- [ ] 场景的价值从开篇到结尾有变化吗？
- [ ] 场景有转折点吗？
- [ ] 观众能移情于人物吗？

### 人物检查
- [ ] 人物有明确的欲望吗？
- [ ] 人物面临的选择是两难的吗？
- [ ] 人物的选择是否揭示了他的深层性格？

## 🎯 Prompt模板

当需要设计故事结构时，使用以下框架：

```
【故事结构设计】

1. 激励事件：
   - 发生了什么打破主人公生活平衡的事件？
   - 这个事件是随机还是有因？
   - 它激发了主人公什么欲望？

2. 进展纠葛：
   - 内心冲突：主人公内心有什么矛盾？
   - 个人冲突：与谁产生冲突？
   - 外界冲突：社会/环境带来什么阻碍？

3. 危机：
   - 主人公面临什么两难选择？
   - 两个选项分别是什么？
   - 为什么两个选项同等重要/痛苦？

4. 高潮：
   - 主人公做出了什么决定？
   - 这个决定导致了什么价值剧变？
   - 从正面到负面，还是负面到正面？

5. 结局：
   - 高潮之后发生了什么？
   - 人物/世界有什么变化？
   - 观众获得了什么情感满足？
```
