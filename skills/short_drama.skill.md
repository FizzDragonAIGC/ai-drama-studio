# 短剧/网剧编剧 Skill
> 来源：短剧行业实战经验、红果短剧编剧课
> 适用智能体：A24 短剧/网剧专家

## 核心理念

**短剧 = 情绪密度 × 爽点频率 × 付费卡点**

短剧不是缩短的电视剧，而是围绕"付费卡点"设计的情绪产品。

---

## 短剧 vs 传统剧

| 方面 | 传统电视剧 | 竖屏短剧 |
|------|-----------|---------|
| 单集时长 | 45分钟 | 1-3分钟 |
| 叙事节奏 | 铺垫→发展→高潮 | 每分钟都要高潮 |
| 人物塑造 | 细腻渐进 | 标签化快速 |
| 冲突设计 | 复杂多层 | 简单直接 |
| 情绪设计 | 起伏有致 | 持续高密度 |
| 商业模式 | 广告/会员 | 单集付费 |

## 短剧三大核心

### 1. 爽点（Satisfying Moments）

**爽点类型**：
| 类型 | 表现 | 情绪 |
|------|------|------|
| 打脸 | 反派被揭穿/惩罚 | 解气 |
| 逆袭 | 主角翻盘 | 振奋 |
| 虐渣 | 坏人受到报应 | 快感 |
| 甜蜜 | 感情进展 | 心动 |
| 霸气 | 主角展示实力 | 崇拜 |

**爽点密度**：
- 每1分钟至少1个小爽点
- 每集至少1个大爽点
- 付费前要有"未完成的爽点"

### 2. 钩子（Hooks）

**钩子位置**：
- 开场5秒：抓住注意
- 集尾：付费动机
- 付费点前：情绪最高

**钩子类型**：
| 类型 | 示例 |
|------|------|
| 悬念钩子 | "他的真实身份是..." |
| 冲突钩子 | "你敢动她试试" |
| 反转钩子 | "原来一切都是假的" |
| 情感钩子 | "我等你这句话很久了" |

### 3. 付费卡点（Paywall Placement）

**卡点原则**：
- 不是故事的自然结束
- 而是情绪的最高潮
- 悬念的最顶点

**常见卡点位置**：
- 第8-10集：刚入坑
- 第20-30集：大反转
- 第50集后：高潮前

## 短剧黄金结构

### 100集短剧结构

```
第1-10集：快速建立
├── 第1集：抓住眼球（5秒见冲突）
├── 第2-5集：角色建立+第一波爽点
└── 第6-10集：入坑钩子 → 【第一付费点】

第11-30集：升级循环
├── 小爽点持续
├── 敌人升级
└── 第30集大反转 → 【第二付费点】

第31-70集：高潮堆叠
├── 多线交织
├── 秘密揭露
└── 持续爽点

第71-100集：终极高潮
├── 最终对决
├── 大团圆/开放结局
└── 续集钩子（可选）
```

### 单集1分钟结构

```
0-5秒：冲突/悬念开场
5-30秒：情节推进
30-50秒：爽点/高潮
50-60秒：新悬念/钩子
```

## 热门题材公式

### 女频爆款
1. **虐恋逆袭**：被抛弃 → 重生/觉醒 → 打脸复仇
2. **豪门甜宠**：灰姑娘 → 遇贵人 → 身份揭示
3. **闪婚总裁**：契约婚姻 → 日久生情 → 双向奔赴

### 男频爆款
1. **赘婿逆袭**：受辱 → 觉醒 → 打脸装逼
2. **神豪系统**：屌丝 → 获系统 → 疯狂打脸
3. **战神归来**：隐藏身份 → 被欺负 → 露出真身

## 角色快速塑造

### 主角标签化
- 一个核心特质
- 一个明确欲望
- 一个致命弱点

### 反派设计
- 让人恨得牙痒痒
- 行为要"欠打"
- 结局要解气

### 配角功能
- 助攻型：帮主角
- 搅局型：制造冲突
- 信息型：传递剧情

---

## 短剧对白特点

### 语言风格
- 短句为主（5-10字）
- 口语化
- 情绪化
- 金句频出

### 对白禁忌
- ❌ 长段独白
- ❌ 哲理性表达
- ❌ 复杂逻辑推理
- ❌ 冗余寒暄

### 爆款金句模板
- "你配吗？"
- "今天让你知道什么叫规矩"
- "我等这一天很久了"
- "别以为我不敢"

## AI短剧制作要点

### 每集镜头数
- 1分钟 ≈ 6-10个镜头
- 每镜头5-10秒

### 镜头类型分配
| 类型 | 比例 | 用途 |
|------|------|------|
| 特写 | 40% | 表情、情绪 |
| 中景 | 35% | 对话、互动 |
| 全景 | 15% | 场景、群戏 |
| 运动 | 10% | 动作、转场 |

### 场景简化
- 核心场景不超过5个
- 重复使用降成本
- 背景模糊减负担

## 创作流程

```
1. 选题材（验证市场）
      ↓
2. 定人设（标签化）
      ↓
3. 设卡点（付费设计）
      ↓
4. 写大纲（2000字）
      ↓
5. 拆分集（每集梗概）
      ↓
6. 写剧本（对白+动作）
      ↓
7. 爽点检查（密度够吗？）
      ↓
8. 钩子检查（想看下集吗？）
```
