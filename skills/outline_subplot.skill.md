# 📝 副線編織技能

## 元信息
- **技能ID**: outline_subplot
- **所屬Agent**: 大綱Agent
- **參考書籍**: 《故事》麥基
- **難度**: ⭐⭐⭐⭐

## 核心理念

副線 = 主線的延伸和對照
- 副線必須與主題相關
- 副線豐富主角/主題
- 副線不是「另一個故事」

## 副線類型

### 1. 愛情副線
```
功能：軟化主角/增加賭注
與主題關係：從另一角度探討主題
例：動作片中的愛情線
```

### 2. 配角成長線
```
功能：豐富世界/對照主角
與主題關係：配角走不同路徑
例：朋友的平行故事
```

### 3. 反派線
```
功能：製造威脅/複雜化衝突
與主題關係：反派代表反主題
例：反派的計劃推進
```

### 4. 懸疑線
```
功能：保持好奇/增加層次
與主題關係：謎底揭曉支持主題
例：逐漸揭露的秘密
```

### 5. 喜劇緩和線
```
功能：調節情緒/增加趣味
與主題關係：用輕鬆方式探討主題
例：搞笑配角的故事
```

## 副線權重

### 12集番劇建議
```
A線（主線）：70%的螢幕時間
B線（主要副線）：20%
C線（次要副線）：10%

規則：
- 不超過3條明確副線
- 副線必須與主線交叉
- 副線有自己的起承轉合
```

## 副線與主線的交織

### 平行交叉
```
集1：A線場景1 → B線場景1 → A線場景2
集2：A線場景3 → B線場景2 → A線場景4

兩條線獨立推進，偶爾交匯
```

### 因果交叉
```
A線事件 → 影響B線 → B線反應 → 影響A線

兩條線互相影響
```

### 匯合型
```
A線和B線各自發展
→ 中點或高潮時匯合
→ 成為同一條線
```

## 副線節奏

### 副線進度安排
```
集1-2：A線為主，B線引入
集3-4：A線發展，B線發展
集5-6：A線升級，B線到達中點
集7-8：A線中點，B線開始與A線交叉
集9-10：A線危機，B線支援/複雜化
集11-12：A+B線匯合，共同高潮
```

### 副線payoff
```
規則：
- 副線必須有結局
- 副線結局應該在主線之前或同時
- 不能讓副線無頭無尾
```

## 副線設計清單

```
□ 這條副線與主題有關嗎？
□ 這條副線讓主角更豐富嗎？
□ 這條副線有自己的弧光嗎？
□ 這條副線會與主線交叉嗎？
□ 這條副線有清晰的結局嗎？
□ 刪除這條副線會損失什麼？
```
