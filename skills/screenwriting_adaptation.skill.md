# 📚 文学改编 Skill
> 来源：《Adaptation》Linda Seger / 《The Art of Adaptation》

---

## 📐 改编核心原则

### 忠实 vs 背叛
> "忠实于原作精神，而非字面。"

| 层面 | 可改 | 不可改 |
|------|------|--------|
| 情节 | ✅ 删减/合并 | ❌ 核心事件 |
| 人物 | ✅ 减少数量 | ❌ 主角本质 |
| 对白 | ✅ 现代化 | ❌ 金句/名言 |
| 主题 | ⚠️ 可微调 | ❌ 核心主题 |
| 结局 | ⚠️ 需谨慎 | ❌ 悲喜性质 |

---

## 🎯 改编六大策略

### 1. 提取核心 (Extract Core)
找到故事的"不可改变的内核"

**问自己**：
- 这个故事最重要的是什么？
- 删掉什么读者会愤怒？
- 什么是这个故事独有的？

### 2. 视觉化 (Visualize)
把内心描写变成可看的画面

| 原著 | 改编 |
|------|------|
| "她很紧张" | 手指不停敲桌子 |
| "他深爱着她" | 默默为她准备早餐 |
| "城市在衰落" | 空荡的街道，关闭的商店 |

### 3. 压缩 (Compress)
小说的内容太多，必须取舍

**压缩方法**：
- 合并相似人物
- 删除副线
- 压缩时间跨度
- 删除过渡场景

### 4. 扩展 (Expand)
短篇需要扩展

**扩展方法**：
- 增加人物背景
- 发展配角
- 增加障碍
- 丰富世界观

### 5. 结构重组 (Restructure)
小说结构不一定适合电影

**常见调整**：
- 线性→倒叙
- 多视角→单视角
- 慢热→动作开场

### 6. 现代化 (Modernize)
让古典作品与当代观众连接

**方法**：
- 更新时代背景
- 调整文化元素
- 强化普世主题

## 📊 改编难度评估

### 改编友好型
- 类型明确（惊悚、悬疑）
- 外部动作多
- 对白精彩
- 结构清晰

### 改编困难型
- 意识流
- 大量内心独白
- 非线性叙事
- 哲学/抽象主题
- 多时间线

## 🎭 人物改编

### 人物删减原则
| 保留 | 删除/合并 |
|------|----------|
| 主角 | 功能重复的配角 |
| 核心关系 | 单一功能人物 |
| 反派 | 过多家族成员 |
| 关键配角 | 过渡性人物 |

### 人物合并技巧
```
原著：A提供信息 + B给予鼓励 + C揭示秘密
改编：合并为一个"导师"角色，完成三个功能
```

## 📋 改编检查清单

### 分析原著
- [ ] 核心主题是什么？
- [ ] 不可删除的情节？
- [ ] 主角的人物弧光？
- [ ] 最精彩的场景？
- [ ] 读者最爱的元素？

### 改编决策
- [ ] 保留什么？
- [ ] 删除什么？
- [ ] 合并什么？
- [ ] 改变什么？
- [ ] 增加什么？

### 验证改编
- [ ] 精神是否忠实？
- [ ] 电影独立成立？
- [ ] 原著粉能接受？
- [ ] 新观众能理解？

## 🔧 改编工作流程

### Phase 1：分析 (1-2周)
```
1. 读原著3遍以上
2. 标记关键场景
3. 提取核心元素
4. 分析结构
5. 确定主题
```

### Phase 2：决策 (1周)
```
1. 确定改编策略
2. 决定保留/删除
3. 设计结构
4. 规划人物
```

### Phase 3：创作 (4-8周)
```
1. 写大纲
2. 写处理/剧本
3. 对照原著检查
4. 修改调整
```

## 🎬 改编类型

### 忠实改编
尽可能接近原著
- **例子**：《指环王》《哈利波特》
- **适用**：经典作品，粉丝基础大

### 自由改编
借用概念，大幅修改
- **例子**：《银翼杀手》《闪灵》
- **适用**：作者同意，或版权允许

### 现代改编
更新时代背景
- **例子**：《克鲁拉》《狮子王》
- **适用**：公版作品，经典重述

### 类型转换
改变类型/风格
- **例子**：《傲慢与偏见与僵尸》
- **适用**：创意改编

---

## 📖 公版文学改编要点

### 优势
- 无版权费用
- 知名度高
- 创作自由度大

### 注意点
- 与现有改编区分
- 找到新角度
- 保持经典魅力

### 常见改编角度
| 角度 | 例子 |
|------|------|
| 换视角 | 从配角讲述 |
| 换时代 | 现代/未来版 |
| 换类型 | 加入新元素 |
| 前传/后传 | 延展故事 |
| 换性别 | 性别转换 |
