## 📐 竖屏基础参数

竖屏与横屏的根本差异：

| 参数 | 横屏(16:9) | 竖屏(9:16) | 变化 |
|------|-----------|-----------|------|
| 宽高比 | 1.78:1 | 0.56:1 | 倒置 |
| 分辨率 | 1920×1080 | 1080×1920 | 倒置 |
| 视野 | 宽广 | 纵深 | 聚焦 |
| 主体 | 多人/环境 | 单人/特写 | 集中 |
| 信息区 | 自由 | 上下留白 | 受限 |

## 🎯 竖屏构图原则

### 1. 垂直主导
| 横屏思维 | 竖屏思维 |
|----------|----------|
| 横向排列 | 纵向排列 |
| 宽场景 | 纵深场景 |
| 横向运动 | 纵向运动 |
| 左右构图 | 上下构图 |

### 2. 中心聚焦
```
┌─────────────┐
│   安全区    │ ← 字幕/标题区
├─────────────┤
│             │
│    主体     │ ← 核心内容区
│    区域     │
│             │
├─────────────┤
│   安全区    │ ← 互动按钮区
└─────────────┘
```

### 3. 三分法（竖版）
```
┌─────────────┐
│     1/3     │ ← 天空/背景
├─────────────┤
│     1/3     │ ← 人物主体
├─────────────┤
│     1/3     │ ← 地面/前景
└─────────────┘
```

## 🎬 竖屏构图类型

### 人物构图
| 景别 | 构图 | AI Prompt | 适用 |
|------|------|-----------|------|
| 特写 | 脸部填充画面 | extreme close-up, face filling frame, vertical | 情绪表达 |
| 近景 | 胸部以上 | close-up, upper body, 9:16 aspect ratio | 对话 |
| 中景 | 膝部以上 | medium shot, vertical composition | 动作 |
| 全身 | 头顶到脚 | full body, vertical frame | 展示 |

### 双人构图
| 类型 | 描述 | AI Prompt |
|------|------|-----------|
| 上下排列 | 一人在上一人在下 | two people, vertical arrangement |
| 前后纵深 | 前景后景 | foreground background, depth |
| 对角线 | 斜向对位 | diagonal composition, two subjects |
| 分屏 | 上下各半 | split screen, vertical |

### 场景构图
| 类型 | 描述 | AI Prompt |
|------|------|-----------|
| 纵深走廊 | 向远方延伸 | corridor perspective, vertical depth |
| 高楼仰视 | 向上看建筑 | low angle, looking up, tall building |
| 楼梯/电梯 | 垂直空间 | staircase, vertical space |
| 瀑布/高塔 | 自然纵向 | waterfall, tower, vertical element |

## 📱 安全区域

### 平台UI避让
```
┌─────────────────────┐
│ ████████████████ │ ← 顶部状态栏(约5%)
│                     │
│                     │
│    有效内容区域     │ ← 中间85%
│                     │
│                     │
│     @账号名称       │ ← 底部信息(约5%)
│ ♡ 💬 ➤ ⋮          │ ← 互动按钮(约5%)
└─────────────────────┘
```

### 关键信息位置
| 区域 | 位置 | 适合放置 |
|------|------|----------|
| 上10% | 顶部 | 可被遮挡的内容 |
| 中70% | 核心 | 主要画面/人物 |
| 下20% | 底部 | 避免重要信息 |

## 🎯 竖屏运镜

### 适合竖屏的运镜
| 运镜 | 描述 | 效果 |
|------|------|------|
| 垂直摇 | 上下摇动 | 展示高度 |
| 纵向推 | 向前推进 | 走入场景 |
| 升降 | 垂直升降 | 气势/转场 |
| 跟拍 | 跟随移动 | 动态感 |

### 避免的运镜
| 运镜 | 问题 |
|------|------|
| 大横摇 | 画面信息流失 |
| 宽横移 | 不适合竖屏 |
| 过快平移 | 模糊/眩晕 |

## 🎨 竖屏美学特点

### 利用纵向空间
| 技法 | 描述 | 示例 |
|------|------|------|
| 纵向引导线 | 利用垂直线条 | 路灯、树木、建筑 |
| 上下对比 | 天地对比 | 天空vs地面 |
| 层次叠加 | 前中后景 | 人物前景+环境后景 |
| 镜像构图 | 上下对称 | 水面倒影 |

### 增强纵深
| 技法 | 描述 |
|------|------|
| 前景元素 | 添加近处物体 |
| 景深控制 | 主体清晰背景虚 |
| 透视线 | 汇聚于画面中心 |

## 🚫 竖屏常见错误

| 错误 | 问题 | 解决 |
|------|------|------|
| 横屏素材裁切 | 主体不完整 | 重新拍摄/构图 |
| 过多留白 | 画面空洞 | 填充有意义内容 |
| 字幕遮挡 | 盖住人物 | 预留字幕区 |
| 横向排列 | 人小看不清 | 改为纵向排列 |
| 忽视安全区 | 被UI遮挡 | 遵守安全区 |
