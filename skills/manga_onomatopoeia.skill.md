## 📐 拟声词基础

### 日语拟声词分类

| 类型 | 日语 | 描述 | 例子 |
|------|------|------|------|
| **拟声语** | 擬音語 | 实际声音 | ドカン(爆炸)、ザー(雨声) |
| **拟态语** | 擬態語 | 状态/感觉 | キラキラ(闪亮)、ドキドキ(心跳) |

## 🎯 常用拟声词速查

### 动作类

| 日语 | 发音 | 含义 | 场景 |
|------|------|------|------|
| **ドン！** | DON | 冲击/登场 | 强势登场 |
| **バン！** | BAN | 撞击/枪声 | 打击、开枪 |
| **ドカ！** | DOKA | 重击 | 拳击、踢打 |
| **バキ！** | BAKI | 骨折声 | 暴力打击 |
| **ガッ！** | GA | 抓取 | 抓住某物 |
| **ザッ！** | ZA | 落地/停止 | 急停 |
| **シュッ！** | SHU | 快速移动 | 挥剑、冲刺 |
| **ドドド** | DODODO | 持续轰鸣 | 逼近压力 |

| 日语 | 发音 | 含义 | 场景 |
|------|------|------|------|
| **ドキドキ** | DOKIDOKI | 心跳 | 紧张、心动 |
| **ゴゴゴ** | GOGOGO | 威压 | 气场、杀意 |
| **シーン** | SHIIN | 寂静 | 尴尬沉默 |
| **ガーン** | GAAN | 震惊 | 打击受挫 |
| **ビクッ** | BIKU | 惊吓 | 被吓到 |
| **ニヤ** | NIYA | 奸笑 | 阴险微笑 |
| **キラキラ** | KIRAKIRA | 闪亮 | 美丽、闪耀 |

### 环境类

## 📊 字体风格与情感

| 情感 | 字体风格 | 边缘处理 |
|------|----------|----------|
| **愤怒** | 粗重、锐利 | 尖刺、锯齿 |
| **恐惧** | 颤抖、扭曲 | 不规则 |
| **可爱** | 圆润、柔软 | 圆角、气泡 |
| **冲击** | 超大、立体 | 放射线 |
| **神秘** | 细长、古典 | 阴影 |
| **搞笑** | 夸张、变形 | 波浪、弹跳 |

## 🔧 拟声词设计要素

### 尺寸与重要性
```
超大 ═══════════════════ 核心冲击
 │
大   ═══════════════ 重要效果
 │
中   ═════════ 普通效果
 │
小   ═════ 背景音
 │
微   ═══ 环境细节
```

### 字体变形
```
【扩张型】           【收缩型】
  ド                   ド
 ド ド                ドド
ドドドド              ドドドドド
爆发、扩散            聚焦、冲击

【倾斜型】           【震动型】
   ド                 ド̴̤
  ド                  ド̵̤
 ド                   ド̴̤
速度、方向            颤抖、恐惧
```

## 🎬 拟声词位置布局

### 冲击效果
```
┌─────────────────────────────┐
│     ド                      │
│       カ                    │
│         ン！！               │
│           【冲击点】         │
│                             │
└─────────────────────────────┘
```
沿冲击方向排列

### 包围效果
```
┌─────────────────────────────┐
│ ゴ                      ゴ  │
│   ゴ    【角色】      ゴ    │
│     ゴ            ゴ        │
│ ゴ      威压感        ゴ    │
└─────────────────────────────┘
```
环绕增强压迫感

### 背景效果
```
┌─────────────────────────────┐
│ザーザーザーザーザーザーザー │
│  【角色】                   │
│ザーザーザーザーザーザーザー │
│     雨声铺满背景            │
└─────────────────────────────┘
```
