# 高概念设计 (High Concept Design)

## 概述
高概念是用一句话概括整个故事的核心卖点，是好莱坞判断项目商业价值的关键工具。

## 核心理论

### 什么是高概念
- **定义**：能用一句话（25字以内）说清楚的故事前提
- **特点**：新颖、有冲突、有画面感、有商业潜力
- **公式**：[熟悉元素] + [新颖转折] = 高概念

### 高概念三要素
1. **钩子 (Hook)** - 让人立刻想知道更多
2. **讽刺性 (Irony)** - 内在矛盾或反差
3. **视觉化 (Visual)** - 能立刻想象出画面

### 经典高概念案例
- 《侏罗纪公园》：科学家用恐龙DNA复活恐龙，建了一个主题公园
- 《泰坦尼克》：穷小子和富家女在沉船上的爱情
- 《让子弹飞》：土匪冒充县长去当官
- 《流浪地球》：人类带着地球逃离太阳系

## 高概念拆解流程

### 第一步：识别核心冲突
- 主角想要什么？
- 什么阻止他？
- 风险是什么？

### 第二步：提炼独特卖点
- 这个故事和其他故事有什么不同？
- 什么是从未见过的？
- 观众为什么要看？

### 第三步：验证商业潜力
- 目标观众是谁？
- 类似题材的市场表现？
- 是否有续集潜力？

### 第四步：一句话总结
模板：
- 当[主角]遇到[事件]，他必须[行动]，否则[后果]
- [职业/身份]的[主角]发现[秘密]，必须在[时限]内[目标]
- 在[世界观]中，[主角]是唯一能[特殊能力]的人

## Prompt模板

### 高概念提取
```
分析以下故事，提取高概念：

1. 核心前提（一句话）：
2. 钩子元素：
3. 独特卖点：
4. 目标观众：
5. 类比参考：「X meets Y」格式
```

### 高概念验证
```
验证这个高概念的商业潜力：

高概念：[内容]

检查清单：
□ 能用一句话说清吗？
□ 有内在矛盾/讽刺吗？
□ 能立刻想象画面吗？
□ 观众会好奇后续吗？
□ 有明确目标观众吗？
```

## 案例分析

### 《小王子的打工日记》高概念拆解
- **一句话**：小王子的星球被AI接管后，他来地球当快递员
- **钩子**：童话人物+现代打工人
- **讽刺**：纯真王子vs996社畜
- **视觉化**：小王子穿快递服骑电动车
- **目标观众**：年轻打工人
- **类比**：《小王子》meets《中国奇谭》
