## 📐 上瘾模型 (Hook Model)

### 四步循环
```
┌────────────────────────────────────────┐
│                                        │
│    触发 ──→ 行动 ──→ 奖励 ──→ 投入    │
│     ↑                           │      │
│     └───────────────────────────┘      │
│                                        │
└────────────────────────────────────────┘
```

### 应用到番剧
| 阶段 | 含义 | 番剧应用 |
|------|------|----------|
| 触发 | 让人想起你 | 悬念、更新提醒 |
| 行动 | 简单开始 | 一键播放、短集 |
| 奖励 | 获得满足 | 爽点、燃点、甜点 |
| 投入 | 付出时间 | 看完、追更、评论 |

## 🎯 追剧成瘾设计

### 1. 悬念成瘾
```
技巧：
- 每集结尾留悬念
- 重要揭示放下一集
- 多条线交织
- 让观众猜测

心理原理：
蔡格尼克效应——未完成的事情会一直惦记
```

### 2. 情感成瘾
```
技巧：
- 让观众爱上角色
- 持续给糖/给虐
- 制造情感起伏
- CP发展吊人

心理原理：
情感投资越多，越难割舍
```

### 3. 升级成瘾（爽文模式）
```
技巧：
- 持续成长可见
- 打脸-升级循环
- 新能力/新境界
- 更强的敌人

心理原理：
成就感+多巴胺
```

### 4. 社交成瘾
```
技巧：
- 制造讨论话题
- 设置争议点
- CP党之争
- 剧透管理

心理原理：
社交货币，参与感
```

## 📊 奖励类型

### 即时奖励
| 奖励 | 效果 | 应用 |
|------|------|------|
| 爽感 | 多巴胺 | 打脸、逆袭 |
| 笑点 | 愉悦 | 搞笑场面 |
| 甜蜜 | 心动 | 撒糖、暧昧 |
| 感动 | 催泪 | 真情、牺牲 |

### 延迟奖励
| 奖励 | 效果 | 应用 |
|------|------|------|
| 伏笔回收 | 满足感 | 前后呼应 |
| 真相揭示 | 恍然大悟 | 悬念解开 |
| CP确认 | 期待满足 | 终于在一起 |
| 复仇成功 | 大快人心 | 大Boss倒台 |

### 奖励节奏
```
短视频：每30秒一个小奖励
普通剧：每3-5分钟一个奖励
长剧：每集多个小奖励+一个大奖励
```

## 🎭 间歇性强化

### 最强成瘾机制
```
不是每次都给奖励
有时候给，有时候不给
→ 期待感最强

应用：
- 甜宠剧：不是每次互动都发糖
- 悬疑剧：不是每次都有进展
- 爽文：有时候主角也会吃亏
```

### 变量奖励类型
| 类型 | 说明 | 例子 |
|------|------|------|
| 族群奖励 | 社交互动 | CP互动、战友情 |
| 猎物奖励 | 获得资源 | 升级、获得宝物 |
| 自我奖励 | 掌控感 | 主角变强、掌控局面 |

## 📋 成瘾设计清单

### 每集检查
- [ ] 有没有让人想看下一集的悬念？
- [ ] 有没有至少3个情绪奖励点？
- [ ] 有没有让角色更讨喜的时刻？
- [ ] 观众投入了情感吗？
- [ ] 有没有可讨论的话题？

### 整体检查
- [ ] 有没有成长/进展可见？
- [ ] 有没有持续的悬念线？
- [ ] 有没有变量奖励（有时给有时不给）？
- [ ] 有没有社交分享点？
- [ ] 有没有角色情感投资？

## 🔧 成瘾公式

### 单集公式
```
开篇钩子
↓
小奖励（3-5分钟一个）
↓
中奖励（中点高潮）
↓
大奖励（接近结尾）
↓
悬念钩子（结尾）
```

### 系列公式
```
第1集：大量奖励，建立期待
第2-3集：建立投资，制造牵挂
中期：起伏，有收有放
后期：提升赌注，加强悬念
大结局：释放所有积累的期待
```
