## 標誌性技法

### 1. 步移鏡頭（Step-printing）
```
特徵：抽幀造成的殘影效果
效果：時間凝固、記憶質感
應用：都市遊蕩、情緒高潮

Prompt關鍵詞：
motion blur, step-printing effect,
streaky movement, time-lapse trails
```

### 2. 框中框構圖
```
特徵：用窗戶、門框、鏡子裝框
效果：偷窺感、距離感、孤獨
應用：親密場景、回憶

Prompt關鍵詞：
framed through doorway, window frame,
mirror reflection, voyeuristic angle
```

### 3. 霓虹光影
```
特徵：霓虹燈、混合光源
效果：都市感、曖昧、頹廢美
應用：夜景、酒吧、街道

Prompt關鍵詞：
neon lights, mixed color lighting,
red and green neon, Hong Kong night
```

### 4. 慢動作特寫
```
特徵：放慢的動作、聚焦細節
效果：時間靜止、凝視
應用：關鍵動作、情感時刻

Prompt關鍵詞：
slow motion, close-up detail,
frozen moment, intimate gaze
```
