## 📐 注意力衰减曲线

观众注意力随时间**指数衰减**：

| 时间点 | 注意力水平 | 流失率 |
|--------|-----------|--------|
| 0-1秒 | 100% | 基准 |
| 1-3秒 | 70% | 30%流失 |
| 3-7秒 | 50% | 50%流失 |
| 7-15秒 | 35% | 65%流失 |
| 15-30秒 | 25% | 75%流失 |
| 30-60秒 | 20% | 80%流失 |

**关键点**：每个时间节点都需要新的刺激来挽留观众

## 🎯 注意力维持机制

### 1. 新奇刺激
| 类型 | 示例 | 频率 |
|------|------|------|
| 视觉变化 | 场景切换/角度变化 | 每3-5秒 |
| 信息投放 | 新角色/新信息 | 每5-10秒 |
| 情绪波动 | 情绪转折 | 每10-15秒 |
| 悬念更新 | 新问题产生 | 每15-30秒 |

### 2. 情感钩子
| 情感 | 触发机制 | 注意力效果 |
|------|----------|-----------|
| 好奇 | 信息缺口 | 持续关注 |
| 期待 | 预期奖励 | 等待揭晓 |
| 共鸣 | 自我投射 | 情感投入 |
| 紧张 | 威胁存在 | 警觉状态 |

### 3. 认知参与
| 方式 | 描述 | 效果 |
|------|------|------|
| 猜测 | 让观众预测 | 验证欲望 |
| 选择 | 心理站队 | 结果关心 |
| 解谜 | 提供线索 | 思考参与 |
| 回忆 | 关联经验 | 个人连接 |

## 🎬 注意力设计节点

### 30秒短剧
```
0s  → [Hook]视觉冲击
3s  → [Info]人物/情境建立
8s  → [Turn]冲突产生
15s → [Peak]情绪高点
22s → [Twist]反转/意外
28s → [End]悬念/留白
```

### 1分钟短剧
```
0s  → [Hook]强力开场
5s  → [Setup]快速建置
15s → [Conflict]冲突展开
30s → [Escalate]冲突升级
45s → [Climax]高潮
55s → [End]结局/悬念
```

### 3分钟短剧
```
0s   → [Hook]
10s  → [Setup]
30s  → [First Turn]
60s  → [Development]
90s  → [Second Turn]
120s → [Climax]
150s → [Resolution]
170s → [Cliffhanger]
```

## 📋 注意力刺激类型

### 视觉刺激
| 类型 | 描述 | AI Prompt |
|------|------|-----------|
| 运动 | 快速移动元素 | dynamic movement, action |
| 对比 | 明暗/颜色对比 | high contrast, dramatic lighting |
| 面孔 | 人脸特写 | close-up face, expressive |
| 反常 | 不寻常画面 | unusual, unexpected visual |

### 听觉刺激
| 类型 | 描述 | 效果 |
|------|------|------|
| 突然音效 | 意外声响 | 惊醒注意 |
| 音乐变化 | 节奏/旋律变化 | 情绪引导 |
| 静音 | 突然安静 | 紧张感 |
| 人声 | 对白/呼喊 | 聚焦信息 |

### 情节刺激
| 类型 | 描述 | 效果 |
|------|------|------|
| 意外 | 出乎意料 | 重新聚焦 |
| 威胁 | 危险出现 | 紧张关注 |
| 情感 | 感人时刻 | 情感投入 |
| 悬念 | 未解答问题 | 等待揭晓 |

## 🚫 注意力杀手

| 问题 | 表现 | 解决方案 |
|------|------|----------|
| 节奏拖沓 | 无事发生 | 删减/加速 |
| 信息重复 | 反复说同一件事 | 精简合并 |
| 视觉单调 | 固定镜头太久 | 切换角度 |
| 逻辑混乱 | 看不懂在干嘛 | 简化叙事 |
| 情感平淡 | 没有波动 | 增加冲突 |
| 预期过度 | 猜到结局 | 增加意外 |
