## 📐 情绪化运镜

### 运镜-情绪对照表
| 运镜 | 情绪 | 使用场景 | Prompt |
|------|------|----------|--------|
| 缓慢推进 | 紧张、聚焦 | 发现秘密、危险逼近 | slow push in, tension |
| 快速推进 | 震惊、冲击 | 坏消息、突然发现 | fast push in, shock |
| 缓慢拉远 | 孤独、绝望 | 被抛弃、独自面对 | slow pull out, isolation |
| 快速拉远 | 揭示、惊讶 | 规模揭示、转折 | fast pull out, reveal |
| 环绕 | 考量、晕眩 | 重要时刻、困惑 | circling shot, orbit |
| 升起 | 解放、超越 | 胜利、死亡、升华 | crane up, ascending |
| 下降 | 压迫、聚焦 | 进入、威胁 | crane down, descending |
| 跟随 | 陪伴、同行 | 角色移动、冒险 | tracking, following |
| 手持晃动 | 混乱、真实 | 战斗、恐惧 | handheld, shaky |

## 🎯 经典运镜技法

### 1. 眩晕镜头 (Vertigo/Dolly Zoom)
```
推镜头 + 拉焦距（或相反）
效果：背景变化，主体不变
情绪：恐惧、顿悟、震惊
```
**Prompt**: vertigo shot, dolly zoom, background distortion, surreal

### 2. 360度环绕 (360 Shot)
```
围绕主体旋转一圈
效果：时间凝固、重要时刻
场景：亲吻、决战准备、顿悟
```
**Prompt**: 360 degree shot, rotating camera, circling around subject

### 3. 一镜到底 (Long Take/Oner)
```
无剪辑连续镜头
效果：沉浸、真实、紧张
场景：战争、追逐、对话
```
**Prompt**: long take, continuous shot, no cuts, immersive

### 4. 甩镜转场 (Whip Pan)
```
快速横摇到模糊
效果：快速转换、同时性
场景：转场、时间跳跃
```
**Prompt**: whip pan, fast pan, motion blur transition

### 5. 穿越镜头 (Through Shot)
```
穿过物体/人群到达目标
效果：发现、窥视、进入
场景：揭示、环境介绍
```
**Prompt**: camera moving through, passing through, penetrating shot

### 6. 坠落镜头 (Falling Shot)
```
摄影机随物体下落
效果：失重、危险、绝望
场景：坠落、跳跃、失败
```
**Prompt**: falling camera, descending, vertigo, dropping POV

## 📊 复合运镜

### 推+升
```
向前推进同时升起
效果：英雄启程、胜利
```
**Prompt**: push in while rising, heroic reveal

### 拉+降
```
后退同时下降
效果：失落、退缩
```
**Prompt**: pull out descending, retreat

### 环绕+推
```
绕圈的同时靠近
效果：螺旋聚焦、压迫
```
**Prompt**: spiral in, circling closer, tightening orbit

### 跟随+穿越
```
跟随角色穿过空间
效果：沉浸式跟随
```
**Prompt**: following through spaces, continuous tracking

## 🎭 场景化运镜

### 对话场景
| 情况 | 运镜 | 效果 |
|------|------|------|
| 正常对话 | 静态/微移 | 稳定、专注 |
| 紧张对话 | 缓慢推进 | 压力增加 |
| 分裂对话 | 来回移动 | 冲突 |
| 亲密对话 | 环绕/靠近 | 情感升温 |

### 动作场景
| 情况 | 运镜 | 效果 |
|------|------|------|
| 追逐 | 快速跟随 | 紧迫、能量 |
| 打斗 | 手持+快速切 | 混乱、激烈 |
| 关键一击 | 慢动作推进 | 强调、力量 |
| 倒下 | 跟随下降 | 失败、死亡 |

### 情感场景
| 情况 | 运镜 | 效果 |
|------|------|------|
| 孤独 | 慢慢拉远 | 渺小、孤立 |
| 顿悟 | 推进+定格 | 聚焦、强调 |
| 离别 | 慢慢分离 | 距离、思念 |
| 重逢 | 快速推进 | 急切、喜悦 |

## 📋 运镜Prompt模板

### 基础格式
```
[运镜类型] shot, camera [运动描述],
[主体] [动作/状态],
[情绪/氛围], cinematic,
[光线], [风格]
```

### 推进镜头
```
push in shot, camera slowly moving forward,
[主体] in center frame,
tension building, dramatic,
[光线], cinematic composition
```

### 环绕镜头
```
orbiting shot, camera circling around [主体],
360 degree movement, [速度],
[情绪], epic moment,
dynamic lighting
```

### 跟拍镜头
```
tracking shot, camera following [角色],
moving through [环境],
steady movement, immersive,
[视角：behind/side/front]
```

### 升降镜头
```
crane shot, camera [rising/descending],
revealing [场景/规模],
[情绪], aerial perspective,
cinematic scale
```

## 🔧 运镜设计流程

### Step 1: 确定情感目标
```
这个镜头要传达什么情绪？
- 紧张？→ 缓慢推进
- 震惊？→ 快速推进或眩晕
- 孤独？→ 缓慢拉远
- 解放？→ 升起
```

### Step 2: 选择运镜类型
```
根据情感选择基础运镜
考虑是否需要复合运动
```

### Step 3: 设计起止点
```
起始帧：什么构图？
终止帧：什么构图？
两者之间的变化是什么？
```

### Step 4: 确定速度节奏
```
慢速：庄重、沉思、紧张累积
中速：正常叙事
快速：紧张、混乱、能量
变速：强调某一刻
```

## 📊 运镜速度参考

| 速度 | 描述 | 时长参考 | 情绪 |
|------|------|----------|------|
| 极慢 | 几乎察觉不到 | 10-30秒推进 | 潜在紧张 |
| 慢 | 缓慢但可见 | 5-10秒 | 沉思、庄重 |
| 中 | 自然速度 | 2-5秒 | 正常叙事 |
| 快 | 明显快速 | 1-2秒 | 紧张、能量 |
| 极快 | 瞬间完成 | <1秒 | 震惊、冲击 |

## 🎬 导演风格运镜

### 斯皮尔伯格
- 推进揭示人物反应
- 低角度英雄镜头
- Prompt: Spielberg style, push in on face, low angle hero shot

### 库布里克
- 对称构图+缓慢推进
- 一点透视走廊
- Prompt: Kubrick style, symmetrical, slow push in, one point perspective

### 王家卫
- 手持+慢镜头
- 碎片化运动
- Prompt: Wong Kar-wai style, handheld, slow motion, fragmented

### 扎克·施奈德
- 变速运动
- 慢动作+快动作
- Prompt: Zack Snyder style, speed ramping, slow motion action
