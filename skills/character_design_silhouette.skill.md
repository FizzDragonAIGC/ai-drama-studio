# 👤 剪影设计 Skill
> 来源：《The Art of Character Design》/ Disney Animation / Pixar

---

## 📐 剪影设计核心

> **剪影法则**：一个好的角色设计，仅凭轮廓就能辨认。

### 为什么剪影重要
- 远距离识别
- 动画中快速识别
- 建立角色独特性
- 视觉记忆点

---

## 🎯 剪影设计原则

### 1. 清晰可辨 (Readable)
```
✅ 好剪影           ❌ 差剪影
   ┌───┐               ┌───┐
   │   │               │   │
 ──┼───┼──           ──┼───┼──
   │   │               │   │
  ╱     ╲             │   │
 ╱       ╲            │   │
                      └───┘
独特、有特征          平庸、难辨认
```

### 2. 独特性 (Unique)
每个角色的剪影应该不同

```
角色A    角色B    角色C
  △        ○        □
 ╱ ╲      ╱ ╲      ╱ ╲
高瘦     圆润     方正
```

### 3. 角色性格外化
```
英雄：高大、稳定、肩宽
反派：尖锐、不规则、阴影
智者：上大下小（头部突出）
小丑：夸张、不对称
```

## 📊 形状语言

### 基础形状含义

| 形状 | 性格 | 角色类型 |
|------|------|----------|
| 圆形 ○ | 友善、柔和、可爱 | 主角、萌角色 |
| 方形 □ | 稳定、可靠、强壮 | 英雄、保护者 |
| 三角 △ | 危险、攻击、动态 | 反派、战士 |

### 形状组合
```
○ + □ = 友善且可靠（大白）
△ + □ = 强壮且危险（灭霸）
○ + △ = 可爱但有攻击性（皮卡丘）
```

### 形状变体
```
正三角 ▲ = 稳定的力量
倒三角 ▼ = 不稳定、危险
尖锐 ◇ = 敏捷、狡猾
波浪 ～ = 柔软、有机
```

## 🎭 体型设计

### 体型对比
```
高瘦     标准     矮胖
 │        │        ○
 │       ╱ ╲      ╱ ╲
 │      │   │    │   │
╱ ╲     │   │   ─┴───┴─
│   │   │   │
│   │   └───┘
```

### 体型含义
| 体型 | 印象 | 例子 |
|------|------|------|
| 高瘦 | 优雅、脆弱、知性 | 杰克船长 |
| 矮胖 | 可爱、滑稽、温暖 | 加菲猫 |
| 肌肉 | 力量、英雄、威胁 | 浩克 |
| 纤细 | 敏捷、神秘、危险 | 黑寡妇 |
| 标准 | 亲近、普通 | 主角常用 |

## 📋 剪影检查法

### 五步检查
1. **黑色填充**：将角色填满黑色
2. **辨认测试**：能认出是谁吗？
3. **动作测试**：做不同动作还能认吗？
4. **对比测试**：与其他角色放一起能区分吗？
5. **缩小测试**：缩小后还清晰吗？

### 检查清单
- [ ] 头部形状独特？
- [ ] 身体比例独特？
- [ ] 有标志性元素（帽子/武器/发型）？
- [ ] 与其他角色有对比？
- [ ] 姿态是否强化个性？

## 🔧 设计流程

### Step 1：形状探索
```
画出20-30个小剪影缩略图
不追求细节
只追求形状和比例
```

### Step 2：筛选优化
```
选择3-5个有潜力的
略微细化
测试不同姿势
```

### Step 3：细节添加
```
在选定的剪影基础上
添加特征、服装、道具
保持剪影清晰
```

### Step 4：对比验证
```
与其他角色放在一起
确保每个都可辨认
调整过于相似的设计
```

## 📊 标志性元素

### 增强识别度
| 元素类型 | 例子 |
|----------|------|
| 发型 | 悟空的刺猬头 |
| 帽子 | 路飞的草帽 |
| 武器 | 索隆的三刀 |
| 服装 | 蝙蝠侠的披风 |
| 配饰 | 哈利波特的眼镜 |
| 姿态 | 柯南的小身体 |

### 设计原则
- 标志性元素要简洁
- 影响剪影轮廓
- 与角色性格相关
- 便于重复绘制

## 🎬 案例分析

### 《超人总动员》
- **弹力女**：曲线、柔软→弹性
- **巴小杰**：圆润、可爱→婴儿
- **衣夫人**：三角、尖锐→时尚/危险
- **所有超人**：清晰剪影、各不相同

### 《海贼王》
- **路飞**：草帽是标志
- **索隆**：三把刀的剪影
- **娜美**：发型和身材曲线
- **乔巴**：小身体大帽子
