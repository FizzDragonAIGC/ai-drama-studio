## 核心理念

主題 ≠ 話題
- 話題：關於什麼（愛情、戰爭、成長）
- 主題：想說什麼（愛情是值得冒險的）

主題 = 一個完整的句子，帶有觀點

## 主題挖掘五問

### 問題1：關於什麼
```
「這個故事是關於什麼的？」
（答案通常是話題，不是主題）
```

### 問題2：想說什麼
```
「關於[話題]，你想說什麼？」
「你對[話題]的看法是什麼？」
「故事結束時，觀眾應該相信什麼？」
```

### 問題3：為什麼重要
```
「為什麼這個觀點值得一個故事？」
「這個觀點對現代觀眾有什麼意義？」
「你希望觀眾看完後改變什麼？」
```

### 問題4：怎麼證明
```
「故事中哪些事件證明了這個觀點？」
「主角的旅程如何體現這個主題？」
「反派的存在如何襯托主題？」
```

### 問題5：反面是什麼
```
「有人會反對這個觀點嗎？」
「反對的理由是什麼？」
「故事如何回應這個反對？」

反主題 → 讓主題更有力
```

## 主題類型

### 1. 存在類
```
人為什麼活著？
- 尋找意義
- 面對死亡
- 存在的孤獨
```

### 2. 道德類
```
什麼是對的？
- 正義vs法律
- 個人vs集體
- 目的vs手段
```

### 3. 關係類
```
人與人如何連接？
- 愛的本質
- 背叛與原諒
- 家庭的意義
```

### 4. 成長類
```
人如何改變？
- 克服恐懼
- 接受自我
- 責任與自由
```

### 5. 社會類
```
社會如何運作？
- 權力與腐敗
- 階級與不平等
- 傳統vs變革
```

## 主題清晰度測試

### 電梯測試
```
「用一句話說出主題」
❌ 「關於愛情」 → 話題
❌ 「愛情很複雜」 → 太模糊
✅ 「真愛需要放下佔有」 → 清晰主題
```

### 行動測試
```
「如果觀眾真的相信這個主題，
 他們會做出什麼不同的選擇？」

如果答案是「不知道」→ 主題太抽象
```

### 衝突測試
```
「這個主題的反面觀點是什麼？」
「有人會為反面觀點辯護嗎？」

如果反面太弱 → 主題太安全/無聊
好主題 = 有爭議、有張力
```

## 主題與結構的關係

```
開頭：主題隱含在設定中
      （世界觀暗示了什麼？）

中段：主題受到挑戰
      （反主題的聲音出現）

高潮：主題被證明
      （通過主角的選擇）

結尾：主題被強化
      （餘韻讓觀眾思考）
```

## 主題表達層次

| 層次 | 方式 | 例子 |
|------|------|------|
| 直接 | 對白 | 角色直接說出主題 |
| 象徵 | 視覺 | 意象/符號承載主題 |
| 行動 | 選擇 | 角色的決定體現主題 |
| 結構 | 對比 | 不同角色命運的對照 |
| 暗示 | 留白 | 讓觀眾自己領悟 |

**最高境界：不說出來，但所有人都感受到**

## 常見陷阱

| 陷阱 | 問題 | 解決 |
|------|------|------|
| 說教 | 主題太直白 | 用行動代替台詞 |
| 模糊 | 主題不清晰 | 逼自己寫出一句話 |
| 安全 | 主題沒爭議 | 找到真正的張力 |
| 矛盾 | 故事不支持主題 | 調整情節或主題 |
| 多重 | 太多主題 | 找出核心主題 |
