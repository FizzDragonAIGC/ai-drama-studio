# ✍️ 場景標題技能

## 元信息
- **技能ID**: script_scene_heading
- **所屬Agent**: 劇本Agent
- **參考書籍**: 《電影劇本寫作基礎》
- **難度**: ⭐⭐

## 標準格式

```
內/外 場景名稱 - 時間

INT. 咖啡廳 - 日
EXT. 城市街道 - 夜
INT./EXT. 車內 - 黃昏
```

## 場景標題元素

### 1. 內外標識
```
INT. = 內景（室內）
EXT. = 外景（室外）
INT./EXT. = 兩者兼有（如車內看窗外）
```

### 2. 場景名稱
```
具體明確：
✅ 小明的臥室
✅ 舊金山金門大橋
✅ 醫院急診室

避免模糊：
❌ 一個房間
❌ 某處
❌ 城市
```

### 3. 時間
```
基本時間：
- 日
- 夜
- 黃昏
- 黎明

特殊時間：
- 同時（連續場景）
- 稍後（同場景時間跳躍）
- 閃回
- 夢境
```

## AI番劇場景標題

### 中文標準格式
```
場景 01 | 內 小鎮碼頭 - 黃昏
場景 02 | 外 湘西街道 - 夜
場景 03 | 內/外 渡船上 - 日
```

### 增強格式（含更多信息）
```
【場景 01】
位置：內 翠翠家門口
時間：黃昏
天氣：晴朗
情緒：寧靜
```

## 常見錯誤

| 錯誤 | 修正 |
|------|------|
| 「客廳」（哪個客廳？） | 「李家客廳」 |
| 「外面」 | 「學校操場」 |
| 「室內 - 白天」 | 「INT. 辦公室 - 日」 |
| 每個小變化都開新場景 | 同一時空可以合併 |
