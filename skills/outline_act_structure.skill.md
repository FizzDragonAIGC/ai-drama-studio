# 📝 幕結構設計技能

## 元信息
- **技能ID**: outline_act_structure
- **所屬Agent**: 大綱Agent
- **參考書籍**: 《電影劇本寫作基礎》Syd Field
- **難度**: ⭐⭐⭐

## 核心理念

幕 = 故事的大段落
- 每一幕有獨特的功能
- 幕之間有清晰的轉折點

## 三幕結構

### 經典比例
```
第一幕：25% | 第二幕：50% | 第三幕：25%

12集：集1-3 | 集4-9 | 集10-12
24集：集1-6 | 集7-18 | 集19-24
```

### 第一幕：建立
```
功能：
- 建立世界觀
- 介紹人物
- 設定日常
- 觸發事件
- 鎖定方向

結束標誌：主角做出決定，進入「新世界」
```

### 第二幕：對抗
```
功能：
- 遭遇障礙
- 嘗試→失敗→再試
- 關係發展
- 中點轉折
- 最低谷

第二幕A（上半）：
- 主角還抱有希望
- 障礙可以克服
- 學習新技能

中點：
- 重大轉折/揭露
- 改變故事方向
- 賭注升級

第二幕B（下半）：
- 主角失去希望
- 障礙越來越大
- 最黑暗時刻

結束標誌：主角做出最終決定
```

### 第三幕：解決
```
功能：
- 最終對決
- 解決衝突
- 展示成長
- 建立新秩序

結束標誌：新的平衡達成
```

## 四幕結構（電視劇常用）

```
第一幕：鉤子+建立（15%）
第二幕：複雜化（30%）
第三幕：危機（35%）
第四幕：高潮+解決（20%）

12集：
- 集1-2：第一幕
- 集3-5：第二幕
- 集6-9：第三幕
- 集10-12：第四幕
```

## 五幕結構（複雜敘事）

```
第一幕：引入
第二幕：上升
第三幕：高潮（中點）
第四幕：下降
第五幕：結局

莎士比亞結構
適合：權謀/史詩/複雜多線
```

## 關鍵轉折點

### Plot Point 1（第一幕結束）
```
位置：25%處
功能：打破日常，進入冒險
例：「你願意拯救世界嗎？」→「我願意」
```

### Midpoint（中點）
```
位置：50%處
功能：改變遊戲規則
類型：
- 虛假勝利（以為成功，其實沒有）
- 虛假失敗（以為失敗，其實沒有）
- 重大揭露（改變理解）
```

### Plot Point 2（第二幕結束）
```
位置：75%處
功能：最低谷後的覺醒
例：「我知道該怎麼做了」
```

### Climax（高潮）
```
位置：85-95%處
功能：最終對決
規則：主角主動解決問題
```

## 結構檢查清單

```
□ 第一幕25%之前是否有觸發事件？
□ 中點是否有重大轉折？
□ 最低谷是否足夠黑暗？
□ 高潮是否由主角主導？
□ 結局是否建立新秩序？
□ 每一幕的功能是否明確？
```
