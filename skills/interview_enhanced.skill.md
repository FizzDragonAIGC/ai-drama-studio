# 增強型創意訪談 (Enhanced Creative Interview)

## 概述
專業訪談系統，每個問題必須包含：具體故事引用、提問原因、影響範圍、觀眾視角。

## 🚨 核心規則

### 問題四要素（缺一不可）
1. **question** - 針對具體角色/情節的問題
2. **why_asking** - 為什麼要問這個問題
3. **impact** - 答案會影響哪些章節/場景
4. **audience_perspective** - 觀眾會如何感受

## 問題模板

### 模板A：角色心理探索
```json
{
  "question": "在《故事名》第X章，當[角色名]發現[具體事件]時，他選擇了[具體行動]。這個選擇背後，他經歷了怎樣的內心掙扎？",
  "why_asking": "這個決定是角色弧線的轉折點，理解他的心理過程能讓後續行為更有說服力",
  "impact": ["第5集的獨白設計", "第7集與對手的對峙", "角色的視覺符號設計"],
  "audience_perspective": "觀眾會問：他為什麼這麼做？這個答案決定觀眾是同情還是疏離",
  "story_reference": "引用原文：「他站在雨中，看著遠去的背影，手中的信封已經被雨水浸濕...」"
}
```

### 模板B：場景視覺化
```json
{
  "question": "第X章的[場景名]戲，你為什麼選擇在[時間/天氣/地點]進行？這個環境對情感有什麼象徵意義？",
  "why_asking": "場景設計直接影響美術風格和光影設計",
  "impact": ["場景美術設計", "光影方案", "配樂情緒"],
  "audience_perspective": "觀眾的潛意識會讀取視覺符號，這影響他們的情緒投入",
  "story_reference": "原文描寫：「暴雨如注，閃電劈過天際...」"
}
```

### 模板C：情節轉折
```json
{
  "question": "故事在第X章突然[轉折描述]。這個轉折是否需要更多鋪墊？或者刻意要製造突然感？",
  "why_asking": "轉折的處理方式決定是「意料之外情理之中」還是「突兀」",
  "impact": ["前幾集的伏筆設計", "轉折點的節奏處理", "觀眾情緒曲線"],
  "audience_perspective": "如果鋪墊夠，觀眾會有「原來如此」的滿足感；如果不夠，會覺得被欺騙",
  "story_reference": "具體轉折：「沒有人想到，那個一直微笑的人，就是幕後黑手...」"
}
```

### 模板D：關係探索
```json
{
  "question": "[角色A]和[角色B]在第X章有一場[描述場景]的戲。他們之間的潛台詞是什麼？表面說的和心裡想的有什麼落差？",
  "why_asking": "潛台詞是對白設計的核心，決定演員的微表情和語氣設計",
  "impact": ["對白風格", "表演指導", "鏡頭語言（是否用特寫捕捉微表情）"],
  "audience_perspective": "好的潛台詞讓觀眾感覺角色「有層次」，壞的讓角色顯得「單薄」",
  "story_reference": "原文對話：「你回來了。」「嗯，回來了。」（看似平淡，實則波濤洶湧）"
}
```

## 必問8大類

### 1. 開場設計 (Episode 1)
- 第一個鏡頭應該是什麼？為什麼？
- 觀眾看完前3分鐘，應該記住什麼？

### 2. 角色首次亮相
- 主角第一次出場，觀眾的第一印象應該是什麼？
- 反派首次出場，如何不落俗套？

### 3. 核心衝突
- 這個故事的「不可調和的矛盾」是什麼？
- 如果只能保留一個衝突，保留哪個？

### 4. 情感高潮
- 哪一場戲要讓觀眾哭/笑/緊張？
- 達到情感高潮需要哪些前置情感積累？

### 5. 視覺符號
- 有沒有貫穿全劇的視覺符號（如《肖申克》的雨）？
- 這個符號在不同場景如何變化？

### 6. 改編取捨
- 原著哪個場景絕對不能刪？
- 哪些可以大幅簡化或合併？

### 7. 風格基調
- 如果用3個形容詞描述這部作品的氣質？
- 參考哪些作品的視覺風格？

### 8. 結局設計
- 最後一個鏡頭是什麼？
- 留白多少？觀眾帶走什麼思考？
