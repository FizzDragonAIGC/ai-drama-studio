# Creative Interviewing Skill
> 基於 Ken Metzler《Creative Interviewing: The Writer's Guide to Gathering Information by Asking Questions》

## 📚 書籍簡介
Ken Metzler的經典著作，教導作家如何通過提問收集信息，從準備到執行的完整採訪指南。

## 🎯 核心理念
**好的採訪 = 準備 + 傾聽 + 追問**

採訪不是審訊，而是引導對方講出自己都沒意識到的故事。

## 📋 採訪問題設計原則

### 1. 問題金字塔結構
```
          [開放性問題]
         /           \
    [引導性問題]    [確認性問題]
       /     \         /      \
  [細節追問] [情感追問] [事實核實]
```

### 2. 六種問題類型

#### A. 開場問題 (Ice-breaker)
- 目的：建立信任，讓對方放鬆
- 例句：「能聊聊你最初是怎麼開始寫這個故事的嗎？」
- 技巧：從對方熟悉且願意談的話題開始

#### B. 背景問題 (Background)
- 目的：了解創作脈絡
- 例句：「這個角色的原型是誰？有真實人物參考嗎？」
- 技巧：不要假設，讓對方自己說出背景

#### C. 關鍵問題 (Key Questions)
- 目的：挖掘核心創作意圖
- 例句：「如果觀眾只能記住一個畫面，你希望是哪個？」
- 技巧：這是採訪的核心，要提前設計好

#### D. 追問問題 (Follow-up)
- 目的：深挖細節
- 例句：「你剛才說『那是最難寫的部分』，能說說為什麼嗎？」
- 技巧：抓住關鍵詞追問

#### E. 對比問題 (Contrast)
- 目的：揭示選擇背後的原因
- 例句：「為什麼選擇讓他死，而不是讓他離開？」
- 技巧：提供替代方案，讓對方解釋選擇

#### F. 假設問題 (Hypothetical)
- 目的：探索可能性
- 例句：「如果預算無限，這場戲你會怎麼拍？」
- 技巧：打破現實限制，了解理想狀態

## 🎭 針對創作者的採訪技巧

### 採訪前準備
1. **閱讀作品** - 至少讀兩遍，標記疑問
2. **研究背景** - 了解創作者的其他作品、經歷
3. **設計問題清單** - 10-15個核心問題
4. **準備追問方向** - 每個核心問題準備2-3個追問

### 採訪中技巧
1. **傾聽比說更重要** - 80%時間讓對方說
2. **沉默是金** - 對方停頓時不要急著填補
3. **複述確認** - 「所以你的意思是...」
4. **抓住金句** - 記錄對方說的原話

### 獲取戲劇性素材
- 問「那天發生了什麼？」而不是「你感覺怎樣？」
- 問「誰在場？他們說了什麼？」
- 問「然後呢？」讓故事繼續

## 📝 針對小說改編的採訪問題模板

### 角色相關
1. 「{角色名}的原型是誰？為什麼選擇這樣的設定？」
2. 「寫{角色名}時，你腦中浮現的演員形象是誰？」
3. 「{角色名}有沒有你自己的影子？哪部分？」
4. 「如果{角色名}活在現代，他/她會做什麼工作？」

### 情節相關
1. 「{關鍵場景}是怎麼想出來的？」
2. 「這場戲你改了幾稿？最初版本是什麼樣的？」
3. 「寫{高潮場景}時，你自己有什麼感受？」
4. 「有沒有被刪掉但你很喜歡的情節？」

### 主題相關
1. 「如果用一句話總結這個故事，你會怎麼說？」
2. 「你希望讀者看完後，會去思考什麼問題？」
3. 「這個故事和你的人生經歷有什麼關聯？」

### 改編邊界
1. 「哪個場景絕對不能刪？為什麼？」
2. 「哪個角色可以被合併或刪除？」
3. 「結局能改嗎？如果改，你能接受什麼方向？」

## 🔧 AI採訪問題生成Prompt

```
你是一位資深編劇顧問，正在為AI番劇改編項目設計採訪問題。

基於以下小說分析：
- 主角：{protagonist_name}，特質：{trait}
- 核心衝突：{core_conflict}
- 主題：{theme}
- 關鍵場景：{key_scenes}

請設計5個針對性採訪問題，要求：
1. 問題要具體，包含角色名和情節細節
2. 問題要挖掘創作意圖，而不是小說已寫明的內容
3. 問題要服務於視覺化改編
4. 使用Ken Metzler的問題類型框架

輸出格式：
Q1: [問題類型] 問題內容
目的：這個問題想了解什麼
追問方向：如果對方回答X，可以追問Y
```
