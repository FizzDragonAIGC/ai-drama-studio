# 🎬 情感曲线 Skill
> 来源：《Story》Robert McKee + 《情感设计》

---

## 📐 什么是Emotional Arc（情感曲线）

**定义**：观众在观看过程中情感变化的轨迹

### 情感曲线 vs 剧情曲线
| 类型 | 关注点 | 测量 |
|------|--------|------|
| 剧情曲线 | 事件发展 | 发生了什么 |
| 情感曲线 | 观众感受 | 感觉如何 |

两者应该配合，但不必完全一致

## 🎯 基础情感曲线

### 经典三幕情感曲线
```
情感强度
    │
    │              ╭───╮ ← 高潮（感动/激动/释放）
    │             ╱     ╲
    │       ╭────╯       ╲
    │      ╱              ╲
    │   ╭─╯                ╲
    │  ╱                    ╲──── 结局
    │ ╱ ← 开场                    （满足/惆怅）
    │╱   （好奇）
    └─────────────────────────────────→ 时间
      第一幕    第二幕     第三幕
```

### 情感类型
| 阶段 | 目标情感 | 实现手段 |
|------|----------|----------|
| 开场 | 好奇、兴趣 | 吸引人的开头 |
| 发展 | 投入、认同 | 角色塑造 |
| 冲突 | 紧张、担忧 | 障碍设置 |
| 低谷 | 悲伤、绝望 | 最黑暗时刻 |
| 高潮 | 激动、释放 | 关键时刻 |
| 结局 | 满足、感动 | 收尾 |

## 🔧 情感节拍设计

### 什么是情感节拍
```
节拍 = 情感发生变化的最小单位

例：
希望 → 失望 = 1个节拍
紧张 → 释放 = 1个节拍
悲伤 → 愤怒 = 1个节拍
```

### 节拍链示例
```
场景：告别

节拍1：平静（不知要分别）
节拍2：惊讶（得知消息）
节拍3：否认（不能接受）
节拍4：愤怒（为什么不告诉我）
节拍5：悲伤（接受现实）
节拍6：释然（理解对方）
节拍7：不舍（最后拥抱）
节拍8：希望（相信会再见）
```

## 🎯 情感对比技巧

### 情感反差
```
高情感时刻前，安排低情感段落

例：
平静的日常 → 突然的噩耗
→ 反差增强冲击力

快乐的庆祝 → 意外的悲剧
→ 反差增强悲伤
```

### 情感韵律
```
不要连续相同情感

❌ 紧张 → 紧张 → 紧张 → 紧张
   观众：疲劳

✓ 紧张 → 释放 → 温馨 → 紧张
   观众：持续参与
```

### 情感余韵
```
重要情感时刻后，给足反应时间

告白成功！
├─ 不要立刻切走
├─ 给2-3个反应镜头
├─ 让观众沉浸在这个情感中
└─ 然后才过渡
```

## 🎬 AI番剧应用

### 情感可视化Prompt

| 情感 | Prompt关键词 | 色调建议 |
|------|--------------|----------|
| 快乐 | joyful, bright, smiling | 暖色、高饱和 |
| 悲伤 | melancholic, blue tones | 冷色、低饱和 |
| 紧张 | tense, dramatic | 高对比、暗调 |
| 浪漫 | romantic, soft lighting | 暖色、柔光 |
| 恐惧 | fearful, dark, ominous | 暗色、冷调 |
| 希望 | hopeful, dawn light | 暖色、渐亮 |

### 情感转场示例
```
从悲伤到希望：

镜头1："sad scene, rain, dark colors"
镜头2："rain stopping, clouds parting"
镜头3："sunlight breaking through, hopeful atmosphere"

视觉逐步过渡，带动情感变化
```

### 情感曲线分镜
```
| 段落 | 情感目标 | 色调 | 节奏 | 配乐 |
|------|----------|------|------|------|
| 开场 | 温馨 | 暖黄 | 慢 | 轻快 |
| 发展 | 紧张 | 冷蓝 | 加速 | 紧张 |
| 低谷 | 悲伤 | 灰暗 | 慢 | 悲伤 |
| 高潮 | 激动 | 高对比 | 快 | 高潮 |
| 结局 | 感动 | 暖橙 | 慢 | 感动 |
```

## 💡 情感曲线检查

### 设计检查
- [ ] 有明确的情感高潮吗？
- [ ] 情感有足够变化吗？
- [ ] 有反差增强效果吗？
- [ ] 给了足够的情感余韵吗？

### 观看测试
- [ ] 自己看的时候有情感波动吗？
- [ ] 高潮时刻真的感动了吗？
- [ ] 有没有无聊/拖沓的段落？
- [ ] 结局感觉满足吗？
