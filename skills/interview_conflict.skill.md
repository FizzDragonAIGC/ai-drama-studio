# 🎤 衝突發現技能

## 元信息
- **技能ID**: interview_conflict
- **所屬Agent**: 訪談Agent
- **參考書籍**: 《故事》麥基 - 衝突章節
- **難度**: ⭐⭐⭐⭐

## 核心理念

沒有衝突就沒有故事
- 衝突 = 欲望 + 障礙
- 挖掘衝突 = 找到故事的引擎

## 衝突三層次

### 1. 內在衝突（人物 vs 自己）
```
「主角內心最大的矛盾是什麼？」
「他想要什麼但又害怕得到？」
「他必須克服自己的什麼缺陷？」
「他的信念會在什麼情況下動搖？」

例：想要自由 vs 害怕孤獨
```

### 2. 人際衝突（人物 vs 他人）
```
「誰最阻礙主角實現目標？」
「這個人為什麼要阻礙他？」
「他們之間有沒有複雜的情感？」
「最後誰會改變？」

例：父子兩代的價值觀衝突
```

### 3. 外在衝突（人物 vs 世界）
```
「這個世界的規則是什麼？」
「這個規則如何壓迫主角？」
「主角要改變世界還是適應它？」
「社會/體制/命運如何成為障礙？」

例：小人物對抗不公的體制
```

## 衝突挖掘問題清單

### 欲望線（Want Line）
```
「主角表面上想要什麼？」
「他願意為此付出什麼代價？」
「如果得不到會怎樣？」
「得到之後他會滿足嗎？」
```

### 需求線（Need Line）
```
「主角真正需要什麼？（他可能自己不知道）」
「這個需求會在什麼時候浮現？」
「他會意識到自己真正的需求嗎？」
「想要和需要之間的矛盾是什麼？」
```

### 障礙清單
```
「列出阻礙主角的所有因素」
- 外部障礙：敵人、環境、規則
- 內部障礙：恐懼、缺陷、誤解
- 人際障礙：關係、誤會、背叛

「哪個障礙最難克服？」
「障礙會升級嗎？怎麼升級？」
```

## 衝突強度檢驗

### 賭注測試
```
「如果主角失敗，會失去什麼？」

低賭注 → 不緊張
- 「可能會尷尬」

中等賭注 → 有張力
- 「會失去工作」

高賭注 → 緊張
- 「會死/會失去摯愛的人」

最高賭注 → 生存級
- 「整個世界會毀滅」

建議：至少是「無法挽回」的損失
```

### 對手測試
```
「對手的目標是什麼？」
「對手為什麼認為自己是對的？」
「對手和主角有什麼共同點？」

好對手 = 主角的鏡像
壞對手 = 純粹的壞
```

## 衝突結構模板

```
主角：[名字]
Want（想要）：[表面目標]
Need（需要）：[深層需求]

主要衝突：
├── 內在：[恐懼/缺陷] vs [成長方向]
├── 人際：[對手] 想要 [對立目標]
└── 外在：[世界規則] 壓迫 [主角]

賭注：如果失敗，[後果]
升級：衝突會如何加劇？
轉折：什麼事件會逼主角做出選擇？
```

## 衝突矩陣

| 角色A | 角色B | 衝突點 | 背後原因 | 可能和解？ |
|-------|-------|--------|----------|------------|
| 主角 | 父親 | 職業選擇 | 期望vs自我 | 理解後和解 |
| 主角 | 情敵 | 爭奪愛人 | 嫉妒 | 一方退出 |
| 主角 | 體制 | 自由vs規訓 | 價值觀 | 改變規則 |

## 常見問題

| 問題 | 診斷 | 建議 |
|------|------|------|
| 衝突太弱 | 賭注不夠高 | 提高失敗的代價 |
| 衝突太多 | 失去焦點 | 找出核心衝突線 |
| 對手太弱 | 沒有真正威脅 | 讓對手更強/更聰明 |
| 內在衝突不明 | 角色太完美 | 增加缺陷和恐懼 |
