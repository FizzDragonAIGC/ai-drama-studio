# 🎬 建立镜头 Skill
> 来源：《电影镜头设计》Steven D. Katz + 好莱坞叙事传统

---

## 📐 什么是Establishing Shot（建立镜头）

**定义**：场景开始时展示环境全貌的广角镜头，告诉观众"我们在哪里"

### 建立镜头的作用
| 作用 | 说明 |
|------|------|
| **地点交代** | 这是哪里 |
| **时间交代** | 白天/夜晚/季节 |
| **氛围设定** | 场景情绪基调 |
| **空间建立** | 后续镜头的空间参照 |

## 📋 建立镜头模式

### 传统三段式
```
远 → 中 → 近

镜头1：城市远景（大环境）
镜头2：建筑外观（具体地点）
镜头3：室内/人物（进入场景）
```

### 逆向建立
```
近 → 中 → 远

镜头1：细节特写（悬念）
镜头2：中景（揭示更多）
镜头3：远景（完整揭示）

用途：制造悬念、揭示
```

### 单镜建立
```
一个移动镜头完成建立
从远景推进到近景
或从细节拉出到远景
```

## 🎯 建立镜头时长

| 情况 | 建议时长 | 原因 |
|------|----------|------|
| 熟悉场景 | 1-2秒 | 观众知道这里 |
| 新场景 | 3-5秒 | 需要时间理解 |
| 重要场景 | 4-8秒 | 强调此地重要 |
| 转场用途 | 2-3秒 | 足够过渡 |

## 🎬 AI番剧应用

### 建立镜头Prompt模板
```
基础模板：
"establishing shot, [地点描述], 
[时间线索], [天气/氛围], 
wide angle, cinematic, [风格]"
```

### 场景分类Prompts

**城市建立**
```
"establishing shot tokyo cityscape,
tokyo tower visible, evening sky,
city lights beginning to glow,
anime style, wide angle"
```

**学校建立**
```
"establishing shot japanese high school,
cherry blossoms in bloom, spring morning,
students walking to school, anime style"
```

**室内建立**
```
"wide shot japanese apartment interior,
evening light through window,
cozy living room, anime style"
```

**自然环境**
```
"establishing shot countryside,
rice fields, mountains in distance,
summer afternoon, studio ghibli style"
```

### 时间变化序列
```
同一地点的时间变化：

早晨：
"school building exterior, morning,
students arriving, soft morning light"

中午：
"same school building, midday,
bright sunlight, empty courtyard"

傍晚：
"same school building, sunset,
orange sky, students leaving"
```

## 💡 建立镜头检查

### 信息检查
- [ ] 地点清晰吗？
- [ ] 时间明确吗？
- [ ] 氛围传达了吗？
- [ ] 与后续场景连贯吗？

### 技术检查
- [ ] 景别足够宽吗？
- [ ] 时长足够吗？
- [ ] 画面质量OK吗？
- [ ] 风格一致吗？

---

## ⚠️ 常见问题

| 问题 | 原因 | 解决 |
|------|------|------|
| 观众不知在哪 | 缺少建立镜头 | 加入建立 |
| 建立太久 | 时长过长 | 缩短到必要时长 |
| 风格不一致 | 建立与场景风格断裂 | 统一视觉风格 |
| 信息不清 | 地标不明显 | 选择更明确的地标 |
