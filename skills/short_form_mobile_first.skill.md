## 📐 手机观看特性

手机观看与电视/电脑的根本差异：

| 维度 | 电视/电脑 | 手机 | 设计影响 |
|------|----------|------|----------|
| 屏幕尺寸 | 20-65英寸 | 5-7英寸 | 元素要大 |
| 观看距离 | 2-5米 | 20-40厘米 | 细节放大 |
| 观看姿势 | 固定坐姿 | 随意姿势 | 画面简洁 |
| 注意力 | 专注 | 碎片化 | 节奏快 |
| 环境 | 安静 | 嘈杂 | 字幕必要 |
| 音频 | 扬声器 | 可能静音 | 视觉叙事 |

---

## 🎯 手机优先设计原则

### 1. 视觉元素放大
| 元素 | 传统尺寸 | 手机优化 |
|------|----------|----------|
| 人物脸部 | 画面30% | 画面50%+ |
| 字幕 | 小字 | 大字加粗 |
| 关键道具 | 正常比例 | 特写强调 |
| 文字信息 | 标准字号 | 放大1.5-2倍 |

### 2. 景别偏大
| 推荐 | 谨慎使用 |
|------|----------|
| 特写 | 大远景 |
| 近景 | 远景 |
| 中近景 | 大全景 |

### 3. 信息聚焦
```
画面元素数量：3个以内最佳
同时在场人物：2-3人为宜
背景复杂度：简洁为主
```

## 📱 静音观看适配

### 字幕设计
| 参数 | 建议值 |
|------|--------|
| 字号 | 画面高度的4-6% |
| 位置 | 底部1/4区域 |
| 颜色 | 白字黑边/黄字 |
| 行数 | 最多2行 |
| 字数 | 每行15字以内 |

### 视觉叙事
| 传统依赖 | 静音替代 |
|----------|----------|
| 对白解释 | 表情+动作 |
| 旁白铺垫 | 字幕卡片 |
| 音效暗示 | 视觉提示 |
| BGM情绪 | 色调+节奏 |

### 关键信息字幕化
```
✓ 人物名称首次出场
✓ 地点时间变化
✓ 关键对白
✓ 心理活动
✓ 情绪标签
```


### 手势优化
| 手势 | 功能 | 设计考虑 |
|------|------|----------|
| 上下滑 | 切换视频 | 不中断体验 |

| 左右滑 | 进度调节 | 保持画面连贯 |

## 🔊 音频分层设计

### 有声观看
| 层级 | 内容 | 音量 |
|------|------|------|
| 对白 | 台词 | 100% |
| 音效 | 强调 | 70-80% |
| BGM | 氛围 | 30-50% |

### 无声观看
| 信息 | 替代方案 |
|------|----------|
| 对白 | 字幕 |
| 音效 | 视觉震动/闪烁 |
| BGM情绪 | 色调变化 |

### 公共场所友好
- 即使静音也能理解剧情
- 字幕包含所有关键信息
- 视觉表演到位

## 📋 手机优先检查表

```
【手机观看检查】

□ 视觉清晰度
  - 人物脸部在小屏幕上清晰可辨
  - 关键道具/文字放大展示
  - 背景简洁不干扰主体

□ 静音观看
  - 有完整字幕
  - 关键信息有视觉提示
  - 纯视觉也能理解剧情

□ 碎片化观看
  - 每集独立可懂
  - 开头快速回顾
  - 结构清晰易跟

□ 竖屏适配
  - 9:16比例
  - 安全区预留
  - 纵向构图

□ 网络适配
  - 码率合适
  - 加载流畅
  - 压缩后画质可接受
```

## 🎯 手机优先拍摄提示

### 拍摄设置
| 参数 | 建议 |
|------|------|
| 分辨率 | 1080×1920 |
| 帧率 | 30/60fps |
| 比例 | 9:16 |
| 稳定 | 使用稳定器 |

### 构图要点
| 要点 | 原因 |
|------|------|
| 主体居中 | 安全区考虑 |
| 上下留白 | 平台UI避让 |
| 大特写优先 | 小屏幕可见 |
| 简洁背景 | 不分散注意 |

### 灯光要点
| 要点 | 原因 |
|------|------|
| 充足光线 | 压缩后不糊 |
| 面部补光 | 表情清晰 |
| 避免逆光 | 手机宽容度低 |

## 📊 不同场景的手机优化

### 对话场景
```
- 使用近景/特写
- 正反打清晰
- 字幕跟随说话人
```

### 动作场景
```
- 中近景为主
- 关键动作放慢
- 音效视觉化
```

### 环境场景
```
- 快速切过
- 字幕说明
- 或用分屏展示
```

### 多人场景
```
- 焦点人物突出
- 依次特写介绍
- 避免群像全景
```
