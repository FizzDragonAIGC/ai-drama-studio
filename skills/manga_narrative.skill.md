# 漫画叙事技巧 Skill
> 来源：《理解漫画》《制造漫画》斯科特·麦克劳德
> 适用智能体：A22 漫画叙事专家, A08 漫画分镜师

## 核心定义

**漫画 = 有意识排列的并置图片和其他图像**

漫画是一种通过时间与空间的视觉表现来传达信息、理念与情感的艺术形式。

---

## 漫画的六大关键概念

### 1. Closure（闭合/脑补）
读者在格与格之间自动补全缺失的动作和时间。

**这是漫画最核心的魔法！**

### 2. 格子与留白（Gutter）
- 格子 = 时间的容器
- 留白 = 想象的空间
- 格间距控制时间流逝感

### 3. 时间 = 空间
- 横向：时间流逝
- 纵向：同时发生
- 格子大小 = 时间长度感知

### 4. 图像抽象光谱
```
写实 ←————————————→ 抽象
照片  素描  简化  符号  文字
```

### 5. 图文关系
- 文字主导型
- 图像主导型
- 互补型
- 蒙太奇型

### 6. 线条的表现力
线条风格直接影响情绪：
- 粗犷线条 → 力量、紧张
- 细腻线条 → 精致、安静
- 断续线条 → 不安、神秘

## 六种格间转换

| 类型 | 说明 | Closure难度 | 示例 |
|------|------|------------|------|
| 1. 瞬间到瞬间 | 同一动作的分解 | 最易 | 眨眼的分解 |
| 2. 动作到动作 | 同一主体不同动作 | 易 | 挥拳→击中 |
| 3. 主体到主体 | 同一场景不同主体 | 中 | 对话双方 |
| 4. 场景到场景 | 时空跨越 | 难 | 白天→夜晚 |
| 5. 视角到视角 | 同一场景不同角度 | 中 | 俯视→平视 |
| 6. 非线性 | 无逻辑关联 | 最难 | 抽象表达 |

## 视线流设计

### 阅读方向

**页漫（日漫）**：
```
┌───┬───┐
│ 1 │ 2 │  右→左
├───┼───┤  上→下
│ 3 │ 4 │
└───┴───┘
```

**条漫（韩漫/网漫）**：
```
┌───┐
│ 1 │
├───┤
│ 2 │  上→下
├───┤  滚动阅读
│ 3 │
└───┘
```

### 视线引导技巧

1. **角色视线** → 引导读者看向
2. **动作方向** → 顺势移动
3. **对话框位置** → 先读的放右上
4. **留白区域** → 视觉休息点

## 格子设计原则

### 格子大小与时间

| 格子 | 感知 |
|------|------|
| 大格 | 时间长、重要 |
| 小格 | 时间短、快节奏 |
| 横长格 | 稳定、叙述 |
| 竖长格 | 紧张、动态 |
| 跨页 | 高潮、震撼 |

### 格子边框

| 边框 | 情绪 |
|------|------|
| 实线方框 | 正常叙述 |
| 虚线/无框 | 回忆、梦境 |
| 锯齿边框 | 惊讶、爆炸 |
| 圆角边框 | 温柔、回忆 |
| 出血（无边） | 无限延伸感 |

---

## 漫画节奏控制

### 节奏类型

| 节奏 | 格子密度 | 留白 | 效果 |
|------|---------|------|------|
| 快节奏 | 密集小格 | 少 | 紧张、动作 |
| 慢节奏 | 稀疏大格 | 多 | 抒情、沉思 |
| 停顿 | 单格特写 | 最多 | 强调、情感高点 |

### 翻页效果

- **右页最后一格** = 悬念点
- **左页第一格** = 揭示/反转
- **跨页** = 视觉高潮

## 条漫特殊技巧

### 滚动叙事

1. **纵向构图** → 利用滚动
2. **长条留白** → 制造停顿
3. **分段高潮** → 每屏一个小钩子
4. **最后一格** → 集尾钩子

### 条漫节奏

```
图 → 图 → 图 → [留白] → 转折 → 图 → 图 → [钩子]
```

## AI漫画生成要点

### 单格Prompt模板

```
[主体描述], [动作/表情], [场景/背景],
manga panel style, [效果线/漫符],
[构图：close-up/wide shot/dutch angle],
[情绪：dramatic/comedic/melancholic]
```

### 一致性控制

- 角色参考图（--cref）
- 风格参考图（--sref）
- 固定角色描述词
- 固定画风描述词

### 分镜到Prompt流程

```
1. 设计格子布局
2. 确定每格内容
3. 写出视觉描述
4. 转换为AI Prompt
5. 添加风格/效果词
6. 生成并调整
```
