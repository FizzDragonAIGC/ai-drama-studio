## 📊 构图基础原则

### 三分法 (Rule of Thirds)
```
┌───────┬───────┬───────┐
│       │       │       │
│   ●   │       │   ●   │ ← 焦点位置
├───────┼───────┼───────┤
│       │       │       │
│   ●   │       │   ●   │
├───────┼───────┼───────┤
│       │       │       │
│       │       │       │
└───────┴───────┴───────┘
```

- 主体放在交叉点
- 地平线放在上/下三分线
- 制造动态平衡

### 中心构图 (Center Framing)
```
┌─────────────────────┐
│                     │
│         ●           │
│                     │
└─────────────────────┘
```

- 强调对称/庄重
- 直面观众
- 权力/重要性
- 例：韦斯·安德森、库布里克

### 对称构图 (Symmetry)
```
┌─────────────────────┐
│    ←─────┼─────→    │
│  □       │       □  │
│         ○           │
│  □       │       □  │
└─────────────────────┘
```

- 秩序、控制、权力
- 不安、压迫（过度对称）
- 宗教、皇室、机构

## 🎯 高级构图技巧

### 引导线 (Leading Lines)
```
┌─────────────────────┐
│     ╲               │
│       ╲             │
│         ╲ ○         │ ← 引导视线
│          ╲          │
└───────────╲─────────┘
```

**类型**：
- 道路、走廊
- 栏杆、桥梁
- 建筑线条
- 光线方向

### 框中框 (Frame within Frame)
```
┌─────────────────────┐
│   ┌───────────┐     │
│   │           │     │
│   │     ○     │     │
│   │           │     │
│   └───────────┘     │
└─────────────────────┘
```

**含义**：
- 被困/囚禁
- 孤立
- 窥视
- 过去/记忆

**元素**：门框、窗户、镜子、拱门

### 负空间 (Negative Space)
```
┌─────────────────────┐
│                     │
│                     │
│                     │
│               ○     │
└─────────────────────┘
```

**使用**：
- 孤独感
- 渺小感
- 等待/预期
- 留白美学

## 📊 平衡与张力

### 视觉平衡
```
重量感分布：
大体积 = 重
亮色 = 重
暖色 = 重
清晰 = 重
前景 = 重
```

### 对称平衡
```
┌─────────────────────┐
│  ■             ■    │
│       稳定、正式    │
└─────────────────────┘
```

### 非对称平衡
```
┌─────────────────────┐
│  ■■■               ●│
│    动态、自然       │
└─────────────────────┘
```

### 失衡
```
┌─────────────────────┐
│  ■■■                │
│    紧张、不安       │
└─────────────────────┘
```

## 🎭 前景-中景-后景

### 深度层次
```
前景          中景          后景
  │            │             │
  ▼            ▼             ▼
┌─────────────────────────────┐
│  ██       ○              █  │
│  ██                      █  │
│  ██                      █  │
└─────────────────────────────┘
```

**作用**：
- 创造深度
- 引导视线
- 增加层次
- 提供信息

## 📋 构图检查清单

### 基础
- [ ] 主体位置恰当？
- [ ] 有视觉焦点？
- [ ] 画面平衡？
- [ ] 使用引导线？

### 进阶
- [ ] 前中后景设计？
- [ ] 负空间有意义？
- [ ] 框中框增强叙事？
- [ ] 构图支持情感？

### 避免
- ❌ 主体切边
- ❌ 杂乱背景
- ❌ 无焦点
- ❌ 忽略头部空间/视线空间

## 🎬 导演构图风格

### 韦斯·安德森
- 极致对称
- 中心构图
- 平面感
- 糖果色

### 库布里克
- 一点透视
- 对称构图
- 冷静、压迫
- 长走廊

### 王家卫
- 碎片化
- 框中框
- 模糊/反射
- 都市负空间
