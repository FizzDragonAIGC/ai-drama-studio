# 世界观圣经 (Worldbuilding Bible)

## 概述
为原创或改编作品建立完整、一致的世界观设定文档。

## 世界观圣经结构

### 一、核心设定

**1.1 世界观一句话**
- 这是一个怎样的世界？
- 和现实世界的关键区别？

**1.2 时间设定**
- 历史背景
- 当前时间点
- 重大历史事件

**1.3 空间设定**
- 地理范围
- 关键地点
- 地图（如需要）

### 二、社会系统

**2.1 政治结构**
- 统治方式
- 权力分布
- 主要势力/国家

**2.2 经济系统**
- 货币/交易方式
- 主要产业
- 贫富差距

**2.3 社会阶层**
- 阶层划分
- 流动性
- 冲突来源

### 三、特殊规则

**3.1 魔法/科技系统**
- 能力来源
- 使用限制
- 代价/副作用

**3.2 种族/物种**
- 有哪些种族
- 各自特点
- 种族关系

**3.3 特殊规则**
- 这个世界独有的规则
- 不能违反的设定

### 四、文化细节

**4.1 语言**
- 官方语言
- 方言/暗语
- 禁忌词

**4.2 宗教/信仰**
- 主要宗教
- 仪式
- 禁忌

**4.3 日常生活**
- 饮食
- 服饰
- 娱乐

### 五、视觉风格

**5.1 建筑风格**
- 参考时代/文化
- 关键元素
- 材质/色彩

**5.2 服饰风格**
- 各阶层服饰
- 职业特征
- 色彩含义

**5.3 科技水平**
- 交通工具
- 通讯方式
- 武器

## Prompt模板

### 世界观提取
```
从以下故事中提取世界观设定：

【原文】
{story_content}

请输出：
1. 世界观一句话总结
2. 时间背景
3. 关键地点
4. 社会结构
5. 特殊规则/能力系统
6. 需要补充的设定
```

### 世界观扩展
```
基于以下核心设定，扩展完整世界观：

【核心设定】
{core_setting}

需要扩展：
1. 这个世界的历史（关键事件）
2. 社会运作方式
3. 日常生活细节
4. 视觉风格参考

要求：保持内在一致性。
```

## 一致性检查

### 每场戏检查
- [ ] 场景设定是否符合世界观？
- [ ] 角色能力是否遵守规则？
- [ ] 科技/魔法使用是否一致？
- [ ] 社会规则是否被遵守？

### 全剧检查
- [ ] 是否有设定冲突？
- [ ] 规则是否被打破？
- [ ] 世界观是否自洽？
