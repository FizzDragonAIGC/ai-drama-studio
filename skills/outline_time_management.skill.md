# 📝 時間管理技能

## 元信息
- **技能ID**: outline_time_management
- **所屬Agent**: 大綱Agent
- **參考書籍**: 《電影語言》
- **難度**: ⭐⭐⭐

## 核心理念

故事時間 ≠ 螢幕時間
- 故事可能跨越10年
- 觀眾只看12集

## 時間處理類型

### 1. 實時
```
故事時間 = 螢幕時間
例：24小時（每集=1小時）

效果：緊張、沉浸
限制：很難長期維持
```

### 2. 壓縮時間
```
故事時間 > 螢幕時間
例：1年的故事 = 12集

技巧：
- 選擇關鍵時刻
- 時間跳躍
- 蒙太奇
```

### 3. 延展時間
```
故事時間 < 螢幕時間
例：10分鐘的決戰 = 整集

技巧：
- 多角度呈現
- 細節放大
- 主觀時間
```

## 時間跳躍技巧

### 明確跳躍
```
字幕：「三年後」
旁白：「那個夏天過後...」
日曆翻頁
```

### 視覺跳躍
```
季節變化：雪→花
環境變化：老房子→新建築
外觀變化：短髮→長髮、孩子→成人
```

### 暗示跳躍
```
對話暗示：「這三年你去哪了？」
道具暗示：新照片、獎杯、信件日期
行為暗示：熟練度、習慣變化
```

## 時間線設計

### 線性時間線
```
過去 → 現在 → 未來

最易理解
適合：大多數故事
```

### 非線性時間線
```
現在 ↔ 過去（閃回）
現在 ↔ 未來（閃前）

增加層次
適合：懸疑、角色研究
```

### 多時間線
```
時間線A（過去） ╱
                 ╲→ 匯合
時間線B（現在） ╱

複雜但有深度
適合：史詩敘事
```

## 閃回使用原則

### 何時使用閃回
```
✅ 揭露關鍵背景
✅ 解釋當前行為
✅ 製造情感衝擊
✅ 打破線性單調

❌ 僅僅因為有這段故事
❌ 可以用對話替代時
❌ 打斷緊張節奏時
```

### 閃回技術
```
觸發物：照片、歌曲、地點
視覺區分：色調變化、畫幅變化
聲音先行：先聽到過去的聲音
```

## 時間節奏控制

### 每集時間跨度
```
高強度集：數小時-1天
中等集：數天-1週
舒緩集：可以跳躍較大

規則：重要時刻放慢，過渡加速
```

### 季度時間規劃示例
```
集1-2：第1天
集3-4：第1週
集5-6：第1個月
集7：時間跳躍（半年後）
集8-10：第2天（高潮期緊湊）
集11：數小時（決戰）
集12：結局+數月後的尾聲
```
