## 📐 默奇剪辑六法则

沃尔特·默奇（《教父》《现代启示录》剪辑师）提出的剪辑优先级：

| 优先级 | 法则 | 权重 | 说明 |
|--------|------|------|------|
| 1 | **情感** | 51% | 剪辑点是否**符合情感**？ |
| 2 | **故事** | 23% | 剪辑是否**推进故事**？ |
| 3 | **节奏** | 10% | 剪辑是否**有节奏感**？ |
| 4 | **视线** | 7% | 观众的**视线**是否被引导？ |
| 5 | **二维空间** | 5% | 构图在**屏幕上**是否连贯？ |
| 6 | **三维空间** | 4% | 是否符合**物理空间**逻辑？ |

> 💡 **情感永远是第一位的**。技术规则可以为情感让步。

## 🎯 剪辑的理想切点

### "眨眼"理论
**剪辑就像眨眼**——人在思考转换时会眨眼，剪辑点应该在观众"内心眨眼"的时刻。

### 理想切点特征
- 情感**转折**的瞬间
- 观众**获得信息**的瞬间
- 动作**完成**或**开始**的瞬间
- 视线**转移**的瞬间

## 🔄 剪辑节奏类型

### 1. 快节奏剪辑
| 特征 | 效果 |
|------|------|
| 镜头 < 2秒 | 紧张、激烈 |
| 快速切换 | 混乱、动作 |
| 跳切 | 时间压缩 |

**适用场景**：动作、追逐、惊悚

### 2. 慢节奏剪辑
| 特征 | 效果 |
|------|------|
| 镜头 > 5秒 | 沉浸、思考 |
| 长镜头 | 真实、连续 |
| 缓慢过渡 | 抒情、史诗 |

**适用场景**：情感、文艺、冥想

### 3. 变节奏剪辑
| 特征 | 效果 |
|------|------|
| 快慢交替 | 张弛有度 |
| 关键时刻放慢 | 强调重点 |
| 逐渐加快 | 紧张升级 |

## 🎬 常用剪辑技巧

### 1. 硬切 (Hard Cut)
直接从A切到B，最基本的剪辑。

### 2. 跳切 (Jump Cut)
同一场景跳过时间，制造不连续感。
**用途**：时间流逝、紧张

### 3. 匹配剪辑 (Match Cut)
两个相似元素的切换（形状、动作、声音）。
**例子**：骨头飞向天空 → 太空飞船（2001太空漫游）

### 4. 交叉剪辑 (Cross-Cutting)
两个不同空间的场景交替。
**用途**：平行动作、紧张感

### 5. 叠化 (Dissolve)
两个画面重叠过渡。
**用途**：时间流逝、梦境

### 6. 淡入淡出 (Fade In/Out)
渐显/渐隐到黑场。
**用途**：段落结束、重大转折

### 7. L剪辑 / J剪辑
| 类型 | 特征 | 效果 |
|------|------|------|
| **L剪辑** | 画面先切，声音延续 | 流畅过渡 |
| **J剪辑** | 声音先切，画面后切 | 预示、期待 |

### 8. 蒙太奇 (Montage)
一系列短镜头快速组合。
**用途**：时间流逝、情感升华、训练过程

## 📋 节奏设计模板

### 场景节奏曲线
```
紧张度
  ^
  │      ┌───┐
  │     /     \     ┌───┐
  │    /       \   /     \
  │   /         \ /       \
  │  /                     \
  │ /                       \
  └───────────────────────────→ 时间
    开始  铺垫  高潮  回落  结束
```

### 镜头时长规划
| 段落 | 情绪 | 建议镜头长度 |
|------|------|-------------|
| 开场 | 建立 | 3-5秒 |
| 铺垫 | 正常 | 2-4秒 |
| 高潮 | 紧张 | 1-2秒 |
| 回落 | 缓和 | 4-8秒 |

## 🔧 实操检查清单

- [ ] 每个剪辑点是否符合情感需求？
- [ ] 剪辑是否推进了故事？
- [ ] 节奏是否有变化（不单调）？
- [ ] 关键时刻是否有节奏变化？
- [ ] 视线引导是否清晰？
- [ ] 空间关系是否清楚？
