# 📱 条漫/Webtoon设计 Skill
> 适用于：竖屏短视频、韩式条漫、快看/腾讯动漫风格

---

## 📐 条漫 vs 传统漫画

| 维度 | 传统漫画 | 条漫 |
|------|----------|------|
| 阅读设备 | 纸/横屏 | 手机竖屏 |
| 阅读方向 | 翻页 | 无限滚动 |
| 单位 | 页（固定） | 条（无限） |
| 节奏控制 | 翻页 | 滚动速度 |
| 格子关系 | 紧密 | 分散+留白 |
| 分辨率 | 印刷 | 屏幕(800-1200px宽) |

## 🎯 条漫三大核心原则

### 1. 一屏一重点
```
┌─────────────┐
│             │
│   一个核心   │  ← 一屏只传达一个信息
│   信息/情感  │
│             │
└─────────────┘
```

### 2. 垂直叙事
```
时间/情绪
    ↓
┌───────┐
│ 起    │
├───────┤
│ 承    │
├───────┤
│ 转    │
├───────┤
│ 合    │
└───────┘
```
利用滚动方向制造节奏

### 3. 悬念滚动（Scroll Hook）
```
┌─────────────┐
│  "你知道吗  │
│   其实..."  │
│             │
├─────────────┤  ← 屏幕边缘（必须滚动）
│             │
│   "我喜欢   │
│    你"      │
└─────────────┘
```
关键信息放在需要滚动才能看到的位置

## 🎭 条漫节奏设计

### 留白控制节奏

**快节奏**（紧张/动作）
```
┌─────┐
│ 格1 │
├─────┤
│ 格2 │
├─────┤
│ 格3 │
└─────┘
紧密排列，快速滚动
```

**慢节奏**（情感/转折）
```
┌─────┐
│ 格1 │
│     │
│     │
│     │ ← 大量留白
│     │
├─────┤
│ 格2 │
└─────┘
留白延长阅读时间
```

### 情感放大技巧

**1. 逐步放大**
```
┌─────────┐
│ [远景]  │
├─────────┤
│ [中景]  │
├─────────┤
│ [近景]  │
├─────────┤
│         │
│ [特写]  │  ← 情感爆发
│  😢     │
│         │
└─────────┘
```

**2. 垂直下坠**
```
┌─────────┐
│   ↓     │
│   角色   │
│   ↓     │
│   坠落   │
│   ↓     │
│   中...  │
│   ↓     │
└─────────┘
利用滚动模拟下坠感
```

## 🪝 钩子设计（Hook）

### 每话开头钩子
前3屏必须抓住读者！

| 类型 | 例子 |
|------|------|
| **悬念** | "三天后，他死了" |
| **冲突** | 开场就是争吵/打斗 |
| **反差** | "我是全校最丑的人"（配图：美女） |
| **金句** | 震撼人心的台词 |

### 每话结尾钩子（让人追更）
| 类型 | 例子 |
|------|------|
| **悬念** | "但她不知道的是..." |
| **反转** | 最后一格揭示真相 |
| **表情** | 意味深长的眼神 |
| **问句** | "你猜，她会怎么选？" |

### 滚动中的微钩子
每10-15屏设置一个小钩子，保持阅读动力

## 💕 韩漫类型公式

### 恋爱类（Romance）
```
相遇 → 误会 → 心动 → 阻碍 → 告白 → 在一起
  ↑                              ↓
  └────── 竞争者/三角恋 ←────────┘
```

**节奏**：
- 每5话一个小高潮（心动/误会）
- 每15-20话一个大转折
- 告白通常在30-50话

### 复仇类（Revenge）
```
悲惨过去 → 重生/觉醒 → 筹备 → 复仇开始 → 一个个击破 → 最终BOSS
```

**爽点设计**：
- 每话至少一个"打脸"场景
- 恶人必须先嚣张再被打脸
- 打脸要分层次（小喽啰→中层→大BOSS）

### 重生/穿越类
```
惨死/意外 → 回到过去/穿越 → 利用先知 → 改变命运 → 新结局
```

**核心爽点**：
- 预知未来带来的优势
- 曾经错过的机会这次抓住
- 复仇+恋爱双线并行

## 📊 条漫与AI番剧的对应

| 条漫元素 | AI番剧对应 |
|----------|-----------|
| 一话 | 一集（3-8分钟） |
| 一屏 | 一个镜头（3-8秒） |
| 滚动钩子 | 镜头切换悬念 |
| 话末钩子 | 集末预告 |
| 留白 | 慢镜头/静止 |
| 特写放大 | 推镜头特写 |
