## 📋 8要素专业分镜格式

### 镜号：001

#### 1. Image Description（画面描述）
```
• 详细描述：清晨6点，城市天际线在晨雾中若隐若现。主角站在公寓顶楼边缘，
  背影孤独而决绝。风吹动他的衣角，远处教堂钟声响起。
• 情感传达：孤独、抉择、临界状态、城市的冷漠与生机并存
```

#### 2. Style Painting（画风参考）
```
• 参考画家：Edward Hopper的孤独都市感 + Gregory Crewdson的戏剧光影
• 色彩基调：蓝灰色主调，暖橙色晨光从左侧切入
• 质感要求：胶片颗粒感，高光柔和，阴影细节丰富
```

#### 3. Perspective（视角）
```
• 视角类型：低角度仰拍，增强主角的孤立感和决定的重要性
• 视点高度：从地面向上30度角
• 透视效果：轻微鱼眼效果，强调环境对主角的压迫感
```

#### 4. Shot Size（景别）
```
• 主要景别：Medium Long Shot（中远景）
• 人物占比：占画面高度2/3
• 环境占比：1/3天空，1/3城市，1/3楼顶
• 景深：浅景深，主角清晰，背景城市虚化但可辨认轮廓
```

#### 5. Video Prompt（AI生成提示词）
```
cinematic shot, early morning, man standing on rooftop edge, back view,
孤独的背影，城市天际线在晨雾中，blue-grey color palette, 
warm orange sunlight from left, Edward Hopper style, dramatic lighting, 
shallow depth of field, 35mm film grain, realistic details, 
cinematic atmosphere, 4K, professional photography
```

#### 6. Camera Movement（镜头运动）
```
• 运动类型：极缓慢的推镜头（3秒推进20%）
• 开始状态：Medium Long Shot
• 结束状态：Medium Shot
• 运动速度：0.5秒/百分比推进
• 稳定要求：三脚架稳定，无手持抖动
```

#### 7. Dialogue（对白）
```
• 角色：主角（内心独白）
• 内容："这就是终点了吗？还是...另一个起点？"
• 语气：平静中带着颤抖，自言自语的疑问
• 时机：在镜头推进到Medium Shot时开始
```

#### 8. Duration（时长）
```
• 镜头时长：7秒
• 分解：
  - 静止画面：2秒（建立环境）
  - 缓慢推进：3秒（营造紧张）
  - 静止特写：2秒（情感停留）
• 剪辑提示：与下个镜头的匹配剪辑（动作连贯）
```

## ✅ 分镜质量标准
- 视觉冲击力：每个画面都有明确的视觉焦点和情感指向
- 叙事清晰性：通过画面能理解故事进展
- 情感准确性：画面传递的情感与剧本要求一致
- 技术可行性：在预算和技术范围内可实施
