# 🎬 高潮设计 Skill
> 来源：《Story》Robert McKee + 《救猫咪》Blake Snyder

---

## 📐 什么是Climax（高潮）

**定义**：故事中情感和戏剧张力达到最高点的时刻

### 高潮的类型
| 类型 | 说明 | 示例 |
|------|------|------|
| **大高潮** | 全片最高点 | 最终决战 |
| **幕高潮** | 每幕结束 | 第二幕末的最低点 |
| **段落高潮** | 段落小高潮 | 发现关键线索 |
| **场景高潮** | 场景最强点 | 对话中的关键句 |

## 🔧 高潮设计要素

### 1. 赌注（Stakes）
```
高潮时刻，角色会失去/获得什么？

低赌注：可能迟到
中赌注：可能失去友谊
高赌注：可能失去生命
最高赌注：世界毁灭

赌注越高 → 高潮越有力
```

### 2. 不可逆性
```
高潮后，一切都不同了
├─ 关系改变（无法回头）
├─ 真相揭露（无法遗忘）
├─ 选择做出（无法收回）
└─ 牺牲付出（无法挽回）
```

### 3. 主动选择
```
✓ 好的高潮：主角做出选择
✗ 差的高潮：事情自己发生了

高潮 = 角色用行动回答核心问题
```

### 4. 情感释放
```
铺垫的所有情感在这里释放

积累的：
- 悬念解答
- 冲突解决
- 情感爆发
- 期待满足
```

## 📋 高潮场景设计

### 高潮三段式
```
┌─────────────────────────────────────────┐
│ 1. 高潮前：最后准备                     │
│    - 最后的犹豫/决心                    │
│    - 暴风雨前的宁静                     │
│    - 倒计时感                           │
├─────────────────────────────────────────┤
│ 2. 高潮中：行动时刻                     │
│    - 关键动作/对话                      │
│    - 最高张力                           │
│    - 决定性一刻                         │
├─────────────────────────────────────────┤
│ 3. 高潮后：立即反应                     │
│    - 情感释放                           │
│    - 后果初现                           │
│    - 新平衡开始                         │
└─────────────────────────────────────────┘
```

### 高潮节奏控制
| 阶段 | 节奏 | 镜头 | 音乐 |
|------|------|------|------|
| 高潮前 | 减速 | 长镜头/特写 | 降低/消失 |
| 高潮中 | 适中 | 强调性剪辑 | 高潮点 |
| 高潮后 | 看情况 | 反应镜头 | 情感释放 |

## 🎯 不同类型高潮

### 动作高潮
```
特点：物理对抗的顶点
要素：
- 最大规模的动作
- 最危险的时刻
- 关键一击

镜头：快剪 + 关键慢动作
```

### 情感高潮
```
特点：情感的顶点
要素：
- 关键台词
- 情感爆发
- 真相时刻

镜头：特写为主，停留够久
```

### 悬念高潮
```
特点：答案揭晓
要素：
- 所有线索汇聚
- 真相大白
- 反转（如有）

镜头：反应镜头链
```

### 道德高潮
```
特点：价值观的选择
要素：
- 两难困境
- 牺牲或妥协
- 定义角色的选择

镜头：沉默、凝视、决定瞬间
```

### 高潮场景Prompt

```
"epic confrontation, hero vs villain,

```

```
"emotional climax, tears streaming,

```

**告白/表白高潮**
```

```

```
|------|------|------|------|------|
| 01 | 远 | 对峙全景 | 3秒 | 紧张 |

| 05 | 特写 | 结果 | 3秒 | 释放 |
| 06 | 远 | 余波 | 4秒 | 沉淀 |
```

## 💡 高潮设计检查

### 必须有
- [ ] 前面有足够铺垫
- [ ] 赌注清晰且高
- [ ] 角色做出选择
- [ ] 有情感释放

### 避免
- ❌ 反高潮（突然泄气）
- ❌ 拖太长失去张力
- ❌ 结果太容易
- ❌ 没有后果
