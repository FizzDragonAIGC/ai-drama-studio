## 📐 集中线基础

### 什么是集中线
```
        ╲    │    ╱
         ╲   │   ╱
          ╲  │  ╱
           ╲ │ ╱
    ────────【焦点】────────
           ╱ │ ╲
          ╱  │  ╲
         ╱   │   ╱
        ╱    │    ╲
```

**作用**：将视线强制引导到焦点，表现震惊、冲击、重要时刻

---

## 🎯 集中线类型

### 1. 全方位集中线
```
    ╲   │   ╱
     ╲  │  ╱
      ╲ │ ╱
  ─────【焦点】─────
      ╱ │ ╲
     ╱  │  ╲
    ╱   │   ╲
```
**用途**：最强冲击、震惊表情

### 2. 水平集中线
```
  ══════════【焦点】══════════
  ══════════        ══════════
  ══════════        ══════════
```
**用途**：紧张对峙、横向冲击

### 3. 垂直集中线
```
       │ │ │
       │ │ │
       │ │ │
      【焦点】
       │ │ │
       │ │ │
       │ │ │
```
**用途**：压迫感、坠落、升华

### 4. 半圆集中线（背后发散）
```
         │ │ │
        ╱ │ │ │ ╲
       ╱  │ │ │  ╲
      ╱   │ │ │   ╲
     ╱ 【人物/焦点】 ╲
    ─────────────────
```
**用途**：角色登场、背后光芒

### 5. 不规则集中线
```
    ╲    ╲│ │╱    ╱
     ╲╲   ││╱   ╱╱
       ╲╲ ││ ╱╱
   ──────【焦点】──────
       ╱╱ ││ ╲╲
     ╱╱   ││╲   ╲╲
    ╱    ╱│ │╲    ╲
```
**用途**：混乱、疯狂、不安

## 📊 集中线与情感

| 情感 | 集中线特征 |
|------|------------|
| **震惊** | 全方位、密集、尖锐 |
| **愤怒** | 粗重、不规则 |
| **恐惧** | 锯齿状、抖动 |
| **觉醒** | 放射状、发光效果 |
| **紧张** | 水平、紧凑 |
| **压迫** | 垂直、向下 |
| **浪漫** | 柔和、稀疏 |

## 🔧 集中线参数

### 线条密度

| 密度 | 效果 | 线条数 |
|------|------|--------|
| **稀疏** | 轻微关注 | 8-12根 |
| **中等** | 普通强调 | 16-24根 |
| **密集** | 强烈冲击 | 32-48根 |
| **极密** | 最大冲击 | 64+根 |

### 焦点空白区

```
密集线条 ────→ 【空白圈】 ←──── 密集线条
                ↑
           焦点所在区域
         （保持清晰可见）
```

| 空白大小 | 效果 |
|----------|------|
| **小** | 压迫、窒息 |
| **中** | 标准强调 |
| **大** | 神圣、重要 |

## 📐 集中线绘制技巧

### 中心定位
```
1. 确定焦点中心（人物眼睛/关键物品）
2. 画出空白区域（不被线条覆盖）
3. 从边缘向中心画线
4. 线条在空白区边缘停止
```

### 线条变化
```
外 ══════════════════════╱ 内
粗                        细
（线条从外到内渐细）
```

### 空白区形状
| 形状 | 适用 |
|------|------|
| **圆形** | 人脸、物品 |
| **椭圆** | 人物半身 |
| **人形** | 全身剪影 |
| **不规则** | 配合动作 |

## 🎨 集中线变体

### 1. 闪光集中线
```
    ╲ ✦│✦ ╱
     ╲ │ ╱
  ─✦──【焦点】──✦─
     ╱ │ ╲
    ╱ ✦│✦ ╲
```
**效果**：神圣、觉醒

### 2. 锯齿集中线
```
    ╱╲ │ ╱╲
    ╲╱ │ ╲╱
  ────【焦点】────
    ╱╲ │ ╱╲
    ╲╱ │ ╲╱
```
**效果**：恐怖、危险

### 3. 双重集中线
```
外层（粗）
  ╲   │   ╱
   ╲ 内层（细） ╱
    ╲│ │ │╱
  ───【焦点】───
    ╱│ │ │╲
   ╱         ╲
  ╱   │   ╲
```
**效果**：更强冲击

## ⚠️ 集中线 vs 速度线

| 特征 | 集中线 | 速度线 |
|------|--------|--------|
| **方向** | 向心/离心 | 平行于运动 |
| **目的** | 强调焦点 | 表现速度 |
| **焦点** | 有明确焦点 | 跟随物体 |
| **情感** | 震惊、重要 | 动态、速度 |
