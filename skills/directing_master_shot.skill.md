# 🎬 主镜头法 Skill
> 来源：《电影导演创作》Sidney Lumet + 好莱坞传统拍摄法

---

## 📐 什么是Master Shot（主镜头）

**定义**：一个连续拍摄的长镜头，覆盖整场戏的全部动作和对白

### 主镜头的作用
| 作用 | 说明 |
|------|------|
| **安全网** | 保证有完整素材可用 |
| **空间建立** | 让观众理解场景布局 |
| **表演连贯** | 记录演员完整表演 |
| **剪辑锚点** | 随时可以回切的"家" |

## 🎯 主镜头类型

### 按功能分类

```
┌─────────────────────────────────────┐
│         主镜头类型                  │
├─────────────────────────────────────┤
│ 1. 全景主镜头 (Wide Master)         │
│    - 看到所有人物                   │

├─────────────────────────────────────┤
│ 2. 移动主镜头 (Moving Master)       │
│    - 摄影机跟随动作                 │
│    - 更动态                         │
├─────────────────────────────────────┤
│ 3. 近景主镜头 (Close Master)        │
│    - 从近景开始                     │
│    - 更亲密                         │
├─────────────────────────────────────┤
│ 4. 单人主镜头 (One-Shot Master)     │
│    - 一镜到底用作成片               │
│    - 需精确排练                     │
└─────────────────────────────────────┘
```

## 🔧 主镜头拍摄流程

### Step 1: 走位确认
```
1. 演员走位彩排
2. 确定镜头位置
3. 标记地板（mark）
4. 灯光调整
```

### Step 2: 技术彩排
```
1. 摄影机跟焦测试
2. 收音位置确认
3. 完整走一遍
4. 确认无技术问题
```

### Step 3: 拍摄
```
1. 从头到尾不停
2. 多拍几条备选
3. 表演细节可以调整
4. 确保有完整无误的一条
```

## 📋 主镜头设计要素

| 要素 | 考虑点 |
|------|--------|
| **景别** | 多宽才能包含所有人？ |
| **角度** | 哪个角度最能展示空间？ |
| **高度** | 平视还是稍高？ |
| **运动** | 固定还是跟随？ |
| **时机** | 何时适合回切主镜头？ |

## 🎬 主镜头剪辑策略

### 何时使用主镜头
- ✅ 场景开始（建立空间）
- ✅ 重要动作需要完整看到
- ✅ 特写太多需要"呼吸"
- ✅ 戏剧节奏需要放缓

### 何时避免主镜头
- ❌ 需要强调细节时
- ❌ 紧张高潮时刻
- ❌ 隐藏信息时

## 🎯 AI番剧应用

### 传统方法 vs AI方法

| 传统拍摄 | AI生成 |
|----------|--------|
| 真的拍一个长镜头 | 生成全景作为"锚点" |
| 演员真的走位 | 描述角色位置关系 |
| 可以直接用 | 通过一致性保持连贯 |

### 主镜头Prompt模板
```
wide shot establishing scene, [角色A] and [角色B] in [场景],
[角色位置关系], [光线], [氛围]
```

### 示例
```
"wide shot establishing scene, teacher standing at blackboard,
students sitting in rows, afternoon light through windows,
japanese high school classroom, anime style"
```

## 💡 大师技巧

### Sidney Lumet的建议
> "主镜头不只是保险，它定义了场景的基调。"

### 检查清单
- [ ] 所有人物都在画面内吗？
- [ ] 重要道具可见吗？
- [ ] 光线一致吗？
- [ ] 有足够头部空间吗？
- [ ] 场景的地理关系清晰吗？
