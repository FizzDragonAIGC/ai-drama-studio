# 🎬 越轴 Skill
> 来源：《电影剪辑技巧》Karel Reisz + 创意越轴案例分析

---

## 📐 什么是Crossing the Line（越轴）

**定义**：摄影机从轴线一侧移动到另一侧，打破180度法则

### 越轴的结果
| 效果 | 说明 |
|------|------|
| **位置反转** | 原本在左的角色变到右边 |
| **方向混乱** | 运动方向突然反转 |
| **空间迷失** | 观众不知道谁在哪 |

---

## 🎯 越轴类型

### 1. 无意越轴（错误）
```
导演/剪辑师失误导致
结果：观众困惑
处理：后期尽量补救
```

### 2. 有意越轴（技巧）
```
刻意打破规则制造效果
结果：特定心理效果
处理：需要铺垫或缓冲
```

## 🔧 越轴的视觉效果

### 对话场景越轴
```
正常：                越轴后：
┌──────────┐          ┌──────────┐
│ A    B   │    →     │ B    A   │
│ →    ←   │          │ ←    →   │
└──────────┘          └──────────┘
→ 观众：等等，谁是谁？
```

### 追逐场景越轴
```
正常：                越轴后：
┌──────────┐          ┌──────────┐
│ 追 → 逃  │    →     │ 逃 ← 追  │
└──────────┘          └──────────┘
→ 观众：他们反过来了？
```

## 📋 合理越轴的情况

### 1. 表达心理变化
| 场景 | 越轴效果 |
|------|----------|
| 角色心理崩溃 | 世界观颠倒感 |
| 关系反转 | 强弱交换 |
| 真相揭露 | "一切都不同了" |

### 2. 增强混乱感
```
适用场景：
- 打斗场面
- 恐慌人群
- 噩梦/幻觉
- 时间扭曲
```

### 3. 空间揭示
```
越轴可以揭示之前看不到的空间
类似"原来那边还有..."的效果
```

## 🎯 安全越轴技巧

### 技巧1：中性镜头缓冲
```
镜头1：轴线北侧
┌──────────┐
│ A    B   │
│ →    ←   │
└──────────┘
        ↓
镜头2：中性（正面/背面）
┌──────────┐
│    A     │
│    ↓     │  ← 在轴线上
│    B     │
└──────────┘
        ↓
镜头3：轴线南侧
┌──────────┐
│ B    A   │
│ →    ←   │
└──────────┘
→ 有缓冲，不那么突然
```

### 技巧2：运动越轴
```
一个连续镜头内移动过轴线
┌─────────────────────────┐
│   📹→  →  →  📹        │
│                         │
│───A═══════════B─────────│ 轴线
│                         │
└─────────────────────────┘
→ 观众"跟着"摄影机过去，可以接受
```

### 技巧3：角色运动
```
角色移动建立新轴线
原轴线：A ═══ B
        ↓ A移动
新轴线：A ═══ B（新位置）
```

### 技巧4：插入镜头
```
镜头1：轴线一侧
镜头2：特写细节（手、物品等）
镜头3：轴线另一侧
→ 插入镜头"重置"了观众感知
```

## 🎬 AI番剧应用

### 刻意越轴场景设计
```
场景：真相揭露时刻

镜头1-5：正常轴线（稳定）
"A on left, B on right, facing each other"

镜头6：揭露瞬间 - 越轴！
"B on left, A on right, reversed positions"

→ 视觉上传达"一切都不同了"
```

### 混乱场景prompt
```
打斗混乱：
"chaotic fight scene, disorienting angles,
dynamic camera positions, action confusion"

心理崩溃：
"psychological breakdown, tilted frame,
disorienting perspective, visual confusion"
```

### 越轴检查清单
```
刻意越轴时检查：
[ ] 有明确的叙事理由吗？
[ ] 用了缓冲技巧吗？
[ ] 效果是否如预期？
[ ] 之后有重建立空间吗？
```

## 💡 经典越轴案例

### 《闪灵》走廊场景
```
库布里克刻意越轴
效果：空间迷失、不安感
观众不知道酒店布局
→ 与角色一起迷失
```

### 《黑客帝国》打斗
```
高速打斗中越轴
效果：动作混乱、超现实
符合"超出正常物理"的设定
```

## ⚠️ 何时避免越轴

| 场景 | 原因 |
|------|------|
| 正常对话 | 会让观众困惑 |
| 需要清晰的追逐 | 方向感重要 |
| 观众需要理解空间 | 信息传递优先 |
| 新场景开始 | 尚未建立空间 |
