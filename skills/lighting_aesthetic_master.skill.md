## 💡 十大电影级光影类型

### 1. 侧逆光 (Rim Light / Back Light)
**效果**：神秘、圣洁、轮廓分明
**情绪**：神圣登场、命运时刻

```
侧逆光勾勒[主体]轮廓，金色光晕环绕发丝边缘，
面部保持深沉阴影，仅眼睛反射一点高光，
背景[光源]过曝形成光芒四射效果，
营造神秘圣洁的仪式感氛围
```

**参数**：
- 主光位置：主体侧后方45°-90°
- 光比：1:8 或更高
- 补光：仅眼神光

---

### 2. 伦勃朗光 (Rembrandt Lighting)
**效果**：戏剧性、古典、心理深度
**特征**：脸颊形成三角形光斑

```
伦勃朗式布光，[主体]面部形成经典三角光斑，
一侧脸颊沐浴在温暖的[色温]光线中，
另一侧沉入深邃阴影，仅眼眶下方保留三角形高光，
戏剧性明暗对比强化角色的内心挣扎与复杂性
```

**经典应用**：《教父》、《辛德勒的名单》

---

### 3. 丁达尔效应 (God Rays / Volumetric Light)
**效果**：神圣、希望、史诗感
**条件**：需要空气介质（雾/尘/烟）

```
丁达尔效应光束从[高窗/云层/树叶缝隙]间穿透而下，
可见光柱中漂浮的尘埃微粒闪烁金色光点，
光束精准照亮[主体/物体]，周围保持相对黑暗，
体积光营造神圣降临的仪式感，
仿佛命运之手在指引方向
```

**经典应用**：《肖申克的救赎》图书馆、《角斗士》竞技场

---

### 4. 低调光 (Low Key Lighting)
**效果**：压抑、悬疑、黑色电影感
**特点**：大面积暗部，小面积高光

```
低调光布局，画面90%沉入深邃黑暗，
仅[局部]被一束精准光线照亮，
强烈明暗对比营造压迫窒息感，
阴影中隐约可见[模糊轮廓]，
黑色电影质感，危险潜伏于暗处
```

**经典应用**：《七宗罪》、《唐人街》

---

### 5. 高调光 (High Key Lighting)
**效果**：明亮、纯净、梦幻
**特点**：均匀照明，阴影极少

```
高调光均匀照明，画面整体明亮通透，
阴影柔和几乎不可见，
[主体]沐浴在纯净的白色光芒中，
营造梦幻、圣洁、超脱现实的氛围
```

**经典应用**：回忆闪回、天堂场景、浪漫时刻

---

### 6. 冷暖对比 (Color Temperature Contrast)
**效果**：情感张力、空间分割、内心冲突

```
冷暖色温强烈对比，[主体]左侧沐浴在冷蓝色[光源A]中，
右侧被暖橙色[光源B]包裹，
两种色温在面部中央交汇形成微妙过渡，
视觉上分割内心的矛盾与挣扎，
冷色代表[理性/外界/未来]，暖色象征[情感/内心/过去]
```

**经典组合**：
- 🔵 冷蓝（月光/霓虹）vs 🟠 暖橙（灯光/火焰）
- 🟢 青绿（科技/外星）vs 🔴 红色（危险/激情）

---

### 7. 手电筒光 (Flashlight / Practical Light)
**效果**：恐怖、探索、悬疑
**特点**：单一移动光源

```
手持手电筒，圆形光束亮区集中在[焦点]，
边缘晕染自然朦胧的淡紫暗金色散色，
光束中可见漂浮的尘埃微粒，
光束外万物隐没于绝对黑暗，
营造未知恐惧与探索紧张感
```

---

### 8. 硬光直射 (Hard Light)
**效果**：残酷、审判、无情
**特点**：边缘锐利、阴影硬朗

```
硬光从[正上方/正前方]直射，
[主体]面部形成锐利分明的阴影边界，
眼窝深陷于黑暗中仅见瞳孔反光，
下巴阴影如刀切般利落投射在颈部，
无情的审判感，没有柔和余地
```

**经典应用**：审讯室、法庭、末日审判

---

### 9. 漫反射柔光 (Soft Diffused Light)
**效果**：温暖、亲密、治愈
**特点**：边缘柔和、过渡自然

```
柔和的漫反射光线笼罩整个空间，
阴影边缘如水墨晕染般自然过渡，
肤色呈现健康的暖调质感，
没有刺眼高光，一切温柔包裹，
营造安全、亲密、治愈的氛围
```

---

### 10. 剪影光 (Silhouette)
**效果**：神秘、孤独、史诗
**特点**：主体全黑，背景明亮

```
纯粹剪影构图，[主体]完全隐没于黑色轮廓中，
面部细节无法辨认，仅存姿态与轮廓，
背景[日落/火焰/爆炸]提供强烈逆光，
形成戏剧性的光影反差，
匿名感与孤独感并存
```

## 🎬 大师光影哲学

### Roger Deakins 四原则
1. **动机优先** - 每个光源都要有存在的理由
2. **自然主义** - 尽量使用场景内光源
3. **简约克制** - 用最少的光达到最好的效果
4. **服务叙事** - 光影是讲故事的工具

### Gordon Willis (教父摄影师)
- **顶光压眼** - 眼眶沉入阴影，增加神秘感
- **极致低调** - 宁可暗也不可亮
- **光即人格** - 光影直接表达角色性格

### 新海诚 光影特征
- **光斑泛滥** - 大量镜头眩光
- **体积云** - 光穿透云层
- **黄金时刻** - 日出日落极致利用
- **窗光教科书** - 室内自然光大师

## ✅ 光影审美检查清单

- [ ] 光源有明确动机（窗户/灯/火焰/月光）
- [ ] 光影服务情绪（不只是照亮）
- [ ] 光比符合叙事需求
- [ ] 色温与情感基调一致
- [ ] 阴影区有层次（不死黑）
- [ ] 高光区有细节（不过曝）
- [ ] 氛围词汇丰富（至少3个维度）
- [ ] 有参考风格方向（导演/电影）
- [ ] 考虑动态变化（光线随时间/情节变化）
