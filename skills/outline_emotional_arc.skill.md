## 核心理念

情感弧線 = 觀眾的情緒旅程
- 不是隨機的情緒波動
- 是精心設計的體驗曲線

## 情感節拍設計

### 單集情感節拍（24分鐘）
```
0-2分：情感錨點（建立基調）
2-8分：情緒發展
8-12分：情緒轉折
12-18分：情緒升級
18-22分：情緒高點
22-24分：情緒落點（懸念/餘韻）

規則：至少2次情緒轉變
```

### 季度情感節拍（12集）
```
集1-2：好奇+期待（建立）
集3-4：緊張+擔憂（衝突）
集5-6：希望+不安（發展）
集7：震驚+困惑（中點轉折）
集8-9：絕望+痛苦（低谷）
集10：決心+憤怒（反擊）
集11：緊張+激動（高潮）
集12：釋放+滿足（解決）
```

## 情緒對比技術

### 對比增強
```
悲傷前放喜悅 → 悲傷更悲傷
緊張前放平靜 → 緊張更緊張
勝利前放絕望 → 勝利更勝利

規則：情緒差距越大，衝擊越強
```

### 情緒轉場
```
硬切：突然轉變（震驚效果）
漸變：慢慢過渡（自然流暢）
對比切：A情緒→B情緒（諷刺/強調）
```

## 觀眾情感投資

### 建立投資的時機
```
集1-3：必須讓觀眾在乎主角
方法：
- 展示脆弱
- 展示善良
- 展示獨特
- 製造共鳴

如果集3觀眾還不在乎 → 後面的情感戲都無效
```

### 情感payoff
```
投資：觀眾在乎角色關係
payoff：關係的高潮時刻（重聚/決裂/告白）

投資：觀眾希望主角成功
payoff：主角實現目標的時刻

規則：payoff的分量 ≤ 投資的積累
```
