# 分集大纲设计 (Series Outline Design)

## 概述
将完整故事拆分为多集，每集都有完整结构同时服务整体叙事。

## 核心理论

### 分集的本质
- 每集是一个**迷你电影**：有开始、发展、高潮
- 每集同时是**整体的一部分**：推进主线
- 每集结尾是**钩子**：让观众想看下一集

### 时长与集数对照表

| 总内容量 | 3分钟/集 | 5分钟/集 | 8分钟/集 | 10分钟/集 |
|----------|----------|----------|----------|-----------|
| 5万字小说 | 20集 | 12集 | 8集 | 6集 |
| 10万字小说 | 40集 | 24集 | 15集 | 12集 |
| 20万字小说 | 80集 | 48集 | 30集 | 24集 |

### 分集节奏公式

**短剧（3-5分钟）**
- 开场钩子：30秒
- 快速发展：2分钟
- 小高潮/反转：1分钟
- 留悬念：30秒

**标准集（8-10分钟）**
- 开场回顾+新钩子：1分钟
- 发展A：3分钟
- 中点反转：1分钟
- 发展B：3分钟
- 高潮+悬念：2分钟

## 分集设计流程

### Step 1: 故事拆解
```
将原著故事按照以下维度拆解：

1. 列出所有【重大事件】
2. 标记【情感高点】
3. 识别【角色转变点】
4. 找出【信息揭示点】
```

### Step 2: 确定结构
```
【全剧结构】

第一幕（1-3集）：建立世界
- 主角日常
- 打破平衡
- 踏上旅程

第二幕A（4-7集）：试炼之路
- 新世界规则
- 结识伙伴
- 初次失败

中点（第8集）：重大反转
- 真相揭示
- 或重大失败
- 或虚假胜利

第二幕B（9-12集）：黑暗时刻
- 敌人反击
- 失去重要东西
- 主角最低点

第三幕（13-15集）：最终决战
- 内心觉醒
- 终极对决
- 新的平衡
```

## 钩子设计技巧

### 10种结尾钩子类型

1. **悬崖式** - 主角陷入危险，画面定格
2. **揭示式** - 揭示一个惊人真相
3. **反转式** - 刚才的理解全错了
4. **分离式** - 重要角色离开/被抓
5. **威胁式** - 更大的危险出现
6. **选择式** - 主角面临艰难选择
7. **谜题式** - 抛出新的谜题
8. **回归式** - 以为解决的问题卷土重来
9. **情感式** - 强烈情感时刻戛然而止
10. **时间式** - 倒计时开始

### 钩子强度分级
- ⭐ 轻钩：小悬念，可接受等待
- ⭐⭐ 中钩：想知道发生什么
- ⭐⭐⭐ 强钩：必须立刻看下一集

## Prompt模板

### 分集规划
```
为以下故事设计分集大纲：

【故事概要】
{story_summary}

【要求】
- 总集数：X集
- 每集时长：X分钟
- 目标平台：抖音/B站/YouTube

请输出：
1. 全剧三幕结构
2. 每集一句话概要
3. 主要反转点位置
4. 每集钩子类型
```

### 单集大纲
```
根据以下信息生成第X集详细大纲：

【前集结尾】
{previous_ending}

【本集需要完成】
- 主线任务：
- 角色发展：
- 信息揭示：

【下集需要铺垫】
{next_episode_setup}

【时长】X分钟

请按模板输出完整大纲。
```

## 检查清单

### 单集检查
- [ ] 开场是否有钩子？
- [ ] 是否有中点转折？
- [ ] 高潮是否足够强？
- [ ] 结尾钩子强度？
- [ ] 信息量是否合适？

### 全剧检查
- [ ] 节奏是否有变化？
- [ ] 是否有足够的高潮点？
- [ ] 主角成长是否清晰？
- [ ] 中点反转是否够大？
- [ ] 结局是否满足期待？
