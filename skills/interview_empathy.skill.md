## 元信息
- **技能ID**: interview_empathy
- **所屬Agent**: 訪談Agent

- **難度**: ⭐⭐⭐⭐

## 核心理念

共情式追問 ≠ 同情
- 不是「我理解你的感受」
- 而是「幫我進入你的世界」

## 共情追問四層次

### Level 1: 事實層
```
「當時具體發生了什麼？」
「你是在什麼情境下想到這個的？」
「能描述一下那個場景嗎？」
```

### Level 2: 行為層
```
「你當時做了什麼反應？」
「如果再來一次，你會怎麼做？」
「這個習慣是什麼時候開始的？」
```

### Level 3: 感受層
```
「那個時刻你的感受是什麼？」
「現在回想起來，情緒有變化嗎？」
「如果用一個顏色形容當時的心情？」
```

### Level 4: 意義層
```
「這件事對你意味著什麼？」
「為什麼這個故事值得被講述？」
「你希望觀眾從中得到什麼？」
```

## 追問觸發詞

### 當用戶說... → 你追問...

| 用戶說 | 追問 |
|--------|------|
| 「很有趣」 | 「有趣在哪裡？能具體說說嗎？」 |
| 「感覺不對」 | 「哪個部分讓你覺得不對？」 |
| 「就是那種感覺」 | 「那種感覺像什麼？冷的還是熱的？」 |
| 「大概是...」 | 「除了大概，還有什麼是確定的？」 |
| 「不知道怎麼說」 | 「沒關係，用比喻也行」 |
| 「別人都說...」 | 「那你自己覺得呢？」 |

## 沉默的力量

```
技巧：問完問題後，等待5秒
- 用戶會用更深層的答案填補沉默
- 不要急著追問或換話題
- 點頭、眼神接觸，表示「我在聽」
```

## 鏡像技術

```
用戶：「我想做一個關於孤獨的故事」
鏡像：「孤獨...」（重複關鍵詞，等待）
用戶：「對，就是那種在人群中的孤獨」
鏡像：「人群中的孤獨...」
用戶：「像是明明身邊都是人，但沒人真正理解你」
= 挖掘到了具體的孤獨類型
```

## 情感地圖繪製

```
步驟：
1. 請用戶描述故事的情感旅程
2. 用曲線圖標註高低點
3. 追問每個轉折點的原因

模板：
「如果把主角的情緒畫成一條線，
 開頭是什麼位置？
 最低點在哪裡？發生了什麼？
 最高點呢？
 結尾停在哪裡？」
```

## 身體語言觀察

| 信號 | 含義 | 追問 |
|------|------|------|
| 眼睛發亮 | 興奮點 | 「這個部分你特別有感覺？」 |
| 皺眉 | 困惑/衝突 | 「這裡有什麼讓你糾結的？」 |
| 語速加快 | 熱情 | 「聽起來這對你很重要」 |
| 停頓變長 | 深度思考 | （保持沉默，等待） |
| 手勢增多 | 試圖表達複雜概念 | 「可以畫給我看嗎？」 |


## 禁忌

- ❌ 打斷用戶的思路
- ❌ 過早給出解決方案
- ❌ 評判用戶的想法
- ❌ 用專業術語壓制
- ❌ 假裝理解（不懂就問）
