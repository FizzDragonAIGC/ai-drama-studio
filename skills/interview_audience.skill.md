# 🎤 目標受眾分析技能

## 元信息
- **技能ID**: interview_audience
- **所屬Agent**: 訪談Agent
- **參考書籍**: 《故事》麥基 + 《引爆點》格拉德威爾
- **難度**: ⭐⭐⭐

## 核心理念

受眾分析 ≠ 討好所有人
- 找到你的1000個鐵粉
- 不是「誰都能看」而是「專門為誰而做」

## 受眾畫像五維度

### 1. 人口統計
```
「你想像中的典型觀眾是誰？」
「年齡範圍？」
「什麼職業/生活狀態？」
「在哪裡看這部作品？（手機/電腦/電視）」
```

### 2. 心理特徵
```
「這群人平時喜歡看什麼？」
「他們的精神狀態？（焦慮/無聊/尋找意義）」
「什麼話題能引起他們的共鳴？」
「他們最近在擔心什麼？」
```

### 3. 觀看習慣
```
「他們一次能看多長時間？」
「喜歡追劇還是等完結再看？」
「會邊看邊刷手機嗎？」
「會二刷嗎？為什麼？」
```

### 4. 社交行為
```
「看完會分享嗎？分享到哪裡？」
「會截圖/做表情包嗎？」
「會寫長評還是打分走人？」
「會參與同人創作嗎？」
```

### 5. 付費意願
```
「願意為什麼付費？」
「多少錢會猶豫？多少錢會直接買？」
「會買周邊嗎？什麼類型的？」
```

## 受眾共鳴測試

### 「所以」測試
```
你說：「這是一個關於XX的故事」
觀眾心想：「所以呢？跟我有什麼關係？」

必須回答：
「如果你曾經 [經歷/感受]，這個故事就是為你準備的」
```

### 痛點共鳴
```
這群觀眾的痛點是什麼？
- 職場壓力？
- 感情困惑？
- 身份認同？
- 未來焦慮？

你的故事如何觸及這個痛點？
```

### 爽點設計
```
觀眾想要的「爽」是什麼？
- 逆襲打臉？
- 甜蜜撒糖？
- 燒腦解謎？
- 視覺奇觀？
- 情緒宣洩？
```

## 受眾層級

```
     [核心粉絲] 
        1%
    會安利、會創作
    
    [忠實觀眾]
       10%
    追完全系列、買周邊
    
   [普通觀眾]
      30%
    看完、評價、可能推薦
    
  [路人觀眾]
     60%
   刷到就看、看完就忘
```

**策略：先征服核心，再輻射外圈**

## 競品受眾分析

```
「你的目標觀眾現在在看什麼？」
「那些作品哪裡做得好？」
「哪裡讓觀眾不滿意？」
「你能填補什麼空白？」
```

## 反面受眾

```
「誰絕對不是你的目標觀眾？」
「如果他們不喜歡，你無所謂？」
「哪些評價你可以忽略？」

明確反面受眾 = 更清晰的定位
```
