# 😂 喜剧设计 Skill
> 如何让观众笑

---

## 📐 喜剧核心原理

### 笑的本质
```
预期 ≠ 结果 → 笑

铺垫 → 建立预期 → 打破预期 → 笑

关键：意外感 + 安全感
```

### 喜剧三要素
| 要素 | 说明 |
|------|------|
| 反差 | 预期vs现实 |
| 时机 | 对的时候说 |
| 节奏 | 快铺垫慢包袱 |

## 🎯 喜剧类型

### 1. 反差喜剧
```
类型：
- 外表vs内心
- 形象vs行为
- 期待vs现实

例子：
- 看起来很酷，其实很怂
- 以为是高手，其实是菜鸟
- 准备大干一场，结果秒败
```

### 2. 误会喜剧
```
类型：
- A以为的 ≠ 实际的
- 两人说的不是一件事
- 第三视角看到全貌

例子：
- 以为在告白，其实在说别人
- 两人互相误会在吃醋
- 观众知道但角色不知道
```

### 3. 夸张喜剧
```
类型：
- 表情夸张
- 反应过度
- 把小事说大

例子：
- 被吓到魂飞天外（字面意思）
- 一点小事全员震惊
- 内心OS画面化
```

### 4. 吐槽喜剧
```
类型：
- 对不合理的吐槽
- 打破第四面墙
- 毒舌

例子：
- "这明显是flag啊！"
- "剧本都不会这么写"
- 对同伴的无厘头吐槽
```

### 5. 重复喜剧
```
类型：
- 相同的梗重复出现
- 每次重复升级
- Running gag

例子：
- 每次出场都摔跤
- 每次说"不会出事"就出事
- 固定的搞笑口头禅
```

## 📊 喜剧节奏

### 铺垫-包袱节奏
```
铺垫：快速建立预期
停顿：吊一下
包袱：打破预期
反应：留时间让观众笑
```

### 三连笑结构
```
第一次：建立笑点
第二次：重复升级
第三次：再升级或反转
```

### 时机控制
```
包袱太快 → 观众没反应过来
包袱太慢 → 失去新鲜感

最佳时机：
观众刚建立预期的瞬间
```

## 🎭 角色喜剧类型

### 角色定位
| 类型 | 特点 | 功能 |
|------|------|------|
| 傻瓜 | 天然呆、不懂常识 | 制造天然笑料 |
| 吐槽役 | 理性、毒舌 | 吐槽傻瓜 |
| 高冷 | 面瘫、冷漠 | 反差制造 |
| 中二 | 自以为是 | 自嘲笑料 |
| 暴走 | 容易情绪失控 | 夸张反应 |

### 经典组合
```
傻瓜 + 吐槽役 = 经典漫才组合
高冷 + 暴走 = 反差CP
中二 + 普通人 = 一个唱一个看笑话
```

## 📊 喜剧禁忌

### 不好笑的原因
```
❌ 解释笑点（笑点一解释就死了）
❌ 笑点太老（陈旧的梗）
❌ 伤害性幽默（讽刺弱势群体）
❌ 节奏太慢
❌ 铺垫太明显
```

### 笑果减弱的原因
```
- 重复太多次
- 笑点透支
- 观众已经猜到
- 没有新意
```
