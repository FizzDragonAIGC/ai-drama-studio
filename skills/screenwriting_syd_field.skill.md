## 📊 三幕范式 (Paradigm)

悉德·菲尔德是好莱坞剧本结构的奠基人，他的三幕范式是所有编剧的基础。

### 基本结构

```
┌─────────────────────────────────────────────────────────────────────┐
│                          120页/分钟                                  │
├────────────────┬────────────────────────────────┬──────────────────┤
│    第一幕      │           第二幕               │      第三幕      │
│    建置        │           对抗                 │      解决        │
│   (1-30页)     │         (30-90页)              │    (90-120页)    │
├────────────────┼────────────────────────────────┼──────────────────┤
│                │                                │                  │
│    ↓           │              ↓                 │        ↓         │
│  情节点1       │           中点                 │     情节点2      │
│  (25-27页)     │          (60页)                │    (85-90页)     │
│                │                                │                  │
└────────────────┴────────────────────────────────┴──────────────────┘
```

---

## 🎬 三幕详解

### 第一幕：建置 (Set-Up)
**时长**：约30分钟/页
**功能**：
- 介绍主角
- 建立戏剧前提
- 创造戏剧情境
- 确立角色关系

**关键元素**：
| 元素 | 说明 | 位置 |
|------|------|------|
| 开场 | 视觉化呈现故事世界 | 前10页 |
| 激励事件 | 启动故事的事件 | 10-15页 |
| 情节点1 | 改变行动方向的事件 | 25-27页 |

---

### 第二幕：对抗 (Confrontation)
**时长**：约60分钟/页
**功能**：
- 主角追求目标
- 遭遇障碍
- 上升动作

**关键元素**：
| 元素 | 说明 | 位置 |
|------|------|------|
| 障碍1 | 第一个重大阻碍 | 35-40页 |
| 中点 | 故事方向转变 | 60页 |
| 障碍2 | 更大的阻碍 | 70-75页 |
| 情节点2 | 推向高潮的事件 | 85-90页 |

---

### 第三幕：解决 (Resolution)
**时长**：约30分钟/页
**功能**：
- 高潮
- 解决冲突
- 结局

---

## 🎯 情节点 (Plot Point)

### 定义
> "情节点是一个事件或事故，它'钩住'动作并把它转向另一个方向。"

### 情节点1
**位置**：第一幕结尾 (25-27页)
**作用**：
- 改变故事方向
- 将主角推入第二幕
- 提升赌注

**例子**：
- 《唐人街》：杰克发现委托人是假的
- 《洛基》：洛基获得拳王争霸机会

### 情节点2
**位置**：第二幕结尾 (85-90页)
**作用**：
- 再次改变方向
- 推向高潮
- 让主角面对最终考验

---

## 📝 场景构建

### 场景四要素
1. **目的**：这场戏为什么存在？
2. **地点**：在哪里发生？
3. **时间**：什么时候？
4. **角度**：从谁的视角？

### 场景结构
```
开头 → 发展 → 结尾
 ↓       ↓      ↓
建立   冲突   转折/揭示
```

---

## 🔧 角色塑造四维度

### 1. 戏剧性需求 (Dramatic Need)
- 主角想要什么？
- 这是故事的驱动力

### 2. 观点 (Point of View)
- 主角如何看待世界？
- 这决定了他的选择

### 3. 态度 (Attitude)
- 主角的行为方式
- 乐观/悲观/讽刺/热情

### 4. 变化 (Change)
- 主角如何改变？
- 人物弧光

---

## 📋 前10页检查清单

菲尔德强调：**前10页必须抓住读者**

- [ ] 主角登场？
- [ ] 戏剧前提清晰？
- [ ] 视觉化开场？
- [ ] 故事世界建立？
- [ ] 基调确立？
- [ ] 有悬念/疑问？
- [ ] 读者想继续看？

---

## 🎯 写作练习

### 练习1：定义戏剧前提
> 用一句话概括：谁 + 做什么 + 结果如何

### 练习2：设计情节点
> 列出两个关键转折点，检验是否：
> - 改变方向？
> - 提升赌注？
> - 产生后果？

### 练习3：场景卡片法
> 每个场景写一张卡片：
> - 场景号
> - 内/外景
> - 日/夜
> - 目的
> - 3-5行描述

|------|-------------|--------|----------|

| 学习曲线 | 简单 | 中等 | 复杂 |

## 🎬 应用示例：《唐人街》分析

### 第一幕 (1-30分钟)
- 杰克·吉特斯接受假莫雷太太的委托
- 调查莫雷先生的婚外情
- **情节点1**：真莫雷太太出现，威胁起诉

### 第二幕 (30-90分钟)
- 杰克深入调查水务局阴谋
- 发现莫雷的死亡真相
- **中点**：发现巨大的土地阴谋
- **情节点2**：发现伊芙琳的秘密（女儿/妹妹）

### 第三幕 (90-130分钟)
- 试图揭露真相
- 对抗诺亚·克洛斯
- **高潮**：唐人街的悲剧结局
