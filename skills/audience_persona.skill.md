## 📐 观众画像框架

### 基础画像维度
| 维度 | 问题 | 示例 |
|------|------|------|
| 年龄 | 几岁到几岁？ | 15-25岁 |
| 性别 | 主要性别？ | 60%女性 |
| 地域 | 哪里的人？ | 一二线城市 |
| 职业 | 做什么工作？ | 学生/上班族 |
| 收入 | 消费能力？ | 中等 |
| 平台 | 在哪看？ | B站/抖音 |

### 心理画像维度
| 维度 | 问题 |
|------|------|
| 核心需求 | 他们为什么看这个？逃避？学习？娱乐？ |
| 痛点 | 他们生活中什么让他们难受？ |
| 爽点 | 什么能让他们感到满足？ |
| 价值观 | 他们相信什么？ |
| 情感缺口 | 现实中缺什么，要从内容里补？ |

## 🎯 常见观众类型

### 按年龄
| 类型 | 特征 | 内容偏好 |
|------|------|----------|
| 00后 | 网感强、注意力短 | 快节奏、梗多、反转 |
| 95后 | 追求个性、怀旧 | 情怀、独特、有深度 |
| 90后 | 压力大、需解压 | 治愈、爽文、逆袭 |
| 80后 | 有家庭、时间少 | 精品、短平快 |
| 70后 | 传统、重剧情 | 正剧、历史、家庭 |

### 按需求
| 类型 | 核心需求 | 典型作品 |
|------|----------|----------|
| 爽文党 | 打脸、逆袭、升级 | 《斗破苍穹》 |
| 甜宠党 | 恋爱、撒糖、心动 | 《亲爱的热爱的》 |
| 虐心党 | 情感冲击、哭 | 《东京喰种》 |
| 燃向党 | 热血、奋斗、梦想 | 《灌篮高手》 |
| 治愈党 | 温暖、平静、美好 | 《夏目友人帐》 |
| 脑洞党 | 反转、悬疑、智斗 | 《死亡笔记》 |
| 硬核党 | 专业、考究、深度 | 《进击的巨人》 |

## 📊 画像模板

```
【目标观众画像】

一、基础信息
- 年龄段：
- 性别比：
- 地域：
- 职业：
- 平台：

二、心理特征
- 核心需求（为什么看）：
- 生活痛点：
- 内容爽点：
- 价值观：
- 情感缺口：

三、观看习惯
- 观看时间：
- 观看时长：
- 付费意愿：
- 互动习惯：

四、内容偏好
- 喜欢的类型：
- 喜欢的元素：
- 讨厌的元素：
- 参考作品：

五、关键洞察
- 一句话描述这群人：
- 他们最想看到什么：
- 他们最怕看到什么：
```

## 🎭 画像验证问题

### 检验画像准确性
1. 这个人会在什么情况下打开我们的内容？
2. 他看完第一集会继续看吗？为什么？
3. 他会分享给谁？怎么推荐？
4. 他愿意付费吗？为什么？
5. 他会二刷吗？

### 常见错误
- ❌ 目标观众是"所有人"
- ❌ 只看年龄性别不看心理
- ❌ 自己喜欢当观众喜欢
- ❌ 忽略观看场景和情绪状态

## 📋 不同IP的观众设计

### 《小王子》改编
```
核心观众：
- 25-35岁职场人
- 怀念童真、渴望简单
- 在现实中感到疲惫
- 需要：治愈、思考人生意义

内容策略：
- 保留哲理性
- 加入职场元素（小王子去打工）
- 治愈为主，反思为辅
- 画面温暖柔和
```

### 《西游记》改编
```
核心观众：
- 15-30岁男性为主
- 喜欢热血、战斗、逆袭
- 对传统文化有兴趣
- 需要：爽感、燃

内容策略：
- 强化战斗场面
- 加入升级系统
- 悟空要够强、够帅、够叛逆
- 画面要燃、要炸
```
