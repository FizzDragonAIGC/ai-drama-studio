# 😎 帅气设计 Skill
> 如何让角色/场面看起来超帅

---

## 📐 帅气核心要素

### 帅=什么
```
帅 = 能力 + 态度 + 时机 + 呈现

能力：做到常人做不到的事
态度：从容、淡定、不在意
时机：关键时刻出手
呈现：视觉+音效+节奏
```

---

## 🎯 帅气类型

### 1. 实力帅
```
特点：强者的从容
表现：
- 轻松解决难题
- 一招制敌
- 压倒性力量
- 面对危险不慌

Prompt: confident, powerful, overwhelming strength
```

### 2. 反差帅
```
特点：表面vs实际
表现：
- 看起来弱，实际超强
- 平时搞笑，战斗认真
- 装傻充愣，关键时刻帅

Prompt: unexpected coolness, sudden serious
```

### 3. 牺牲帅
```
特点：为他人付出
表现：
- 挡在前面
- 默默保护
- 放弃自己
- 背影离去

Prompt: protective, sacrificial, heroic
```

### 4. 沉默帅
```
特点：少说多做
表现：
- 话不多
- 用行动证明
- 关键时刻一句话
- 冷酷外表

Prompt: stoic, cool, silent type
```

### 5. 觉醒帅
```
特点：突破极限
表现：
- 濒死时爆发
- 保护重要的人
- 突破自我
- 变身/升级

Prompt: awakening, power up, breaking limits
```

## 📊 帅气场面设计

### 登场帅
```
元素：
- 逆光/剪影
- 从暗处走出
- 风吹动头发/披风
- 缓慢出场
- 一句简短台词

Prompt:
dramatic entrance, backlit silhouette,
wind blowing hair, slow walk,
confident, intimidating presence
```

### 战斗帅
```
元素：
- 动态姿势
- 速度线
- 冲击波
- 从容表情
- 一击必杀

Prompt:
dynamic action pose, speed lines,
impact effect, confident expression,
overwhelming attack
```

### 逆袭帅
```
元素：
- 从倒地到站起
- 擦去嘴角血
- 眼神变化
- 气场爆发
- 反杀

Prompt:
rising from defeat, wiping blood,
intense eyes, aura flaring,
counter attack, comeback
```

### 离去帅
```
元素：
- 背影
- 夕阳/暗光
- 风中站立
- 不回头
- 一句话后离开

Prompt:
cool exit, walking away,
back view, sunset, not looking back,
dramatic departure
```

## 🎭 让角色帅的技巧

### 设计层面
```
- 独特的剪影
- 标志性服装
- 标志性动作/姿势
- 冷酷/自信的表情
- 帅气的发型
```

### 行为层面
```
- 关键时刻出手
- 保护同伴
- 说到做到
- 不解释自己
- 承担责任
```

### 对白层面
```
- 话少但有分量
- 金句型台词
- 霸气宣言
- 安慰人简短有力
```

## 📋 帅气场面检查清单

### 必备要素
- [ ] 有仪式感吗？（不是随便帅）
- [ ] 有铺垫吗？（帅之前先蓄势）
- [ ] 节奏对吗？（慢→快→定格）
- [ ] 配乐到位吗？
- [ ] 有反差吗？（之前vs现在）

### 视觉检查
- [ ] 姿势够帅吗？
- [ ] 角度够好吗？（低角度更显帅）
- [ ] 光线够戏剧吗？
- [ ] 有特效吗？（风、光、气场）
- [ ] 值得截图吗？
