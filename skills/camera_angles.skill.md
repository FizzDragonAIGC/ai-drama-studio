# 📷 镜头角度 Skill
> 摄影角度与构图Prompt指南

---

## 📐 垂直角度

### 角度类型
| 角度 | 效果 | 使用场景 | Prompt |
|------|------|----------|--------|
| 仰拍 | 高大、英雄、威胁 | 英雄登场、反派、建筑 | low angle, looking up |
| 平拍 | 中性、客观 | 对话、日常 | eye level, straight on |
| 俯拍 | 渺小、脆弱、全局 | 孤独、俯瞰、弱势 | high angle, looking down |
| 鸟瞰 | 上帝视角、战略 | 全景、地图、战场 | bird's eye view, overhead |
| 虫眼 | 极端仰视 | 震撼、压迫 | worm's eye view, extreme low |

### 角度情感
```
仰拍 → 敬畏、力量、威胁
平拍 → 平等、客观、亲近
俯拍 → 怜悯、弱小、掌控
鸟瞰 → 疏离、全知、客观
```

## 🎯 水平角度

### 角度类型
| 角度 | 效果 | 使用场景 | Prompt |
|------|------|----------|--------|
| 正面 | 直接、对话 | 独白、面对观众 | front view, facing camera |
| 3/4侧面 | 立体、常用 | 多数场景 | three-quarter view, 3/4 angle |
| 侧面 | 轮廓、奔跑 | 行走、跑步 | side view, profile |
| 背面 | 神秘、离去 | 离开、思考 | back view, from behind |
| 过肩 | 对话、窥视 | 对话场景 | over the shoulder, OTS |

### 荷兰角
```
Dutch angle, tilted frame,
倾斜 15-45 度,
效果：不安、混乱、疯狂
使用：恐怖、动作、心理扭曲
```

## 📊 景别分类

### 标准景别
| 景别 | 范围 | 用途 | Prompt |
|------|------|------|--------|
| 大远景 | 全景+环境 | 建立场景、规模 | extreme wide shot, establishing |
| 远景 | 全身+环境 | 位置、空间 | wide shot, full shot |
| 全景 | 全身 | 动作、姿态 | full body shot |
| 中景 | 膝上 | 对话、互动 | medium shot, waist up |
| 中近景 | 胸上 | 情感、表演 | medium close-up, chest up |
| 近景 | 肩上 | 情绪、反应 | close-up, shoulders up |
| 特写 | 脸部 | 情感细节 | close-up face, facial |
| 大特写 | 眼睛/局部 | 极端情感 | extreme close-up, eyes only |
| 细节 | 物品/手 | 重要道具 | detail shot, insert |

## 🎭 组合角度

### 仰拍+近景
```
low angle close-up,
效果：威胁、权力、英雄
使用：反派、决断时刻
```

### 俯拍+远景
```
high angle wide shot,
效果：孤独、渺小、绝望
使用：角色孤立、困境
```

### 荷兰角+特写
```
Dutch angle close-up,
效果：疯狂、崩溃、不稳定
使用：心理压力、恐怖
```

### 平拍+中景
```
eye level medium shot,
效果：客观、日常、亲近
使用：对话、日常场景
```

## 📋 特殊镜头角度

### POV镜头
```
first person view, POV shot,
through character's eyes,
效果：代入感、主观体验
```

### 反射镜头
```
reflection shot, mirror view,
water reflection, window reflection,
效果：内心、双重、观察
```

### 透过物体
```
shot through [object],
foreground framing,
效果：窥视、层次、距离
```

### 主观运动
```
subjective camera, shaky POV,
running/walking perspective,
效果：紧张、追逐、恐惧
```

## 🔧 角度Prompt模板

### 基础格式
```
[景别] [角度] shot,
[主体], [状态/动作],
[光线], [风格],
cinematic, [情绪]
```

### 人物镜头
```
[景别], [垂直角度],
[角色描述], [表情],
[光线], [背景],
[风格], portrait
```

### 动作镜头
```
[角度] action shot,
[角色] [动作],
dynamic, motion,
[景别], dramatic angle
```

### 场景建立
```
establishing shot, wide angle,
[场景描述], [时间],
[天气], cinematic,
panoramic view
```

## 📊 角度-场景匹配

### 对话场景
| 内容 | 建议角度 |
|------|----------|
| 平等对话 | 平拍中景交替 |
| 权力对话 | 仰拍/俯拍对比 |
| 亲密对话 | 平拍近景/特写 |
| 对抗对话 | 低角度过肩 |

### 动作场景
| 内容 | 建议角度 |
|------|----------|
| 追逐 | 侧拍跟随 |
| 打斗 | 多角度快切 |
| 决斗 | 低角度、对称 |
| 坠落 | 仰拍/俯拍交替 |

### 情感场景
| 内容 | 建议角度 |
|------|----------|
| 悲伤 | 俯拍、特写 |
| 孤独 | 远景、负空间 |
| 希望 | 仰拍、光线 |
| 恐惧 | 荷兰角、阴影 |
