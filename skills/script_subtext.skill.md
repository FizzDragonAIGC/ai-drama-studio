## 核心理念

潛台詞 = 角色說的 ≠ 角色真正的意思
- 好對白有兩層：表面意思 + 真正意思
- 觀眾需要「讀懂」而非「被告知」

## 潛台詞三層次

### 層次1：直接表達
```
「我愛你。」= 我愛你

沒有潛台詞
適合：關鍵時刻、高潮場景
```

### 層次2：輕度潛台詞
```
「今天天氣真好。」= 我不知道該說什麼 / 我在迴避話題

稍微不直接
觀眾能輕易理解
```

### 層次3：深度潛台詞
```
「你的衣服很好看。」= 我注意到你了 / 我想接近你

需要上下文理解
更有味道
```

## 潛台詞技術

### 1. 說反話
```
表面：「我很好。」
真意：我很糟糕。

表面：「隨便你。」
真意：我很在意。

表面：「你真厲害。」
真意：（諷刺）你這個白痴。
```

### 2. 答非所問
```
A：「你愛我嗎？」
B：「今天月亮真圓。」
= 迴避問題 / 不敢回答

A：「你去哪了？」
B：「餓了嗎？我做飯。」
= 不想解釋 / 轉移話題
```

### 3. 過度禮貌
```
「打擾了，請問您有時間嗎？」
= 我們之間有距離 / 我很不舒服

「謝謝你的坦誠。」
= 你傷害了我
```

### 4. 沉默
```
A：「你能原諒我嗎？」
B：（長時間沉默）
= 還在掙扎 / 無法原諒

沉默是最強的潛台詞
```

### 5. 舊話重提
```
「記得你說過...」
= 我記得每一個細節
= 那件事對我很重要
```

## 潛台詞與情境

### 同一句話，不同潛台詞
```
「你回來了。」

場景1（分手後重逢）：我一直在等你
場景2（吵架後）：你終於回來道歉了
場景3（發現出軌）：我知道你去哪了
場景4（平靜日常）：（無潛台詞）
```

## 潛台詞檢查清單

```
□ 這句話表面說的是什麼？
□ 角色真正想表達的是什麼？
□ 為什麼角色不直說？
□ 觀眾能讀懂嗎？
□ 是否有足夠的情境暗示？
```

## 危險區域

```
❌ 太隱晦：觀眾完全無法理解
❌ 太直白：失去潛台詞的魅力
❌ 過度使用：角色像在打啞謎
✅ 平衡：關鍵時刻直接，日常有潛台詞
```
