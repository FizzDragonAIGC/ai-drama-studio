# 电视剧编剧技巧 Skill
> 来源：《美剧编剧入行手册》艾伦·桑德勒
> 适用智能体：A23 电视剧编剧

## 核心理念

**电视剧 = 角色驱动的长篇叙事**

你的故事必须围绕中心人物展开，角色的选择推动情节进展。

---

## 电视剧 vs 电影

| 方面 | 电影 | 电视剧 |
|------|------|--------|
| 结构 | 完整三幕 | 多幕+系列 |
| 角色 | 变化弧光 | 持续发展 |
| 冲突 | 单一解决 | 持续升级 |
| 时长 | 90-180分钟 | 每季8-24集 |
| 结局 | 封闭 | 开放+延续 |

## 美剧结构类型

### 四幕剧（1小时剧集）

```
每幕约12分钟，每幕5-7场戏

第一幕（建立）
├── 冷开场（Teaser）
├── 问题引入
└── 第一个转折 → 广告

第二幕（复杂化）
├── 障碍出现
├── B线展开
└── 中点危机 → 广告

第三幕（危机）
├── 情况恶化
├── 看似无解
└── 最低点 → 广告

第四幕（解决）
├── 最终对决
├── 解决
└── 新问题暗示
```

### 五/六幕剧

更多广告间隔，每幕更短，节奏更紧凑。

### 半小时喜剧

```
A故事（主线）+ B故事（副线）+ C故事（小品）
每幕约10分钟
```

---

## A-B-C故事线

### A故事（主线）
- 占60-70%篇幅
- 主角的核心冲突
- 贯穿全集

### B故事（副线）
- 占20-30%篇幅
- 配角或主角的次要冲突
- 与A线互动或对照

### C故事（调味）
- 占10%篇幅
- 轻松调剂
- 可选，主要用于喜剧

### 交织技巧
```
A开场 → B引入 → A发展 → C片段 → B危机 
    → A高潮 → B解决 → A结尾
```

## 感叹词式结构指南

桑德勒的"感叹词"节奏：

| 位置 | 感叹词 | 情绪 |
|------|--------|------|
| 开场 | "哇！" | 抓住注意 |
| 转折1 | "哦不！" | 问题出现 |
| 中点 | "天哪！" | 情况复杂 |
| 最低点 | "完蛋了！" | 绝望 |
| 高潮 | "YES！" | 胜利/解决 |
| 结尾 | "嗯？" | 新悬念 |

## 剧本写作流程

### 1. 故事梗概（Logline）
一句话概括：
> [角色] + [冲突] + [特殊设定]

### 2. 故事大纲（Outline）
2000字左右：
- 故事走向
- 核心主线
- 人物关系图
- **标注付费卡点**（如适用）

### 3. 分场大纲（Beat Sheet）
每场戏一段：
- 场景/时间
- 在场角色
- 发生什么
- 情绪/目的

### 4. 剧本初稿
- 1分钟 ≈ 1页剧本
- 50分钟剧集 ≈ 50页

### 5. 改写，改写，再改写

## 集钩子设计

### 集头钩子（Cold Open）
- 冲击性画面
- 悬念问题
- 时间跳跃
- 意外事件

### 广告前钩子（Act Break）
- 危机升级
- 反转揭示
- 情感高点
- "会怎样？"问题

### 集尾钩子（Tag/Cliffhanger）
- 新威胁出现
- 关系转变
- 秘密揭露
- "下周见！"悬念

## 季度规划

### 单季12集示例

| 集 | 功能 | 主要内容 |
|----|------|---------|
| 1 | 试播集 | 世界观+角色+核心问题 |
| 2-3 | 建立 | 日常+配角+规则 |
| 4 | 转折 | 第一次重大挑战 |
| 5-6 | 发展 | 深入角色+次要冲突 |
| 7 | 中季高潮 | 重大转折/揭示 |
| 8-9 | 升级 | 风险提高+关系变化 |
| 10 | 危机 | 一切崩塌 |
| 11 | 高潮前 | 准备最终对决 |
| 12 | 季终 | 解决+新悬念 |

### 人物弧线规划

```
角色A：恐惧 → 面对 → 克服 → 成长
       ↓      ↓      ↓      ↓
集数：  1-3    4-6    7-10   11-12
```

## 编剧室工作

### 角色分工
- **Showrunner**：总负责
- **执行制片人**：管理层
- **主笔**：核心故事
- **故事编辑**：修改润色
- **Staff Writer**：执行写作

### 故事会议流程
1. 提出想法
2. 头脑风暴
3. 投票选择
4. 分场讨论
5. 分配写作

## AI番剧长篇设计

### 每集60-80镜头
- 叙事镜头：50-60%
- 戏剧镜头：25-30%
- 情感镜头：15-20%

### 分集Prompt管理
```
集1_场景01_镜头01.prompt
集1_场景01_镜头02.prompt
...
```

### 连续性检查
- 角色造型一致
- 场景延续正确
- 服装道具连贯
- 时间线无矛盾
