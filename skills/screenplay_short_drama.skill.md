# 短劇/微短劇劇本寫作 - Short Drama Writing

## 短劇類型定義

| 類型 | 單集時長 | 總集數 | 總時長 | 平台 |
|------|----------|--------|--------|------|
| 微短劇 | 1-3分鐘 | 60-100集 | 2-5小時 | 抖音/快手/紅果 |
| 短劇 | 3-10分鐘 | 20-40集 | 2-4小時 | 愛奇藝/騰訊/B站 |
| 迷你劇 | 10-20分鐘 | 6-12集 | 2-4小時 | Netflix/YouTube |

## 微短劇核心法則

### 1. 黃金三秒法則
```
前3秒 = 生死線
- 必須有視覺衝擊或懸念
- 不要片頭、不要鋪墊
- 直接進入衝突

❌ 錯誤開場：「那是一個普通的早晨...」
✅ 正確開場：「簽字吧，離婚協議書。」
```

### 2. 付費卡點設計
```
免費集 (1-10集): 建立世界，製造爽點
付費牆 (第10-15集): 最大懸念處

卡點類型：
- 身份揭露：「你不知道我是誰？我是...」→ 下集揭曉
- 命運轉折：刀架脖子上 → 下集見分曉
- 情感高潮：「我愛的人是...」→ 未說完
- 打臉時刻：反派囂張到極點 → 打臉在下集
```

### 3. 每集結構 (1-3分鐘)
```
0:00-0:03 - 鉤子開場 (Hook)
0:03-0:30 - 衝突升級 (Escalation)
0:30-1:30 - 核心場景 (Core Scene)
1:30-2:00 - 反轉/爽點 (Payoff)
2:00-2:30 - 下集懸念 (Cliffhanger)
```

## 爆款短劇套路

### 1. 霸總復仇類
```
結構：
1. 女主被欺負（建立同情）
2. 霸總出現（神秘身份）
3. 反派囂張（製造爽點期待）
4. 霸總打臉（情緒釋放）
5. 身份揭曉（更大爽點）

關鍵：每3-5集一個打臉高潮
```

### 2. 穿越重生類
```
結構：
1. 前世悲慘結局
2. 重生回到關鍵時刻
3. 改變命運的選擇
4. 報復/拯救行動
5. 新結局

關鍵：觀眾有「上帝視角」的爽感
```

### 3. 甜寵類
```
結構：
1. 歡喜冤家相遇
2. 被迫共處
3. 漸生情愫
4. 誤會分離
5. 甜蜜HE

關鍵：每集至少一個「甜點」
```

### 4. 逆襲打臉類
```
結構：
1. 主角被看不起
2. 隱藏身份/能力
3. 反派不斷挑釁
4. 逐步揭示實力
5. 終極打臉

關鍵：「裝X打臉」節奏要穩
```

## 短劇對白技巧

### 1. 金句設計
```
每集必須有1-2句「金句」：
- 可截圖傳播
- 情緒爆發點
- 角色標誌性台詞

例：
「三年前你讓我滾，三年後我讓你跪。」
「你配嗎？」
「我從不威脅人，我只陳述事實。」
```

### 2. 衝突對白公式
```
A說狠話 → B更狠回擊 → A愣住 → 反轉

例：
反派：「就憑你？也配和我爭？」
主角：「我不和你爭。」
反派：（得意）
主角：「因為你根本不是我的對手。」
```

### 3. 避免的對白
```
❌ 長篇大論的解釋
❌ 無衝突的日常閒聊
❌ 重複信息
❌ 書面語/文藝腔

✅ 短句、有力
✅ 潛台詞
✅ 情緒張力
✅ 口語化
```

## 節奏設計

### 集數節奏分配
```
第1-5集：建立（免費區）
- 介紹主角
- 建立衝突
- 製造期待

第6-15集：升級（付費牆附近）
- 衝突升級
- 小高潮不斷
- 付費卡點在此

第16-40集：爆發
- 主線推進
- 副線交織
- 每5集一個大高潮

最後5集：收尾
- 大boss對決
- 所有伏筆回收
- 滿足感結局
```

### 單集節奏（3分鐘版）
```
0:00 - 承接上集/新衝突
0:30 - 第一個小高潮
1:00 - 轉折或深化
1:30 - 第二個高潮
2:00 - 反轉
2:30 - 懸念/鉤子
```

```
請為以下短劇生成劇本：

【類型】：霸總/穿越/甜寵/逆襲
【單集時長】：1-2分鐘 / 3-5分鐘
【總集數】：60集 / 80集 / 100集
【付費卡點】：第10-15集

【一句話概念】：
【主角】：名字、身份、目標
【反派】：名字、身份、與主角關係
【核心爽點】：（打臉/逆襲/甜寵/復仇）

要求：
1. 每集必須有鉤子開場和懸念結尾
2. 付費卡點設計在最大懸念處
3. 每3-5集一個打臉/爽點高潮
4. 對白簡短有力，含金句
5. 輸出：
   - 60集分集大綱
   - 每集核心衝突
   - 付費卡點位置
   - 3句代表性金句
```

## 行業數據參考

### 爆款標準
```
完播率 > 40%
互動率 > 5%
付費轉化 > 3%
```

### 平台偏好
```
抖音：節奏快，1-2分鐘，強反轉
快手：接地氣，3-5分鐘，情感向
紅果：爽文改編，霸總/逆襲
B站：年輕向，創意/反套路
```
