# 蒙太奇写作 Skill
> 来源：爱森斯坦蒙太奇理论、影视编剧技巧
> 适用智能体：编剧助手、分镜设计师

## 核心理念

**"1+1>2"** —— 两个镜头的组合产生超越单独镜头的新意义

蒙太奇（Montage）是通过镜头组接创造意义的艺术。

---

## 核心理论

### 蒙太奇的本质

```
镜头A + 镜头B = 新意义C
示例：哭泣的脸 + 棺材 = 悲痛
      哭泣的脸 + 彩票 = 狂喜
```

### 蒙太奇类型

| 类型 | 说明 | 示例 |
|------|------|------|
| 叙事蒙太奇 | 推进故事时间 | 训练过程、时间流逝 |
| 对比蒙太奇 | 形成对照反差 | 贫vs富、战vs和 |
| 隐喻蒙太奇 | 象征性表达 | 羊群→人群 |
| 节奏蒙太奇 | 制造节奏感 | 战斗、追逐 |
| 心理蒙太奇 | 表达内心状态 | 回忆、幻觉 |
| 积累蒙太奇 | 情感累积 | 多角色同时反应 |

### 蒙太奇 vs 场景

| 蒙太奇 | 场景 |
|--------|------|
| 多时空碎片 | 单一时空连续 |
| 压缩时间 | 接近实时 |
| 通过组接产生意义 | 通过动作产生意义 |
| 通常无对话或少对话 | 对话驱动 |

## 实操技巧

### 叙事蒙太奇（时间压缩）

**适用场景**：训练、成长、旅程、准备

**结构模板**：
```
开始状态 → [多个片段] → 结束状态
          ↑
        通常3-7个镜头
        每个镜头3-5秒
        可配音乐
```

**示例：训练蒙太奇**
```
1. 俯卧撑做不起来
2. 跑步摔倒
3. 剑术被打败
   --- 时间流逝暗示 ---
4. 俯卧撑流畅
5. 跑步领先
6. 剑术打平
7. 最终一击取胜
```

### 对比蒙太奇

**适用场景**：展示差距、讽刺、戏剧张力

**结构模板**：
```
镜头A（状态1）→ 镜头B（状态2）→ 镜头A'（深化1）→ 镜头B'（深化2）
```

**示例：贫富对比**
```
1. 富人餐桌：满汉全席
2. 穷人餐桌：一碗白粥
3. 富人孩子：挑食
4. 穷人孩子：珍惜每一粒米
5. 富人倒掉剩菜
6. 穷人舔干净碗
```

### 平行蒙太奇

**适用场景**：同时发生的事件、紧张感

**结构模板**：
```
线A镜头1 → 线B镜头1 → 线A镜头2 → 线B镜头2 → 汇合
```

**示例：营救**
```
1. 炸弹倒计时：59:00
2. 主角在路上狂奔
3. 炸弹：45:00
4. 被困者试图挣脱
5. 炸弹：30:00
6. 主角遇到障碍
7. 炸弹：10:00
8. 主角到达，剪线
```

### 心理蒙太奇

**适用场景**：内心世界、情感爆发、回忆

**技巧**：
- 可以打破物理法则
- 意象化表达
- 快速闪回
- 碎片化

**示例：创伤闪回**
```
1. 触发物（某个声音/画面）
2. 闪回片段1（0.5秒）
3. 回到现实，角色反应
4. 闪回片段2（0.5秒）
5. 闪回片段3（1秒）
6. 回到现实，角色崩溃
```

## Prompt模板

### 蒙太奇生成

```
请为以下需求设计蒙太奇序列：

目的：[时间压缩/对比/平行/心理...]
开始状态：[蒙太奇开始时的状态]
结束状态：[蒙太奇结束后的状态]
时长约束：[预计多少秒/多少个镜头]

请输出：
1. 蒙太奇类型选择及理由
2. 镜头列表（每个镜头3-5秒）
3. 每个镜头的内容描述
4. 音乐/音效建议
5. 整体节奏设计
```

### 训练蒙太奇

```
设计一段训练/成长蒙太奇：

角色：[谁在训练]
训练内容：[学什么]
起点：[完全不会？基础薄弱？]
终点：[达到什么程度]
情感曲线：[挫折→突破？平稳上升？]

要求：
1. 5-7个镜头
2. 每个镜头内容+时长
3. 有挫折节点
4. 有突破时刻
5. 配乐建议
```

### 对比蒙太奇

```
设计一段对比蒙太奇：

对比主题：[什么vs什么]
对比维度：[列出几个对比角度]
情感目标：[想让观众感受什么]

请输出：
1. 6-8个镜头
2. AB交替结构
3. 强化对比的视觉元素
4. 最终结束在哪方
```

## 检查清单

### 蒙太奇设计检查
- [ ] 类型选择正确？
- [ ] 镜头数量合适（不要太多太少）？
- [ ] 有明确的开始和结束状态？
- [ ] 节奏有变化？
- [ ] 不需要额外解释？

### 蒙太奇写作检查
- [ ] 每个镜头描述清晰简洁？
- [ ] 镜头之间有逻辑/情感连接？
- [ ] 有音乐/音效配合？
- [ ] 时间节奏标注？

## 案例分析

### 案例：《洛奇》训练蒙太奇

**目的**：展示洛奇从业余拳手到挑战者的蜕变

**镜头序列**：
| 序号 | 内容 | 时长 | 节奏 |
|------|------|------|------|
| 1 | 凌晨喝生鸡蛋 | 3s | 慢 |
| 2 | 跑步，喘不上气 | 3s | 中 |
| 3 | 打沙袋，动作笨拙 | 3s | 中 |
| 4 | 跳绳绊倒 | 2s | 快 |
| 5 | 继续跑步 | 3s | 中 |
| 6 | 打沙袋，开始有力 | 3s | 中 |
| 7 | 跳绳流畅 | 2s | 快 |
| 8 | 跑上台阶，艰难 | 4s | 慢 |
| --- | 时间过渡 | --- | --- |
| 9 | 跑上台阶，轻松 | 3s | 快 |
| 10 | 冲上顶端，举拳欢呼 | 5s | 高潮 |

**配乐**：《Gonna Fly Now》全程

**技巧分析**：
- 相同动作对比（跑台阶）体现进步
- 音乐贯穿统一
- 从慢到快的节奏
- 最后定格在高潮画面

### 番剧蒙太奇示例

**场景**：日常相处蒙太奇

```
INT./EXT. 多场景 - 日/夜 - 蒙太奇

【配乐：轻快的日常BGM】

1. 教室。小红传纸条给小明。小明看了偷笑。(3秒)

2. 便利店。两人一起挑泡面。为了最后一包争抢。(3秒)

3. 河边。并肩走着。各自吃着冰棍。(3秒)

4. 图书馆。小明睡着了。小红帮他盖上外套。(4秒)

5. 游戏厅。两人激烈对战。小明输了，一脸不甘。(3秒)

6. 天台。夕阳下。两人背靠背坐着，什么都没说。(5秒)

【音乐淡出】
```

## 常见错误

### ❌ 蒙太奇太长
超过60秒容易疲劳

### ❌ 镜头无关联
随机拼凑，没有逻辑或情感线

### ❌ 缺少变化
所有镜头节奏一样

### ❌ 结束弱
没有定格在有力的画面

### ❌ 过度依赖
太多蒙太奇让故事碎片化
