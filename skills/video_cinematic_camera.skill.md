# 电影级AI视频运镜提示词 Skill

> 参考即梦AI / Jimeng 3.5 Pro 专业运镜提示词库

## 核心理念
**首帧 + 运镜描述 + 尾帧 = 电影级视频**

不只描述画面，更要描述**镜头如何移动**、**情绪如何变化**、**视觉重心如何转移**。

---

## 🎬 8大电影级运镜类型

### 1. 平滑升镜环绕 (Pedestal Up + Orbit)
**效果**：优雅、神秘、仪式感

**提示词模板**：
```
以低角度跟随镜头开始，追踪[地面元素]的[动态]。镜头顺着[主体]的身体平滑上升(Pedestal Up)到面部，然后无缝过渡到一个缓慢的180度环绕镜头(Orbit)。随着镜头的环绕，由于长焦镜头效果，背景的[大型元素]看起来被压缩并巨大地显现在[主体]身后。
```

**示例**：
```
以低角度跟随镜头开始，追踪地面附近白色丝带的飘动。镜头顺着她的身体平滑上升(Pedestal Up)到面部，然后无缝过渡到一个缓慢的180度环绕镜头(Orbit)。随着镜头的环绕，由于长焦镜头效果，背景的月亮看起来被压缩并巨大地显现在她身后。
```

---

### 2. 荷兰角滚转推 (Dutch Angle + Camera Roll)
**效果**：失衡、压迫、末日感

**提示词模板**：
```
（电影级失衡运镜，荷兰角构图），画面处于倾斜状态，镜头在[环境]中缓慢向前推进，同时围绕画面中心进行（顺时针/逆时针 Z轴滚转 Camera Roll）。两侧[高耸元素]随着镜头的旋转仿佛正在向中间倒塌，挤压观众的视线。处于画面中心的[主体]保持相对稳定，仿佛它是混乱世界的唯一主宰。这种旋转带来的眩晕感与[环境元素]的混乱交织，营造出一种噩梦般的、世界崩塌的压迫氛围，光影随着旋转剧烈变幻。
```

---

### 3. 螺旋极速俯冲 (Spiral Dive)
**效果**：震撼、史诗、冲击力

**提示词模板**：
```
镜头从[主体]饱经沧桑且充满侵略性的头部特写开始，向外进行螺旋式快速移动。在穿过层层迷雾并短暂升高后，镜头进行一次极具戏剧性的高速俯冲，相机直指下方[场景元素]，产生强烈的透视变化感，周围飞速掠过的[细节]因极速而剧烈模糊。随后，镜头流畅地过渡到围绕[场景]的低角度环绕拍摄。最后，镜头平稳地调整构图并向后拉远，将[全部元素]一同框入画面，呈现[场景]令人窒息的史诗对峙全景。
```

### 4. FPV主观视角 (First Person View)
**效果**：沉浸、探索、悬疑

**提示词模板**：
```
FPV第一人称主观视角，镜头伴随步行持续轻微颤抖扭动；[场景描述]。手持手电筒，向左缓步深入，镜头猛地向右急晃，手电筒光束骤然扫到[发现物]，[发现物描述]撞入视野，雕像保持绝对静止不动状态，背景无配乐，仅有白噪音（[环境音描述]）、同步放大的粗重急促呼吸声、脚步声。
```

**图片提示词**：
```
第一人称视角，POV视角，手持手电筒，圆形光束亮区集中在人物身上，边缘晕染自然朦胧的淡紫暗金色散色，光束可见浮尘，光束外万物隐没于绝对黑暗。与参靠图角色一致的石像，高角度视角，中景，侧视，他的所有皮肤（包括脸部）已石化，身上结着真实细腻的蛛网和灰尘，黄金盔甲已无光泽，布满尘埃，他双眼平静闭合。[细节描述]。超写实高细节质感，戏剧性强烈光影对比，营造神秘废弃、末日静谧的诡异氛围，科幻与衰败交织，尘埃、蛛网都极具真实触感。
```

### 5. FPV穿越俯冲 (FPV Dive Through)

```
电影级FPV飞行视角，极速贴地感。从高空向[目标]方向发起侵略性的快速降落，距离感瞬间拉近。画面边缘呈现强烈的径向模糊效果，视觉重心紧锁快速逼近的[目标细节]。伴随高频震动的镜头语言，展现出如同战斗机俯冲般的视觉冲击力。
```

### 7. 跟随推进 (Follow Push)
**效果**：紧张、追逐、悬疑

**提示词模板**：
```
镜头从[主体]背后开始，以稳定的速度跟随[主体]向前移动。随着[主体]的脚步，镜头逐渐推近，从中景过渡到近景。[环境变化描述]，光影在[主体]身上流动变化。最终镜头停在[主体]肩膀位置，与[主体]一同面对[前方场景]。
```


## 📋 视频提示词结构

### 标准结构（5层）
```
1. 【运镜类型】电影级FPV飞行视角/荷兰角失衡运镜/...
2. 【镜头动态】从X开始，向Y移动，最终到达Z
3. 【主体描述】主角/物体的详细描述
4. 【环境氛围】场景、光影、天气、质感
5. 【情绪强化】营造XX氛围，增强XX感
```

### 首尾帧设计
```
首帧：[静态构图描述，作为视频起点]
运镜：[详细的镜头运动过程]
尾帧：[静态构图描述，作为视频终点]
```

## 🔧 输出格式

```json
{
  "shot_type": "运镜类型",
  "first_frame": "首帧描述（可直接用于图片生成）",
  "camera_movement": "详细运镜描述",
  "last_frame": "尾帧描述",
  "duration": "建议时长(秒)",
  "mood": "情绪基调",
  "full_prompt": "完整视频提示词（中文）",
  "full_prompt_en": "完整视频提示词（英文）"
}
```
