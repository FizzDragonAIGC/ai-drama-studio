# 📚 長篇小說分析技能

## 元信息
- **技能ID**: adaptation_novel_analysis
- **所屬Agent**: 改編Agent
- **參考書籍**: 《改編的藝術》Linda Seger
- **難度**: ⭐⭐⭐⭐

## 核心理念

長篇改編 ≠ 壓縮
- 不是刪減內容，而是轉換媒介
- 小說是內心的藝術，影視是外在的藝術

## 小說解構六步法

### Step 1: 全局掃描
```
基本信息：
- 字數/章節數
- 時間跨度
- 空間跨度
- 人物數量
- 敘事視角

快速判斷：
- 這是情節驅動還是角色驅動？
- 這是單線還是多線？
- 這是線性還是非線性？
```

### Step 2: 核心提取
```
核心問題（必答）：
- 這個故事最核心的是什麼？（一句話）
- 如果只能保留三個場景，是哪三個？
- 去掉什麼會讓故事崩塌？
- 什麼是「只有這個故事才有」的獨特性？

核心情感：
- 主要情緒是什麼？
- 情感最高點在哪？
- 觀眾會為什麼哭/笑/震驚？
```

### Step 3: 結構映射
```
原著結構：
├── 開端（哪幾章）：建立什麼
├── 發展（哪幾章）：主要衝突是什麼
├── 高潮（哪幾章）：決定性時刻
└── 結局（哪幾章）：如何收尾

映射到番劇：
原著結構 → 番劇集數
開端 → 第1-2集
發展 → 第3-X集
高潮 → 第X-Y集
結局 → 最後1-2集
```

### Step 4: 可視化評估
```
每個場景評估：
□ 是否有視覺動作？（能拍出來嗎）
□ 是否有外在衝突？（不只是心理活動）
□ 是否有空間感？（在哪裡發生）
□ 是否有對話？（還是大段內心獨白）

高可視化：✅ 直接轉化
低可視化：⚠️ 需要改造
無法視覺化：❌ 考慮刪除或重寫
```

### Step 5: 人物梳理
```
人物重要性排序：
A級（必保留）：主角、核心配角
B級（可合併）：功能性角色
C級（可刪除）：一次性出現、純敘事功能

人物合併策略：
- 相似功能的角色合併
- 不同時間的角色合併
- 給主角更多戲份（減少視角切換）
```

### Step 6: 難點標記
```
【心理描寫多】
- 難度：⭐⭐⭐⭐
- 解法：轉化為外在行動、象徵、對話

【時間跨度大】
- 難度：⭐⭐⭐
- 解法：選擇關鍵時間點、用字幕/旁白過渡

【多視角敘事】
- 難度：⭐⭐⭐⭐
- 解法：選擇主要視角、合併或刪除次要視角

【大量旁白/議論】
- 難度：⭐⭐⭐⭐⭐
- 解法：刪除或轉化為對話

【夢境/幻覺/意識流】
- 難度：⭐⭐⭐
- 解法：視覺化、但控制篇幅
```

## 改編可行性評分

| 維度 | 1分（難） | 5分（易） |
|------|-----------|-----------|
| 情節清晰度 | 意識流 | 情節分明 |
| 對話豐富度 | 純描寫 | 大量對話 |
| 場景數量 | 少且抽象 | 多且具體 |
| 人物數量 | >20 | <10 |
| 時間跨度 | >50年 | <5年 |
| 視覺潛力 | 日常現實 | 奇觀場景 |

總分>20：輕鬆改編
總分15-20：需要技巧
總分<15：高難度改編
