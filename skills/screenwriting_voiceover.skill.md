# 旁白技巧 Skill
> 来源：影视编剧理论、番剧叙事实践
> 适用智能体：编剧助手、剧情设计师

## 核心理念

**"旁白是特权，要谨慎使用"** —— 能用画面就不用旁白

旁白（Voiceover/Narration）是画面之外的解说声音，是强大但危险的叙事工具。

---

## 核心理论

### 旁白的本质

```
旁白 = 画面之外的声音 + 提供画面无法展示的信息
好旁白 = 补充画面 + 增强情感 + 独特视角
坏旁白 = 解释画面 + 偷懒说明 + 破坏节奏
```

### 旁白的功能

| 功能 | 说明 | 示例 |
|------|------|------|
| 内心展示 | 角色的思想和感受 | "那一刻，我知道一切都变了" |
| 时间桥接 | 跨越时间的叙述 | "三年后..." |
| 世界观说明 | 大量设定信息 | 《进击的巨人》开场 |
| 反讽对比 | 旁白与画面矛盾 | 说"一切都很好"，画面是废墟 |
| 情感升华 | 提升情感高度 | 结尾总结性独白 |
| 悬念制造 | 预告或暗示 | "我不知道，这是最后一次见他" |

### 旁白类型

```
┌─────────────────────────────────────────────┐
│ 1. 主角旁白 - 第一人称，最常见              │
│ 2. 回顾旁白 - 未来的自己回忆过去            │
│ 3. 全知旁白 - 上帝视角，知道一切            │
│ 4. 角色旁白 - 非主角的旁白视角              │
│ 5. 不可靠旁白 - 可能在说谎的旁白            │
└─────────────────────────────────────────────┘
```

## 实操技巧

### 旁白的黄金法则

**1. 能不用就不用**
```
✗ 旁白："他很伤心"
✓ 画面：他独自坐在雨中
```

**2. 不要解释画面**
```
✗ 画面：他握紧拳头
   旁白："我握紧了拳头"
✓ 画面：他握紧拳头
   旁白："那一刻，我下定决心"
```

**3. 创造画面与旁白的张力**
```
画面：热闹的派对
旁白："这是我人生中最孤独的一天"
```

### 番剧旁白使用场景

| 场景 | 适合度 | 说明 |
|------|--------|------|
| 开场世界观 | ★★★★★ | 番剧传统，快速建立设定 |
| 战斗解说 | ★★★★ | 解释能力/战术 |
| 内心独白 | ★★★★ | 日式番剧特色 |
| 结尾升华 | ★★★★★ | 情感收束 |
| 日常叙述 | ★★ | 容易懒惰使用 |

### 旁白节奏设计

```
【松】纯画面叙事
     ↓
【紧】旁白进入
     ↓
【松】旁白退出，画面接管
     ↓
【紧】关键时刻旁白点睛
```

### 旁白与画面的关系

| 关系 | 效果 | 示例 |
|------|------|------|
| 同步 | 强化理解 | 说看到什么，画面就是什么 |
| 对位 | 产生张力 | 说"幸福"，画面是悲剧 |
| 补充 | 增加信息 | 画面是行动，旁白是内心 |
| 预告 | 制造悬念 | 旁白暗示未来，画面是现在 |

## Prompt模板

### 旁白生成

```
为以下场景写旁白：

场景描述：[画面上发生什么]
旁白者：[谁在说旁白]
时间视角：[正在发生/回忆/预告]
情感基调：[感伤/兴奋/平静/紧张...]

要求：
1. 旁白不解释画面，而是补充内心
2. 简洁有力，不要冗长
3. 与画面形成对位或补充
4. 标注每段旁白的画面位置
```

### 开场旁白设计

```
为番剧第一集设计开场旁白：

世界观要点：
1. [要点1]
2. [要点2]
3. [要点3]

主角特征：[简述主角]
整体基调：[番剧风格]

要求：
1. 30秒内能说完
2. 配合画面节奏
3. 建立世界观但保留神秘
4. 结尾有钩子
5. 标注配套画面建议
```

### 战斗旁白

```
为以下战斗场景写旁白解说：

战斗双方：[A vs B]
关键技能/战术：[需要解释的设定]
观众已知信息：[之前已经知道什么]

要求：
1. 解释清楚能力系统
2. 不破坏战斗节奏
3. 穿插在动作间隙
4. 增强战斗理解和紧张感
```

## 案例分析

### 案例：《你的名字》开场旁白

**旁白文本**：
```
曾经，梦到过某个人。
那是谁呢？
是很重要的人，
不能忘记的人。
但是，已经不记得了。
```

**技巧分析**：

| 技巧 | 应用 |
|------|------|
| 悬念制造 | "是谁"引发好奇 |
| 情感铺垫 | "重要的人"建立情感 |
| 主题暗示 | 遗忘与记忆的主题 |
| 简洁有力 | 短句，留白 |

**配套画面**：
- 彗星划过夜空
- 两个模糊的身影
- 晨光中的城市

### 经典旁白对比

**好的旁白**：
```
画面：年迈的主角看着照片
旁白："如果能重来一次...不，
      我不会改变任何事。
      因为每一个错误，
      都把我带向了你。"
```
✓ 不解释画面，表达内心
✓ 情感升华
✓ 与画面形成时间对照

**差的旁白**：
```
画面：年迈的主角看着照片
旁白："我拿起了旧照片。
      照片里是我年轻时的样子。
      旁边是我的妻子。
      我很想念她。"
```
✗ 解释观众已经看到的
✗ 没有内心深度
✗ 毫无诗意

## 番剧旁白风格库

### 热血番风格
```
"我发过誓！绝不会再让任何人在我面前死去！"
"这一拳，承载了所有人的希望！"
```

### 治愈番风格
```
"那个夏天的风，直到现在还记得。"
"原来，平凡的日常，才是最珍贵的。"
```

### 悬疑番风格
```
"那时的我还不知道，这只是噩梦的开始。"
"真相，有时候比谎言更加残酷。"
```

### 校园番风格
```
"青春，就是做一些长大后会后悔的事。"
"如果那天，我说出口了的话..."
```

## 常见错误

### ❌ 解说式旁白
```
旁白："他走进了房间，看到了桌上的信。"
```
这是在描述画面，不是旁白

### ❌ 过度使用旁白
每个镜头都有旁白，观众疲劳

### ❌ 旁白与画面不匹配
旁白在说A，画面是无关的B

### ❌ 旁白太长
一段旁白超过30秒，打断节奏

### ❌ 旁白风格不一致
同一个角色的旁白语气时而文艺时而粗犷
