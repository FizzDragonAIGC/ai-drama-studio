# 📋 剧本评审 Coverage (Script Coverage)
> 好莱坞标准剧本评估方法

---

## 📊 Coverage 是什么

Coverage 是好莱坞制片公司用来评估剧本的标准报告格式。每天有数百个剧本提交，Reader（剧本评审员）会写Coverage来筛选。

---

## 📝 标准 Coverage 格式

```
【剧本信息】
标题：
作者：
类型：
页数/字数：
时长：

【Logline】一句话概括（25字内）

【总评】RECOMMEND / CONSIDER / PASS
        推荐      /  考虑   / 不通过

【评分表】
          差  一般  良好  优秀  卓越
          1   2    3    4    5
前提/概念  □   □    □    □    □
结构      □   □    □    □    □  
角色      □   □    □    □    □
对白      □   □    □    □    □
视觉化    □   □    □    □    □
商业潜力  □   □    □    □    □

【梗概】300字故事概要

【优点】3-5个亮点

【问题】3-5个需要改进的地方

【建议】具体的修改方向
```

## 🔍 六大评估维度

### 1. 前提/概念 (Premise)
| 等级 | 标准 |
|------|------|
| ⭐ | 陈词滥调，毫无新意 |
| ⭐⭐ | 有想法但执行普通 |
| ⭐⭐⭐ | 概念清晰，有一定吸引力 |
| ⭐⭐⭐⭐ | 高概念，一句话就能卖出去 |
| ⭐⭐⭐⭐⭐ | 前所未见，颠覆性创新 |

**检查点：**
- [ ] 能用一句话概括吗？
- [ ] 有"如果...会怎样"的假设吗？
- [ ] 能让人立刻想看吗？

### 2. 结构 (Structure)
| 等级 | 标准 |
|------|------|
| ⭐ | 结构混乱，没有节奏 |
| ⭐⭐ | 有基本框架但松散 |
| ⭐⭐⭐ | 三幕清晰，节拍基本准确 |
| ⭐⭐⭐⭐ | 结构紧凑，张弛有度 |
| ⭐⭐⭐⭐⭐ | 结构创新，每个节拍都精准 |

**检查点：**
- [ ] 激励事件在前25%？
- [ ] 中点有转折？
- [ ] 75%有最低点？
- [ ] 高潮有力？

### 3. 角色 (Character)
| 等级 | 标准 |
|------|------|
| ⭐ | 扁平角色，没有动机 |
| ⭐⭐ | 有基本设定但不鲜活 |
| ⭐⭐⭐ | 主角有弧光，配角有功能 |
| ⭐⭐⭐⭐ | 角色立体，有内在矛盾 |
| ⭐⭐⭐⭐⭐ | 角色独特且令人难忘 |

**检查点：**
- [ ] 主角有明确欲望？
- [ ] 主角有内在缺陷？
- [ ] 角色有成长弧线？
- [ ] 配角有独特功能？

### 4. 对白 (Dialogue)
| 等级 | 标准 |
|------|------|
| ⭐ | 直白无趣，像说明书 |
| ⭐⭐ | 功能性对白，无特色 |
| ⭐⭐⭐ | 自然流畅，角色有差异 |
| ⭐⭐⭐⭐ | 有潜台词，金句频出 |
| ⭐⭐⭐⭐⭐ | 每句话都能单独传播 |

**检查点：**
- [ ] 说的和想的不一样？（潜台词）
- [ ] 蒙住名字能分辨谁说的？
- [ ] 有值得记住的台词？
- [ ] 删掉一半对白还能看懂？

### 5. 视觉化 (Visual)
| 等级 | 标准 |
|------|------|
| ⭐ | 全是对话，没有画面 |
| ⭐⭐ | 有场景描述但平淡 |
| ⭐⭐⭐ | 有视觉想象空间 |
| ⭐⭐⭐⭐ | 画面感强，能"看见"故事 |
| ⭐⭐⭐⭐⭐ | 视觉语言叙事，每帧都是海报 |

**检查点：**
- [ ] 读剧本时脑海有画面？
- [ ] 有标志性视觉符号？
- [ ] 光影/色彩有设计？
- [ ] 动作描写简洁有力？

### 6. 商业潜力 (Commercial)
| 等级 | 标准 |
|------|------|
| ⭐ | 小众，无市场 |
| ⭐⭐ | 有受众但规模小 |
| ⭐⭐⭐ | 主流类型，有竞争力 |
| ⭐⭐⭐⭐ | 卖点清晰，能破圈 |
| ⭐⭐⭐⭐⭐ | 爆款潜质，现象级 |

**检查点：**
- [ ] 目标受众明确？
- [ ] 有可营销的卖点？
- [ ] 有对标成功作品？
- [ ] 有衍生价值？

---

## 🚦 总评标准

### RECOMMEND (推荐)
- 总分 ≥ 24（满分30）
- 无单项 ≤ 2
- 有至少1项 = 5

### CONSIDER (考虑)
- 总分 18-23
- 无单项 ≤ 1
- 有明确的修改方向

### PASS (不通过)
- 总分 < 18
- 或有单项 = 1
- 或根本性问题无法修改

## 🔧 AI番剧特别评估项

除了标准Coverage，AI番剧还需要评估：

| 维度 | 评估内容 |
|------|----------|
| **AI可生成性** | Prompt是否清晰？能生成预期画面？ |
| **分镜效率** | 每集镜头数合理？节奏匹配？ |
| **风格一致性** | 视觉风格统一？角色可识别？ |
| **情感节奏** | 每集有独立情感弧线？有钩子？ |
| **批量可行性** | 能否批量生成？模板化程度？ |
