# 🎭 Skill: 斯坦尼斯拉夫斯基表演體系

> 來源：《演員的自我修養》斯坦尼斯拉夫斯基

## 核心理念

**體驗派表演**：演員必須真正「成為」角色，而非「表演」角色。

## 七大支柱

### 1. 情感記憶 (Emotional Memory)
- 調用自身經歷的真實情感
- 將個人經驗移植到角色情境
- AI應用：描述角色此刻應喚起的情感類型

### 2. 魔術假如 (Magic If)
- 「假如我是這個角色，我會怎麼做？」
- 建立角色的行為邏輯
- AI應用：為每個動作提供心理依據

### 3. 規定情境 (Given Circumstances)
- 時間、地點、社會背景
- 角色的過去、現在、未來
- AI應用：明確鏡頭中的情境設定

### 4. 超級任務 (Super-objective)
- 角色在整個故事中的終極目標
- 所有行動的動力來源
- AI應用：在角色設計中明確超級任務

### 5. 貫穿動作 (Through-line of Action)
- 為達成超級任務的一連串行動
- 每場戲的具體目標
- AI應用：分鏡中標註角色當前目標

### 6. 交流與適應 (Communion & Adaptation)
- 角色之間的真實互動
- 根據對手反應調整自己
- AI應用：雙人/群戲的互動設計

### 7. 節奏與節拍 (Tempo-rhythm)
- 內心節奏外化為動作節奏
- 不同情緒有不同節拍
- AI應用：動作速度與情緒匹配

## 實用檢查清單

為每個角色/鏡頭回答：
- [ ] 我是誰？（身份）
- [ ] 我從哪裡來？（過去）
- [ ] 我要去哪裡？（目標）
- [ ] 我現在想要什麼？（當下目標）
- [ ] 什麼阻礙我？（障礙）
- [ ] 我用什麼方法？（策略）

## AI Prompt 應用

### 表演描述模板
```
[角色名] [情緒狀態], motivated by [內心目標],
[具體動作] while [潛台詞/真實想法],
[身體姿態] reflecting [心理狀態]
```

### 示例
```
小猴子 tense but curious, motivated by survival instinct,
crouching on beam while calculating escape routes,
body coiled like spring reflecting years of street survival
```
