# 📺 番剧单集结构 (Anime Episode Structure)
> 5-8分钟AI番剧叙事设计

---

## 📐 标准结构 (8分钟)

```
┌─────────────────────────────────────────┐
│ [0:00-0:30]  开场钩子/前情提要           │
├─────────────────────────────────────────┤
│ [0:30-4:00]  A Part - 建立与发展         │
│   • 场景1: 日常/建立 (30秒-1分)         │
│   • 场景2: 问题出现 (1分)               │
│   • 场景3: 初步应对 (1分30秒)           │
│   • 小高潮/悬念 → Eyecatch              │
├─────────────────────────────────────────┤
│ [4:00-4:05]  Eyecatch 过场画面           │
├─────────────────────────────────────────┤
│ [4:05-7:30]  B Part - 高潮与解决         │
│   • 场景4: 深入问题 (1分)               │
│   • 场景5: 危机升级 (1分30秒)           │
│   • 场景6: 情感高潮 (1分)               │
│   • 余韵/下集伏笔                       │
├─────────────────────────────────────────┤
│ [7:30-8:00]  下集预告/悬念               │
└─────────────────────────────────────────┘
```

---

## 🎭 A Part vs B Part

| 部分 | 功能 | 情绪 | 节奏 |
|------|------|------|------|
| **A Part** | 建立问题 | 平静→紧张 | 渐进 |
| **Eyecatch** | 呼吸点 | 暂停 | 停顿 |
| **B Part** | 解决问题 | 紧张→释放 | 加速 |

---

## 🎬 每场戏的结构

### 场景模板
```
【场景X】地点 - 时间 (XX秒)

[开头] 建立氛围（5秒）
- 环境音/光线/空间感

[中间] 核心内容（主体时长）
- 动作/对白/冲突

[结尾] 转场钩子（5秒）
- 悬念/情绪/下一场引子
```

---

## 💓 情感曲线设计

```
情绪强度
  ▲
  │           ★ 情感高潮
  │          /│\
  │         / │ \
  │    ↗   /  │  \
  │   /   /   │   \  
  │  /   /    │    \
  │ /   ★     │     \★ 余韵
  │/   小高潮  │      │
  ├─────┬─────┼──────┼────→ 时间
  0    2min  4min  6min   8min
       A Part │ B Part
```

### 情感节拍
| 时间点 | 情绪状态 | 设计目的 |
|--------|----------|----------|
| 0:00 | 好奇 | 钩住观众 |
| 1:00 | 平静 | 建立日常 |
| 2:00 | 紧张 | 问题出现 |
| 3:30 | 小高潮 | A Part收尾 |
| 5:00 | 压力 | B Part发展 |
| 6:30 | 高潮 | 情感爆发 |
| 7:30 | 余韵 | 情绪落地 |
| 8:00 | 期待 | 下集悬念 |

---

## 🎣 每集钩子设计

### A Part开场钩子（前30秒）
| 类型 | 示例 |
|------|------|
| **悬念开场** | 结局片段→倒叙 |
| **冲突开场** | 直接进入矛盾 |
| **视觉开场** | 震撼画面 |
| **情绪开场** | 强烈情感 |

### B Part结尾钩子（最后30秒）
| 类型 | 效果 |
|------|------|
| **悬崖** | 危机未解决 |
| **揭示** | 新信息曝光 |
| **反转** | 出乎意料 |
| **抉择** | 面临选择 |
| **情感** | 情绪高点戛然而止 |

## 🔍 场景功能分类

| 场景类型 | 功能 | 节奏 | 时长建议 |
|----------|------|------|----------|
| **日常场景** | 建立角色/世界 | 慢 | 30-60秒 |
| **过渡场景** | 连接情节 | 中 | 15-30秒 |
| **冲突场景** | 推进剧情 | 快 | 45-90秒 |
| **情感场景** | 打动观众 | 慢 | 60-90秒 |
| **高潮场景** | 释放张力 | 快→慢 | 60-120秒 |

## ✅ 检查清单

### 结构检查
- [ ] A Part有小高潮？
- [ ] Eyecatch位置合适？
- [ ] B Part有情感高潮？
- [ ] 结尾有下集钩子？

### 节奏检查
- [ ] 每场戏有明确功能？
- [ ] 情绪有起伏变化？
- [ ] 没有拖沓段落？

### 情感检查
- [ ] 观众能共情角色？
- [ ] 高潮能打动人？
- [ ] 余韵有回味空间？
