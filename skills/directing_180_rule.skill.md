# 🎬 180度法则 Skill
> 来源：《电影镜头设计》Steven D. Katz + 好莱坞连贯性剪辑法则

---

## 📐 什么是180度法则

**定义**：在两个角色之间画一条想象的线（轴线），摄影机必须保持在这条线的同一侧

### 为什么重要
| 原因 | 说明 |
|------|------|
| **方向一致** | 角色视线方向不会突然反转 |
| **空间感** | 观众知道谁在左谁在右 |
| **对话逻辑** | A看右 ↔ B看左 = 对话 |

## 🎯 180度法则图解

### 基本原理
```
        ✓ 摄影机可放置区域
    ┌─────────────────────────┐
    │                         │
    │   📹    📹    📹        │
    │   1     2     3         │
    │                         │
────┼──────A═════════B────────┼──── 轴线
    │        ↑                │
    │    ✗ 禁止区域           │
    │                         │
    └─────────────────────────┘
```

### 正确示例
```
摄影机在轴线上方：

📹1 看到的：          📹2 看到的：          📹3 看到的：
┌──────────┐          ┌──────────┐          ┌──────────┐
│ A面向右  │          │ A    B   │          │ B面向左  │
│   →      │          │ →    ←   │          │   ←      │
└──────────┘          └──────────┘          └──────────┘
   单人A                 双人                  单人B

→ 视线方向一致：A看右，B看左
```

### 错误示例（越轴）
```
如果📹从上方移到下方：

原来📹看到的：         越轴后📹看到的：
┌──────────┐          ┌──────────┐
│ A    B   │          │ B    A   │
│ →    ←   │          │ →    ←   │
└──────────┘          └──────────┘

→ A和B的位置对调了！观众混乱
```

## 🔧 轴线类型

### 1. 对话轴线
```
两人对话时，连接两人的线就是轴线
        A ══════════ B
              ↑
            轴线
```

### 2. 运动轴线
```
角色运动方向决定轴线
        ────────→
              ↑
        角色移动方向 = 轴线
```

### 3. 视线轴线
```
角色看向某物的视线就是轴线
        👁️ ════════════ 🎯
        角色            看的东西
```

## 📋 合法越轴方法

### 1. 中性镜头过渡
```
镜头1：轴线一侧
镜头2：正面/背面（轴线上）← 中性
镜头3：轴线另一侧
```

### 2. 镜头内越轴
```
摄影机在一个连续镜头内移过轴线
观众跟随移动，不会混乱
```

### 3. 角色移动
```
角色在画面内移动，建立新轴线
```

### 4. 插入镜头缓冲
```
镜头1：轴线一侧
镜头2：特写细节/空镜 ← 缓冲
镜头3：轴线另一侧
```

### 5. 刻意制造混乱
```
打斗、混乱场景可以有意越轴
增加混乱感
```

## 🎯 多人场景的轴线

### 三人场景
```
           A
          ╱ ╲
         ╱   ╲
        B ─── C
        
轴线1：A-B
轴线2：A-C  
轴线3：B-C

规则：对话在谁之间，就遵守那条轴线
```

### 圆桌场景
```
        A
     ╱     ╲
    D   🔵   B    （🔵 = 桌子）
     ╲     ╱
        C

技巧：以桌子中心为参考点
     保持旋转方向一致（顺时针或逆时针）
```

## 🎬 AI番剧应用

### Prompt中的方向控制
```
场景：A和B对话

如果决定A在左，B在右：

建立镜头：
"two people talking, person A on left side,
person B on right side, facing each other"

A的单人镜头：
"close-up person A, looking towards right"

B的单人镜头：
"close-up person B, looking towards left"
```

### 方向一致性检查表
```
[ ] 确定A在画面哪一侧
[ ] 确定B在画面哪一侧
[ ] A的视线方向指向B
[ ] B的视线方向指向A
[ ] 背景元素与建立镜头一致
```

### 分镜规划模板
```
| 镜号 | 类型 | 轴线位置 | A位置 | B位置 | A视线 | B视线 |
|------|------|----------|-------|-------|-------|-------|
| 01 | 建立 | 上方 | 左 | 右 | → | ← |
| 02 | 单人A | 上方 | - | - | → | - |
| 03 | 单人B | 上方 | - | - | - | ← |
| 04 | 过肩 | 上方 | 前景 | 焦点 | → | ← |
```

## 💡 常见问题

| 问题 | 原因 | 解决 |
|------|------|------|
| 角色位置对调 | 无意越轴 | 检查摄影机位置 |
| 两人看同方向 | 视线未对望 | 修正视线方向 |
| 空间感混乱 | 频繁越轴 | 加入建立镜头 |
| 背景穿帮 | 越轴后背景变 | 保持轴线一侧 |

---

## ⚠️ 何时可以打破

| 情况 | 效果 |
|------|------|
| 打斗混乱 | 增加混乱感 |
| 心理崩溃 | 表现失衡 |
| 梦境/幻觉 | 表现不真实 |
| 刻意迷失 | 让观众也迷失 |
