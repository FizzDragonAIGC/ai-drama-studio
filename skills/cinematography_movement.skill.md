# 🎥 运动镜头 Skill
> 来源：《Film Directing Shot by Shot》/ 《Cinematography: Theory and Practice》

## 📐 运动镜头分类

### 1. 摇镜 (Pan/Tilt)
**摄影机固定，只旋转**

| 类型 | 运动 | 用途 |
|------|------|------|
| Pan（横摇） | 左右旋转 | 跟随、展示环境 |
| Tilt（纵摇） | 上下旋转 | 展示高度、揭示 |
| Whip Pan | 快速横摇 | 转场、紧张 |
| Swish Pan | 超快速 | 时间跳跃 |

### 2. 推轨 (Dolly/Track)
**摄影机在轨道上移动**

| 类型 | 运动 | 用途 |
|------|------|------|
| Dolly In | 推进 | 强调、聚焦、紧张 |
| Dolly Out | 拉远 | 揭示、孤立、结束 |
| Track | 横向 | 跟随、平行 |

### 3. 摇臂 (Crane/Jib)
**摄影机升降移动**

| 类型 | 运动 | 用途 |
|------|------|------|
| Crane Up | 升起 | 解放、结束、全景 |
| Crane Down | 降落 | 进入、聚焦、威胁 |
| Boom | 斜向 | 动态揭示 |

### 4. 手持 (Handheld)
**手持摄影机**

| 风格 | 效果 |
|------|------|
| 轻微晃动 | 真实、纪录片感 |
| 剧烈晃动 | 混乱、紧张、战争 |
| 奔跑跟拍 | 追逐、紧迫 |

### 5. 稳定器 (Steadicam/Gimbal)
**稳定的移动镜头**

| 用途 | 效果 |
|------|------|
| 长镜头跟随 | 沉浸、流畅 |
| 空间穿越 | 介绍环境 |
| 第一人称 | 主观体验 |

### 6. 无人机 (Drone)
**航拍视角**

| 用途 | 效果 |
|------|------|
| 建立镜头 | 展示地点 |
| 跟拍 | 追逐场面 |
| 俯瞰 | 规模感 |

## 🎯 运动镜头含义

### 情感对照表
| 运动 | 情感/含义 |
|------|----------|
| 推进 | 聚焦、紧张、进入 |
| 拉远 | 疏离、揭示、孤独 |
| 跟随 | 同行、陪伴 |
| 环绕 | 观察、考量、晕眩 |
| 上升 | 解放、希望、超越 |
| 下降 | 压迫、进入、聚焦 |
| 快速 | 紧张、混乱、能量 |
| 缓慢 | 庄重、沉思、悲伤 |

## 📊 眩晕镜头 (Vertigo Effect)

**同时推进+拉变焦（或反向）**

```
视觉效果：
背景变化，前景主体不变
───────────────────→
空间扭曲感
```

**用途**：
- 恐惧、晕眩
- 顿悟、震惊
- 时间扭曲感

---

## 🎭 运动的节奏

### 速度变化
```
加速 → 减速
快 → 慢
情感：紧张 → 释放
```

### 运动组合
```
推进 + 升起 = 解放的聚焦
拉远 + 下降 = 压抑的远离
环绕 + 推进 = 螺旋聚焦
```

## 📋 运动镜头检查清单

### 选择运动
- [ ] 为什么要移动？
- [ ] 运动有情感目的？
- [ ] 与角色状态匹配？
- [ ] 与节奏匹配？

### 执行运动
- [ ] 起点和终点设计？
- [ ] 速度恰当？
- [ ] 与演员动作配合？
- [ ] 与音乐/节奏配合？

### 避免
- ❌ 无目的移动
- ❌ 过度运动造成晕眩
- ❌ 破坏空间连续性
- ❌ 技术炫耀

## 🎬 经典运动镜头

### 《好家伙》科帕卡巴纳
- **技术**：Steadicam长镜头
- **运动**：从后门穿过厨房到舞台
- **含义**：黑帮世界的深入，主角的特权

### 《闪灵》走廊
- **技术**：Steadicam跟拍
- **运动**：跟随三轮车
- **含义**：迷宫般的困境，不安

### 《拯救大兵瑞恩》开场
- **技术**：手持
- **运动**：混乱、晃动
- **含义**：战争的混乱与恐惧

### 《赎罪》海滩
- **技术**：Steadicam 5分钟长镜头
- **运动**：穿越敦刻尔克海滩
- **含义**：战争的规模与荒诞
