# 🎬 北野武風格技能

## 元信息
- **技能ID**: director_kitano
- **所屬Agent**: 導演風格Agent
- **大師**: 北野武 Takeshi Kitano
- **代表作**: 《花火》《菊次郎的夏天》《座頭市》

## 標誌性技法

### 1. 靜止與爆發
```
特徵：長時間靜止→突然暴力
效果：張力、意外、真實
應用：暴力場景、情緒爆發

Prompt關鍵詞：
static shot, moment before violence,
calm before storm, sudden action
```

### 2. 藍色調
```
特徵：標誌性的北野藍
效果：冷漠、詩意、孤獨
應用：海邊、夜景、沉思

Prompt關鍵詞：
Kitano blue, cold blue tone,
melancholic atmosphere, ocean blue
```

### 3. 沉默的情感
```
特徵：不說話但情感滿溢
效果：內斂、深沉、餘韻
應用：告別、情感場景

Prompt關鍵詞：
silent emotion, understated feeling,
wordless communication, stoic expression
```

### 4. 固定長鏡頭
```
特徵：固定機位，角色進出畫面
效果：疏離、觀察、留白
應用：日常場景、等待

Prompt關鍵詞：
static camera, fixed frame,
characters entering and exiting,
observational distance
```

## 北野武的沉默

```
北野武電影中的沉默：
- 不是沒話說
- 而是說不出口
- 或者無需說

畫面替代對話：
- 一起看海 = 和解
- 沉默抽菸 = 思考
- 低頭不語 = 愧疚
- 直視前方 = 決心
```
