# 章節鉤子與懸念設計
> 讓觀眾無法停止觀看的技巧

## 🎯 核心理念
**每章結尾都要讓觀眾產生「再看一章」的衝動。**

---

## 📋 10種章末鉤子類型

### 1. 懸崖式 (Cliffhanger)
角色處於物理或情感危險中。
```
❌ 「他爬上了懸崖。」
✅ 「他的手指開始滑落...」
```

### 2. 揭露式 (Revelation)
驚人的信息被揭露。
```
❌ 「她發現了真相。」
✅ 「那封信上寫著：『我是你的親生父親。』」
```

### 3. 決定式 (Decision Point)
角色必須做出重大選擇。
```
❌ 「她需要做個決定。」
✅ 「兩扇門。一扇通往自由，一扇通往真相。她只能選一個。」
```

### 4. 反轉式 (Twist)
事情的發展完全出乎意料。
```
❌ 「他發現情況不對。」
✅ 「『救命！』他回頭。喊叫的人是他自己的倒影。」
```

### 5. 倒計時式 (Countdown)
時間壓力創造緊迫感。
```
❌ 「他必須趕快。」
✅ 「炸彈顯示：00:47。電梯還在第28層。」
```

### 6. 謎團式 (Mystery)
提出一個讓人好奇的問題。
```
❌ 「有些事情很奇怪。」
✅ 「如果她從未離開過這個房間，那照片裡站在艾菲爾鐵塔前的人是誰？」
```

### 7. 承諾式 (Promise)
預示即將發生的精彩。
```
❌ 「明天會很重要。」
✅ 「『明天，』他說，『我會讓你見識什麼叫真正的戰爭。』」
```

### 8. 情感式 (Emotional)
強烈的情感衝擊。
```
❌ 「她很傷心。」
✅ 「『我愛你。』這是她聽到他說的最後一句話。」
```

### 9. 對抗式 (Confrontation)
衝突即將爆發。
```
❌ 「他們要對峙了。」
✅ 「她轉過身，發現他正站在門口。『我們談談。』」
```

### 10. 視角切換式 (POV Switch)
切換到另一個角色的危機。
```
❌ 「與此同時，另一邊...」
✅ 「與此同時，三百公里外的監獄裡，那個本該被處決的人睜開了眼睛。」
```

---

## 🎬 每集首尾設計

### 開場 (Cold Open)
- 用30秒抓住觀眾
- 可以是：動作場景 / 懸疑場景 / 情感衝擊
- 目的：讓觀眾不願意切走

### 結尾 (Tag)
- 最後15秒留下鉤子
- 必須製造「我要知道接下來怎樣」的渴望
- 可以是：懸崖 / 揭露 / 倒計時

## 💡 寫鉤子的黃金法則

1. **具體而非抽象** - 「一把槍」比「危險」更有畫面
2. **行動而非描述** - 「她跑」比「她害怕」更緊張
3. **留白而非解釋** - 讓觀眾自己填補空白
4. **意外而非預期** - 打破觀眾的預測
5. **情感而非事件** - 觀眾記住的是感受
