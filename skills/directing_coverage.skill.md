# 🎬 机位覆盖 Skill
> 来源：《电影导演技术》Mike Crisp + 《Film Directing Shot by Shot》Steven D. Katz

---

## 📐 什么是Coverage（机位覆盖）

**定义**：从多个角度和景别拍摄同一场景，确保后期剪辑有足够素材

### 覆盖的目的
| 目的 | 说明 |
|------|------|
| **剪辑自由度** | 提供多种切换可能 |
| **修复错误** | 表演失误可用其他角度替代 |
| **节奏控制** | 后期可调整对话节奏 |
| **强调选择** | 可选择最佳表演瞬间 |

## 🎯 标准覆盖模式

### 对话场景基础覆盖

```
┌─────────────────────────────────────┐
│                                     │
│     📹 Master Shot (主镜头)         │
│              ↓                      │
│     A ←──────→ B                    │
│              ↓                      │
│   📹 ────→ A    B ←──── 📹          │
│   (A单人)        (B单人)            │
│              ↓                      │
│   📹 ──→ A肩B    A肩B ←── 📹        │
│   (过肩A)         (过肩B)           │
│              ↓                      │
│   📹 → A特写    B特写 ← 📹          │
└─────────────────────────────────────┘
```

### 5镜头覆盖法
| 镜号 | 类型 | 内容 |
|------|------|------|
| 1 | Master | 全景交代双方 |
| 2 | Single A | A的中景/近景 |
| 3 | Single B | B的中景/近景 |
| 4 | OTS A | 从A肩看B |
| 5 | OTS B | 从B肩看A |

## 🔧 覆盖策略类型

### 1. 最大覆盖（Maximum Coverage）
- 每个角度、每个景别都拍
- 适用：大预算、复杂场景
- 优点：后期选择多
- 缺点：耗时、耗钱

### 2. 最小覆盖（Minimal Coverage）
- 只拍必要镜头
- 适用：独立电影、时间紧张
- 优点：快速、节约
- 缺点：后期受限

### 3. 计划覆盖（Planned Coverage）
- 根据分镜精确拍摄
- 适用：大师级导演
- 优点：高效、有预见性
- 缺点：需要经验

## 📋 覆盖检查清单

### 对话场景
- [ ] 主镜头（完整走一遍）
- [ ] A角色单人镜头（台词完整）
- [ ] B角色单人镜头（台词完整）
- [ ] 过肩镜头（双向）
- [ ] 特写（关键情感时刻）
- [ ] 反应镜头（听者反应）
- [ ] 插入镜头（手部、物品等）

### 动作场景
- [ ] 全景（建立空间）
- [ ] 中景（动作清晰）
- [ ] 特写（关键细节）
- [ ] 多角度（同一动作）
- [ ] 慢动作（关键时刻）

## 🎯 AI番剧应用

### Prompt结构
```
场景：[描述]
覆盖镜头清单：
- Master: [全景描述]
- Single A: [A的中景]
- Single B: [B的中景]
- Insert: [细节镜头]
- Reaction: [反应镜头]
```

### 生成示例
```
场景：两人咖啡馆对话

覆盖镜头：
1. "wide shot, two people sitting across cafe table, warm lighting"
2. "medium shot, young woman speaking, coffee cup in hand"
3. "medium shot, young man listening intently, cafe background"
4. "close-up, hands holding coffee cup, nervous gesture"
5. "close-up, man's surprised expression, reaction shot"
```

## ⚠️ 常见错误

| 错误 | 后果 | 解决 |
|------|------|------|
| 跳轴 | 方向混乱 | 遵守180度法则 |
| 景别跳变过小 | 跳切感 | 景别变化>30% |
| 缺少反应 | 单调 | 补拍反应镜头 |
| 时长不一致 | 剪辑困难 | 每个角度跑完整 |
