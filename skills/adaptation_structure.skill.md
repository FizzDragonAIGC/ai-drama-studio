# 📚 改編結構映射技能

## 元信息
- **技能ID**: adaptation_structure
- **所屬Agent**: 改編Agent
- **參考書籍**: 《改編的藝術》+ 《故事》
- **難度**: ⭐⭐⭐⭐

## 核心理念

結構映射 = 小說的章節 → 番劇的集數
- 不是按比例壓縮
- 而是按戲劇邏輯重組

## 番劇集數規劃

### 短番（6-8集）
```
適合：
- 10萬字以內
- 單一主線
- 時間跨度小

結構：
集1：建立世界+觸發
集2-3：發展衝突
集4-5：危機加深
集6-7：高潮
集8：解決+收尾
```

### 標準番（12-13集）
```
適合：
- 10-20萬字
- 一主一副線
- 中等複雜度

結構：
集1-2：建立+觸發
集3-6：第一幕發展
集7（中點）：大轉折
集8-10：第二幕危機
集11-12：高潮
集13：解決
```

### 長番（24-26集）
```
適合：
- 20萬字以上
- 多線敘事
- 史詩級故事

結構：
第一季（1-12）：
- 集1-3：建立
- 集4-8：發展
- 集9-11：第一次高潮
- 集12：季終懸念

第二季（13-24）：
- 集13-15：重啟+新挑戰
- 集16-20：升級衝突
- 集21-23：最終高潮
- 集24：大結局
```

## 章節→集數轉換公式

### 基礎公式
```
原著章節數 ÷ 2-3 = 預估集數

例：30章 ÷ 2.5 = 12集
```

### 調整因子
```
+集數 情況：
- 有重要支線需要展開
- 情感戲需要更多時間
- 原著太壓縮需要舒展

-集數 情況：
- 大量心理描寫可壓縮
- 重複場景可合併
- 過渡章節可跳過
```

## 場景密度規劃

| 集類型 | 場景數 | 節奏 | 適合內容 |
|--------|--------|------|----------|
| 高密度集 | 15-20 | 快 | 動作、追逐、緊張 |
| 中密度集 | 10-15 | 中 | 常規敘事 |
| 低密度集 | 5-10 | 慢 | 情感、沉思、回憶 |

**建議：高低密度交替，不要連續3集同節奏**

## 結構重組技巧

### 1. 閃回重組
```
原著：線性敘事
改編：倒敘開場+閃回填充

效果：
- 開場更有衝擊力
- 製造懸念
- 保持觀眾好奇
```

### 2. 平行剪輯
```
原著：A故事→B故事（順序）
改編：A場景→B場景→A場景→B場景（交叉）

效果：
- 製造張力
- 暗示聯繫
- 節奏更緊湊
```

### 3. 合併時間線
```
原著：多個時間點分開描述
改編：壓縮到更短時間跨度

例：
原著10年 → 改編3年
原著寒暑假分別發生 → 同一個夏天
```

### 4. 拆分大場景
```
原著：一個長場景
改編：拆成多個小場景，中間插入其他線

效果：
- 避免單一場景太長
- 增加懸念
- 保持節奏
```

## 集結構模板

### 單集標準結構
```
開場（2分鐘）：
- 懸念or回顧or場景建立

ACT 1（5分鐘）：
- 問題出現

ACT 2（10分鐘）：
- 嘗試解決
- 障礙升級

ACT 3（5分鐘）：
- 小高潮or反轉

結尾（2分鐘）：
- 懸念or情感落點

總時長：24分鐘（動畫）
```

### 每集必備元素
```
□ 有開場鉤子（5秒內抓住）
□ 有一個核心問題（這集要解決什麼）
□ 有一個轉折點（意外發生）
□ 有一個情感時刻（觀眾投入感情）
□ 有結尾懸念（想看下一集）
```
