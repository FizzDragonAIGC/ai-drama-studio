## 🎯 十种开场类型

### 1. 动作开场 (Action Opening)
从高潮场面开始
- **优点**：立即抓住注意力
- **适合**：动作片、惊悚片
- **例子**：《黑暗骑士》银行抢劫

### 2. 日常开场 (Ordinary World)
展示主角正常生活
- **优点**：建立对比
- **适合**：成长片、冒险片
- **例子**：《黑客帝国》尼奥的办公室

### 3. 倒叙开场 (In Medias Res)
从故事中间开始
- **优点**：制造悬念
- **适合**：悬疑片、传记片
- **例子**：《日落大道》死人独白

### 4. 旁白开场 (Narration)
角色讲述引入
- **优点**：快速建立背景
- **风险**：可能太被动
- **例子**：《肖申克的救赎》

### 5. 序幕开场 (Prologue)
展示背景/起源
- **优点**：提供必要信息
- **风险**：可能太长
- **例子**：《指环王》戒指历史

### 6. 符号开场 (Symbolic)
象征性画面
- **优点**：艺术性强
- **适合**：文艺片
- **例子**：《2001》骨头变飞船

### 7. 对话开场 (Dialogue)
从对话中揭示信息
- **优点**：自然
- **风险**：可能无聊
- **例子**：《低俗小说》

### 8. 悬念开场 (Mystery)
提出谜题
- **优点**：强烈好奇心
- **适合**：悬疑片、惊悚片
- **例子**：《七宗罪》

### 9. 时间节点开场 (Specific Moment)
"那一天改变了一切"
- **优点**：明确转折
- **例子**：《泰坦尼克号》

### 10. 对比开场 (Contrast)
展示即将失去的美好
- **优点**：情感铺垫
- **例子**：《飞屋环游记》蒙太奇

## 📊 开场三层结构

```
┌─────────────────────────────────────────┐
│  第一层：钩子 (Hook)                     │
│  0-30秒：抓住注意力                      │
├─────────────────────────────────────────┤
│  第二层：建置 (Setup)                    │
│  1-10分钟：建立世界和角色                │
├─────────────────────────────────────────┤
│  第三层：激励 (Inciting)                 │
│  10-15分钟：激励事件发生                 │
└─────────────────────────────────────────┘
```

## 🎭 主角出场技巧

### 角色登场四法
1. **英雄时刻**：做一件帅的事
2. **救猫咪**：做一件好的事
3. **麻烦时刻**：展示缺陷
4. **普通时刻**：平凡中见特别

### 快速建立角色
| 手法 | 例子 |
|------|------|
| 职业 | 医生、杀手、教师 |
| 环境 | 房间透露性格 |
| 关系 | 与他人互动 |
| 选择 | 面对选择时的反应 |
| 对白 | 说话方式 |

## 📋 开场黄金法则

### DO ✅
- 视觉化呈现，而非解释
- 提出问题，引发好奇
- 展示"改变前"的状态
- 确立游戏规则
- 给观众一个关心的理由

### DON'T ❌
- 过多背景解释
- 无关紧要的场景
- 太多角色同时登场
- 节奏太慢
- 与后面脱节

## 🎬 经典开场分析

### 《黑暗骑士》
- **类型**：动作开场
- **钩子**：银行抢劫
- **悬念**：小丑是谁？
- **基调**：黑暗、紧张

### 《飞屋环游记》
- **类型**：对比开场
- **钩子**：两个孩子相遇
- **情感**：爱情蒙太奇
- **功能**：让观众爱上角色，理解动机

### 《社交网络》
- **类型**：对话开场
- **钩子**：快速对话
- **建立**：马克的聪明和社交缺陷
- **激励**：被女友甩
