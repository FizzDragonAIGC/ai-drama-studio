## 標誌性技法

### 1. 匹配剪輯（Match Cut）
```
特徵：連續動作跨越不同場景/時空
效果：夢與現實模糊、流暢
應用：回憶、夢境、幻覺

Prompt關鍵詞：
match cut composition, seamless transition,
reality blending, connected poses
```

### 2. 現實與幻想交織
```
特徵：無縫切換真實與想像
效果：迷幻、心理深度
應用：心理驚悚、夢境

Prompt關鍵詞：
reality merging with dream,
blurred boundary, surreal transition,
Satoshi Kon dream logic
```

### 3. 面部變形
```
特徵：表情極端變化
效果：心理張力、不安
應用：瘋狂、恐懼、情緒爆發

Prompt關鍵詞：
exaggerated facial expression,
face distortion, emotional intensity,
Satoshi Kon expression
```

### 4. 都市碎片
```
特徵：東京都市的細節拼貼
效果：現代焦慮、資訊過載
應用：都市場景、心理壓力

Prompt關鍵詞：
Tokyo urban fragments, neon signs,
crowded visual, information overload,
modern anxiety
```
