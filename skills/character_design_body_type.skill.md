> 来源：《人体解剖与造型》+ 动画角色体型设计理论

---

## 📐 基础体型分类

### 三种基础体型（索莫托分类）
| 体型 | 特征 | 性格暗示 | AI Prompt |
|------|------|----------|-----------|
| 外胚型 | 瘦长、四肢细长 | 聪明、敏感 | ectomorph, thin lanky build |
| 中胚型 | 肌肉发达、宽肩 | 活跃、强壮 | mesomorph, athletic muscular build |
| 内胚型 | 圆润、脂肪多 | 和蔼、舒适 | endomorph, soft rounded build |

## 🎭 动画体型设计

### 男性体型
| 体型 | 特征 | 适用角色 | AI Prompt |
|------|------|----------|-----------|
| 肌肉型 | 宽肩窄腰、肌肉分明 | 英雄、战士 | muscular build, broad shoulders |
| 精瘦型 | 瘦削但有力 | 刺客、忍者 | lean athletic build, wiry |
| 壮硕型 | 大块头、厚实 | 力量型、保镖 | bulky build, massive frame |
| 标准型 | 匀称普通 | 普通人、主角 | average build, normal proportions |
| 瘦弱型 | 骨感、单薄 | 书生、病患 | thin frail build, skinny |
| 圆润型 | 胖乎乎、可爱 | 喜剧、厨师 | chubby build, rotund |

### 女性体型
| 体型 | 特征 | 适用角色 | AI Prompt |
|------|------|----------|-----------|
| 苗条型 | 纤细、曲线柔和 | 少女、精灵 | slender build, graceful |
| 运动型 | 健美、肌肉线条 | 战士、运动员 | athletic female build, toned |
| 丰满型 | 曲线明显 | 成熟女性 | curvy build, hourglass figure |
| 娇小型 | 小巧玲珑 | 萝莉、妖精 | petite build, small frame |
| 高挑型 | 修长、模特感 | 女王、贵族 | tall slender build, model proportions |

## 🔄 体型与剪影

### 剪影识别度
| 体型组合 | 对比效果 | 例子 |
|----------|----------|------|
| 高瘦+矮胖 | 经典喜剧搭档 | 劳雷尔与哈代 |
| 肌肉+纤细 | 力量与敏捷 | 坦克与刺客 |
| 巨大+迷你 | 反差萌 | 大个子与小精灵 |

### 夸张体型
| 类型 | 特征 | AI Prompt |
|------|------|-----------|
| 超级肌肉 | 夸张的肌肉量 | hyper muscular, exaggerated muscles |
| 极度瘦长 | 面条人比例 | extremely thin, stick figure |
| 巨型身材 | 远超正常比例 | giant build, massive scale |
| 迷你身材 | 极度矮小 | tiny build, miniature |

## 💡 体型细节设计

### 肩部设计
| 类型 | 特征 | AI Prompt |
|------|------|-----------|
| 宽肩 | 力量感、可靠 | broad shoulders, wide |
| 窄肩 | 弱势、学者 | narrow shoulders, slim |
| 斜肩 | 优雅、柔和 | sloped shoulders |
| 平肩 | 强势、运动 | square shoulders |

### 腰部设计
| 类型 | 特征 | AI Prompt |
|------|------|-----------|
| 细腰 | 优雅、敏捷 | slim waist, narrow |
| 粗腰 | 力量、稳定 | thick waist, solid |
| 无腰 | 壮汉、笨重 | no waist definition |

### 四肢设计
| 类型 | 特征 | AI Prompt |
|------|------|-----------|
| 长腿 | 优雅、速度 | long legs, leggy |
| 短腿 | 稳定、可爱 | short legs, stubby |
| 粗壮手臂 | 力量型 | thick muscular arms |
| 纤细手臂 | 敏捷型 | thin graceful arms |

## 📋 体型设计模板

```
【角色体型设计】

角色名：[名字]
性别：[男/女/其他]
年龄：[岁数]

基础体型：[外胚/中胚/内胚]

详细参数：
- 身高定位：[矮/中/高/极高]
- 体重定位：[瘦/中/胖]
- 肌肉量：[无/少/中/多/极多]

部位特征：
- 肩膀：[宽/中/窄]
- 腰部：[细/中/粗]
- 臀部：[小/中/大]
- 腿型：[长/中/短][粗/中/细]
- 手臂：[粗/中/细]

整体印象：[一句话描述]

AI Prompt：
[整合体型描述]
```
