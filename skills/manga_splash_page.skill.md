## 📐 跨页类型

### 1. 单页满版（Splash Page）
```
┌─────────────────────┐
│                     │
│    一整页单图       │
│    无格子分割       │
│                     │
└─────────────────────┘
```
**用途**：重大时刻、角色登场、震撼场面

### 2. 双页跨版（Double-Page Spread）
```
┌───────────────────────────────────────────┐
│                                           │
│         两页合并为一张大图                 │
│                                           │
└───────────────────────────────────────────┘
```
**用途**：史诗场面、战斗高潮、宏大场景

### 3. 跨页带格（Spread with Panels）
```
┌───────────────────────────────────────────┐
│  ┌───┐                           ┌───┐   │
│  │ 1 │     大场景背景            │ 2 │   │
│  └───┘                           └───┘   │
│           主画面跨越两页                  │
│  ┌───────────────────────────────────┐   │
│  │              3                    │   │
│  └───────────────────────────────────┘   │
└───────────────────────────────────────────┘
```
**用途**：场景+反应结合

## 📊 跨页构图法则

### 中心焦点构图
```
┌───────────────────────────────────────────┐
│ ╲                                     ╱   │
│   ╲                                 ╱     │
│     ╲         焦点人物           ╱       │
│       ╲         ★           ╱           │
│         ╲                 ╱               │
└───────────────────────────────────────────┘
```
**视线引导**：四周向中心聚焦

### 对角线构图
```
┌───────────────────────────────────────────┐
│ ╲                                         │
│   ╲   角色A                               │
│     ╲                                     │
│       ╲═════════════════════╲             │
│                               ╲   角色B   │
└───────────────────────────────────────────┘
```
**用途**：对峙、冲突

### 透视冲击构图
```
┌───────────────────────────────────────────┐
│               ╱│╲                         │
│             ╱  │  ╲                       │
│           ╱    │    ╲                     │
│         ╱      ★      ╲    ← 消失点     │
│       ╱                  ╲                │
│     ╱                      ╲              │
└───────────────────────────────────────────┘
```
**用途**：冲击感、纵深感

## 🔧 装订线处理

### 问题：书脊遮挡
```
┌──────────┐║┌──────────┐
│          │║│          │
│    左页  │║│  右页    │
│          │║│          │
└──────────┘║└──────────┘
           ↑
        书脊（会被折叠遮挡）
```

### 解决方案

| 方案 | 处理方式 |
|------|----------|
| **避开中线** | 重要元素不放中间 |
| **延伸处理** | 中间放可延伸的背景 |
| **断裂效果** | 利用断裂创造效果 |
| **数字阅读** | 无需考虑（无书脊） |

## 🎨 跨页视觉元素

### 必备元素

| 元素 | 作用 |
|------|------|
| **主焦点** | 视觉中心 |
| **引导线** | 视线引导 |
| **空间感** | 前中后景 |
| **动态线** | 运动感 |
| **光影** | 氛围感 |

### 效果层次
```
【前景】速度线/碎片/特效
   ↓
【中景】主要角色/动作
   ↓  
【远景】背景/环境
```

## 📋 跨页设计检查表

- [ ] 时机选择恰当（真正重要的时刻）
- [ ] 构图有冲击力
- [ ] 视觉焦点明确
- [ ] 书脊处理妥当
- [ ] 与前后页节奏呼应
- [ ] 不滥用（物以稀为贵）

---

## ⚠️ 常见错误

| 错误 | 后果 |
|------|------|
| **滥用跨页** | 失去冲击力 |
| **重点在书脊** | 被遮挡 |
| **缺乏焦点** | 散乱无力 |
| **前后脱节** | 节奏断裂 |
| **构图平庸** | 浪费版面 |
