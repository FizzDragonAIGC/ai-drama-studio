# ⚔️ 短剧快速冲突 Skill
> 来源：剧作冲突理论 + 短视频爆款分析

---

## 📐 冲突核心原则

短剧冲突必须**即时、激烈、可视化**：

| 传统剧 | 短剧 | 原因 |
|--------|------|------|
| 渐进铺垫 | 开场即冲突 | 抓注意力 |
| 含蓄暗示 | 直接爆发 | 时间有限 |
| 内心冲突 | 外化对抗 | 视觉优先 |
| 复杂背景 | 简单对立 | 易于理解 |

---

## 🎯 冲突类型

### 1. 人际冲突（最常用）
| 类型 | 描述 | 示例 |
|------|------|------|
| 地位冲突 | 强弱对抗 | 老板vs员工，婆婆vs媳妇 |
| 情感冲突 | 爱恨交织 | 情敌、三角恋 |
| 价值冲突 | 理念对立 | 传统vs现代 |
| 利益冲突 | 资源争夺 | 遗产、职位 |
| 信任冲突 | 欺骗/误会 | 谎言被揭穿 |

### 2. 社会冲突
| 类型 | 描述 | 示例 |
|------|------|------|
| 阶层冲突 | 贫富对立 | 穷小子vs富家女 |
| 职场冲突 | 权力斗争 | 职场PUA |
| 家庭冲突 | 代际矛盾 | 逼婚、啃老 |
| 圈层冲突 | 群体对立 | 本地vs外地 |

### 3. 内心冲突（需外化）
| 类型 | 描述 | 外化方式 |
|------|------|----------|
| 选择困境 | 两难抉择 | 通过对话展现 |
| 自我否定 | 缺乏自信 | 镜子对话/内心画外音 |
| 道德挣扎 | 对错两难 | 天使恶魔对话 |

## 🎬 快速冲突公式

### 3秒建立冲突
```
[人物A] + [对立立场/态度] + [人物B]
```
示例：
- 霸道婆婆 + 指着鼻子骂 + 隐忍媳妇
- 嚣张同事 + 抢功劳 + 被欺负员工

### 冲突升级公式
```
初始冲突 → 加入新因素 → 升级 → 再加因素 → 爆发
```

### 冲突密度
| 时长 | 冲突数量 | 节奏 |
|------|----------|------|
| 30秒 | 1个主冲突 | 极快 |
| 1分钟 | 1主+1小 | 快 |
| 2-3分钟 | 1主+2-3小 | 中快 |

## 🎯 冲突表演要点

### 表情强度
| 情绪 | 弱 | 中 | 强 |
|------|-----|-----|-----|
| 愤怒 | 眉头微皱 | 咬牙切齿 | 暴跳如雷 |
| 委屈 | 眼眶泛红 | 泪流不止 | 崩溃大哭 |
| 不屑 | 冷笑 | 嗤之以鼻 | 狂笑讽刺 |
| 震惊 | 愣住 | 瞪大眼睛 | 说不出话 |

### 肢体语言
| 态度 | 肢体表现 |
|------|----------|
| 进攻 | 身体前倾、指向对方 |
| 防御 | 后退、双臂交叉 |
| 威胁 | 压迫性靠近 |
| 崩溃 | 后退跌倒、抱头 |

## 🚫 冲突常见问题

| 问题 | 表现 | 解决 |
|------|------|------|
| 太弱 | 观众无感 | 提高强度/更直接 |
| 太假 | 不合逻辑 | 建立合理动机 |
| 太快 | 没铺垫 | 加3秒前置 |
| 太久 | 拖沓 | 精简到关键点 |
| 无果 | 冲突后无结果 | 给出阶段性结果 |
