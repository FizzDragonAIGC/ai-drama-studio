## 📐 Q版基础比例

### 头身比对照

| 头身比 | 名称 | 特点 | 适用 |
|--------|------|------|------|
| **2头身** | 超Q版 | 极度可爱、简化 | 表情包、搞笑 |
| **3头身** | 标准Q版 | 平衡可爱与表现力 | 最常用 |
| **4头身** | 小头身 | 略有身材、更多动作 | 动作场景 |
| **5-6头身** | 萌系 | 介于Q版与正常之间 | 萌系作品 |

## 🎯 Q版头部设计

### 头部特征
```
正常 → Q版 变化：
┌─────────────────────────────────┐
│ 头部：放大（约1.5-2倍）         │
│ 眼睛：放大、位置下移            │
│ 鼻子：简化为点或省略            │
│ 嘴巴：简化、表情夸张            │
│ 脸型：圆润化                    │
└─────────────────────────────────┘
```

### 眼睛简化
```
【正常】          【Q版】
 ┌───────┐        ●  ●    ← 简单圆形
 │ 瞳孔  │   →     
 │ 高光  │        ◉  ◉    ← 保留高光
 └───────┘        
                  ╲  ╱    ← 表情线条
```

### 表情放大
```
普通表情：微调
Q版表情：夸张5倍

笑 → 大笑 → 狂笑
泪 → 瀑布泪 → 喷泉泪
怒 → 全脸青筋 → 燃烧
```

## 📊 Q版身体简化

### 身体特征变化

| 部位 | 正常 | Q版处理 |
|------|------|---------|
| **躯干** | 区分胸腰 | 简化为圆柱/方块 |
| **手** | 五指分明 | 简化为拳头/三指 |
| **脚** | 正常比例 | 小短腿/简化鞋 |
| **关节** | 正常弯曲 | 减少关节、直线化 |

## 🔧 Q版动作设计

### 动作夸张原则
```
正常动作幅度 × 1.5~2 = Q版动作幅度

跳跃：腿更弯、跳更高
跑步：手脚摆幅更大
战斗：姿势更夸张
```

### 常用Q版动作
```
【开心跳跃】      【愤怒跺脚】      【可爱拜托】
    ●                ●                ●
   ╲╱              ╔╗              ╱│╲
   ╱╲        💢    ║║              │
   ∧∧              ╚╝             ╱ ╲
                   ▼▼             🙏
```

### 表情与动作配合
| 情绪 | 表情 | 身体动作 |
|------|------|----------|
| 开心 | 眯眼大笑 | 跳跃、挥手 |
| 愤怒 | 青筋、尖牙 | 跺脚、挥拳 |
| 悲伤 | 瀑布泪 | 蹲下、手捂脸 |
| 惊讶 | 大眼、张嘴 | 后仰、手举起 |
| 害羞 | 脸红、手捂脸 | 扭动 |

## 🎬 Q版应用场景

### 何时使用Q版

| 场景 | 目的 |
|------|------|
| **搞笑桥段** | 增加喜剧效果 |
| **日常互动** | 轻松氛围 |
| **反差萌** | 严肃角色的反差 |
| **章节封面** | 可爱吸引 |
| **额外内容** | 四格、特典 |
| **情感爆发** | 夸张表现 |

### Q版切换时机
```
【正常】→ 搞笑台词 → 【Q版】→ 回归正常 → 【正常】

示例：
严肃对决 → 突然肚子饿 → Q版崩坏 → 回归严肃
```

## 📐 Q版服装简化

### 简化原则
```
保留特征，减少细节

【正常】                【Q版】
华丽铠甲（50处细节）→   简化铠甲（10处特征）
复杂花纹 → 代表性图案
飘逸长发 → 简化发型保留轮廓
```

### 特征提取
| 服装元素 | 保留 | 简化 |
|----------|------|------|
| **颜色** | ✓保留 | - |
| **轮廓** | ✓保留 | - |
| **标志** | ✓保留 | - |
| **褶皱** | - | ✗简化 |
| **细节** | - | ✗减少 |
| **配件** | ✓重点 | ✗次要 |

## 🎨 Q版风格变体

### 日式SD风
```
特点：圆脸、大眼、精致
眼睛：保留高光和细节
线条：流畅圆润
```

### 简笔Q版
```
特点：极简、图标化
眼睛：简单点或线
线条：最少线条表达
```

### 欧美Chibi
```
特点：头略小、表情夸张
眼睛：更大的表现力
线条：可略带西式风格
```

## ⚠️ 常见错误

| 错误 | 问题 |
|------|------|
| **头太小** | 不够Q |
| **身体太细节** | 失去简化感 |
| **表情平淡** | 不够生动 |
| **比例不一致** | 头身比混乱 |
| **失去特征** | 认不出角色 |
