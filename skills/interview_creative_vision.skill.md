# 创意愿景访谈 (Creative Vision Interview)

## 概述
深度访谈创作者（作者/导演/制片人），挖掘他们的真实意图、情感内核和艺术追求。

## 核心理论

### 访谈的重要性
- 表面故事 ≠ 真正想表达的
- 创作者的个人经历往往是灵感来源
- 商业需求和艺术追求需要平衡
- 避免"我以为你想要"的误解

### 访谈三层次
1. **事实层** - 故事讲什么
2. **情感层** - 想传达什么感受
3. **主题层** - 想探讨什么问题

## 必问的10个核心问题

### 第一部分：起源与动机（3题）

**Q1: 这个故事的灵感来源是什么？**
- 个人经历？
- 看到的新闻？
- 其他作品的启发？
- 一个"如果...会怎样"的问题？

**Q2: 为什么是现在要讲这个故事？**
- 和当下社会有什么关联？
- 你个人生活中发生了什么？
- 为什么这个故事对你重要？

**Q3: 你希望观众看完后有什么感受或行动？**
- 单纯娱乐？
- 思考某个问题？
- 改变某种观念？
- 产生某种情绪？

### 第二部分：角色与主题（4题）

**Q4: 主角身上有多少是你自己？**
- 性格特点？
- 经历过的困境？
- 你的恐惧或欲望？

**Q5: 这个故事的核心主题用一句话怎么说？**
- 不是情节，是主题
- 关于什么的故事？爱？自由？成长？

**Q6: 故事里最让你心痛/兴奋的一场戏是哪个？**
- 为什么是这场？
- 你写这场时的感受？

**Q7: 如果只能保留一个场景，你会保留哪个？**
- 这个场景承载了什么？
- 删掉它故事还成立吗？

### 第三部分：风格与商业（3题）

**Q8: 有没有参考的作品？你希望作品像什么？**
- 视觉风格参考？
- 叙事风格参考？
- 情绪氛围参考？
- 「X meets Y」的组合？

**Q9: 你最担心什么？不希望作品变成什么样？**
- 避免的风格？
- 避免的误解？
- 底线是什么？

**Q10: 商业上的期待是什么？**
- 目标平台？
- 目标观众？
- 成功的定义？

## 进阶追问技巧

### 当回答模糊时
- "能具体举个例子吗？"
- "当时是什么感觉？"
- "如果是你，你会怎么做？"

### 当回答矛盾时
- "我注意到之前你说...现在又说...能帮我理解一下吗？"
- "这两个想法如何共存？"

### 当触及真实情感时
- 暂停，给空间
- "这似乎对你很重要..."
- "谢谢你分享这个"

## Prompt模板

### 访谈引导
```
我将扮演一位资深制片人/编剧顾问，对你进行创意愿景访谈。

请放松，没有对错之分，我只是想真正理解你想讲的故事。

我们开始吧。

Q1: 这个故事的灵感最初是怎么来的？有没有某个具体的时刻或事件？
```

### 访谈总结
```
基于以上访谈，总结创作者愿景：

【高概念】一句话版本

【情感内核】这个故事真正想表达的是...

【目标观众】最能产生共鸣的人群

【核心场景】必须保留的关键时刻

【绝对禁区】不能碰的红线

【商业定位】平台×时长×风格
```

## 检查清单
- [ ] 是否了解了灵感来源？
- [ ] 是否明确了核心主题？
- [ ] 是否知道最重要的场景？
- [ ] 是否了解风格参考？
- [ ] 是否知道底线和禁区？
- [ ] 是否明确商业目标？
