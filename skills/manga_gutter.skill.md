# 📖 漫画格间设计 Skill
> 来源：《Understanding Comics》Scott McCloud + 《漫画技法》

---

## 📐 格间基础概念

### 什么是格间（Gutter）
```
┌─────────┐ ← 格子（Panel）
│         │
│  画面   │
│         │
└─────────┘
     ↑
   格间（Gutter）= 格子之间的空白
     ↓
┌─────────┐
│         │
│  画面   │
│         │
└─────────┘
```

**格间的作用**：读者在格间中"脑补"时间流逝和动作过程

---

## 🎯 格间宽度与时间感

| 格间宽度 | 时间感知 | 适用场景 |
|----------|----------|----------|
| **无间距** | 同时/瞬间 | 同一动作多角度 |
| **极窄 (1mm)** | 0.1秒 | 快速连续动作 |
| **窄 (2-3mm)** | 1秒内 | 对话、反应 |
| **标准 (4-5mm)** | 数秒 | 正常叙事 |
| **宽 (6-8mm)** | 数分钟 | 场景转换 |
| **超宽 (10mm+)** | 数小时/天 | 时间跳跃 |

## 🔄 格间过渡六种类型

### 1. 时刻到时刻
```
【格1】         【格2】
刀举起    →    刀挥下
（极窄格间）
```
**时间跳跃**：几分之一秒
**脑补内容**：最少

### 2. 动作到动作
```
【格1】         【格2】
起跑      →    冲过终点
（窄格间）
```
**时间跳跃**：数秒
**脑补内容**：中等

### 3. 主体到主体
```
【格1】         【格2】
角色A说话  →   角色B反应
（标准格间）
```
**时间跳跃**：瞬间（同一场景）
**脑补内容**：最常用

### 4. 场景到场景
```
【格1】         【格2】
室内       →    户外
（宽格间）
```
**时间跳跃**：可长可短
**脑补内容**：空间+时间

### 5. 方面到方面
```
【格1】         【格2】         【格3】
眼睛特写  →   手部特写   →   全身
（标准格间）
```
**时间跳跃**：同一时刻
**脑补内容**：拼凑整体

### 6. 非连续
```
【格1】         【格2】
现在       →    10年前
（超宽格间或特殊处理）
```
**时间跳跃**：随意
**脑补内容**：需要提示

## 🎨 格间视觉处理

### 标准格间
```
┌─────┐   ┌─────┐
│     │   │     │
└─────┘   └─────┘
    白色空白
```
**效果**：中性、标准

### 黑色格间
```
┌─────┐███┌─────┐
│     │███│     │
└─────┘███└─────┘
    黑色填充
```
**效果**：时间跳跃、回忆、紧张

### 出血格间
```
┌─────────────────┐
│                 │
│    图像延伸到   │
│    页面边缘     │
└─────────────────┘
```
**效果**：打破边界、冲击力

### 无格间（画面相连）
```
┌─────────────────┐
│ A画面 │ B画面   │
│    融合过渡     │
└─────────────────┘
```
**效果**：同时发生、梦境

## 📊 格间设计原则

### 节奏控制
| 场景类型 | 格间处理 |
|----------|----------|
| **动作戏** | 窄格间、密集 |
| **对话戏** | 标准格间 |
| **抒情戏** | 宽格间、留白多 |
| **高潮** | 极窄→突然宽 |
| **转场** | 黑格间/宽格间 |

### 信息量原则
```
格间窄 → 读者脑补少 → 作者控制多
格间宽 → 读者脑补多 → 读者想象多
```

## 🔧 特殊格间技法

### 1. 悬念格间
```
┌─────┐
│ ？  │ ← 悬念画面
└─────┘
  很宽格间（留白增加期待）
┌─────┐
│ ！  │ ← 揭示画面
└─────┘
```

### 2. 速度格间
```
┌──┐┌──┐┌──┐┌──┐  极窄间距 = 极快速度
└──┘└──┘└──┘└──┘
```

### 3. 情感格间
```
┌─────────┐
│ 悲伤表情 │
└─────────┘

   （大量留白）

┌─────────┐
│ 独自远去 │
└─────────┘
```
**效果**：沉重、感伤

## 🎯 AI Prompt指导

### 格间风格描述
```
manga panel layout with [gutter style]:
- tight gutters: 紧凑格间，快节奏
- standard gutters: 标准格间
- wide gutters: 宽格间，慢节奏
- bleeding panels: 出血格，冲击力
- borderless: 无边框，梦幻感
```

### 示例Prompt
```
manga page, 6 panels with tight gutters,
action sequence, dynamic panel shapes,
black and white, shonen style
```
