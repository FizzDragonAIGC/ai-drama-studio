> 来源：《电影镜头设计》Steven D. Katz + 视觉心理学研究

---

## 📐 什么是Eye Trace（视线引导）

**定义**：通过画面设计控制观众视线移动的路径

### 为什么重要
| 原因 | 说明 |
|------|------|
| **聚焦注意力** | 让观众看到该看的 |
| **叙事引导** | 按顺序接收信息 |
| **情感操控** | 引导看到情感重点 |
| **剪辑流畅** | 切换时视线不跳 |

## 🔧 视线引导技巧

### 1. 构图引导
| 技巧 | 方法 | 效果 |
|------|------|------|
| 引导线 | 道路、栏杆、光束 | 引向焦点 |
| 框中框 | 门框、窗框 | 聚焦主体 |
| 三分法 | 主体放在交点 | 自然吸引 |
| 对角线 | 沿对角线安排 | 动态感 |

### 2. 光线引导
```
┌─────────────────────────────────┐
│                                 │
│    暗区  ░░░░░█████░░░░  暗区   │
│           ↓ 光照区 ↓            │
│         ░░ 人物 ░░              │
│                                 │
└─────────────────────────────────┘
→ 明暗对比自然引导视线
```

### 3. 焦点引导
| 技巧 | 实现 |
|------|------|
| 前景虚化 | 前景模糊→视线跳过 |
| 背景虚化 | 背景模糊→关注主体 |
| 焦点转移 | 动态引导视线 |

### 4. 运动引导
```
运动方向 = 视线方向

移动物体 ────→ 👁️观众视线跟随

人物视线 ────→ 👁️观众视线跟随
```

## 📋 镜头间视线连贯

### 剪辑时视线位置
```
镜头1结束时：
┌─────────────────────────────────┐
│                                 │
│              👁️ ← 角色A在这     │
│                                 │
└─────────────────────────────────┘

镜头2开始时：
┌─────────────────────────────────┐
│                                 │
│              👁️ ← 角色B也在这   │
│                                 │
└─────────────────────────────────┘
→ 视线位置一致 = 剪辑流畅
```

### 视线跳跃问题
```
镜头1结束时：
┌─────────────────────────────────┐
│  👁️ ← 角色在左上                │
│                                 │
│                                 │
└─────────────────────────────────┘

镜头2开始时：
┌─────────────────────────────────┐
│                                 │
│                                 │
│                        👁️       │ ← 角色在右下
└─────────────────────────────────┘
→ 视线需要大跳跃 = 不舒适
```

## 🎯 视线引导模式

### 1. Z型阅读模式
```
┌─────────────────────────────────┐
│  1 ──────────────→ 2            │
│         ╲                       │
│          ╲                      │
│           ↘                     │
│  3 ──────────────→ 4            │
└─────────────────────────────────┘
→ 适合多信息画面
```

### 2. 中心辐射模式
```
┌─────────────────────────────────┐
│    ↗     ↑     ↖                │
│      ╲   │   ╱                  │
│   ←──── 🎯 ────→                │
│      ╱   │   ╲                  │
│    ↙     ↓     ↘                │
└─────────────────────────────────┘
→ 适合单主体
```

### 3. 路径引导模式
```
┌─────────────────────────────────┐
│  START                          │
│    ↓                            │
│   ╭──→ 信息1                    │
│   │      ↓                      │
│   │   ╭──→ 信息2                │
│   │   │      ↓                  │
│         🎯 焦点                  │
└─────────────────────────────────┘
→ 适合叙事性画面
```

## 🎬 AI番剧应用

### 构图Prompt技巧
```
引导线：
"path leading to character, road perspective, 
lines converging on subject"

框中框：
"character framed by doorway, 
window frame around subject"

光线引导：
"spotlight on character, 
dramatic lighting, dark surroundings"
```

### 剪辑规划
```
镜头1：角色A在画面中央
↓ 切
镜头2：角色B也在画面中央

或者：

镜头1：角色A视线向右看
↓ 切
镜头2：从左侧入画的物体
→ 视线方向匹配
```

### 分镜设计模板
```
| 镜号 | 主体位置 | 视线引导元素 | 切入下一镜 |
|------|----------|--------------|------------|
| 01 | 中央 | 门框框住 | 主体中央 |
| 02 | 中央 | 光照高亮 | 视线向右 |
| 03 | 左入 | 跟随运动 | 主体中央 |
```

## 💡 常见问题

| 问题 | 原因 | 解决 |
|------|------|------|
| 观众不看主体 | 有更吸引的元素 | 降低干扰物 |
| 剪辑不顺 | 视线位置跳跃 | 规划主体位置 |
| 画面混乱 | 无引导 | 加入引导线 |
| 信息漏看 | 停留时间短 | 延长镜头 |
