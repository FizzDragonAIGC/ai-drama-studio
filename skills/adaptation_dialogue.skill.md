# 📚 對話轉化技能

## 元信息
- **技能ID**: adaptation_dialogue
- **所屬Agent**: 改編Agent
- **參考書籍**: 《對白的藝術》Robert McKee
- **難度**: ⭐⭐⭐⭐

## 核心理念

小說對話 ≠ 影視對話
- 小說對話可以有大段內心旁白
- 影視對話必須「聽」起來自然

## 對話轉化三原則

### 原則1：壓縮
```
原著：
「我覺得我們應該好好談談，」他說，「這件事情已經困擾我很久了，
每次想到你那天說的話，我就會想，我們之間是不是出了什麼問題，
也許是我做錯了什麼，也許是我們都需要一些時間來思考...」

改編：
「我們需要談談。」
（沉默，眼神交流）
「是我的問題嗎？」
```

### 原則2：外化
```
原著（內心獨白）：
他想，她一定是在隱瞞什麼。她的眼神飄忽，
從不直視他，這讓他感到一陣寒意。

改編（外在行動+對話）：
「你在看哪裡？」
（她移開視線）
「沒什麼。」
「你在撒謊。」
```

### 原則3：留白
```
原著（解釋性對話）：
「我愛你，但我不能和你在一起，因為我有責任，
我的家人需要我，我不能為了自己的幸福...」

改編（潛台詞）：
「我愛你。」
「但是？」
「...」
（轉身離開）
```

## 文學語言→口語轉化

| 文學風格 | 口語風格 |
|----------|----------|
| 「他感到一陣強烈的愧疚湧上心頭」 | 「對不起」/「都是我的錯」 |
| 「她意識到時間正在流逝」 | 「已經很晚了」/「我該走了」 |
| 「他們的關係陷入了一種微妙的緊張」 | （長時間的沉默+肢體語言） |
| 「這是一個令人心碎的時刻」 | （不說，只演） |

## 對話功能檢查

### 每句對話必須服務於至少一個功能

```
□ 推動劇情
  - 傳達新信息
  - 觸發行動
  - 揭示轉折

□ 塑造人物
  - 展示性格
  - 暴露慾望
  - 體現背景

□ 建立關係
  - 權力動態
  - 親密程度
  - 衝突張力

□ 營造氛圍
  - 幽默/緊張/浪漫
  - 時代/地域特色
```

**無功能對話 = 刪除**

## 對話節奏設計

### 快節奏（緊張/衝突）
```
- 短句
- 互相打斷
- 幾乎無停頓
- 音量逐漸升高

例：
「你知道嗎——」
「我知道。」
「那你為什麼——」
「因為我必須。」
「你根本不——」
「夠了！」
```

### 慢節奏（親密/沉思）
```
- 長句+停頓
- 給對方反應時間
- 允許沉默
- 輕聲細語

例：
「你還記得那個夏天嗎？」
（停頓）
「嗯。」
「我一直想告訴你...」
（長停頓）
「我知道。」
```

## 潛台詞設計

### 表面意思 vs 真正意思

| 說的話 | 真正的意思 |
|--------|------------|
| 「你最近好嗎？」 | 我很想你 |
| 「隨便你」 | 我很生氣 |
| 「我沒事」 | 我很受傷 |
| 「你變了」 | 我不再了解你 |
| 「保重」 | 再見（永別） |

### 潛台詞技巧
```
1. 說反話
   表面：「真好」 → 實際：很差

2. 轉移話題
   問A答B，迴避真相

3. 重複
   「好的...好的...」（不好）

4. 沉默
   最重要的話不說出來
```

## 角色聲音差異化

### 每個角色要有獨特的說話方式

| 維度 | 角色A | 角色B |
|------|-------|-------|
| 句子長度 | 短促 | 冗長 |
| 用詞 | 粗俗 | 文雅 |
| 態度 | 直接 | 迂迴 |
| 口頭禪 | 有 | 無 |
| 幽默感 | 諷刺 | 冷面 |
