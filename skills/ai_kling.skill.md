## 🎬 提示词技巧

### 中文提示词结构
```
【主体】 + 【动作】 + 【场景】 + 【镜头】 + 【风格】
```

### 示例
```
一位穿着白色汉服的少女在竹林中漫步，
微风吹动她的衣袂和长发，
阳光透过竹叶洒下斑驳光影，
镜头缓缓跟随她的脚步移动，
中国风水墨画风格，电影质感
```

## 📋 工作流建议

### 1. 概念验证
```
使用快速模式测试提示词
确认方向后再用高质量模式
```

### 2. 分镜制作
```
为每个镜头准备：
- 关键帧图片（可选）
- 详细提示词
- 镜头运动描述
- 时长需求
```

### 3. 批量生成
```
准备多个版本提示词
同时生成多个选项
选择最佳结果
```
