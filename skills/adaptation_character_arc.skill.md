# 📚 人物弧光改編技能

## 元信息
- **技能ID**: adaptation_character_arc
- **所屬Agent**: 改編Agent
- **參考書籍**: 《角色的藝術》+ 《故事》
- **難度**: ⭐⭐⭐⭐

## 核心理念

人物弧光 = 內在變化的軌跡
- 番劇比小說更依賴外在行動來展示內在變化
- 改編核心：讓內心旅程「看得見」

## 弧光類型

### 1. 正向弧光（最常見）
```
缺陷 → 成長 → 克服

例：
怯懦 → 勇氣
自私 → 利他
封閉 → 開放

設計要點：
- 明確起點缺陷
- 設計催化事件
- 漸進式改變
- 最終考驗驗證
```

### 2. 負向弧光（悲劇）
```
優點 → 腐蝕 → 墮落

例：
理想主義 → 妥協 → 腐敗
信任 → 背叛 → 偏執

設計要點：
- 起點是可共情的
- 墮落有「合理」的理由
- 觀眾既惋惜又理解
```

### 3. 平坦弧光（英雄改變世界）
```
堅定 → 考驗 → 更堅定

主角不變，但改變了周圍的人

例：
超人、哆啦A夢

設計要點：
- 主角從一開始就「對」
- 他的考驗是堅持而非改變
- 重點在他影響的人的弧光
```

## 弧光可視化設計

### 原著心理 → 改編行動

| 原著描寫 | 改編呈現 |
|----------|----------|
| 「他開始反思」 | 凝視窗外/沉默/道具象徵 |
| 「她終於原諒了」 | 擁抱/刪除信息/回歸舊地 |
| 「他不再害怕」 | 主動出擊/直視敵人/改變姿態 |
| 「她決定離開」 | 收拾行李/撕照片/關門特寫 |

### 弧光里程碑

```
集1-2：建立缺陷
└── 通過行動展示（不是說出來）

集3-4：意識萌芽
└── 開始質疑自己

中點：重要覺醒
└── 標誌性事件觸發變化

集X-Y：反覆考驗
└── 新方式 vs 舊習慣的拉扯

高潮：最終證明
└── 用行動證明已經改變

結局：新的常態
└── 展示改變後的生活
```

## 配角弧光設計

### 配角弧光規模
```
主角：完整弧光（6-8個里程碑）
核心配角：中等弧光（3-4個里程碑）
次要配角：微弧光（1-2個變化）
功能配角：無弧光（保持不變）
```

### 配角弧光與主角的關係
```
鏡像型：
- 配角走相反的路
- 襯托主角的選擇

催化型：
- 配角的變化促進主角改變
- 或主角促進配角改變

對照型：
- 相似處境，不同選擇
- 不同結局說明主題
```

## 弧光壓縮技巧

### 小說30萬字 → 12集番劇
```
1. 選擇核心弧光
- 只保留1-2個主要人物的完整弧光
- 其他角色簡化或合併

2. 濃縮轉變時刻
- 原著可能有10個覺醒時刻
- 改編選擇3個最有力的

3. 強化外在表現
- 每個內在轉變都有外在對應
- 服裝/髮型/習慣/語言的變化

4. 設計視覺錨點
- 用重複的意象標記變化
- 開頭和結尾的對比鏡頭
```

## 弧光檢驗清單

```
□ 起點清晰嗎？（觀眾知道主角缺什麼）
□ 終點清晰嗎？（觀眾知道主角得到什麼）
□ 變化有外在表現嗎？（不只是旁白說）
□ 變化是漸進的嗎？（不是突然的）
□ 有反覆嗎？（不是一帆風順）
□ 最終考驗夠大嗎？（證明變化是真的）
□ 觀眾會在乎嗎？（有情感投資）
```
