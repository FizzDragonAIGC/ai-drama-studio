## 核心理念

視覺化寫作 = 讓文字「可見」
- 讀劇本時，腦中自動播放畫面
- 每一個字都服務於視覺呈現

## 視覺寫作原則

### 1. 具體勝過抽象
```
❌ 她很傷心
✅ 她的淚水滴在信紙上，墨字暈開

❌ 房間很亂
✅ 衣服散落一地，咖啡杯發霉，窗簾緊閉
```

### 2. 動態勝過靜態
```
❌ 他站在那裡
✅ 他的手指輕輕敲著桌面

❌ 夜色很美
✅ 月光穿過雲層，灑在湖面上
```

### 3. 細節勝過概述
```
❌ 他緊張
✅ 他的腳不停抖動，咬著指甲

選擇一個有力的細節，勝過三個平淡的描述
```

### 4. 暗示勝過說明
```
❌ 他是個老兵
✅ 他的夾克上有褪色的軍章，右手習慣性地放在腰間
```

## 感官寫作

### 視覺
```
顏色、光影、動作、表情
「暖黃的燈光照在她臉上」
```

### 聽覺
```
環境音、對話、音樂暗示
「遠處傳來火車的汽笛聲」
```

### 其他感官（轉化為視覺）
```
氣味 → 人的反應：「她皺起鼻子」
觸感 → 動作：「他的手指滑過粗糙的樹皮」
溫度 → 環境：「哈出的氣變成白霧」
```

## 空間感營造

### 建立鏡頭（Establishing）
```
從大到小：
城市遠景 → 街道 → 建築 → 窗戶 → 室內 → 角色
```

### 空間描寫
```
不要列清單：
❌ 房間裡有床、桌子、椅子、窗戶...

選擇性描寫：
✅ 床頭的照片框裡，兩個人的笑容已經褪色
（暗示了時間、關係、情感）
```

## 時間感營造

### 快速時間
```
短句、動作連續
「他抓起鑰匙。衝出門。車子發動。」
```

### 慢速時間
```
細節放大、描寫豐富
「她的手指一寸寸地移向他的臉，
 燈光在她眼中跳動，
 嘴唇微微顫抖...」
```

## 符號與意象

### 反覆意象
```
開頭：完整的瓷杯
中間：杯子有裂縫
結尾：杯子破碎
= 關係的變化
```

### 環境映射
```
角色開心 → 陽光明媚
角色絕望 → 暴風雨
（但避免太刻意）
```
