# 🇨🇳 中国影视剧本格式 (Chinese Script Format)
> 专业中国影视剧本写作标准

---

## 📋 标准格式

### 基本单位
- **场次**：每个场景一个编号
- **场景头**：内/外景 + 地点 + 时间
- **动作行**：描述画面和动作
- **对白**：角色名 + 台词
- **括号语**：语气/小动作

---

## 🎬 场景头格式

```
第X场  内景/外景  地点  日/夜/晨/昏/凌晨
```

### 示例
```
第1场  外景  北京王府井大街  日

第2场  内景  老旧筒子楼 - 张三家客厅  夜

第3场  外景/内景  医院急诊室走廊  凌晨
```

### 时间选项
| 选项 | 说明 |
|------|------|
| 日 | 白天 |
| 夜 | 夜晚 |
| 晨 | 早晨 |
| 昏 | 黄昏 |
| 凌晨 | 深夜到天亮前 |
| 傍晚 | 日落时分 |
| 稍后 | 同一地点，略过时间 |

---

## 📝 动作行规范

### 规则
1. **只写能拍到的** - 不写内心活动
2. **现在时态** - "走进来"而非"走进来了"
3. **简洁有力** - 每段不超过4行
4. **留白** - 动作块之间空一行

### ❌ 错误示范
```
张三心想：这辈子从来没有这么紧张过，
他回忆起小时候父亲对他的期望，
那些压力一直压在他心头多年...
```

### ✅ 正确示范
```
张三站在门口，握着门把手的手微微颤抖。
他深吸一口气，缓缓推开门。
```

---

## 🗣️ 对白格式

### 基本格式
```
角色名
    台词内容

角色名（括号语）
    台词内容
```

### 示例
```
张三
    你这是什么意思？

李四（冷笑）
    什么意思？你自己心里清楚。

张三
    （沉默片刻）
    好，我知道了。
```

### 括号语用法
| 用法 | 示例 |
|------|------|
| **语气** | （怒吼）（低声）（咬牙切齿） |
| **小动作** | （点烟）（转身）（合上文件） |
| **情绪** | （强忍泪水）（故作轻松） |
| **对谁说** | （对小王）（自言自语）（画外音） |

## 🎥 特殊标注

### 镜头建议
```
【特写】张三颤抖的手
【俯拍】空旷的操场
【跟拍】李四穿过人群
```

### 闪回
```
【闪回 - 十年前】

第15场  外景  乡村小学门口  日

八岁的张三背着书包...

【闪回结束】
```

### 蒙太奇
```
【蒙太奇开始】

A) 内景 健身房 - 张三挥汗如雨
B) 内景 图书馆 - 张三埋头苦读  
C) 外景 操场 - 张三跑步
D) 内景 卧室 - 张三对镜练习

【蒙太奇结束】
```

### 画外音/旁白
```
张三（画外音）
    那一年，我二十三岁，以为自己什么都懂...

张三（V.O.）
    后来我才知道，那是我人生的转折点。
```

## 📖 完整场景示例

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
第7场  内景  老旧筒子楼 - 张三家客厅  夜
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

昏暗的灯光下，墙上挂着全家福，照片里的人都笑得很开心。

张三独自坐在沙发上，面前的茶几上放着一个信封。
信封上写着"辞职报告"。

他拿起信封，又放下。
如此反复几次。

门外传来钥匙开门的声音。

张三连忙把信封塞进沙发垫下面。

妻子王芳推门进来，手里提着菜。

王芳
    回来啦？今天下班挺早。

张三（故作轻松）
    嗯...今天没什么事。

王芳把菜放到厨房，回来坐在张三旁边。
她注意到沙发垫有些歪。

王芳
    怎么了？看你脸色不太好。

张三
    没...没事。就是有点累。

王芳伸手摸摸张三的额头。

王芳
    不发烧啊。
    （盯着他看）
    有什么事瞒着我？

【特写】张三的眼神闪躲。

张三
    真没事...
    （突然站起来）
    我去洗把脸。

张三快步走向卫生间。

王芳看着他的背影，若有所思。
她的目光落在歪斜的沙发垫上。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 🎭 中国特色元素

### 称谓系统
| 关系 | 称呼示例 |
|------|----------|
| 夫妻 | 老公/老婆、他爸/他妈、那口子 |
| 父子 | 爸/爹/老头子、儿子/小子/臭小子 |
| 朋友 | 兄弟/哥们儿、姐妹/闺蜜 |
| 同事 | X哥/X姐、老X、小X |
| 上下级 | 领导/老板、X总/X经理 |

### 文化场景
- **饭桌戏** - 中国家庭核心场景
- **送别戏** - 车站/机场的离别
- **婚丧嫁娶** - 人生大事
- **过年戏** - 团圆与冲突

### 中式潜台词
| 表面说 | 实际意思 |
|--------|----------|
| "你还年轻" | 你不懂 |
| "改天再说" | 不想说/拒绝 |
| "都是为你好" | 你必须听我的 |
| "我没事" | 我有事 |
| "随便" | 一点都不随便 |
