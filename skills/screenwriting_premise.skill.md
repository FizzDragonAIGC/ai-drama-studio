# 前提与主题 Skill
> 来源：Robert McKee《故事》、Blake Snyder《救猫咪》
> 适用智能体：编剧助手、故事策划

## 核心理念

**"前提是故事的DNA"** —— 一句话说清故事的核心冲突与主题

前提（Premise）回答"这个故事关于什么"，主题（Theme）回答"这个故事想说什么"。

---

## 核心理论

### 前提的三要素

```
角色 + 冲突 + 结局暗示 = 强力前提
```

**示例解析**：
- 《千与千寻》：一个被困在神灵世界的女孩必须找回自己的名字才能救出父母
- 《进击的巨人》：人类最后的城市面临巨人入侵，一个少年发誓要消灭所有巨人
- 《你的名字》：两个身体互换的陌生人必须在彗星毁灭小镇前找到彼此

### 主题层次

| 层次 | 定义 | 示例 |
|------|------|------|
| 表层主题 | 故事讲的事 | 少年打败魔王 |
| 深层主题 | 故事的意义 | 成长的代价 |
| 哲学主题 | 普世真理 | 失去才懂珍惜 |

### 主题陈述公式

```
[控制性想法] = [价值] + [原因]
示例：正义终将战胜邪恶，因为坚持信念的人永不放弃
```

## 实操技巧

### 一句话测试

**前提必须能在一句话内说清**：
- ❌ "这是一个关于友情、冒险、还有魔法世界、各种种族..."
- ✅ "一个没有魔力的少年进入魔法学院，必须用智慧证明自己"

### Logline公式

```
当[触发事件]发生时，[主角特征]的[主角]必须[目标]，
否则[风险]，但[障碍]。
```

**示例**：
当妹妹被鬼附身后，一个胆小的高中生必须找到驱魔方法，
否则妹妹会死，但他发现真正的鬼是自己的执念。

### 主题提炼法

1. **对立统一**：找到故事中的核心对立
   - 个人vs集体、传统vs变革、爱vs责任

2. **人物弧光**：主角从A到B的变化代表什么？
   - 怯懦→勇敢 = 真正的勇气来自对他人的爱

3. **结局测试**：结局证明了什么真理？

### 主题强化技巧

| 技巧 | 说明 | 示例 |
|------|------|------|
| 角色对比 | 不同选择不同结局 | 两兄弟一善一恶 |
| 符号系统 | 重复的意象承载主题 | 蝴蝶象征自由 |
| 台词点题 | 关键时刻说出主题 | "活下去！" |
| 子情节 | 副线呼应主线主题 | 配角重复主角困境 |

## Prompt模板

### 前提生成

```
请为以下设定生成5个高概念前提（High Concept Premise）：
- 类型：[奇幻/科幻/青春/悬疑...]
- 世界观：[简述]
- 主角类型：[性格/身份]

要求：
1. 每个前提一句话（30字内）
2. 包含：角色特征+核心冲突+独特设定
3. 具备商业吸引力
4. 能延展成12集番剧
```

### 主题深化

```
分析以下前提的主题层次：
前提：[你的故事前提]

请输出：
1. 表层主题（故事讲什么）
2. 深层主题（故事的意义）
3. 哲学主题（普世真理）
4. 控制性想法（价值+原因）
5. 三个强化主题的具体建议
```

### Logline完善

```
请将以下故事想法转化为专业Logline：
想法：[你的原始想法]

输出格式：
当[触发事件]发生时，[主角特征]的[主角]必须[目标]，
否则[风险]，但[障碍]。

额外要求：
- 25-35字最佳
- 包含讽刺性或戏剧性张力
- 能引发好奇心
```

## 检查清单

### 前提检查
- [ ] 一句话能说清？
- [ ] 有明确的主角？
- [ ] 有清晰的目标？
- [ ] 有具体的障碍？
- [ ] 有时间紧迫感？
- [ ] 有独特的钩子/设定？
- [ ] 能引发好奇心？

### 主题检查
- [ ] 三个层次都清晰？
- [ ] 与主角弧光契合？
- [ ] 不说教但能感受到？
- [ ] 有符号系统支撑？
- [ ] 结局能证明主题？

## 案例分析

### 案例：《鬼灭之刃》

**前提**：
一个善良的少年在全家被鬼杀害后，必须成为鬼杀队员拯救变成鬼的妹妹

**前提解析**：
- 主角特征：善良
- 触发事件：全家被杀
- 目标：拯救妹妹
- 独特设定：妹妹变成了鬼

**主题层次**：
| 层次 | 内容 |
|------|------|
| 表层 | 少年打鬼救妹妹 |
| 深层 | 家人羁绊的力量 |
| 哲学 | 善良是最强大的武器 |

**控制性想法**：
善良终将战胜残忍，因为懂得爱的人永远不会放弃希望

**主题强化手法**：
1. **角色对比**：炭治郎vs无惨（共情vs冷漠）
2. **符号系统**：火神神乐（家族传承）
3. **台词点题**："因为我是长子"（责任与爱）
4. **子情节**：每个柱都有关于羁绊的故事

## 常见问题

### Q：前提太复杂怎么办？
**A**：删到只剩核心。问自己：删掉这个元素故事还成立吗？

### Q：主题是否需要明说？
**A**：最好不要。让观众自己感受到比说出来更有力。

### Q：一个故事能有多个主题吗？
**A**：可以，但必须有一个主主题，其他是子主题。
