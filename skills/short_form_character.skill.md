# 👤 短剧角色快建 Skill
> 来源：剧作人物塑造 + 短视频角色设计

---

## 📐 快建核心原则

短剧角色必须**即时可识别**：

| 传统方式 | 短剧方式 | 原因 |
|----------|----------|------|
| 渐进揭示 | 开场定型 | 没时间铺垫 |
| 复杂多面 | 标签鲜明 | 易于记忆 |
| 内心独白 | 外化表现 | 视觉传达 |
| 背景介绍 | 即时展示 | 效率优先 |

---

## 🎯 30秒角色速建法

### 第一印象三要素
| 要素 | 时间 | 示例 |
|------|------|------|
| 视觉标签 | 1秒 | 穿着/造型/特征 |
| 性格展示 | 3秒 | 第一句话/第一个动作 |
| 身份暗示 | 5秒 | 环境/道具/他人态度 |

### 快速人设公式
```
[身份] + [性格标签] + [欲望] + [阻碍]
```

示例：
- 清洁工 + 隐忍 + 供儿子读书 + 被富婆羞辱
- 霸总 + 冷傲 + 找真爱 + 心理创伤
- 女主 + 善良倔强 + 独立 + 家庭压力

## 🎬 角色类型库

### 主角类型
| 类型 | 特点 | 适合题材 |
|------|------|----------|
| 逆袭者 | 从底层崛起 | 爽剧/复仇 |
| 守护者 | 保护他人 | 家庭/情感 |
| 追梦者 | 追求目标 | 励志/成长 |
| 局外人 | 被卷入事件 | 悬疑/冒险 |
| 反叛者 | 打破规则 | 职场/社会 |

### 配角类型
| 类型 | 功能 | 示例 |
|------|------|------|
| 助手 | 帮助主角 | 闺蜜/兄弟 |
| 导师 | 提供智慧 | 长辈/高人 |
| 对手 | 制造冲突 | 反派/情敌 |
| 镜像 | 对照主角 | 另一种选择 |
| 搞笑 | 调节气氛 | 喜剧担当 |

### 反派类型
| 类型 | 特点 | 示例 |
|------|------|------|
| 绝对恶 | 纯粹坏人 | 简单爽剧用 |
| 有理恶 | 有动机的坏 | 复杂剧情用 |
| 隐藏恶 | 表面好人 | 悬疑反转用 |
| 可悲恶 | 值得同情 | 情感深度用 |

## 🎯 角色出场设计

### 首次出场清单
| 要素 | 需要展示 | 方式 |
|------|----------|------|
| 身份 | 职业/地位 | 环境/服装/对话 |
| 性格 | 核心特点 | 行为/表情/台词 |
| 能力 | 强项/弱点 | 展示性场景 |
| 欲望 | 想要什么 | 直接或暗示 |

### 出场强度
| 等级 | 方式 | 适合角色 |
|------|------|----------|
| 强势 | 带气场出场 | 主角/重要角色 |
| 中性 | 自然出场 | 配角 |
| 悬念 | 暗示/背影 | 神秘角色 |
| 平凡 | 不起眼 | 后期反转角色 |

## 🚫 角色设计误区

| 误区 | 问题 | 正确做法 |
|------|------|----------|
| 脸谱化 | 过于扁平 | 加一个反差点 |
| 过于复杂 | 记不住 | 简化到3个标签 |
| 缺乏辨识度 | 分不清谁是谁 | 强化视觉区分 |
| 没有欲望 | 不知道为啥行动 | 明确动机 |
| 完美无缺 | 不真实 | 加入缺点 |
