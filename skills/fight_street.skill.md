# 👊 Skill: 街頭打鬥設計

> 來源：現代動作電影、MMA、街頭實戰

## 街斗美學

### 核心特徵
1. **原始**：沒有套路，本能反應
2. **混亂**：環境互動、借物攻擊
3. **真實**：會痛、會累、會受傷

## 街斗動作類型

### 攻擊動作
| 動作 | 特徵 | 傷害度 |
|-----|------|--------|
| 直拳 | 最基本、快速直接 | 中 |
| 擺拳 | 弧線攻擊、力量大 | 高 |
| 頭槌 | 近距離、突然 | 高 |
| 膝擊 | 纏抱時使用 | 高 |
| 踹踢 | 保持距離 | 中 |

### 防禦動作
| 動作 | 適用場景 |
|-----|---------|
| 格擋 | 預判到攻擊 |
| 閃避 | 反應快時 |
| 護頭 | 被動挨打 |
| 後退 | 拉開距離 |

### 地面動作
| 動作 | 特徵 |
|-----|------|
| 騎乘位砸拳 | 壓制後連續攻擊 |
| 絞殺 | 從背後鎖喉 |
| 掙脫 | 翻滾、頂撞 |

## 環境互動

### 可利用物品
| 物品 | 使用方式 | 視覺效果 |
|-----|---------|---------|
| 椅子 | 砸、擋 | 木屑飛濺 |
| 瓶子 | 砸頭、扎刺 | 玻璃碎裂 |
| 桌子 | 推倒、掀翻 | 物品散落 |
| 牆壁 | 撞擊、借力 | 灰塵/裂痕 |
| 地面 | 摔倒、拖行 | 摩擦痕跡 |

## 打鬥狀態變化

### 體力曲線
```
開始：100% → 衝動、力量大
中期：60% → 動作變慢、喘氣
後期：30% → 勉強支撐、失誤多
極限：10% → 本能反應、孤注一擲
```

### 傷害累積
- 鼻血、嘴角出血
- 眼眶腫脹
- 衣服撕裂、髒污
- 動作變形、護住傷處
