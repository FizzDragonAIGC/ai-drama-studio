# 用光繪畫 Skill

> 基於《Painting with Light》- John Alton
> 黑色電影攝影聖經，奧斯卡最佳攝影得主

---

## 🎯 核心哲學

**"光是攝影師的畫筆，陰影是他的畫布"**

Alton認為：
- 光不是用來"照亮"的，而是用來"雕刻"的
- 陰影比光更重要 — 沒有陰影就沒有立體感
- 少即是多 — 一束精準的光勝過滿屋燈火

## 🖤 黑色電影光影法則

### 法則1：Mystery Lighting（神秘光）
**原則**：讓觀眾只看到你想讓他們看到的

```
單一強光源從側面或後方照射，
主體大部分沉入陰影，
僅局部高光揭示關鍵信息，
觀眾被迫想像暗部內容
```

**應用**：懸疑、驚悚、黑幫片

### 法則2：Mood Lighting（情緒光）
**原則**：光影直接外化角色心理

| 心理狀態 | 光影處理 |
|---------|---------|
| 內心掙扎 | 面部半明半暗（雙面人） |
| 危機壓迫 | 低角度光，投射巨大陰影 |
| 希望覺醒 | 頂光或逆光突然出現 |
| 絕望深淵 | 幾乎全黑，僅眼睛微光 |

### 法則3：Follow Focus Light（聚焦光）
**原則**：光線引導觀眾視線

```
畫面中只有一個最亮點，
這個亮點就是視覺焦點，
其餘區域漸次變暗，
觀眾眼睛無法抗拒亮點吸引
```

## 💡 Alton六種基礎光位

### 1. Key Light（主光）
- **位置**：主體側前方45°
- **作用**：定義主要形態
- **Alton心得**：主光決定整場戲的情緒基調

### 2. Fill Light（補光）
- **位置**：與主光相對
- **作用**：控制陰影深度
- **Alton心得**：補光弱=戲劇性強，補光強=商業感

### 3. Back Light（逆光/輪廓光）
- **位置**：主體正後方
- **作用**：分離主體與背景
- **Alton心得**：逆光是最具電影感的光

### 4. Kicker（側逆光）
- **位置**：主體側後方
- **作用**：勾勒輪廓，增加層次
- **Alton心得**：增加神秘感的秘密武器

### 5. Eye Light（眼神光）
- **位置**：攝影機附近
- **作用**：點亮眼睛
- **Alton心得**：沒有眼神光的角色=沒有靈魂

### 6. Practical（場景光源）
- **來源**：畫面內可見光源
- **作用**：增加真實感
- **Alton心得**：觀眾相信畫面內的光源

## 🎬 Alton經典光影配方

### 配方A：審訊室
```
單一頂燈（hard top light）直射嫌疑人，
面部形成深邃眼窩陰影，
桌面反射光微弱補光，
背景全黑，審訊者隱於暗處，
只聞其聲不見其人
```
**情緒**：壓迫、權力不對等

### 配方B：黑幫老大
```
低位主光從下方45°打亮半張臉，
另半張臉完全隱入黑暗，
雪茄煙霧中可見丁達爾光柱，
背景威尼斯百葉窗投射條紋光影
```
**情緒**：危險、雙面性

### 配方C：蛇蠍美人
```
柔和主光從上方灑落，
強烈逆光勾勒髮絲輪廓，
眼神光特別加強，
嘴唇高光點綴，
背景深邃神秘
```
**情緒**：誘惑、致命吸引

### 配方D：孤獨偵探
```
窗外霓虹燈（紅/藍）閃爍透入，
室內幾乎無光，
偵探面部被霓虹燈間歇照亮，
雨滴在窗玻璃上形成光影圖案投射
```
**情緒**：孤獨、都市疏離

## 🖼️ Alton構圖光影原則

### 三分法則光影版
- 亮區佔1/3，暗區佔2/3
- 或相反，取決於情緒
- 絕對平均=無聊

### 對角線光影
```
光從畫面一角照入，
向對角方向衰減，
形成自然的視覺引導線
```

### 框中框
```
用陰影區域"框住"亮區，
創造自然的畫中畫效果，
增加空間層次感
```
