## 📐 情感符号基础

### 什么是情感符号（漫符）
```
漫画特有的视觉符号，用于表达情绪状态
无需文字说明，读者一看就懂
```

## 🎯 常用情感符号速查

### 愤怒系

| 符号 | 图示 | 含义 | 位置 |
|------|------|------|------|
| **青筋** | 井字形#或Y形 | 生气/愤怒 | 额头/太阳穴 |
| **怒火** | 🔥 | 暴怒 | 头顶/背后 |
| **影线** | 阴影覆盖上脸 | 黑化/杀意 | 眼睛以上 |
| **锐利眼** | ⚡尖锐眼神 | 怒视 | 眼睛 |

```
愤怒表现：
    # ← 青筋
   ／￣￣＼
  │ •̀ω•́ │ ← 锐利眼神
   ＼＿＿／
```

### 惊讶系

| 符号 | 图示 | 含义 | 位置 |
|------|------|------|------|
| **惊叹号** | ！ | 惊讶 | 头顶 |
| **问号** | ？ | 困惑 | 头顶 |
| **汗滴** | 💧 | 紧张/尴尬 | 脸侧/头顶 |
| **黑线** | ─── | 无语/崩溃 | 额头 |
| **魂飞** | 幽灵飘出 | 灵魂出窍 | 嘴边 |

```
震惊表现：
      ！！
   ／￣￣＼
  │ ⊙△⊙ │ ← 眼睛变形
   ＼＿＿／
     💧
```

### 喜悦系

| 符号 | 图示 | 含义 | 位置 |
|------|------|------|------|
| **闪光** | ✦✧ | 眼睛发光 | 眼中 |
| **花朵** | 🌸 | 开心/少女心 | 周围 |
| **音符** | ♪♫ | 愉快 | 头边 |
| **心形** | ♡♥ | 心动/爱意 | 眼中/周围 |
| **脸红线** | 斜线红晕 | 害羞/开心 | 脸颊 |

```
开心表现：
    ♪♫
   ／￣￣＼
  │ ✦ω✦ │ ← 闪亮眼睛
   ＼＿＿／
   //  \\ ← 脸红
```

### 悲伤系

| 符号 | 图示 | 含义 | 位置 |
|------|------|------|------|
| **眼泪** | 💧瀑布/小滴 | 哭泣 | 眼角 |
| **阴影** | 暗色调 | 沮丧 | 脸部 |
| **低落云** | ☁️灰云 | 郁闷 | 头顶 |
| **点点眼** | ·· | 失神 | 眼睛 |

```
悲伤表现：
    ☁️
   ／￣￣＼
  │ ；ω； │ ← 眼泪
   ＼＿＿／
      orz
```

## 🔧 符号组合技巧

### 复合情感

| 组合 | 含义 |
|------|------|
| **汗+青筋** | 强忍愤怒 |
| **心+脸红** | 恋爱中 |
| **问号+汗** | 困惑尴尬 |
| **黑线+汗** | 无语加尴尬 |
| **闪光+脸红** | 被击中心灵 |

### 强度递进

```
愤怒程度：
轻微：小青筋
中度：大青筋 + 锐利眼
重度：多青筋 + 怒火 + 影线
暴怒：全脸阴影 + 火焰背景

喜悦程度：
轻微：微笑 + 小音符
中度：闪光眼 + 花朵
重度：满脸花 + 大心形
狂喜：全身发光 + 背景特效
```

## 🎬 符号位置规范

### 头部周围
```
        ！？♪ ← 状态符号
         │
    ＃ ─ 头 ─ 💧 ← 情绪符号
         │
        汗
```

### 标准位置

| 符号 | 位置 |
|------|------|
| **青筋** | 额头/太阳穴 |
| **汗滴** | 脸侧/后脑 |
| **惊叹** | 正上方 |
| **心形** | 眼中/头侧 |
| **花朵** | 周围漂浮 |
| **黑气** | 身后/周围 |

## 📐 绘制要点

### 青筋绘制
```
   ╱╲
  ╱──╲   或   ─┼─
  ╲  ╱       ─┼─
   ╲╱
 井字形      十字形
```

### 脸红绘制
```
  ////  ← 平行斜线
  ////
```
位置：颧骨处，双侧对称

### 汗滴绘制
```
   ╭─╮
   │ │
   ╰─╯
    ╲
     ▼
```
大汗：大滴 + 轨迹
小汗：简单滴状

## 🎯 AI绘制Prompt模板

### 情感符号
```
manga emotion symbol,
[符号类型: anger vein/sweat drop/sparkle eyes/blush lines],
[风格: shojo/shonen/chibi],
manga expression marker,
Japanese manga style
```

### 角色表情+符号
```
manga character expression,
[情感: angry/surprised/happy/sad],
with [符号描述],
[符号位置],
[风格] manga style
```

### 示例
```
manga character, angry expression,
anger vein on forehead,
sharp eyes, clenched teeth,
dark shadow over face,
shonen manga style,
black and white
```
