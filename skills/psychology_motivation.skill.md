# 🧠 Skill: 角色動機設計

> 來源：編劇心理學、角色弧光理論

## 動機金字塔

### 1. 表層動機 (Want)
- 角色口頭說想要的東西
- 觀眾最先看到的目標
- 例：小猴子想偷寶石

### 2. 深層動機 (Need)
- 角色真正需要但不自知的東西
- 通常與成長/轉變相關
- 例：小猴子需要歸屬感和認同

### 3. 根源動機 (Wound)
- 過去的創傷導致的行為模式
- 角色弧光的起點
- 例：小猴子被父母拋棄，不敢信任他人

## 動機設計公式

```
因為 [過去創傷/經歷]
所以角色 [形成某種信念/行為模式]
在故事中追求 [表層目標]
實際上需要 [深層需求]
最終通過 [關鍵事件] 獲得 [成長/轉變]
```

## 常見動機類型

### 生存類
- 逃離危險
- 保護家人
- 獲取資源

### 歸屬類
- 尋找家庭
- 獲得認同
- 建立關係

### 尊嚴類
- 證明自我
- 復仇雪恥
- 贏得尊重

### 自我實現類
- 追尋真相
- 完成使命
- 創造價值

### 超越類
- 犧牲奉獻
- 拯救他人
- 留下傳承

## 動機衝突設計

### 內在衝突
```
Want vs Need：想要的和需要的相矛盾
例：想獨自逃走 vs 需要學會信任同伴
```

### 外在衝突
```
目標 vs 障礙：追求目標時遇到的阻礙
例：想保護寶石 vs 海盜追殺
```

### 人際衝突
```
我的動機 vs 你的動機：角色之間的目標衝突
例：小猴子想賣寶石 vs 小菩薩蠻要保護寶石
```

## AI Prompt 應用

### 動機標註模板
```
[角色名] in this scene:
WANT: [當前想要達成的具體目標]
NEED: [潛在的深層需求]
OBSTACLE: [阻礙目標的因素]
TACTIC: [採取的策略/方法]
EMOTION: [由此產生的情緒]
```

### 示例
```
小猴子 in EP01_SC03:
WANT: 偷走粉紅寶石賣錢
NEED: 證明自己有價值（被認可）
OBSTACLE: 占卜師警覺，場面詭異
TACTIC: 等待時機，觀察弱點
EMOTION: 緊張中帶著興奮，一絲不安
```
