## 元信息
- **技能ID**: script_montage
- **所屬Agent**: 劇本Agent
- **參考書籍**: 《電影語言》Marcel Martin
- **難度**: ⭐⭐⭐

## 核心理念

蒙太奇 = 多個短鏡頭組合 → 傳達整體概念
- 時間壓縮
- 並行展示
- 情感累積

## 蒙太奇類型

### 1. 時間蒙太奇（最常用）
```
功能：壓縮時間流逝
例：訓練蒙太奇、成長蒙太奇

【蒙太奇 - 小明的訓練】
A) 健身房 - 小明做俯臥撐，汗流浹背
B) 跑道 - 他跑步，氣喘吁吁
C) 健身房 - 數字顯示：Day 15
D) 跑道 - 他跑得更快了
E) 健身房 - Day 30 - 他輕鬆完成訓練

【蒙太奇結束】
```

### 2. 平行蒙太奇
```
功能：同時發生的不同事件
例：警察趕來 + 犯人逃跑

【平行蒙太奇】
A) 警車 - 警笛響起
B) 銀行內 - 劫匪收拾錢
C) 警車 - 越來越近
D) 銀行內 - 劫匪戴上面罩
E) 警車 - 急轉彎
F) 銀行 - 劫匪衝出後門
【蒙太奇結束】
```

### 3. 對比蒙太奇
```
功能：對比兩種狀態/人物
例：富人vs窮人、過去vs現在

【對比蒙太奇】
A) 豪宅 - 富人吃大餐
B) 街頭 - 流浪漢翻垃圾桶
A) 豪宅 - 倒掉剩飯
B) 街頭 - 找到半個麵包
【蒙太奇結束】
```

### 4. 情感蒙太奇
```
功能：傳達情緒（通常配音樂）
例：愛情蒙太奇、回憶蒙太奇

【回憶蒙太奇 - 快閃】
- 兩人初次相遇的微笑
- 一起走在海邊
- 生日蛋糕上的燭光
- 吵架摔門
- 雨中擁抱
【蒙太奇結束】
```

## 蒙太奇格式（劇本）

### 標準格式
```
蒙太奇 - 描述

A) 場景1 - 動作
B) 場景2 - 動作
C) 場景3 - 動作

蒙太奇結束
```

### 中文格式
```
【蒙太奇 - 主題】

場景A：地點 - 動作描述
場景B：地點 - 動作描述
場景C：地點 - 動作描述

【蒙太奇結束】
```

## 蒙太奇設計原則

### 1. 數量控制
```
4-8個片段為佳
太少：不像蒙太奇
太多：拖沓
```

### 2. 遞進
```
有變化趨勢：
- 時間遞進
- 能力提升
- 情緒加深
- 關係變化
```

### 3. 節奏
```
通常配合音樂
每個片段1-3秒
整體30秒-2分鐘
```

### 4. 結束方式
```
自然過渡到下一場景
或用「某天」等標記結束
```
