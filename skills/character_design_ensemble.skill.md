## 📐 群戏设计核心原则

群戏角色需要**既统一又区分**：

| 原则 | 说明 | 设计方向 |
|------|------|----------|
| 统一性 | 属于同一团队/世界 | 共同的设计元素 |
| 差异性 | 每个角色可辨识 | 独特的剪影和配色 |
| 平衡性 | 视觉上形成和谐 | 大小/颜色/形状互补 |
| 功能性 | 角色定位清晰 | 外形暗示能力 |

## 🎭 团队角色分配

### 经典五人组合
| 位置 | 性格 | 设计方向 | AI Prompt |
|------|------|----------|-----------|
| 领袖 | 勇敢、担当 | 中等偏大、暖色 | team leader, heroic design |
| 智囊 | 聪明、冷静 | 偏瘦、冷色 | team strategist, intelligent |
| 力量 | 强壮、直率 | 最大、稳重色 | team muscle, big strong |

| 心灵 | 温暖、治愈 | 圆润、柔和色 | team heart, warm caring |

### 三人组合
| 位置 | 特点 | AI Prompt |
|------|------|-----------|
| 主角 | 平衡型、中心 | main protagonist, balanced |
| 配角A | 与主角对比（如高/冷静） | contrasting partner A |
| 配角B | 与主角互补（如活泼） | complementary partner B |

### 双人组合

## 🎨 统一性设计

### 建立团队视觉语言
| 元素 | 统一方法 | AI Prompt |
|------|----------|-----------|
| 制服 | 相同款式不同颜色 | team uniform, matching outfits |
| 标志 | 共同的队徽/符号 | team emblem, shared symbol |
| 配色 | 共享调色板 | unified color palette |
| 配饰 | 相同的某个配件 | matching accessories |
| 风格 | 同一设计风格 | consistent design style |

### 世界观统一
| 元素 | 统一方法 |
|------|----------|
| 时代感 | 同一时代的服装风格 |
| 技术水平 | 同一科技/魔法水平 |
| 文化背景 | 同一文化的设计元素 |

## 🔄 差异性设计

### 剪影差异化
| 方法 | 说明 | AI Prompt |
|------|------|-----------|
| 身高差 | 5级身高区分 | varying heights, distinct silhouettes |
| 体型差 | 胖瘦壮不同 | different body types |
| 发型差 | 长短卷直各异 | varied hairstyles |
| 姿态差 | 站姿习惯不同 | different postures |

### 配色差异化
| 方法 | 说明 |
|------|------|
| 主色不同 | 每人一个主色 |
| 明度变化 | 同色系不同明度 |
| 冷暖分配 | 冷色暖色分配 |

### 性格差异化
| 性格 | 外化方式 |
|------|----------|
| 外向 | 亮色、开放姿态 |
| 内向 | 暗色、收缩姿态 |
| 乐观 | 上扬线条 |
| 悲观 | 下垂线条 |
