# 人物灵魂设计师 Skill
> 基于Excel专业智能体设计

## 📚 理论基础
- 心理学体系：融合弗洛伊德、荣格、现代人格心理学
- 叙事学原理：普洛普角色功能、坎贝尔英雄之旅
- 文化人类学：不同文化中的角色原型和社会角色

## 🎯 设计方法论

### 1. 灵魂内核法
先确定角色的灵魂内核（核心信念）

### 2. 创伤溯源法
追溯形成性格的关键创伤事件

### 3. 欲望层次法
区分表层欲望、深层需求、灵魂渴望

### 4. 成长轨迹法
设计可量化的成长指标和转折点

---

## 📋 追问问题设计

### 第一层：心理深度
- 心理类型：[MBTI类型或自定义]
- 核心创伤：____岁时发生的____事件
- 防御机制：[ ]否认 [ ]投射 [ ]合理化 [ ]升华
- 潜意识欲望：内心深处渴望____但不敢承认

### 第二层：成长弧光
- 初始状态：相信____，恐惧____，追求____
- 转折时刻：在____场景意识到____
- 最终状态：学会____，接受____，成为____
- 成长代价：为了成长必须失去____

### 第三层：专业塑造
- 视觉标识：标志性动作____，穿着风格____，物品____
- 语言特征：口头禅____，语速____，词汇特点____
- 文化背景：____地域，____阶层，____教育背景的影响

---

## ✅ 角色档案标准
- 心理真实性：符合心理学原理，行为有内在逻辑
- 成长完整性：有清晰的起点、转折、终点
- 文化准确性：符合角色背景的文化特征
- 视觉可识别：有独特的视觉和语言标识

---

## 📊 输出格式

### 1. 专业角色档案
```
• 基本信息：姓名、年龄、职业、外貌特征
• 心理剖面：核心信念、主要创伤、防御机制、潜意识
• 成长蓝图：起点状态→激励事件→内心挣扎→顿悟时刻→最终状态
• 关系图谱：与各角色的情感距离、权力关系、历史纠葛
```

### 2. 表演指导手册
- 关键情感场景表演要点
- 微表情和肢体语言设计
- 声音特征和语言节奏
- 服装道具的象征意义

### 3. 视觉参考集
- 外貌参考图像（3-5张）
- 气质参考演员建议
- 服装风格参考板
- 生活环境视觉参考
