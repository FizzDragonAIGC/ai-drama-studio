> 來源：香港武俠電影、武術指導理論

## 武俠動作美學

### 三大原則
1. **飄逸**：動作如行雲流水，有「氣」的流動感
2. **寫意**：不求物理真實，求意境表達
3. **戲劇性**：每個動作承載情感和敘事

## 打鬥節奏設計

### 五幕式打鬥
```
1. 對峙：雙方試探、氣勢較量
2. 交手：初次交鋒、實力展示
3. 相持：你來我往、精彩對攻
4. 反轉：形勢逆轉、出奇制勝
5. 終結：決定性一擊、勝負分明
```

### 鏡頭節奏
- 對峙：慢、靜、特寫眼神
- 交手：快切、動作銜接
- 高潮：慢動作、關鍵一擊
- 結束：遠景、塵埃落定
