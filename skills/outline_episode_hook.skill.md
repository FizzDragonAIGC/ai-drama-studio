# 📝 分集鉤子設計技能

## 元信息
- **技能ID**: outline_episode_hook
- **所屬Agent**: 大綱Agent
- **參考書籍**: 《救貓咪》Blake Snyder
- **難度**: ⭐⭐⭐

## 核心理念

鉤子 = 讓觀眾必須看下一集的理由
- 每集結尾必須有鉤子
- 每集開頭必須回應上一集的鉤子

## 鉤子類型

### 1. 危機鉤子
```
「主角會死嗎？」
「他們能逃出來嗎？」
「炸彈會爆炸嗎？」

設計：
- 生命危險
- 倒計時
- 懸崖式結尾
```

### 2. 謎題鉤子
```
「這是什麼意思？」
「那個人是誰？」
「為什麼他要這樣做？」

設計：
- 揭露部分信息
- 留下更大的問號
- 暗示反轉
```

### 3. 情感鉤子
```
「他們會在一起嗎？」
「她會原諒他嗎？」
「他會說出真相嗎？」

設計：
- 關係到達臨界點
- 情感決定懸而未決
- 重聚/分離前一刻
```

### 4. 反轉鉤子
```
「原來他是壞人！」
「她還活著！」
「一切都是騙局！」

設計：
- 揭露驚人真相
- 立即切斷（不解釋）
- 留給下集消化
```

### 5. 決定鉤子
```
「他會選擇A還是B？」
「她會接受嗎？」
「他們要去哪裡？」

設計：
- 重大選擇點
- 兩難困境
- 按下「暫停」
```

## 鉤子強度分級

| 等級 | 強度 | 適用 |
|------|------|------|
| S級 | 「必須馬上看」 | 季中/季終 |
| A級 | 「很想看下一集」 | 重要集結尾 |
| B級 | 「好奇接下來」 | 普通集結尾 |
| C級 | 「還行，之後會看」 | 避免使用 |

## 開場回應模板

### 危機→解決
```
上集結尾：槍指著主角
本集開場：鏡頭拉遠，原來是夢/訓練/回憶
```

### 謎題→部分解答
```
上集結尾：神秘人出現
本集開場：揭露身份，但提出新謎題
```

### 情感→延續
```
上集結尾：「我有話要說」
本集開場：被打斷/說出來但引發新問題
```

## 鉤子設計檢查清單

```
□ 這個結尾會讓人立即點擊下一集嗎？
□ 觀眾會想知道「然後呢」嗎？
□ 有沒有核心問題懸而未決？
□ 是否在最高點切斷？（不要多說）
□ 下一集有足夠的payoff嗎？
```

## 輸出格式

```json
{
  "episode_hooks": [
    {
      "episode": 1,
      "hook_type": "謎題鉤子",
      "hook_content": "神秘來電：「你父親還活著」",
      "strength": "A級",
      "next_episode_payoff": "第2集揭露來電者身份"
    }
  ]
}
```
