# 🎬 諾蘭風格技能

## 元信息
- **技能ID**: director_nolan

- **大師**: Christopher Nolan
- **代表作**: 《盜夢空間》《星際穿越》《敦刻爾克》

## 標誌性技法

### 1. IMAX大畫幅
```
特徵：史詩級寬幅畫面
效果：壯闘、沉浸、震撼
應用：大場面、空間展示

Prompt關鍵詞：
IMAX format, ultra-wide shot,
epic scale, massive landscape,
cinematic wide angle
```

### 2. 實景特效
```
特徵：真實感強的視覺效果
效果：可信度高、沉浸
應用：科幻場景、動作

Prompt關鍵詞：
practical effects aesthetic,
realistic CGI, photorealistic,
grounded sci-fi
```

### 3. 時間扭曲視覺
```
特徵：時間流速不同的視覺化
效果：燒腦、層次感
應用：多時間線、夢境

Prompt關鍵詞：
time dilation visual, nested reality,
multiple timelines, dream layers
```

### 4. 冷硬色調
```
特徵：去飽和、金屬質感
效果：嚴肅、科技感
應用：科幻、驚悚

Prompt關鍵詞：
desaturated colors, cold tone,
steel blue, industrial look
```
