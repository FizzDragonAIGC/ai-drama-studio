# 🎤 願景引導技能

## 元信息
- **技能ID**: interview_vision_guide
- **所屬Agent**: 訪談Agent
- **參考書籍**: 《创意，是一笔灵魂交易》Julia Cameron
- **難度**: ⭐⭐⭐⭐

## 核心理念

願景引導 = 幫創作者看見他們還沒看見的東西
- 不是「你告訴我你想要什麼」
- 而是「讓我們一起發現可能性」

## 願景引導三階段

### 階段1: 打開想像空間

**魔法問題**
```
「如果這部作品大獲成功，三年後會發生什麼？」
「想像首映禮那天，觀眾走出影院時在討論什麼？」
「如果有人因為這部作品改變了人生，會是什麼改變？」
```

**移除限制**
```
「假設技術、預算、時間都不是問題...」
「如果你可以請任何導演來拍...」
「如果這個故事發生在任何時代/地點...」
```

### 階段2: 具象化願景

**感官引導**
```
「閉上眼睛，想像最重要的那個場景...
 - 你看到什麼顏色？
 - 聽到什麼聲音？
 - 空氣是什麼味道？
 - 溫度是冷還是熱？」
```

**角色附身**
```
「如果你是主角，此刻站在那裡，你在想什麼？」
「你最害怕什麼？最渴望什麼？」
「你會怎麼對那個人說？」
```

### 階段3: 錨定核心

**電梯演講測試**
```
「用一句話告訴我這是什麼故事」
「如果海報上只能有一行字...」
「觀眾離開時只記住一件事，是什麼？」
```

**對比錨定**
```
「這部作品是 [A] 遇上 [B]」
例：《星際穿越》= 《2001太空漫遊》遇上《父親的情書》
```

## 視覺化工具

### 情緒板引導
```
「從這20張圖片中，選出最接近你想要的感覺的5張」
「為什麼選這張？哪個元素吸引你？」
「這張絕對不要，為什麼？」
```

### 色彩心理引導
```
「如果這部作品是一種顏色，是什麼？」
「故事開始時是什麼顏色？結束時呢？」
「主角的顏色？反派的顏色？」
```

### 音樂引導
```
「如果要選一首歌作為主題曲...」
「這首歌的哪個部分最像你要的感覺？」
「節奏是快還是慢？為什麼？」
```

## 願景陳述模板

```
【願景陳述】

這是一個關於 [主題] 的故事，
講述 [主角] 如何 [核心行動]，
最終 [結局/轉變]。

觀眾看完會感到 [核心情緒]，
並思考 [核心問題]。

視覺風格像 [參考作品]，
但更 [差異化元素]。

如果成功，它會成為 [品類] 的標杆。
```

## 引導話術

| 情況 | 引導語 |
|------|--------|
| 用戶迷茫 | 「沒關係，我們從你確定的開始」 |
| 用戶糾結 | 「兩個方向都有價值，哪個讓你更興奮？」 |
| 用戶保守 | 「如果大膽一點呢？最瘋狂的版本是什麼？」 |
| 用戶發散 | 「很棒的想法，但如果只能選一個...」 |
| 用戶模仿 | 「如果要超越原作，你會怎麼做？」 |

```json
    "success_metric": "成功標準",

  }
}
```

## 願景檢驗清單

- [ ] 能用一句話說清楚？
- [ ] 有明確的情感目標？
- [ ] 有視覺化的錨點？
- [ ] 與現有作品有區隔？
- [ ] 創作者自己興奮嗎？
