## 🎨 画面构图美学

### 1. 对角线构图（Diagonal Composition）
**效果**：动感、张力、不稳定

```
对角线构图，[主体]沿画面左下至右上对角线分布，
打破水平稳定感，营造动态张力，
背景元素呼应对角线走向
```

### 2. 倾斜构图/荷兰角（Dutch Angle）
**效果**：失衡、不安、心理扭曲

```
荷兰角倾斜构图，画面整体向[左/右]倾斜15-30度，
地平线不再水平，建筑物仿佛即将倒塌，
视觉失衡映射角色内心的混乱与不安
```

### 3. 框中框构图（Frame within Frame）
**效果**：聚焦、困境、窥视

```
框中框构图，通过[门框/窗户/拱门]框住[主体]，
自然形成视觉焦点，同时暗示被困或被监视，
前景框架保持暗色剪影效果
```

### 4. 黄金分割/三分法
**效果**：和谐、专业、平衡

```
三分法构图，[主体]置于画面右侧三分线交点，
视线方向留有二分之一的呼吸空间，
地平线位于上/下三分线位置
```

## 🌈 色彩美学体系

### 电影级调色风格

#### 1. 青橙调（Teal & Orange）
**最常用的好莱坞调色**
```
青橙互补色调，皮肤保持温暖橙色调，
阴影和背景偏向冷青色，
形成强烈但和谐的色彩对比，
增强画面层次感和电影质感
```

#### 2. 低饱和度处理（Desaturated）
**文艺、压抑、纪实感**
```
低饱和度处理，整体色彩饱和度降低30-50%，
保留微弱的色彩倾向（偏青/偏黄），
灰调质感营造纪实或压抑氛围
```

#### 3. 墨蓝+青灰冷色调
**悬疑、科幻、夜景**
```
墨蓝+青灰冷色调基底，
整体画面笼罩在深沉的蓝色氛围中，
仅有少量暖色点缀（灯光/火焰），
营造深夜、科技、疏离的冷峻感
```

#### 4. 暖金色调
**怀旧、温暖、黄金时刻**
```
暖金色调，仿佛黄金时刻的温暖阳光，
整体画面笼罩在蜂蜜色的光晕中，
高光区域泛着金色，阴影保持温暖棕色，
营造怀旧、温馨、浪漫的氛围
```

## 📐 画面层次设计

### 三层空间法
```
前景：[遮挡物/虚化元素] - 增加景深和包围感
中景：[主体/焦点] - 清晰对焦，视觉中心
背景：[环境/氛围] - 交代场景，营造氛围
```

### 示例
```
前景：失焦的铁栏杆剪影，营造监禁感
中景：安迪坐在牢房床上，伦勃朗光照亮半边脸
背景：高窗透入丁达尔光束，象征希望
三层空间共同讲述"困境中的希望"主题
```
