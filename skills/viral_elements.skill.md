## 📐 传播六要素 (STEPPS)

### Jonah Berger的STEPPS模型
| 要素 | 英文 | 含义 | 应用 |
|------|------|------|------|
| 社交货币 | Social Currency | 分享让我看起来很酷 | 独特、内行、身份认同 |
| 触发器 | Triggers | 让人想起你 | 关联日常、热点 |
| 情绪 | Emotion | 引发强烈情绪 | 敬畏、愤怒、焦虑、快乐 |
| 公开性 | Public | 容易被看到 | 可视化、可模仿 |
| 实用价值 | Practical Value | 对别人有用 | 干货、教程 |
| 故事 | Stories | 好故事自带传播 | 把信息包装成故事 |

## 🎯 番剧传播点设计

### 1. 名场面设计
```
标准：
- 视觉冲击力极强
- 脱离上下文也能理解
- 容易截图/做表情包
- 情绪极端（超燃/超甜/超虐/超搞笑）

检查：
□ 这个场面能做封面吗？
□ 能做GIF吗？
□ 能做表情包吗？
□ 会被大量转发吗？
```

### 2. 金句设计
```
标准：
- 简短好记
- 有哲理或有趣
- 脱离剧情也成立
- 可以当签名/状态

例子：
"在这个世界上，唯一能杀死我的，是无聊。"
"大人只看利弊，小孩才分对错。"
```

### 3. 梗设计
```
标准：
- 有反差/意外
- 可以玩
- 可以二创
- 有重复使用价值

类型：
- 口头禅梗
- 动作梗
- 表情梗
- 反差梗
- 谐音梗
```

### 4. CP设计
```
让观众磕CP：
- 有张力的互动
- 暧昧但不明确
- 给想象空间
- 二创素材多

检查：
□ 观众会想让他们在一起吗？
□ 会有人画同人吗？
□ 会有人写同人文吗？
```

## 📊 可分享内容设计

### 截图/壁纸
```
设计要点：
- 构图完整美观
- 色彩冲击力
- 角色帅/美
- 适合做壁纸

每集至少设计：
- 1张竖屏壁纸场景
- 1张横屏壁纸场景
```

### 表情包
```
设计要点：
- 表情夸张
- 情绪明确
- 可以加字
- 日常可用

角色设计时考虑：
- 设计几个标志性表情
- Q版形态更易传播
```

### GIF动图
```
设计要点：
- 2-5秒循环
- 动作有特点
- 不需要上下文
- 情绪感染力强
```

## 🎭 情绪传播性

### 高传播情绪（正面）
| 情绪 | 传播力 | 如何引发 |
|------|--------|----------|
| 敬畏 | ★★★★★ | 宏大场面、超越想象 |
| 惊喜 | ★★★★★ | 意外反转、出乎意料 |
| 快乐 | ★★★★ | 搞笑、可爱、温馨 |
| 感动 | ★★★★ | 真情、牺牲、治愈 |

### 高传播情绪（负面）
| 情绪 | 传播力 | 如何引发 |
|------|--------|----------|
| 愤怒 | ★★★★★ | 不公、恶人当道 |
| 焦虑 | ★★★★ | 悬念、威胁 |
| 悲伤 | ★★★ | 分离、死亡 |

### 低传播情绪
| 情绪 | 传播力 | 原因 |
|------|--------|------|
| 满足 | ★★ | 没有分享冲动 |
| 平静 | ★ | 不想打扰别人 |
| 无聊 | ☆ | 不会分享 |

## 📋 传播检查清单

### 每集必检
- [ ] 有没有名场面？
- [ ] 有没有金句？
- [ ] 有没有可以做表情包的画面？
- [ ] 有没有能引发强烈情绪的时刻？
- [ ] 结尾有没有让人想讨论的点？

### 每个角色必检
- [ ] 有没有标志性特征？
- [ ] 有没有标志性口头禅？
- [ ] 有没有CP可能？
- [ ] 会不会有人想Cos？
- [ ] 会不会有人画同人？

### 整体必检
- [ ] 用一句话能安利吗？
- [ ] 有没有独特卖点？
- [ ] 和其他作品有什么不同？
- [ ] 标签/话题是什么？

## 🔧 传播性评分

### 评分表
| 维度 | 问题 | 分数 |
|------|------|------|
| 名场面 | 每集有几个？ | __/10 |
| 金句 | 有几句能记住？ | __/10 |
| 梗 | 有没有能玩的梗？ | __/10 |
| 情绪 | 情绪强度够不够？ | __/10 |
| 讨论度 | 有争议/讨论点吗？ | __/10 |
| 独特性 | 有什么独一无二的？ | __/10 |
| **总分** | | __/60 |

### 传播力判断
- 50-60：爆款潜力
- 40-49：有传播力
- 30-39：一般
- <30：需要增加传播点
