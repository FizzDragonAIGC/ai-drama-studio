## 元信息
- **技能ID**: adaptation_visual
- **所屬Agent**: 改編Agent

- **難度**: ⭐⭐⭐⭐⭐

## 核心理念

視覺化翻譯 = 文字→畫面的轉譯
- 小說用文字描述世界
- 影視用畫面「顯示」世界
- 關鍵：Show, Don't Tell

| 情緒 | 色調 | 構圖 | 運鏡 |
|------|------|------|------|
| 快樂 | 暖色/高飽和 | 開闘/對稱 | 流暢/上升 |
| 悲傷 | 冷色/低飽和 | 留白/不對稱 | 緩慢/下降 |

## 內心戲視覺化技巧

### 1. 行動外化
```
原著：「他內心掙扎著」

視覺化選項：
- 手捏緊又鬆開
- 在房間裡踱步
- 點燃又熄滅的香菸
- 反覆撥打又掛斷電話
```

### 2. 環境投射
```
原著：「她感到絕望」

視覺化選項：
- 窗外下著大雨
- 路燈忽明忽暗
- 枯萎的花/凋零的樹葉
- 空蕩蕩的街道
```

### 3. 視覺象徵
```
原著：「他們的關係破裂了」

視覺化選項：
- 摔碎的杯子/相框
- 斷裂的項鍊/戒指
- 兩人走向不同方向
- 合照被撕裂/燒毀
```

### 4. 閃回/閃前
```
原著：「她想起那個夏天」

視覺化選項：
- 1-2秒的閃回片段
- 疊化+色調變化
- 聲音先行（笑聲/對話）
- 道具觸發（同款物品）
```

## 描寫段落→分鏡轉化

### 示例轉化
```
原著：
「清晨的陽光穿過老舊的木窗灑進房間，照在女孩沉睡的臉上。
她翻了個身，眉頭微皺，似乎做了一個不太好的夢。
床頭的鬧鐘指向七點整，突然響了起來。
她猛地睜開眼睛，一時間不知自己身在何處。」

分鏡轉化：
1. 遠景/外景：清晨的房子，陽光照耀
2. 中景：窗戶，光線灑入
3. 特寫：女孩臉上的光斑
4. 近景：她翻身，眉頭微皺
5. 特寫：鬧鐘7:00
6. 聲音：鬧鐘響起
7. 特寫：眼睛猛然睜開
8. 主觀鏡頭：模糊的天花板
```

## 無法直接視覺化的處理

### 大段議論/哲學思考
```
選項A：刪除（觀眾自己體會）
選項B：轉化為對話
選項C：用旁白（慎用）
選項D：用視覺隱喻承載
```

### 嗅覺/味覺/觸覺
```
嗅覺：角色反應+環境暗示（花/食物/煙）
味覺：表情+動作（品嚐/吐出/微笑）
觸覺：手部特寫+材質呈現
```

### 長時間跨度
```
技巧：
- 蒙太奇（多個短鏡頭）
- 時間標題卡（三年後）
- 視覺對比（同場景不同季節）
- 角色外觀變化（髮型/衣著）
```
