# 📝 場景序列設計技能

## 元信息
- **技能ID**: outline_scene_sequence
- **所屬Agent**: 大綱Agent
- **參考書籍**: 《編劇的藝術》Lajos Egri
- **難度**: ⭐⭐⭐

## 核心理念

場景序列 = 有邏輯連接的場景組
- 單個場景是「磚」
- 場景序列是「牆」
- 整體結構是「房子」

## 場景序列類型

### 1. 因果序列
```
場景A → 導致 → 場景B → 導致 → 場景C

例：
發現線索 → 追蹤嫌疑人 → 對峙 → 逃脫

邏輯：嚴格的因果鏈
```

### 2. 平行序列
```
場景A1 ↔ 場景B1
場景A2 ↔ 場景B2

例：
主角準備 ↔ 反派準備
主角出發 ↔ 反派設伏

邏輯：對比/呼應
```

### 3. 蒙太奇序列
```
場景A + 場景B + 場景C = 整體效果

例：
訓練片段1 + 訓練片段2 + 訓練片段3 = 成長

邏輯：累積效果
```

### 4. 追逐序列
```
場景A（追）→ 場景B（逃）→ 場景C（接近）→...

例：
街道追逐 → 市場穿越 → 屋頂跳躍 → 巷子對峙

邏輯：空間+緊張度遞進
```

### 5. 揭示序列
```
謎題 → 線索1 → 線索2 → 推理 → 揭曉

例：
發現異常 → 調查現場 → 詢問證人 → 拼湊真相

邏輯：信息逐步揭露
```

## 序列設計模板

### 八序列法（全季）
```
序列1：開場+設定（集1-2）
序列2：新世界探索（集2-3）
序列3：第一次嘗試（集4-5）
序列4：反應+調整（集5-6）
序列5：中點+轉折（集6-7）
序列6：後果+困境（集8-9）
序列7：最終準備（集9-10）
序列8：高潮+結局（集11-12）
```

### 單集序列（4序列）
```
序列1：開場+問題（前6分鐘）
序列2：嘗試+阻礙（6-12分鐘）
序列3：危機+轉折（12-18分鐘）
序列4：解決+收尾（18-24分鐘）
```

## 場景順序原則

### 遞進原則
```
情緒遞進：平靜 → 緊張 → 爆發
信息遞進：不知 → 懷疑 → 確認
賭注遞進：小 → 中 → 大
```

### 對比原則
```
緊張後接輕鬆
悲傷後接希望
動作後接對話
```

### 回歸原則
```
離開日常 → 冒險 → 回到日常（已改變）
設置問題 → 展開 → 回答問題
```

## 場景銜接技術

### 硬切
```
場景A結束 → 直接切到場景B
效果：快節奏/對比
```

### 橋接
```
場景A → 過渡畫面/聲音 → 場景B
效果：流暢/時間流逝
```

### 鉤連
```
場景A結尾提到場景B的元素
→ 場景B開始時呈現
效果：邏輯連貫
```

### 聲音先行
```
場景A畫面還在
→ 場景B聲音先進入
→ 切到場景B
效果：流暢/預示
```
