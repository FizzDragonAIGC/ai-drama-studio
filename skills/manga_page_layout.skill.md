## 📐 基础页面结构

### 日式漫画标准页面

| 类型 | 格子数 | 适用场景 |
|------|--------|----------|
| **标准页** | 4-6格 | 日常叙事 |
| **密集页** | 7-10格 | 快速对话、紧张氛围 |
| **疏松页** | 2-3格 | 重要时刻、情感爆发 |
| **单格页** | 1格 | 冲击力、转折点 |

## 🎯 页面布局模式

### 1. 标准四格布局
```
┌─────────┬─────────┐
│    1    │    2    │
├─────────┼─────────┤
│    3    │    4    │
└─────────┴─────────┘
```
**用途**：稳定叙事、日常场景

### 2. 六格经典布局
```
┌─────────┬─────────┐
│    1    │    2    │
├────┬────┴────┬────┤
│ 3  │    4    │ 5  │
├────┴─────────┴────┤
│         6         │
└───────────────────┘
```
**用途**：对话推进、渐进情感

### 3. 对角线布局
```
┌───────────────────┐
│ ╲      1      ╲   │
│   ╲           ╲   │
├─────╲───────────╲─┤
│  2    ╲    3     ╲│
└─────────╲─────────┘
```
**用途**：动态感、动作场景

### 4. 中心扩散布局
```
┌──────┬──────┬──────┐
│  1   │      │  4   │
├──────┤  主  ├──────┤
│  2   │  格  │  5   │
├──────┤      ├──────┤
│  3   │      │  6   │
└──────┴──────┴──────┘
```
**用途**：突出中心、角色登场

## 📊 格子大小与情感

| 格子大小 | 阅读速度 | 情感效果 |
|----------|----------|----------|
| **超大格** | 很慢 | 震撼、重要时刻 |
| **大格** | 慢 | 关注、强调 |
| **中格** | 正常 | 叙事推进 |
| **小格** | 快 | 快速、紧张 |
| **微格** | 极快 | 瞬间、反应 |

## 🎬 页面节奏设计

### 节奏类型

| 节奏 | 格子特征 | 效果 |
|------|----------|------|
| **快节奏** | 小格密集、斜线分割 | 紧张、动作 |
| **慢节奏** | 大格疏散、规则分割 | 抒情、思考 |
| **爆发节奏** | 小→大突变 | 高潮、冲击 |
| **平稳节奏** | 大小均匀 | 日常叙事 |

### 翻页惊喜技巧
```
【右页（当前页）】        【左页（翻页后）】
┌─────────────────┐      ┌─────────────────┐
│   铺垫...       │      │                 │
│   铺垫...       │  →   │    ！冲击画面！  │
│   悬念？        │      │                 │
└─────────────────┘      └─────────────────┘
```
**要点**：在右页末尾设置悬念，翻页后揭示

## 🔧 视线引导

### Z字形阅读（日漫）
```
┌─────────────────┐
│ 1 ─────→ 2     │
│ ↓               │
│ 3 ─────→ 4     │
│ ↓               │
│ 5 ─────→ 6     │
└─────────────────┘
```
从右上到左下（日漫为从右到左）

### 视线锚点
- **人物视线**：引导读者看向目标
- **动作方向**：沿动作线移动视线
- **对话气泡**：按顺序引导阅读

## 📐 格子形状变化

| 形状 | 效果 |
|------|------|
| **矩形** | 稳定、正常叙事 |
| **斜格** | 动态、紧张 |
| **圆形/椭圆** | 回忆、梦境 |
| **不规则** | 混乱、崩溃 |
| **无边框** | 开放、自由 |
| **出血格** | 冲击、延伸 |
