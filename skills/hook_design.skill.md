## 📐 钩子类型

### 开篇钩子（前30秒）
| 类型 | 方法 | 示例 |
|------|------|------|
| 悬念钩 | 抛出谜题 | "那一天，我杀死了自己" |
| 冲突钩 | 直接冲突 | 开场就打架 |
| 奇观钩 | 震撼画面 | 史诗级场面开场 |
| 反差钩 | 出乎意料 | 看似平常突然反转 |
| 情绪钩 | 强烈情感 | 哭/笑开场 |
| 金句钩 | 记忆点 | 经典台词开场 |

### 持续钩子（让人看下去）
| 类型 | 方法 |
|------|------|
| 信息缺口 | 观众知道一点但不是全部 |
| 即将发生 | 暗示大事要发生 |
| 期待满足 | 观众等着看某个时刻 |
| 情感投资 | 关心角色命运 |
| 谜题推进 | 线索一点点揭示 |

### 结尾钩子（想看下一集）
| 类型 | 方法 |
|------|------|
| 悬崖式 | 关键时刻戛然而止 |
| 反转式 | 最后一秒反转 |
| 揭示式 | 揭露新信息 |
| 预告式 | 暗示下一集更精彩 |
| 情感式 | 留下情感悬念 |

## 🎯 钩子公式

### 30秒黄金公式
```
第1秒：视觉冲击/声音冲击
第3秒：核心人物出场
第7秒：抛出问题/冲突
第15秒：加深悬念
第30秒：确立方向
```

### 5秒抓住法则
```
观众每5秒评估一次"要不要继续看"

每5秒必须给一个理由：
- 新信息
- 情绪变化
- 视觉变化
- 声音变化
```

## 🎭 不同类型的钩子设计

### 甜宠剧钩子
```
开篇：两人冲突/误会相遇
持续：暧昧升级、甜蜜撒糖
结尾：差点表白/被打断

观众期待：
"什么时候在一起啊！！"
```

### 爽文剧钩子
```
开篇：主角被欺负/重生
持续：一步步打脸、升级
结尾：更大的敌人出现

观众期待：
"快看他怎么虐回去！"
```

### 悬疑剧钩子
```
开篇：死亡/失踪/奇怪事件
持续：线索、误导、反转
结尾：新发现、新嫌疑人

观众期待：
"真相到底是什么？！"
```

### 热血剧钩子
```
开篇：展示目标/梦想
持续：训练、比赛、突破
结尾：更强的对手出现

观众期待：
"他能打败这个对手吗？"
```

## 📋 钩子检测清单

### 开篇检测
- [ ] 前5秒有吸引力吗？
- [ ] 前30秒观众会划走吗？
- [ ] 有没有建立悬念？
- [ ] 有没有记忆点？
- [ ] 类型是否清晰？

### 整集检测
- [ ] 每3分钟有没有小钩子？
- [ ] 观众有理由继续看吗？
- [ ] 有没有无聊的段落？
- [ ] 节奏是否太慢？

### 结尾检测
- [ ] 有没有悬念？
- [ ] 观众想看下一集吗？
- [ ] 有没有停不下来的感觉？
- [ ] 会不会被骂"太监"？

## 🔧 钩子强化技巧

### 增强悬念
```
不要：说清楚
要：只说一半

不要："他是凶手"
要："他那天晚上去了…不，我不能说"
```

### 增强期待
```
铺垫 → 延迟 → 满足

例：
铺垫：听说有个厉害的人要来
延迟：迟迟不出场
满足：终于登场，够帅
```

### 增强意外
```
建立预期 → 打破预期

例：
预期：以为是好人
打破：结果是坏人
```
