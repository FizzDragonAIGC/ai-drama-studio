## 恐怖類型核心

### 恐懼來源
```
1. 未知：看不見的威脅
2. 失控：無法掌控局面
3. 孤立：無處求援
4. 變異：熟悉事物變陌生
5. 死亡：最終極恐懼
```

### 恐怖類型分類
```
超自然恐怖：鬼、惡魔、詛咒
心理恐怖：瘋狂、幻覺、不可靠敘述
身體恐怖：變異、疾病、gore
怪物恐怖：異形、殭屍、吸血鬼
氛圍恐怖：壓迫感、慢節奏、暗示
```

## 視覺技法

### 光影
```
- 大量陰影
- 低調照明
- 移動光源（手電筒）
- 突然的亮光（閃電）

Prompt: dark shadows, low-key lighting,
        flashlight beam, horror atmosphere
```

### 構圖
```
- 負空間（暗示威脅）
- 傾斜角度（不安）
- 極端視角（脆弱）
- 局部遮擋（想像恐懼）

Prompt: dutch angle, unsettling composition,
        negative space, lurking shadows
```

### 色調
```
- 冷色調（藍、綠）
- 去飽和
- 突然的紅色（血、警告）
- 蒼白膚色
```

## 敘事技法

### 鋪墊
```
- 平靜→異常（打破日常）
- 小異常累積
- 規則暗示（某個時間不能做某事）
- 背景故事（這裡發生過什麼）
```

### 節奏
```
緊張構建：
安靜→聲音→安靜→更大的聲音

假驚嚇（Jump Scare）：
緊張→原來是貓→真正的威脅

慢燒：
不急於揭示，持續壓迫
```

### 威脅呈現
```
第一階段：暗示存在
第二階段：部分顯現
第三階段：完全揭示（可能令人失望，謹慎）

規則：觀眾想像的比你能拍的更可怕
```

## 角色設計

### 受害者
```
有明顯弱點
觀眾能產生共情
做出合理但錯誤的決定
```

### 威脅源
```
有規則（但不完全清楚）
有起源（悲劇性更佳）
幾乎不可戰勝
```

## 恐怖清單

```
✅ 有效技法：
- 讓觀眾等待
- 用聲音暗示
- 限制視野
- 打破安全感

❌ 避免：
- 過多Jump Scare
- 太早揭示怪物
- 邏輯不通的行為
- 過度血腥取代恐懼
```
