# ✍️ 閃回技巧技能

## 元信息
- **技能ID**: script_flashback
- **所屬Agent**: 劇本Agent
- **參考書籍**: 《故事》+ 《剪輯的語法》
- **難度**: ⭐⭐⭐

## 核心理念

閃回 = 現在時間插入過去片段
- 不是偷懶的敘事手段
- 是揭露信息的戲劇工具

## 閃回類型

### 1. 解釋性閃回
```
功能：解釋角色行為的原因
時機：觀眾需要理解「為什麼」

例：
現在：他拒絕上船
閃回：童年溺水經歷
現在：觀眾理解了恐懼原因
```

### 2. 揭露性閃回
```
功能：揭露新信息/真相
時機：劇情轉折點

例：
現在：她說「我早就知道了」
閃回：她看到某個關鍵場景
現在：改變觀眾對劇情的理解
```

### 3. 情感性閃回
```
功能：加深情感衝擊
時機：分離、死亡、告別

例：
現在：她站在墓前
閃回：兩人的快樂時光（蒙太奇）
現在：觀眾感受到失去的重量
```

### 4. 主觀閃回
```
功能：展示角色記憶/心理
時機：角色回憶/創傷觸發

例：
現在：聽到某首歌
閃回：那首歌相關的記憶
現在：角色情緒變化
```

## 閃回格式

### 劇本標準格式
```
【閃回 - 描述】

INT. 小明家 - 日 - 過去

小明（8歲）...

【閃回結束】
```

### 中文AI番劇格式
```
【閃回開始】
時間：10年前
場景：老家院子
情緒：溫馨

動作描述...對話...

【閃回結束 - 回到現在】
```

## 閃回觸發器

### 視覺觸發
```
- 照片
- 相似的場景
- 同款物品
- 某個人的出現
```

### 聽覺觸發
```
- 音樂
- 聲音（雨聲、鐘聲）
- 某句話
- 相似的語調
```

### 嗅覺/觸覺觸發
```
- 香水
- 食物的味道
- 觸碰某物
```

## 閃回視覺區分

```
方法1：色調變化
- 過去：暖黃/褪色
- 現在：正常色彩

方法2：畫幅變化
- 過去：4:3 / 有黑邊
- 現在：16:9

方法3：質感變化
- 過去：顆粒感/膠片感
- 現在：清晰數字感

方法4：濾鏡
- 柔焦
- 夢幻光暈
```

## 閃回使用原則

### ✅ 使用閃回
```
- 信息無法用對話傳達
- 視覺比語言更有力
- 需要情感衝擊
- 結構上的對比需要
```

### ❌ 避免閃回
```
- 對話能說清的信息
- 打斷緊張節奏
- 過度使用（每集超過2次）
- 沒有明確觸發
```
