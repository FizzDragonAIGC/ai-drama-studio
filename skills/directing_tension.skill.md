# 🎬 张力构建 Skill
> 来源：《编剧的艺术》Lajos Egri + 希区柯克悬念理论

---

## 📐 什么是Tension（张力）

**定义**：让观众感到紧张、不安、期待的戏剧效果

### 张力 vs 悬念
| 概念 | 定义 | 效果 |
|------|------|------|
| **张力** | 对抗产生的紧绷感 | "这太紧张了" |
| **悬念** | 对未知的焦虑 | "接下来会怎样" |

两者常常同时存在

## 🎯 张力来源

### 1. 对立张力
```
角色A的目标 ←──对抗──→ 角色B的目标

例：
警察要抓罪犯 ← → 罪犯要逃跑
主角要告白 ← → 害怕被拒绝
```

### 2. 时间张力
```
┌──────────────────────────────────┐
│  💣 倒计时                        │
│  ████████░░░░░░ 剩余时间减少      │
│                                  │
│  时间越少 → 张力越高             │
└──────────────────────────────────┘
```

### 3. 信息不对称张力
```
观众知道的 > 角色知道的
（希区柯克炸弹理论）

观众看到炸弹 → 角色在聊天
→ 观众极度紧张
```

### 4. 道德张力
```
两个选项都有代价

救A还是救B？
说实话还是保护朋友？
牺牲一人还是牺牲全部？
```

## 🔧 视觉张力技巧

### 构图张力
| 技巧 | 效果 | 示意 |
|------|------|------|
| 压缩空间 | 压迫感 | 角色被框在角落 |
| 倾斜构图 | 不稳定 | 荷兰角 |
| 极端景别 | 紧张 | 超近特写 |
| 负空间 | 威胁感 | 大量空白暗示危险 |

### 剪辑张力
```
节奏加速：
████──███──██──█──█─█─█─█─██████
 慢        逐渐加速        高潮

交叉剪辑：
A → B → A → B → A → B → 碰撞！
两条线逐渐接近 → 张力升高
```

### 声音张力
| 技巧 | 效果 |
|------|------|
| 低频隆隆声 | 不安感 |
| 心跳声 | 紧张代入 |
| 滴答声 | 时间压力 |
| 沉默 | 暴风雨前的宁静 |
| 渐强音乐 | 张力升级 |

## 📋 张力节奏

### 张力波浪
```
张力
  │    ╱╲     ╱╲     ╱╲╱╲
  │   ╱  ╲   ╱  ╲   ╱    ╲
  │  ╱    ╲ ╱    ╲ ╱      ╲
  │ ╱      ╲      ╲        ╲
  │╱
  └────────────────────────────→ 时间

每次波峰比前一次高
有起有落（让观众喘息）
最后达到最高点
```

### 避免张力疲劳
```
❌ 错误：
████████████████████████████████
始终高张力 = 观众麻木

✓ 正确：
███░░██░░░███░░░░████░░░█████████
有张有弛 = 每次紧张都有效
```

## 🎯 场景级张力设计

### 张力场景模板
```
┌─────────────────────────────────────────┐
│ 场景：[名称]                            │
│ 张力类型：[对立/时间/信息/道德]         │
├─────────────────────────────────────────┤
│ 张力设置：                              │
│ - 对立双方：[A] vs [B]                  │
│ - 时间限制：[如有]                      │
│ - 信息差：观众知道什么角色不知道的？    │
│ - 道德困境：[如有]                      │
├─────────────────────────────────────────┤
│ 张力升级节点：                          │
│ 1. [事件1] - 张力等级 3                 │
│ 2. [事件2] - 张力等级 5                 │
│ 3. [事件3] - 张力等级 7                 │
│ 4. [高潮] - 张力等级 10                 │
├─────────────────────────────────────────┤
│ 释放方式：[如何解决/打断]               │
└─────────────────────────────────────────┘
```

## 🎬 AI番剧应用

### 张力镜头Prompt
```
压迫感构图：
"character cornered, claustrophobic framing,
dark shadows closing in, tense atmosphere"

对峙张力：
"two characters facing off, intense stare,
dramatic lighting, tension between them"

时间压力：
"character rushing, desperate expression,
motion blur, urgent, time running out"

等待张力：
"character waiting anxiously, 
nervous hands, tense body language,
anticipation, something about to happen"
```

### 张力场景节奏示例
```
| 镜号 | 内容 | 张力级别 | 时长 |
|------|------|----------|------|
| 01 | 门外脚步声 | 3 | 2秒 |
| 02 | 主角停下动作 | 4 | 1秒 |
| 03 | 门把手转动 | 6 | 1秒 |
| 04 | 主角屏息特写 | 7 | 2秒 |
| 05 | 门缝透光 | 8 | 1秒 |
| 06 | 门打开！ | 10 | 0.5秒 |
| 07 | 原来是朋友 | 释放 | 2秒 |
```
