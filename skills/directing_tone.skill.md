---

## 📐 什么是Tone（基调）

**定义**：影片传达的整体情感氛围和态度

### 基调 vs 相关概念
| 概念 | 定义 | 示例 |
|------|------|------|
| **基调(Tone)** | 创作者的态度 | 讽刺的、温暖的 |
| **情绪(Mood)** | 观众的感受 | 紧张、温馨 |
| **风格(Style)** | 视觉呈现方式 | 黑色电影、极简 |
| **氛围(Atmosphere)** | 场景环境感 | 阴郁、明朗 |

## 🔧 基调构建元素

### 1. 视觉基调
| 元素 | 严肃/黑暗 | 轻松/明亮 |
|------|-----------|-----------|
| 色调 | 低饱和、冷色 | 高饱和、暖色 |
| 对比 | 高对比、阴影 | 柔和、平光 |
| 构图 | 封闭、压迫 | 开放、宽松 |
| 景别 | 特写、紧凑 | 全景、宽阔 |

### 2. 听觉基调
| 元素 | 严肃/黑暗 | 轻松/明亮 |
|------|-----------|-----------|
| 配乐 | 小调、缓慢 | 大调、轻快 |
| 音效 | 沉重、回响 | 清脆、明亮 |
| 对白 | 低沉、慢速 | 轻快、活泼 |

### 3. 叙事基调
| 元素 | 严肃/黑暗 | 轻松/明亮 |
|------|-----------|-----------|
| 台词 | 复杂、隐晦 | 直白、幽默 |
| 节奏 | 缓慢、留白 | 快速、紧凑 |
| 结局 | 悲剧、开放 | 圆满、确定 |

## 📋 基调一致性检查

### 同场景元素是否统一？
```
场景：告别时刻
├─ 视觉：黄昏暖光 ✓
├─ 音乐：钢琴独奏 ✓
├─ 台词："我会想你的" ✓
├─ 表演：含泪微笑 ✓
└─ 节奏：缓慢 ✓
→ 基调统一：感伤但温暖
```

### 警示：基调冲突
```
场景：告别时刻
├─ 视觉：黄昏暖光
├─ 音乐：欢快电子乐 ✗ ← 冲突！
├─ 台词："我会想你的"
└─ 结果：观众困惑
```

## 🎯 类型片基调参考

| 类型 | 主基调 | 视觉特征 | 配乐特征 |
|------|--------|----------|----------|
| **浪漫爱情** | 温暖、轻松 | 柔光、暖色 | 钢琴、弦乐 |
| **动作冒险** | 紧张、刺激 | 高对比 | 激昂打击乐 |
| **恐怖** | 黑暗、压抑 | 低调光、冷色 | 不协和音 |
| **喜剧** | 轻松、戏谑 | 明亮、饱和 | 轻快节奏 |
| **文艺** | 沉思、忧郁 | 自然光、留白 | 极简/无 |
| **科幻** | 冷峻、神秘 | 蓝色调、未来感 | 电子合成 |

## 🎬 AI番剧应用

### 基调词汇→Prompt映射
| 基调 | Prompt关键词 |
|------|--------------|
| 温暖 | warm lighting, soft tones, cozy atmosphere |
| 黑暗 | dark, moody, low key lighting, shadows |
| 忧郁 | melancholic, blue tones, rain, lonely |
| 紧张 | tense, dramatic lighting, high contrast |
| 梦幻 | dreamy, soft focus, pastel colors, ethereal |
| 怀旧 | nostalgic, vintage, film grain, sepia |

## 💡 基调转变

### 何时转变基调
- 重大事件发生
- 时间跳转
- 角色成长节点
- 故事转折

### 转变技巧
```
渐变：基调A → 混合 → 基调B（推荐）
突变：基调A → 基调B（制造冲击）
```
