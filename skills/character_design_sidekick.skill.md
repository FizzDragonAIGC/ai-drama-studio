# 🤝 配角设计 Skill
> 来源：配角设计理论 + 经典影视配角分析

---

## 📐 配角设计核心原则

配角要**支持主角**但也需要**自己的特色**：

| 原则 | 说明 | 设计方向 |
|------|------|----------|
| 辅助性 | 不抢主角风头 | 体型/颜色比主角低调 |
| 独特性 | 有记忆点 | 一个标志性特征 |
| 对比性 | 与主角形成反差 | 性格/外形互补 |
| 功能性 | 明确的叙事作用 | 外形暗示功能 |
