# 📝 信息揭示節奏技能

## 元信息
- **技能ID**: outline_information
- **所屬Agent**: 大綱Agent
- **參考書籍**: 《故事》+ 《懸疑小說寫作》
- **難度**: ⭐⭐⭐⭐

## 核心理念

信息揭示 = 控制觀眾「知道什麼」的藝術
- 太早揭露 → 無聊
- 太晚揭露 → 困惑
- 節奏剛好 → 欲罷不能

## 信息類型

### 1. 世界觀信息
```
「這個世界的規則是什麼？」
揭示時機：儘早（集1-2）
方式：展示而非說明
```

### 2. 人物背景
```
「這個角色是誰？」
揭示時機：漸進式
方式：行動暗示 → 回憶揭露
```

### 3. 秘密/謎題
```
「真相是什麼？」
揭示時機：延遲至關鍵時刻
方式：線索堆疊 → 一次揭曉
```

### 4. 動機
```
「為什麼這樣做？」
揭示時機：行動後（解釋）
方式：對話/回憶/行為暗示
```

## 揭示策略

### 洋蔥式揭示
```
層1（表面）：早期揭露
層2（中層）：中期揭露
層3（核心）：後期揭露

每一層的揭露改變對前一層的理解
```

### 拼圖式揭示
```
碎片1（集2）+ 碎片2（集4）+ 碎片3（集7）
→ 集8觀眾自己拼出完整圖案
→ 集9確認猜測

讓觀眾有「我猜到了！」的滿足感
```

### 反轉式揭示
```
信息A（觀眾以為的）
→ 信息B（真相）

兩個信息必須都有足夠鋪墊
反轉要「意外但合理」
```

## 信息不對稱

### 觀眾 > 角色（戲劇性反諷）
```
觀眾知道炸彈在桌下
角色不知道

效果：焦慮/緊張
希區柯克最愛用
```

### 觀眾 < 角色（謎題）
```
角色知道真相
觀眾不知道

效果：好奇/猜謎
適合懸疑片
```

### 觀眾 = 角色（認同）
```
觀眾和主角同時發現真相

效果：共情/沉浸
適合主角視角故事
```

## 信息投放時間線

### 季度信息規劃
```
集1-2：基礎設定（80%世界觀，50%主角背景）
集3-4：深化人物（更多背景，引入謎題）
集5-6：種下線索（但不解釋）
集7：中點揭露（重要信息）
集8-9：進一步揭露（接近真相）
集10：關鍵揭露（改變局面）
集11-12：最終揭露（謎底）
```

### 單集信息預算
```
新信息：2-3個
發展信息：3-5個
暗示/伏筆：1-2個

規則：每集必須有「進展感」
```

## 伏筆與payoff

### 伏筆種類
```
道具伏筆：物品後來重要
對話伏筆：台詞後來有深意
視覺伏筆：畫面後來被回憶
```

### 伏筆規則
```
伏筆到payoff的距離：
- 短伏筆：同集內
- 中伏筆：2-4集
- 長伏筆：整季

規則：不要埋太多，會忘記
```
