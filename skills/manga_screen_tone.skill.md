## 📐 网点基础

### 什么是网点（Screentone）
```
预印刷的点状/纹理图案，用于：
┌─────────────────────────────────┐
│ ░░░░░░░░░░░░░░░░░░░░░░░░░░░░░ │
│ ░░░░░░░░░░░░░░░░░░░░░░░░░░░░░ │ ← 创造灰度
│ ░░░░░░░░░░░░░░░░░░░░░░░░░░░░░ │
│ ░░░░░░░░░░░░░░░░░░░░░░░░░░░░░ │
└─────────────────────────────────┘
```

**作用**：在黑白漫画中创造灰度、纹理、效果

## 🎯 网点类型

### 1. 点状网点（基础灰度）

| 线数(LPI) | 密度 | 效果 |
|-----------|------|------|
| **10%** | 稀疏 | 浅灰 |
| **20%** | 较稀 | 中浅灰 |
| **30%** | 中等 | 中灰 |
| **40%** | 较密 | 中深灰 |
| **50%** | 密集 | 深灰 |
| **60%+** | 极密 | 近黑 |

### 2. 渐变网点
```
░░░░░▒▒▒▒▒▓▓▓▓▓████
稀疏 ───────────→ 密集
浅   ───────────→ 深
```
**用途**：阴影过渡、立体感

### 3. 效果网点

| 类型 | 图案 | 用途 |
|------|------|------|
| **闪光** | ✦✧★ | 少女漫画、惊艳 |
| **花卉** | 🌸🌺 | 浪漫、回忆 |
| **云纹** | ☁️ | 天空、梦境 |
| **沙点** | ∴∵ | 粗糙质感 |
| **线条** | ─│ | 木纹、布料 |
| **噪点** | ░▒ | 颗粒感 |

## 📊 网点与光影

### 基础光影层次
```
【光源方向】↘

┌───────────────────┐
│ 白（高光）         │
├───────────────────┤
│ ░░░░（10%亮面）    │
├───────────────────┤
│ ▒▒▒▒（30%过渡）    │
├───────────────────┤
│ ▓▓▓▓（50%暗面）    │
├───────────────────┤
│ ████（黑色阴影）   │
└───────────────────┘
```

### 圆柱体网点示例
```

│ 白│░░░│▒▒▒│▓▓▓│█│
└─────────────────┘
```

## 🔧 网点应用区域

### 常见应用

| 区域 | 推荐网点 | 密度 |
|------|----------|------|
| **皮肤阴影** | 点状 | 10-20% |
| **头发** | 点状/渐变 | 20-40% |
| **衣服** | 点状/纹理 | 视颜色定 |
| **背景** | 渐变/效果 | 变化多 |
| **夜空** | 高密度+白点 | 50-60% |
| **金属** | 渐变 | 对比强 |

### 深色物体
```
黑发：
┌─────────────────┐
│▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓│ ← 50%底色
│████░████░████░█│ ← 高光留白
│▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓│
└─────────────────┘
```

## 📐 网点层叠技巧

### 叠加规则
```
❌ 错误：相同角度叠加 → 产生摩尔纹
✅ 正确：不同角度叠加 → 自然混合
```

### 角度建议
| 层次 | 角度 |
|------|------|
| **第一层** | 0° |
| **第二层** | 45° |
| **第三层** | 22.5° |

|------|--------|

```
滤镜 → 像素化 → 彩色半调
通道最大半径：4-8像素
网角：45°
```

## ⚠️ 常见错误

| 错误 | 问题 |
|------|------|
| **密度过高** | 印刷变黑块 |
| **同角度叠加** | 摩尔纹 |
| **全页平铺** | 单调无层次 |
| **忽略光源** | 立体感丧失 |
| **风格混乱** | 不协调 |
