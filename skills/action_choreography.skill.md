> 来源：《Action! Choreography for Stage & Screen》/ 好莱坞动作指导

---

## 📐 动作编排原则

### 核心三要素
1. **清晰性** - 观众能看懂发生什么
2. **连贯性** - 动作之间有逻辑
3. **情感性** - 动作服务于故事情感

### 动作节奏公式
```
建立(Setup) → 升级(Escalation) → 高潮(Climax) → 解决(Resolution)

时间分配：
20% → 50% → 20% → 10%
```

---

## 🎯 动作场景类型

### 1. 追逐场景 (Chase)
```
结构：
逃跑 → 追击 → 障碍 → 追近 → 脱险/被抓

要素：
- 速度感
- 环境变化
- 障碍升级
- 近距离时刻
```

**Prompt关键词**：
```
chase scene, running, pursuing,
jumping over obstacles, narrow escape,
parkour, rooftop chase, car chase
```

### 2. 打斗场景 (Fight)
```
结构：
对峙 → 试探 → 激战 → 转折 → 决胜

要素：
- 力量对比
- 攻防节奏
- 情感变化
- 环境利用
```

### 3. 逃脱场景 (Escape)
```
结构：
被困 → 寻找出路 → 遭遇障碍 → 突破 → 逃出

要素：
- 紧迫感
- 倒计时
- 机关/陷阱
- 惊险瞬间
```

### 4. 营救场景 (Rescue)
```
结构：
潜入 → 定位目标 → 遭遇敌人 → 救出 → 撤离

要素：
- 潜行
- 战斗
- 保护
- 团队配合
```

### 5. 决战场景 (Final Battle)
```
结构：
准备 → 对峙 → 激战 → 低谷 → 觉醒 → 终结

要素：
- 情感回顾
- 主题升华
- 绝招/爆发
- 牺牲/代价
```

## 📊 动作镜头设计

### 镜头类型
| 镜头 | 用途 | Prompt |
|------|------|--------|
| 全景 | 展示战场 | wide shot, battlefield overview |
| 中景 | 展示身体动作 | medium shot, full body action |
| 特写 | 表情/细节 | close-up, facial expression, detail |
| POV | 主观体验 | first person view, POV shot |
| 低角度 | 英雄感 | low angle, heroic shot |
| 高角度 | 俯瞰战局 | high angle, overhead view |
| 荷兰角 | 混乱感 | Dutch angle, tilted frame |

### 运动镜头
| 运动 | 效果 | Prompt |
|------|------|--------|
| 跟拍 | 跟随动作 | tracking shot, following action |
| 环绕 | 360度观察 | 360 shot, circling camera |
| 甩镜 | 快速转换 | whip pan, fast pan |
| 推进 | 聚焦冲击 | push in, zoom in |
| 拉远 | 揭示规模 | pull out, reveal shot |
| 摇晃 | 混乱真实 | handheld, shaky cam |

## 🎭 环境利用

### 场景元素
| 元素 | 用途 | Prompt |
|------|------|--------|
| 楼梯 | 高度变化 | staircase fight, falling down stairs |
| 栏杆 | 借力/跳跃 | railing, swinging, vaulting |
| 桌椅 | 武器/障碍 | furniture as weapon, chair throw |
| 窗户 | 破窗/逃脱 | breaking through window |
| 狭窄空间 | 限制战斗 | narrow corridor fight |
| 高处 | 坠落威胁 | rooftop, high ground, ledge |
| 水中 | 特殊战斗 | underwater fight |
| 火场 | 紧迫危险 | fighting in fire, flames |

### 成龙式环境动作
```
- 用梯子当武器
- 从购物车穿越
- 滑下楼梯扶手
- 在狭窄空间闪转
- 利用日常物品
```
**Prompt**: Jackie Chan style, improvised weapons, creative environment use
