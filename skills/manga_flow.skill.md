## 📐 阅读顺序基础

### 日式漫画阅读方向
```
页面顺序：从右向左
┌─────────┬─────────┐
│ ← 2    │   1 ←   │
├─────────┼─────────┤
│ ← 4    │   3 ←   │
└─────────┴─────────┘
```

### 西式漫画阅读方向
```
页面顺序：从左向右
┌─────────┬─────────┐
│   1 →   │ → 2    │
├─────────┼─────────┤
│   3 →   │ → 4    │
└─────────┴─────────┘
```

## 🎯 视线引导原理

### 自然视线路径

| 起点 | 终点 | 引导方式 |
|------|------|----------|
| **角色眼睛** | 视线目标 | 读者跟随角色视线 |
| **动作起点** | 动作终点 | 沿动作轨迹移动 |
| **气泡起始** | 下一气泡 | 按对话顺序阅读 |
| **亮区** | 暗区 | 明暗对比引导 |

### 视线阻断
```
【错误示例】
┌─────────┐
│ 1 ──→   │
│    ↓    │
│ ← 2     │  ← 阻断！需要回头
└─────────┘

【正确示例】
┌─────────┐
│ 1 ──────│
│    ↘    │
│      2  │  ← 自然流动
└─────────┘
```

## 🔄 流程节奏控制

### 阅读速度调节

| 元素 | 快速阅读 | 慢速阅读 |
|------|----------|----------|
| **格子** | 小格密集 | 大格疏松 |
| **线条** | 速度线、动态线 | 细腻描绘 |
| **对话** | 短句、感叹 | 长段独白 |
| **背景** | 简化/省略 | 详细描绘 |
| **留白** | 少 | 多 |

### 节奏变化公式
```
日常 → 紧张 → 高潮 → 舒缓
(中速)  (加速)  (最快/骤停)  (减速)
```

## 🎬 格间过渡类型

### 1. 时刻间过渡
```
【格1】角色举起拳头
【格2】拳头击中目标
```
**效果**：瞬间感、动作连贯

### 2. 动作间过渡
```
【格1】角色在A点
【格2】角色在B点
```
**效果**：时间流逝、空间移动

### 3. 场景间过渡
```
【格1】室内对话
【格2】室外街道
```
**效果**：空间切换

### 4. 方面间过渡
```
【格1】角色表情
【格2】手部特写
【格3】脚步
```
**效果**：多角度观察同一时刻

### 5. 非连续过渡
```
【格1】现在
【格2】回忆/未来
```
**效果**：时间跳跃

## 🎯 对话气泡流程

### 气泡位置原则
```
┌─────────────────────┐
│  (1)先说的在上/右   │
│        ↓            │
│        人物         │
│        ↓            │
│  (2)后说的在下/左   │
└─────────────────────┘
```

### 多人对话顺序
```
┌─────────────────────┐
│ A「1」     「2」B   │
│   ↓                 │
│ A「3」     「4」B   │
│   ↓                 │
│ A「5」              │
└─────────────────────┘
```
**顺序**：1→2→3→4→5

## 🔧 流程设计技巧

### 预留呼吸空间
- 大场面后留白让读者消化
- 快节奏后安排慢节奏

### 信息分配
- 每格只传达1-2个信息点
- 重要信息单独成格

### 翻页管理
- 右页末尾设悬念
- 左页开头做揭示
