## 🎭 三层冲突设计（麦基）

```
┌─────────────────────────────────────────┐
│         第一层：外界冲突                 │
│   社会机构 / 自然环境 / 敌对势力          │
├─────────────────────────────────────────┤
│         第二层：人际冲突                 │
│   亲密关系 / 友情 / 爱情 / 家庭          │
├─────────────────────────────────────────┤
│         第三层：内心冲突                 │
│   欲望 vs 恐惧 / 理智 vs 情感            │
└─────────────────────────────────────────┘

✅ 复杂故事 = 三层冲突同时存在
❌ 简单故事 = 只有一层冲突
```

### 冲突设计清单
- [ ] 外界冲突：主角面对什么外部障碍？
- [ ] 人际冲突：与谁产生冲突？潜台词是什么？
- [ ] 内心冲突：主角内心的矛盾是什么？

## 🐱 "救猫咪"时刻（斯奈德）

### 定义
> 在建置阶段，让主角做一件让观众喜欢他的小事

### 技巧
| 方法 | 例子 |
|------|------|
| **帮助弱小** | 救猫、帮老人、保护孩子 |
| **幽默自嘲** | 调侃自己的缺点 |
| **反派也有情** | 坏人对某人温柔 |
| **专业能力** | 展示高超技艺 |
| **诚实勇敢** | 在压力下说真话 |

## 🎬 情节点设计（菲尔德）

### 情节点1（25%位置）
**作用**：将主角推入新世界

**检查清单**：
- [ ] 改变了故事方向？
- [ ] 是主角的主动选择？
- [ ] 不可逆转？
- [ ] 提升了赌注？

### 中点（50%位置）
**作用**：假胜利或假失败

**两种类型**：
| 类型 | 说明 | 后续发展 |
|------|------|----------|
| **假胜利** | 主角以为成功了 | 更大的挑战来临 |
| **假失败** | 主角以为失败了 | 发现新方向 |

### 情节点2（75%位置）
**作用**：推向最终对决

**检查清单**：
- [ ] 主角触底？
- [ ] 获得关键信息？
- [ ] 准备最终行动？

## 📊 情感曲线设计（麦基）

```
          游戏时间     坏人逼近
              ↓          ↓
正面 ────●────●────●────────────●────● 高潮
          ↗          ↘      ↗
开场 ●───●                ●───●        结局
     ↘                    ↘
负面 ────────────────●────●
                    中点   一无所有
```

### 情感递减定律
> 我们对某事体验越多，效果越少

**解决方案**：正负情感交替
- 不能连续3个正面场景
- 不能连续3个负面场景
- 高潮前必须有低谷

## 🎯 两难选择设计（麦基）

### 真正的选择 = 两难
| 类型 | 定义 | 例子 |
|------|------|------|
| **两善取其一** | 两个都想要，只能选一个 | 事业 vs 家庭 |
| **两恶取其轻** | 两个都不想要，必须选一个 | 背叛朋友 vs 失去生命 |

### 选择揭示人物
> 主角在危机时刻如何选择，揭示其深层性格

**设计检查**：
- [ ] 两个选项同等重要/痛苦？
- [ ] 没有明显的"正确答案"？
- [ ] 选择有不可逆的后果？

## 🔧 场景分析五步法（麦基）

| 步骤 | 问题 | 输出 |
|------|------|------|
| **1. 确定冲突** | 谁想要什么？阻力是什么？ | 冲突描述 |
| **2. 标注开篇价值** | 正面还是负面？ | +/- 标记 |
| **3. 分解节拍** | 动作/反应的交流 | 节拍列表 |
| **4. 标注结尾价值** | 与开篇比较 | +/- 变化 |
| **5. 找转折点** | 期望与结果的偏差 | 转折描述 |

### 关键原则
> 场景必须有价值变化！如果开篇和结尾价值相同，删掉这场戏。

## 📋 10种故事类型（斯奈德）

| 类型 | 核心元素 | 例子 |
|------|----------|------|
| **屋里有怪物** | 密闭空间 + 怪物 | 异形、电锯惊魂 |
| **金羊毛** | 公路/旅程 | 绿野仙踪、指环王 |
| **如愿以偿** | 许愿/魔法 | 大话西游、土拨鼠之日 |
| **哥们帮** | 两人关系 | 雨人、泰囧 |
| **成长仪式** | 人生转折 | 阳光灿烂的日子 |
| **超级英雄** | 特殊能力 | 蜘蛛侠、哪吒 |
| **傻瓜成功** | 小人物逆袭 | 阿甘正传、喜剧之王 |
| **制度困境** | 群体/组织 | 飞越疯人院、肖申克 |
| **侦探推理** | 解谜 | 唐人街、看不见的客人 |
| **家庭伦理** | 家庭矛盾 | 都挺好、饮食男女 |
