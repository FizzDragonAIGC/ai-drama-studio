## 📐 甜宠核心原理

甜宠 = **理想化爱情 + 情绪满足**

| 要素 | 作用 | 观众心理 |
|------|------|----------|
| 完美人设 | 理想伴侣投射 | 代入幻想 |
| 甜蜜互动 | 多巴胺刺激 | 上瘾循环 |
| 独占感 | "只对你特别" | 被爱感 |
| 安全感 | 被保护/被宠 | 依赖感 |

## 📋 甜宠结构模板

### 单集甜宠结构
```
[虐/误会/冲突] → [解决/相遇] → [甜蜜互动] → [更甜/小悬念]
```

### 甜虐配比
| 风格 | 甜:虐 | 适合 |
|------|-------|------|
| 纯甜 | 9:1 | 轻松向 |
| 甜虐交织 | 7:3 | 有深度 |
| 先虐后甜 | 开头虐结尾甜 | 逆袭甜 |

### 关系发展节奏
```
相遇 → 契约/误会 → 暧昧 → 确认心意 → 在一起 → 撒糖
```

## 🎯 撒糖技巧

### 日常撒糖
| 场景 | 糖点 |
|------|------|
| 一起吃饭 | 投喂/擦嘴角 |
| 一起工作 | 偷看/走神 |
| 送礼物 | 记得喜好 |
| 生病 | 照顾/心疼 |
| 睡觉 | 看着入睡 |

### 高甜时刻
| 类型 | 描述 | 甜度 |
|------|------|------|
| 表白 | 直球告白 | ★★★★★ |
| 吻戏 | 深情亲吻 | ★★★★★ |
| 求婚 | 正式承诺 | ★★★★★ |
| 吃醋 | 表现独占 | ★★★★☆ |
| 护短 | 当众维护 | ★★★★☆ |
| 日常撒娇 | 甜蜜互动 | ★★★☆☆ |

### 撒糖频率
| 时长 | 糖点数量 |
|------|----------|
| 30秒 | 1-2个 |
| 1分钟 | 2-3个 |
| 3分钟 | 4-6个 |

## 📊 甜宠情绪设计

### 心动信号
| 信号 | 表现 |
|------|------|
| 眼神 | 深情凝视/移不开眼 |
| 表情 | 不自觉微笑/脸红 |
| 动作 | 靠近/触碰/保护性姿势 |
| 语言 | 语气变化/特殊称呼 |

### 甜蜜氛围
| 元素 | 设计 |
|------|------|
| 灯光 | 暖色调/柔和 |
| BGM | 甜蜜轻快/深情慢歌 |
| 构图 | 两人距离近 |
| 节奏 | 适当放慢 |
