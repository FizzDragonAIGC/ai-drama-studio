# 即梦AI专业提示词模板 Skill

> 从即梦/Jimeng高赞作品中提炼的提示词结构和技巧

---

## 🎯 即梦提示词核心结构

### 图片提示词（5层结构）
```
1. 【主体描述】角色外貌、姿态、表情、服装
2. 【环境场景】地点、天气、时间、背景元素
3. 【光影设计】光源、色温、对比度、氛围
4. 【构图镜头】景别、角度、焦距、景深
5. 【质感风格】渲染器、画质、艺术风格、参考
```

### 视频提示词（3段式）
```
1. 【首帧描述】静态画面起点
2. 【运镜过程】镜头如何移动（必须用专业术语）
3. 【尾帧描述】静态画面终点
+ 【氛围强化】情绪、质感、声音建议
```

## 📝 高赞作品提示词拆解

### 案例1：平滑升镜环绕（1324❤️）
**原文**：
```
以低角度跟随镜头开始，追踪地面附近白色丝带的飘动。
镜头顺着她的身体平滑上升(Pedestal Up)到面部，
然后无缝过渡到一个缓慢的180度环绕镜头(Orbit)。
随着镜头的环绕，由于长焦镜头效果，
背景的月亮看起来被压缩并巨大地显现在她身后。
```

**拆解**：
- 起始：低角度 + 跟随 + 地面元素
- 运动1：Pedestal Up（升镜）
- 运动2：Orbit（环绕）
- 视觉效果：长焦压缩 + 背景放大

## 🔧 即梦提示词公式

### 图片公式
```
[角度] + [景别] + [主体] + [动作/状态] + [光影] + [色调] + [质感] + [氛围]
```

### 视频公式
```
[运镜类型] + [起始状态] + [运动过程（用专业术语）] + [结束状态] + [声音/氛围]
```

| 升镜 | Pedestal Up | 庄重、揭示 |
| 降镜 | Pedestal Down | 压迫、沉重 |
| 环绕 | Orbit | 神秘、聚焦 |
| 推镜 | Dolly In / Push | 紧张、关注 |
| 拉镜 | Dolly Out / Pull | 揭示、震撼 |
| 摇镜 | Pan | 扫视、交代 |

## ✅ 即梦高分提示词检查清单

- [ ] 运镜有明确类型（不是"镜头移动"）
- [ ] 使用专业术语（Pedestal/Orbit/Dolly等）
- [ ] 有首帧和尾帧概念
- [ ] 光影描述具体（不只说"光线好"）
- [ ] 氛围词丰富（至少3个维度）
- [ ] 有色调基底（青蓝/暖金/低饱和等）
- [ ] 质感词具体（Octane/8K/真实触感等）
- [ ] 声音设计建议（环境音+情绪音）
