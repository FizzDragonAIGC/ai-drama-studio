# 🎬 电视剧单集结构 (TV Episode Structure)
> 10-15分钟+ 三幕式剧集叙事

---

## 📐 三幕结构 (15分钟)

```
┌─────────────────────────────────────────────────────┐
│              第一幕：建置 (0:00-4:00)                 │
│  • 开场钩子 (30秒)                                   │
│  • 世界/角色重建 (1分)                               │
│  • 激励事件 (1分30秒)                                │
│  • 进入第二幕的决定 (1分)                            │
├─────────────────────────────────────────────────────┤
│              第二幕：对抗 (4:00-11:00)                │
│  • 上升动作 (2分)                                    │
│  • 中点转折 (1分)                                    │
│  • 坏人逼近 (2分)                                    │
│  • 最低点 (2分)                                      │
├─────────────────────────────────────────────────────┤
│              第三幕：解决 (11:00-15:00)               │
│  • 顿悟/重整 (1分)                                   │
│  • 高潮行动 (2分)                                    │
│  • 解决/新平衡 (1分)                                 │
└─────────────────────────────────────────────────────┘
```

## 🔀 多线叙事

### A/B/C线设计
| 线 | 占比 | 功能 | 示例 |
|----|------|------|------|
| **A线** | 60% | 主情节 | 主角的核心冲突 |
| **B线** | 25% | 情感线 | 爱情/友情/家庭 |
| **C线** | 15% | 伏笔线 | 暗线/下集铺垫 |

### 多线交织方法
```
时间 → 
A线: ●────●────────●────────●────●
B线:    ●────●────────●────────●
C线:         ●──────────────●
     ↑        ↑            ↑
   A引入    ABC交汇     A收束/BC延续
```

## 🎭 人物深度设计

### 角色层次
| 层次 | 展现方式 | 示例 |
|------|----------|------|
| **外在** | 动作、对白 | 他说了什么 |
| **内在** | 潜台词、表情 | 他真正想的 |
| **核心** | 欲望vs恐惧 | 他为什么这样 |

### 单集人物弧光
- **开始**：角色状态A
- **挑战**：面临选择
- **选择**：做出决定（揭示性格）
- **结果**：状态变化A→A'

## 💬 电视剧对白

### 潜台词设计
| 表面说 | 真正意思 | 情境 |
|--------|----------|------|
| "我没事" | 我很痛苦 | 逞强 |
| "你开心就好" | 我不开心 | 退让 |
| "随便你" | 我很在意 | 赌气 |
| "我们聊聊" | 有问题了 | 警告 |

### 对白功能
1. **推进情节** - 传递信息
2. **展现角色** - 体现性格
3. **制造冲突** - 观点碰撞
4. **暗示主题** - 金句升华

## 📝 电视剧剧本格式

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
第X集「标题」(15分钟)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【线索分布】
A线（主线）：xxxxx
B线（情感）：xxxxx  
C线（伏笔）：xxxxx

═══════════════ 第一幕：建置 ═══════════════

【开场】(0:00-0:30) [A线]
场景：内景 - 地点 - 时间
（简洁有力的开场钩子）

---

【场景1】(0:30-1:30) [A线]
场景：内景 - 地点 - 日

动作描述（简洁，只写能拍的）

角色A
    台词内容

角色B
    （括号语）
    台词内容

---

【场景2】(1:30-2:30) [B线插入]
场景：内景 - 地点 - 日
（B线发展，与A线形成对照）

---

【场景3】(2:30-4:00) [A线激励事件]
场景：外景 - 地点 - 日
（打破平衡的事件发生）

★ 第一幕结束：主角做出进入第二幕的决定

═══════════════ 第二幕：对抗 ═══════════════

【场景4】(4:00-5:30) [A线]
（上升动作，初步应对）

---

【场景5】(5:30-6:30) [B线]
（B线深化）

---

【场景6】(6:30-7:30) [A线中点]
★ 中点转折：假胜利或假失败

---

【场景7】(7:30-9:00) [A+C线]
（A线困难加大，C线伏笔露出）

---

【场景8】(9:00-11:00) [A线最低点]
★ 最低点：一切似乎失败

═══════════════ 第三幕：解决 ═══════════════

【场景9】(11:00-12:00) [A线]
（顿悟，找到解决方案）

---

【场景10】(12:00-14:00) [A线高潮]
★ 高潮行动：最终对决/解决问题

---

【结尾】(14:00-15:00) [A+B+C线]
（A线解决，B线进展，C线延续为下集钩子）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 🔄 集与集的连接

### 单集完结型
- 本集问题本集解决
- 适合单元剧

### 连续剧型
- 本集解决小问题，大问题延续
- 每集结尾有钩子

### 钩子类型
| 类型 | 示例 | 效果 |
|------|------|------|
| **悬崖式** | 枪响，屏幕黑 | 强烈好奇 |
| **揭示式** | 发现秘密 | 期待发展 |
| **抉择式** | 面临两难 | 想知道选什么 |
| **反转式** | 意外身份 | 重新理解 |

## ✅ 检查清单

### 结构检查
- [ ] 三幕比例合理？(25%/50%/25%)
- [ ] 有明确的中点转折？
- [ ] 有最低点设计？
- [ ] 高潮有情感冲击？

### 多线检查
- [ ] A/B/C线清晰？
- [ ] 线索有交织点？
- [ ] 各线节奏不同？

### 角色检查
- [ ] 角色有弧光变化？
- [ ] 对白有潜台词？
- [ ] 选择揭示性格？
