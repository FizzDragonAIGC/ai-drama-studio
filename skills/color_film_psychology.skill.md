# 電影色彩心理學 Skill

> 基於《If It's Purple, Someone's Gonna Die》- Patti Bellantoni
> 電影色彩敘事的權威指南

---

## 🎯 核心理念

**色彩不是裝飾，是敘事工具**

每種顏色都有心理暗示，觀眾潛意識會接收這些信號。導演用色彩預示劇情、塑造角色、引導情緒。

---

## 🔴 紅色 (Red) - 激情與危險

### 心理效應
- 加速心跳、提升腎上腺素
- 引發衝動、攻擊性
- 象徵愛情、憤怒、危險

### 電影應用
| 情境 | 用法 | 經典案例 |
|------|------|----------|
| 愛情/慾望 | 女主紅裙、紅唇 | 《美國麗人》玫瑰花瓣 |
| 危險預警 | 場景中紅色元素增加 | 《乘客》危機紅燈 |
| 暴力/血腥 | 直接或隱喻 | 《乱》战争场景 |
| 權力/支配 | 紅色房間、服裝 | 《大紅燈籠高高掛》 |

### Prompt關鍵詞
```
passionate red, blood red, crimson warning, 
scarlet desire, ruby lips, vermillion rage
```

## 🟡 黃色 (Yellow) - 警告與瘋狂

### 心理效應
- 最先被眼睛捕捉的顏色
- 小劑量 = 溫暖、希望
- 大劑量 = 焦慮、瘋狂、背叛

### 電影應用
| 情境 | 用法 | 經典案例 |
|------|------|----------|
| 希望/溫暖 | 陽光、金色光芒 | 《海上鋼琴師》 |
| 瘋狂/不穩定 | 黃色牆壁、燈光 | 《出租車司機》 |
| 背叛/懦弱 | "yellow belly" | 西部片傳統 |
| 疾病/腐敗 | 病態黃 | 《乙蝕》 |

### Prompt關鍵詞
```
warm golden light, sickly yellow, 
amber warning, mustard tension, 
jaundiced atmosphere
```

## 🔵 藍色 (Blue) - 冷靜與憂鬱

### 心理效應
- 降低心率、產生距離感
- 象徵理性、孤獨、憂傷
- 冷藍 vs 暖藍差異巨大

### 電影應用
| 情境 | 用法 | 經典案例 |
|------|------|----------|
| 孤獨/疏離 | 冷藍調場景 | 《藍》基斯洛夫斯基 |
| 科技/未來 | 藍色LED、屏幕光 | 《銀翼殺手》 |
| 憂鬱/悲傷 | blue hour天空 | 《乱》暴風雨前 |
| 理性/權威 | 藍色制服、辦公室 | 《間諜橋》 |

### Prompt關鍵詞
```
melancholic blue, cold steel blue,
midnight blue isolation, cerulean calm,
electric blue technology
```

## 🟢 綠色 (Green) - 自然與腐敗

### 心理效應
- 雙面性最強的顏色
- 正面：生命、希望、治愈
- 負面：嫉妒、毒素、腐敗

### 電影應用
| 情境 | 用法 | 經典案例 |
|------|------|----------|
| 新生/希望 | 植物、春天 | 《肖申克》結尾草地 |
| 嫉妒 | "green with envy" | 《乱世佳人》 |
| 超自然/恐怖 | 綠色迷霧、燈光 | 《乙蝕》《異形》 |
| 矩陣/數字世界 | 代碼綠 | 《黑客帝國》 |

### Prompt關鍵詞
```
verdant hope, sickly green, 
emerald envy, toxic green glow,
matrix green digital, forest healing
```

## 🟣 紫色 (Purple) - 死亡與神秘

### 心理效應
- 最稀有的自然色彩
- 象徵皇室、神秘、死亡
- **Bellantoni定律：紫色出現，必有人亡**

### 電影應用
| 情境 | 用法 | 經典案例 |
|------|------|----------|
| 死亡預兆 | 紫色光/服裝出現 | 《乱》死亡場景 |
| 神秘/魔幻 | 紫色霧氣、光環 | 《阿凡達》 |
| 高貴/腐敗 | 權力與墮落 | 《羅馬帝國艷情史》 |
| 過渡/轉化 | 日出日落紫調 | 《獅子王》 |

### Prompt關鍵詞
```
ominous purple, royal violet,
mystic purple haze, lavender twilight,
death purple glow
```

## ⬛ 黑白對比 - 二元對立

### 應用原則
- 黑 = 未知、邪惡、正式
- 白 = 純潔、空虛、投降
- 高對比 = 道德明確
- 低對比 = 道德模糊

### Prompt關鍵詞
```
stark black and white contrast,
noir shadows, pure white isolation,
chiaroscuro moral ambiguity
```

## 🎨 色彩組合策略

### 互補色（衝突）
- 紅/綠 = 聖誕節 or 內心掙扎
- 藍/橙 = 科幻經典（《銀翼殺手》《瘋狂麥克斯》）
- 紫/黃 = 高度緊張

### 類似色（和諧）
- 藍/紫/青 = 冷靜連貫
- 紅/橙/黃 = 溫暖活力
- 綠/青/藍 = 自然平靜

### 單色調（情緒沉浸）
- 全藍 = 憂鬱包圍
- 全紅 = 激情燃燒
- 去飽和 = 回憶/創傷

## 💡 實戰應用框架

```
場景色彩設計：
1. 確定情緒目標 → 選擇主色
2. 角色狀態 → 服裝/化妝色彩
3. 環境反映內心 → 場景佈置
4. 對比製造張力 → 互補色點綴
5. 預示命運 → 隱藏色彩線索
```
