## 愛情類型核心

### 愛情故事公式
```
相遇 → 相吸 → 障礙 → 分離 → 重聚

核心：兩個人克服障礙在一起
變體：無法在一起（悲劇）
```

### 障礙類型
```
外部障礙：家庭反對、距離、身份
內部障礙：恐懼承諾、過去創傷
誤會障礙：錯誤信息、不溝通
第三者障礙：情敵
```

## 視覺技法

### 光影
```
- 柔光
- 逆光剪影
- 溫暖色調
- 夕陽/黃金時段

Prompt: soft romantic lighting, golden hour,
        warm glow, dreamy atmosphere
```

### 構圖
```
- 二人構圖（框住兩人）
- 對視角度
- 距離變化（遠→近）
- 鏡像構圖

Prompt: couple framing, intimate distance,
        romantic composition
```

### 色調
```
- 溫暖色（粉、橙）
- 高飽和
- 柔焦
- 春天/夏天感
```

## 敘事技法

### 相遇（Meet-Cute）
```
有趣的初次相遇：
- 意外碰撞
- 誤會開始
- 命運安排
- 對立立場

必須讓觀眾覺得「這兩個人屬於彼此」
```

### 化學反應
```
眼神交流
小動作（理頭髮、不自覺微笑）
對話中的試探
共同經歷
```

### 分離與重聚
```
分離原因必須足夠重
重聚必須主角主動
重聚場景要有視覺記憶點
```

## 撒糖技法

### 心動時刻
```
- 第一次牽手
- 第一次對視
- 意外的親近
- 照顧對方
- 說出心意（未完成）
```

### 撒糖鏡頭
```
特寫：手指交纏
中景：依偎
遠景：二人世界
主觀：看著對方的眼神
```

## 愛情子類型

```
甜寵：高糖分、無虐
虐戀：障礙大、痛苦
暗戀：單向、苦澀
青梅竹馬：從小認識
先婚後愛：契約開始
破鏡重圓：分手後復合
宿敵戀人：從對立到相愛
```

## 愛情清單

```
✅ 有效技法：
- 建立足夠的鋪墊
- 讓觀眾先愛上角色
- 化學反應要可見
- 障礙要合理

❌ 避免：
- 無緣無故的愛
- 太多三角戀
- 誤會拖太久
- 結局太突然
```
