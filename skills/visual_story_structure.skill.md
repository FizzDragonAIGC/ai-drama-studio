# 視覺故事結構 Skill

> 基於《The Visual Story》- Bruce Block
> 視覺敘事結構的權威教材

---

## 🎯 Block核心理念

**"視覺元素是獨立的敘事語言"**

- 故事有結構，視覺也有結構
- 視覺變化可以獨立於對白傳達情緒
- 對比與親和是視覺敘事的基礎

---

## 📐 七大視覺要素

### 1. 空間 (Space)
**深度幻覺的創造與打破**

| 深空間 | 平空間 |
|--------|--------|
| 前/中/後景分明 | 壓縮成一個平面 |
| 透視線收斂 | 正面/側面拍攝 |
| 運動感強 | 靜態、圖形化 |
| 冒險、開放 | 困住、壓迫 |

```
深空間Prompt：
deep perspective, foreground middle background layers,
converging lines, depth of field

平空間Prompt：
flat composition, side profile,
graphic design style, compressed depth
```

### 2. 線條 (Line)
**視覺引導與情緒暗示**

| 線條類型 | 情緒暗示 |
|---------|---------|
| 水平線 | 穩定、平靜、死亡 |
| 垂直線 | 力量、尊嚴、正式 |
| 斜線 | 動態、不安、衝突 |
| 曲線 | 柔和、自然、女性 |

```
動態場景Prompt：
diagonal composition, dynamic angles,
tilted frame, chaotic lines

平靜場景Prompt：
horizontal lines, calm water reflection,
stable composition, grounded framing
```

### 3. 形狀 (Shape)
**基礎幾何與心理聯想**

| 形狀 | 聯想 |
|-----|------|
| 圓形 | 安全、有機、柔軟 |
| 方形 | 穩定、人造、理性 |
| 三角形 | 方向、衝突、不穩 |

```
角色設計應用：
- 英雄 = 方+三角（強壯+方向）
- 反派 = 尖銳三角（危險）
- 配角 = 圓形（友好無害）
```

### 4. 色調 (Tone)
**明暗對比控制**

```
高對比 = 戲劇性、衝突、黑白分明
低對比 = 柔和、憂鬱、模糊界限
明調 = 樂觀、開放、喜劇
暗調 = 神秘、壓抑、悲劇
```

### 5. 色彩 (Color)
**見「電影色彩心理學」Skill**

### 6. 運動 (Movement)
**動態的情緒表達**

| 運動方向 | 暗示 |
|---------|------|
| 左→右 | 前進、正常（西方閱讀習慣）|
| 右→左 | 倒退、對抗、困難 |
| 向上 | 積極、勝利 |
| 向下 | 消極、失敗 |

### 7. 節奏 (Rhythm)
**視覺元素的重複與變化**

```
穩定節奏 = 安心、可預測
打破節奏 = 驚訝、轉折點
加速節奏 = 緊張、高潮逼近
```

## 🔄 對比與親和原則

### Block的核心公式

**對比 (Contrast) = 視覺強度增加**
- 大與小
- 亮與暗
- 暖與冷
- 快與慢

**親和 (Affinity) = 視覺統一和諧**
- 相似的大小
- 接近的色調
- 同方向運動

### 情節與視覺對應

| 情節階段 | 視覺策略 |
|---------|---------|
| 建置期 | 高親和（統一、平靜）|
| 衝突升級 | 逐漸增加對比 |
| 高潮 | 最大對比（視覺爆炸）|
| 結局 | 回歸親和（但有改變）|

## 📊 視覺強度圖

Block建議為每個場景畫視覺強度圖：

```
    高
強   ┃      ╱╲        ← 高潮
度   ┃    ╱    ╲
    ┃  ╱        ╲
    ┃╱            ╲
    低─────────────────
      開始  中段  結尾
```

### 應用方法
1. 標記故事情緒高低點
2. 對應設計視覺對比強度
3. 視覺曲線應與故事曲線同步

## 🎬 分鏡視覺設計

### 建置場景（低強度）
```
統一色調、相似構圖、
穩定水平線、柔和光線、
大景別建立環境
```

### 衝突場景（中等強度）
```
開始引入對比元素、
斜線構圖增多、
光影對比加強、
景別開始變化
```

### 高潮場景（高強度）
```
極端對比：最亮vs最暗、
不穩定三角構圖、
快速景別切換、
色彩衝突（互補色）、
運動方向衝突
```

### 結局場景（回歸親和）
```
對比降低但不回到原點、
保留一些變化元素、
新的視覺平衡、
暗示改變已發生
```

## ✏️ AI繪圖整合

### Block視覺強度Prompt

**低強度（平靜）**
```
harmonious composition,
soft diffused lighting,
muted color palette,
horizontal framing,
unified visual elements
```

**高強度（衝突）**
```
high contrast dramatic lighting,
diagonal dynamic composition,
complementary color clash,
extreme angles,
visual tension and conflict
```

### 空間深度Prompt

**深空間（開放）**
```
deep perspective,
foreground framing element,
middle ground subject,
background environment,
atmospheric depth
```

**平空間（壓迫）**
```
flat graphic composition,
compressed depth,
profile silhouette,
minimal perspective,
stage-like framing
```
