> 来源：《电影镜头设计》Steven D. Katz + 《导演的空间概念》

---

## 📐 什么是Screen Geography（空间地理）

**定义**：观众脑中建立的场景空间布局认知

### 为什么重要
| 原因 | 说明 |
|------|------|
| **方向感** | 观众知道谁在哪里 |
| **连贯性** | 切换镜头不迷失 |
| **动作逻辑** | 追逐、打斗有空间感 |
| **情感空间** | 距离传达关系 |

## 🎯 建立空间地理

### 标准流程
```
Step 1: 建立镜头（Establishing Shot）
┌─────────────────────────────────────┐
│                                     │
│    ┌──窗──┐                        │
│    │      │                        │
│    A      B        C               │
│          桌子                       │
│    └─────门───────┘                 │
└─────────────────────────────────────┘
→ 观众理解：A靠窗，B在中间，C靠门

Step 2: 保持一致
之后的镜头必须符合已建立的空间关系
```

### 空间关系示意
```
正确示例：
建立镜头：A在左，B在右
单人A镜头：A面向右看（看B）
单人B镜头：B面向左看（看A）
→ 空间一致

错误示例：
建立镜头：A在左，B在右
单人A镜头：A面向右看
单人B镜头：B面向右看 ← 错！两人看同方向
→ 空间混乱
```

## 🔧 空间建立技巧

### 1. 建立镜头类型
| 类型 | 使用时机 |
|------|----------|
| 大远景 | 新场景、新地点 |
| 全景 | 房间内部空间 |
| 中景主镜头 | 对话空间 |
| 运动建立 | 跟随角色进入 |

### 2. 重建立时机
需要重新建立空间的情况：
- 有角色移动位置
- 新角色进入
- 时间跳转
- 可能产生混乱时

### 3. 空间地标
```
可用作空间参考的元素：
├─ 窗户（光线方向）
├─ 门（出入口）
├─ 家具（固定物）
├─ 背景特征（墙画、时钟）
└─ 光源（台灯、阳光）
```

## 🎯 复杂空间处理

### 追逐场景空间
```
需要让观众知道：
1. 追的人和被追的人相对位置
2. 他们要去哪里
3. 障碍在哪里
4. 安全区在哪里

技巧：
- 定期插入建立镜头
- 使用地标保持方向感
- 保持运动方向一致
```

### 打斗场景空间
```
┌─────────────────────────────────┐
│        战场平面图               │
│                                 │
│    A ←──── 交战 ────→ B        │
│              │                  │
│              ↓                  │
│            障碍物               │
│                                 │
└─────────────────────────────────┘

镜头策略：
- 主镜头确立双方位置
- 特写不要太久
- 定期回到中景/全景
- 保持方向一致
```

### 多人场景空间
```
  A       B       C       D
  │       │       │       │
  ↓       ↓       ↓       ↓
┌───┐   ┌───┐   ┌───┐   ┌───┐
│   │   │   │   │   │   │   │
└───┘   └───┘   └───┘   └───┘
座位安排决定了所有对话镜头的角度
```

## 🎬 AI番剧应用

### 场景描述Prompt
```
背景空间描述需要一致：

建立镜头：
"wide shot classroom, windows on left side,
blackboard at front, students in rows,
afternoon sunlight from left"

后续镜头：
"medium shot student at desk, 
window visible in background left side,
consistent lighting from left"
```

### 空间一致性检查
```
✓ 窗户始终在同一侧
✓ 门的位置不变
✓ 家具布局一致
✓ 光源方向一致
✓ 背景元素相同
```

### 角色位置Prompt模板
```
场景：教室对话

A(老师)位置：画面左侧，面向右
B(学生)位置：画面右侧，面向左

镜头1(建立)：
"two people in classroom, teacher on left facing right,
student on right facing left, blackboard behind teacher"

镜头2(老师特写)：
"close-up teacher, looking towards right, blackboard visible"

镜头3(学生特写)：
"close-up student, looking towards left, window in background"
```

## 💡 常见错误

| 错误 | 后果 | 解决 |
|------|------|------|
| 缺少建立镜头 | 空间混乱 | 场景开始加建立 |
| 背景不一致 | 穿帮感 | 固定背景描述 |
| 光线方向变化 | 视觉不适 | 统一光源位置 |
| 人物位置跳跃 | 逻辑混乱 | 遵守180度法则 |
