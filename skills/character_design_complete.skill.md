# 🎨 完整角色設計 Skill（融合版）

> 融合了表情、體型、比例、形狀語言、年齡、性格等18個子技能的精華

---

## 1️⃣ 剪影設計 (Silhouette)

### 核心原則
- **一眼可辨**：僅憑輪廓就能認出角色
- **形狀語言**：
  - 圓形 → 友善、可愛（如：胖墩墩的配角）
  - 方形 → 穩重、力量（如：戰士、父親）
  - 三角形 → 危險、神秘（如：反派）
  - 混合 → 複雜性格

### 輸出格式
```json
{
  "silhouette": {
    "dominant_shape": "圓形/方形/三角形",
    "distinctive_feature": "獨特特徵（如：斗篷、長髮、帽子）",
    "recognition_distance": "能在多遠識別"
  }
}
```

## 2️⃣ 體型與比例 (Body Type & Proportion)

### 頭身比
- 2-3頭身：Q版、可愛風
- 5-6頭身：寫實兒童
- 7-8頭身：標準成人
- 9頭身+：英雄、模特

### 體型類型
- **瘦削型**：知識分子、藝術家
- **健壯型**：運動員、戰士
- **圓潤型**：廚師、慈祥長輩
- **高挑型**：貴族、神秘人

### 輸出格式
```json
{
  "body": {
    "head_ratio": "7頭身",
    "body_type": "健壯型",
    "height": "180cm",
    "build": "肌肉線條明顯但不誇張"
  }
}
```

## 3️⃣ 面部設計 (Face Design)

### 臉型
- 圓臉：親和、年輕
- 方臉：堅毅、可靠
- 瓜子臉：秀氣、聰明
- 長臉：成熟、嚴肅

### 五官設計
- **眼睛**：大眼=天真，細眼=精明，下垂眼=溫柔
- **眉毛**：濃眉=剛毅，細眉=細膩，挑眉=傲嬌
- **鼻子**：高挺=高貴，圓鼻=可愛，鷹勾鼻=陰險
- **嘴巴**：厚唇=熱情，薄唇=冷淡，上揚=樂觀

### 輸出格式
```json
{
  "face": {
    "shape": "方臉",
    "eyes": "深邃的眼窩，瞳色琥珀",
    "eyebrows": "濃眉，微微上揚",
    "nose": "高挺直鼻",
    "mouth": "薄唇，常抿著",
    "distinguishing_marks": "左眉有疤"
  }
}
```

## 4️⃣ 表情設計 (Expression)

### 基礎6情緒
1. 😊 喜 - 眉毛上揚，嘴角上揚
2. 😢 悲 - 眉毛下垂，嘴角下拉
3. 😠 怒 - 眉毛皺起，牙關緊咬
4. 😨 懼 - 眼睛睜大，嘴巴微張
5. 🤢 厭 - 鼻子皺起，嘴角下拉
6. 😮 驚 - 眼睛睜圓，嘴巴大張

### 角色專屬表情
每個角色應有1-2個標誌性表情：
- 主角：堅定的微笑
- 反派：陰險的冷笑
- 配角：誇張的驚訝

### 輸出格式
```json
{
  "expressions": {
    "default": "平靜但帶著警覺",
    "signature": "戰鬥時的專注眼神",
    "range": ["冷靜", "憤怒", "罕見的溫柔"]
  }
}
```

## 5️⃣ 年齡表現 (Age)

### 年齡特徵
- **兒童**：大頭大眼，圓潤，活潑姿態
- **青少年**：瘦長，表情多變，略顯青澀
- **青年**：比例標準，精力充沛
- **中年**：略有皺紋，體態略寬，沉穩
- **老年**：皺紋明顯，體態縮小，眼神深邃

### 輸出格式
```json
{
  "age": {
    "actual_age": 35,
    "apparent_age": "看起來比實際年輕，約30歲",
    "age_markers": ["眼角細紋", "成熟的氣質"],
    "aging_direction": "隨劇情會顯得更疲憊"
  }
}
```

## 6️⃣ 姿態與肢體語言 (Posture)

### 常見姿態類型
- **自信**：挺胸抬頭，雙手叉腰
- **內向**：縮肩，避免眼神接觸
- **傲慢**：下巴抬高，俯視他人
- **親和**：微微前傾，雙手打開

### 角色專屬姿態
```json
{
  "posture": {
    "standing": "雙腳與肩同寬，重心略向前",
    "sitting": "背部挺直，雙手交疊",
    "walking": "步伐堅定有力",
    "signature_gesture": "思考時會摸下巴"
  }
}
```

## 7️⃣ 服裝道具 (Costume & Props)

### 服裝設計原則
- 反映職業/身份
- 體現性格特點
- 方便動作表演
- 有標誌性元素

### 輸出格式
```json
{
  "costume": {
    "daily_wear": "黑色風衣，白襯衫，牛仔褲",
    "formal_wear": "深藍西裝",
    "battle_gear": "輕便戰術裝",
    "signature_item": "父親留下的懷錶",
    "color_palette": ["黑", "白", "深藍"]
  }
}
```
