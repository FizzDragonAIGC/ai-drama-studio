# 🎬 插入镜头 Skill
> 来源：《电影剪辑技巧》Karel Reisz + 经典剪辑分析

---

## 📐 什么是Insert Shot（插入镜头）

**定义**：插入到主场景中的特写或细节镜头，通常展示物品、手部动作或细节

### 插入镜头的作用
| 作用 | 说明 |
|------|------|
| **信息传递** | 展示重要细节 |
| **强调** | 吸引注意力 |
| **剪辑桥梁** | 连接两个镜头 |
| **时间控制** | 延长或压缩时间感 |


## 📋 插入镜头设计

### 时长选择
| 内容 | 建议时长 | 原因 |
|------|----------|------|
| 简单物品 | 0.5-1秒 | 一眼就懂 |
| 文字信息 | 2-4秒 | 需要阅读时间 |
| 情感细节 | 1-3秒 | 让观众感受 |
| 关键线索 | 2-3秒 | 确保注意到 |

### 构图原则
```
插入镜头的构图原则：
1. 简洁 - 只看到需要的
2. 清晰 - 信息一目了然
3. 聚焦 - 浅景深或干净背景
4. 相关 - 与上下文关联
```

### 插入时机
```
❌ 不好：突然插入，毫无铺垫
✓ 好：角色看向物品 → 插入物品

❌ 不好：插入太久打断节奏
✓ 好：足够读取信息后立即切走
```

## 🎯 插入镜头示例

### 示例1：手机信息
```
镜头1：角色听到手机响
镜头2：INSERT - 手机屏幕显示"爸爸：妈妈住院了"
镜头3：角色震惊表情

→ 观众和角色同时获取信息
```

### 示例2：紧张细节
```
镜头1：两人对峙对话
镜头2：INSERT - 角色手慢慢握向藏在背后的刀
镜头3：对方说话

→ 观众知道角色不知道的事（悬念）
```

### 示例3：时间跳跃
```
镜头1：角色开始工作
镜头2：INSERT - 时钟指向9点
镜头3：INSERT - 时钟指向6点（同构图）
镜头4：角色完成工作

→ 压缩了一天的时间
```

## 🎬 AI番剧应用

### 插入镜头Prompt模板
```
物品插入：
"extreme close-up, [物品], [状态细节],
sharp focus, simple background, insert shot"

动作插入：
"close-up hands, [动作描述],
detail shot, [情绪暗示]"

环境插入：
"close-up [环境细节], [时间/氛围信息],
establishing detail, insert shot"
```

## 💡 注意事项

### Do's
- ✅ 有明确的叙事目的
- ✅ 与主场景视觉风格一致
- ✅ 时长恰当不拖沓
- ✅ 位置安排合理

### Don'ts
- ❌ 无意义的插入
- ❌ 打断情感高潮
- ❌ 光线风格突然变化
- ❌ 过于频繁使用
