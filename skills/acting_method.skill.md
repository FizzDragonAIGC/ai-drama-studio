# 🎭 Skill: 方法派表演 (Method Acting)

> 來源：Lee Strasberg, Stella Adler, Sanford Meisner

## 三大流派

### 1. 斯特拉斯伯格派 (Strasberg)
**核心：情感記憶**
- 挖掘演員自身的真實情感經歷
- 將個人創傷/喜悅移植到角色
- 適合：內心戲、情感爆發場景

### 2. 阿德勒派 (Adler)
**核心：想像力與情境**
- 不依賴個人經歷，而是想像角色的經歷
- 深入研究角色的社會/歷史背景
- 適合：年代戲、與演員經歷差距大的角色

### 3. 邁斯納派 (Meisner)
**核心：真實反應**
- 「活在當下」，真實回應對手
- 重複練習建立本能反應
- 適合：對手戲、即興感強的場景

## 角色準備七步法

### Step 1: 背景研究
- 角色的時代、職業、階級
- 相關的歷史/文化資料

### Step 2: 角色傳記
- 角色出生到故事開始前的完整人生
- 包括未在劇本中出現的細節

### Step 3: 身體改造
- 角色的體態、步態、手勢習慣
- 職業病、身體特徵

### Step 4: 聲音設計
- 口音、語速、音調
- 特殊的說話習慣、口頭禪

### Step 5: 心理地圖
- 角色的恐懼、慾望、創傷
- 防禦機制、行為模式

### Step 6: 關係網絡
- 與其他角色的關係史
- 潛在的衝突點和親密點

### Step 7: 當下目標
- 每場戲的具體目標
- 如何與超級任務連接

## AI Prompt 應用

### 深度角色描述
```
[角色名]: [年齡] [職業/身份]
Physical: [身體特徵、姿態、特殊習慣]
Psychological: [核心創傷/慾望/恐懼]
Behavioral: [標誌性動作、說話方式]
Current state: [此刻的情緒/目標/緊張感]
```

### 示例
```
蕃長: 45歲 大食商人船隊領袖
Physical: 高大魁梧，絡腮鬍，走路沉穩如船，左肩微傾（舊傷）
Psychological: 喪子之痛深埋心底，對少年有複雜父性投射
Behavioral: 說話前必先沉默三秒，眼神銳利但偶爾飄向遠方
Current state: 表面威嚴實則焦慮，察覺船上有異常
```
