# 信息植入 Skill
> 来源：《故事》Robert McKee、编剧实战技巧
> 适用智能体：编剧助手、世界观设计师

## 核心理念

**"最好的说明是观众没注意到的说明"** —— 信息融入故事，而非打断故事

信息植入（Exposition）是编剧最难的技巧之一：如何让观众知道必要信息，又不觉得被说教？

---

## 核心理论

### 信息植入的悖论

```
观众需要知道 → 但不想被告知
作者需要说明 → 但不能直接说明
```

### 信息类型

| 类型 | 内容 | 难度 |
|------|------|------|
| 背景信息 | 世界观设定、历史 | ★★★ |
| 角色信息 | 身份、关系、过去 | ★★ |
| 规则信息 | 魔法体系、社会规则 | ★★★★ |
| 情节信息 | 之前发生了什么 | ★★ |
| 情感信息 | 角色内心状态 | ★ |

### 信息植入的原则

**1. 延迟原则**
- 信息在需要时才给
- 早给的信息会被遗忘

**2. 冲突包装原则**
- 通过冲突传递信息
- 争吵中揭示关系

**3. 视觉优先原则**
- 能演就不说
- Show don't tell

**4. 重复强化原则**
- 重要信息出现三次
- 视觉+台词+动作

## 实操技巧

### 信息植入十大技巧

**1. 新手角色**
让不懂的角色提问，让懂的角色解释
```
新转学生："这个学校的规矩是什么？"
老生："你不知道吗？这里……"
```

**2. 冲突揭示**
通过争吵/对抗传递信息
```
        弟弟
    你凭什么管我！
        哥哥
    因为爸妈死的时候，让我照顾你！
```

**3. 视觉线索**
用画面传递信息
```
墙上挂着全家福——但父亲的脸被划掉了。
```

**4. 角色行为**
通过习惯性动作揭示背景
```
他条件反射地躲开挥来的手——
然后才意识到对方只是想拍他肩膀。
(暗示：曾被虐待)
```

**5. 环境叙事**
场景本身传递信息
```
房间里堆满奖杯——但上面全是灰尘。
(暗示：曾经辉煌，现已放弃)
```

**6. 中断式说明**
开始说明，被打断
```
        老师
    关于魔法的第一定律——
爆炸声！窗户碎裂。
```

**7. 误解与纠正**
通过误解引出正确信息
```
        A
    你是独生子？
        B
    ……我有个哥哥。他死了。
```

**8. 闪回**
关键时刻的回忆片段
- 短而精准
- 有情感触发点

**9. 文字画面**
新闻、报纸、手机信息
```
屏幕上：紧急通告——A市全面封锁
```

**10. 旁白（慎用）**
番剧特有的画外音说明
- 用于大量世界观
- 最好是角色视角的旁白

## Prompt模板

### 信息植入重写

```
请将以下直白的信息说明改写成自然的场景：

需要传达的信息：[具体信息]
当前场景：[场景背景]
在场人物：[角色列表]

要求：
1. 不使用"顺便说一下""你知道吗"等说明式开场
2. 通过冲突/行动/视觉传递信息
3. 信息传递要自然，不生硬
4. 保持场景的戏剧性
```

### 世界观说明设计

```
为以下世界观设计信息植入方案：

世界观设定：[你的世界观]
需要观众了解的关键信息：
1. [信息1]
2. [信息2]
3. [信息3]

请设计：
1. 每条信息使用哪种植入技巧
2. 在哪个场景中植入
3. 通过哪个角色传递
4. 具体的场景片段示例
```

### 背景信息分配

```
以下是需要在第一集传达的背景信息：
[列出所有信息]

请设计12分钟内的信息分配方案：
1. 哪些信息必须在开场5分钟内给
2. 哪些可以延迟到后半段
3. 哪些可以推迟到第2-3集
4. 每条信息的植入方式
```

## 检查清单

### 信息植入检查
- [ ] 是否真的需要这条信息？
- [ ] 观众在这个时间点需要知道吗？
- [ ] 有没有更自然的传达方式？
- [ ] 是否依赖"As you know"式说明？
- [ ] 能用视觉代替对话吗？

### 说明段落检查
- [ ] 有冲突包装吗？
- [ ] 角色有动机说这些话吗？
- [ ] 长度是否合适？
- [ ] 会打断故事节奏吗？

## 案例分析

### 案例：《进击的巨人》世界观植入

**需要传达的信息**：
1. 人类被巨人逼入三重城墙
2. 已经100年和平
3. 巨人会吃人
4. 调查兵团探索墙外

**植入方法分析**：

| 信息 | 技巧 | 实现方式 |
|------|------|----------|
| 三重城墙 | 视觉+旁白 | 开场航拍，艾伦旁白 |
| 100年和平 | 日常对话 | 训练时提到 |
| 巨人吃人 | 行动展示 | 开场母亲被吃 |
| 调查兵团 | 新手角色 | 艾伦仰望归来的兵团 |

**高明之处**：
- 用"巨人出现"的危机包装信息
- 艾伦的渴望为信息传递提供理由
- 视觉冲击让信息刻骨铭心

### 反面教材

**❌ 生硬的说明**：
```
        老师
    同学们，如你们所知，我们的世界有三重城墙，
    分别是玛利亚、罗塞和席娜。一百年前……
```

**✅ 自然的植入**：
```
艾伦趴在窗口，仰望远处的城墙。
        艾伦（旁白）
    一百年了。
墙的阴影覆盖整个城市。
        艾伦（旁白）
    人类缩在这三层牢笼里——假装安全。
他握紧拳头。
        艾伦（旁白）
    但我不想死在这里。
```

## 常见错误

### 1. "As You Know" 综合症
```
❌ "作为你最好的朋友，我知道你父亲去世三年了……"
```
角色不会对对方说对方已知的事

### 2. 信息过载
一个场景塞入太多背景信息

### 3. 无动机说明
角色没有理由在此时说这些

### 4. 低估观众
解释观众已经理解的内容
