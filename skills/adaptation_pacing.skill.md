- **技能ID**: adaptation_pacing

## 核心理念

節奏 = 信息投放的時機控制
- 不是「快就好」或「慢就好」
- 而是「該快時快，該慢時慢」

## 節奏曲線設計

### 理想的集節奏曲線
```
    ▲ 緊張度
    │     ●          ●●●（高潮）
    │    / \        /    \
    │   /   \      /      \
    │  /     \    /        \
    │ /       \  /          \
    │●         ●●            ●
    └─────────────────────────→ 時間
     開場  發展  中點  升級  結尾
```

### 季節奏曲線（12集）
```
集1-2：建立，節奏較慢（讓觀眾進入）
集3-4：加速，開始有衝突
集5-6：維持，發展人物
集7（中點）：大轉折，高潮
集8-9：低谷，情緒下沉
集10-11：快速升級
集12：最高潮+解決
```

## 節奏控制元素

### 加快節奏
```
技巧：
- 縮短場景時長
- 增加場景數量
- 快速剪輯
- 減少對話
- 增加動作
- 使用音樂

效果：緊張/興奮/急迫
```

### 放慢節奏
```
技巧：
- 延長場景時長
- 減少場景數量
- 長鏡頭
- 允許沉默
- 環境鏡頭
- 細節特寫

效果：沉思/浪漫/悲傷
```

### 節奏突變
```
慢→快：
- 突發事件
- 驚嚇效果
- 追逐開始

快→慢：
- 危機解除
- 情緒爆發後
- 大場面後的餘韻
```

## 場景時長規劃

### 場景長度指南
```
極短（<30秒）：過渡/氛圍
短（30秒-2分鐘）：常規敘事
中（2-5分鐘）：重要場景
長（5-10分鐘）：高潮/關鍵對話
超長（>10分鐘）：極少使用

24分鐘動畫：
- 10-15個場景
- 平均1.5-2分鐘/場景
- 最長場景不超過5分鐘
```

### 場景數量與節奏
```
快節奏集：15-20個場景
中等節奏集：10-15個場景
慢節奏集：5-10個場景
```

## 信息投放節奏

### 懸念設置
```
規則：每5-7分鐘需要一個小懸念
- 「他去哪裡了？」
- 「她為什麼這樣說？」
- 「接下來會發生什麼？」

大懸念：每集結尾+季中點
```

### 信息揭露
```
漸進式揭露：
- 先給暗示
- 再給線索
- 最後揭曉

效果：觀眾有參與感

突然揭露：
- 沒有鋪墊
- 直接揭曉

效果：震驚/意外
（慎用，不要太多）
```

## 情感節奏

### 情緒波動圖
```
集1：好奇→緊張→小勝利
集2：輕鬆→衝突→擔憂
集3：希望→挫折→不安
集4：憤怒→覺醒→決心
...

規則：
- 不要連續兩集同一情緒
- 高潮後需要喘息
- 悲傷後需要希望
```

### 喜劇緩和
```
在緊張或悲傷段落中
插入適當的幽默時刻

功能：
- 讓觀眾喘息
- 提高情感對比
- 角色更立體

注意：
- 不要破壞氛圍
- 符合角色性格
- 時機要準確
```

## 改編節奏調整

### 原著慢→改編加速
```
方法：
- 合併場景
- 刪除重複
- 對話壓縮
- 蒙太奇過渡
```

### 原著快→改編舒緩
```
方法：
- 添加情感場景
- 展開環境描寫
- 深化人物關係
- 增加反應時刻
```
