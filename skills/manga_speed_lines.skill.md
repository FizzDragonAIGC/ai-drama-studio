## 📐 速度线基础

### 什么是速度线
```
物体移动方向 →
══════════════════════════→
══════════════════════════→  速度线（平行于运动方向）
══════════════════════════→
        ┌─────┐
        │ 物体│→
        └─────┘
```

**作用**：表现运动、速度、动态感

---

## 🎯 速度线类型

### 1. 平行速度线
```
════════════════════════→
════════════════════════→
════════════════════════→
        【物体】
════════════════════════→
════════════════════════→
```
**用途**：直线运动、冲刺

### 2. 发散速度线
```
        ╲  │  ╱
         ╲ │ ╱
          ╲│╱
     ═════【物体】═════
          ╱│╲
         ╱ │ ╲
        ╱  │  ╲
```
**用途**：冲击、爆发

### 3. 收敛速度线
```
    ╱              ╲
   ╱                ╲
  ╱                  ╲
 ═════→【物体】←═════
  ╲                  ╱
   ╲                ╱
    ╲              ╱
```
**用途**：聚焦、集中

### 4. 曲线速度线
```
    ～～～～～～～╲
      ～～～～～～～╲
        ～～～～～～～【物体】
      ～～～～～～～╱
    ～～～～～～～╱
```
**用途**：曲线运动、挥舞

### 5. 螺旋速度线
```
        ╱ ╲
       │   │
    ╱──┘   └──╲
   │   【中心】  │
    ╲──┐   ┌──╱
       │   │
        ╲ ╱
```
**用途**：旋转、漩涡

---

## 📊 速度线密度与效果

| 密度 | 效果 | 适用场景 |
|------|------|----------|
| **稀疏** | 轻柔、缓慢 | 普通移动 |
| **中等** | 快速、有力 | 跑步、挥拳 |
| **密集** | 极速、爆发 | 冲刺、必杀技 |
| **渐变** | 加速/减速 | 起步、刹车 |

---

## 🔧 速度线绘制参数

### 线条特征

| 参数 | 慢速 | 中速 | 高速 |
|------|------|------|------|
| **粗细** | 细 | 中 | 粗 |
| **间距** | 宽 | 中 | 密 |
| **长度** | 短 | 中 | 长 |
| **尖端** | 圆润 | 渐尖 | 锐利 |

### 粗细变化规律
```
粗 ════════════════════▶ 细
起点                    终点
（物体位置）            （运动方向）
```

## 📐 速度线与背景关系

### 物体运动（背景固定）
```
【固定背景】
┌─────────────────────┐
│ 树 │ 房 │           │
│    │    │ →【人】   │
│════│════│════════→  │
│════│════│════════→  │
└─────────────────────┘
速度线在物体后方
```

### 背景流动（物体固定）
```
【物体在画面中心不动】
←════════════════════════
←════════════════════════
←═══════【人】═══════════
←════════════════════════
←════════════════════════
速度线铺满背景
```

## 🎨 速度线风格变体

| 风格 | 特征 | 适用 |
|------|------|------|
| **传统** | 工整平行线 | 经典漫画 |
| **狂野** | 粗细不一、略带抖动 | 热血动作 |
| **优雅** | 细腻流畅曲线 | 少女漫画 |
| **机械** | 精准等距 | 科幻漫画 |
| **手绘感** | 略有颤抖 | 写实风格 |

## 📋 速度线检查表

- [ ] 方向与运动一致
- [ ] 密度符合速度
- [ ] 粗细有变化
- [ ] 不遮挡主体
- [ ] 增强而非干扰画面

---

## ⚠️ 常见错误

| 错误 | 问题 |
|------|------|
| **方向错误** | 与运动方向不符 |
| **过度使用** | 画面杂乱 |
| **密度不当** | 速度感不对 |
| **遮挡主体** | 看不清角色 |
| **风格不统一** | 画面不协调 |
