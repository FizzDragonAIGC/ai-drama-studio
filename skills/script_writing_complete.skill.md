# 完整剧本写作 (Complete Script Writing)

## 概述
从大纲到完整剧本的写作流程，包括场景描写、对白、动作指示。

## 剧本格式标准

### AI番剧剧本格式
```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
第X集 第Y场
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【场景】内景/外景 - 地点 - 时间

【环境描述】
简短描述环境氛围，2-3句话。

【动作】
角色名 + 动作描述

【对白】
角色名：台词内容
（括号内是语气或小动作）

【镜头建议】
景别 + 角度 + 运动

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### 示例
```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
第1集 第3场
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【场景】内景 - 地下酒吧 - 夜

【环境描述】
昏暗的霓虹灯在烟雾中闪烁。全息广告在墙上循环播放。
角落里几个黑客低头盯着脑机接口。

【动作】
零 推开厚重的铁门，扫视四周。他的机械眼发出微弱红光。

【对白】
酒保Luna：（没抬头，擦着杯子）
又来找工作？

零：（在吧台坐下）
我需要钱。有人要我的命。

Luna：（轻笑）
谁不想要呢。

【动作】
Luna 从柜台下拿出一个全息芯片，滑向零。

【对白】
Luna：
有个活。偷一个AI的核心数据。
报酬：一百万信用点。

【动作】
零 拿起芯片，机械眼闪烁着扫描。

【镜头建议】
- 开场：中景，零推门，逆光剪影
- 对话：双人过肩镜头
- 芯片：特写，红光扫描效果
```

## 场景描写原则

### 只写能"看到"的
❌ 错误：他心里很紧张
✅ 正确：他的手微微颤抖

❌ 错误：她想起了童年
✅ 正确：她看着窗外，眼神空洞（闪回：童年画面）

### 动词优先
❌ 错误：他非常快速地走过去
✅ 正确：他冲了过去

❌ 错误：她慢慢地、小心翼翼地打开门
✅ 正确：她屏住呼吸，轻轻推开门缝

### 建立氛围
- 用天气暗示情绪
- 用光线暗示基调
- 用声音建立空间

## 对白写作原则

### 潜台词优先
❌ 直白：我爱你，但我害怕被伤害
✅ 潜台词：你说你会回来的。上次你也这么说。

### 角色声音差异化
- 每个角色有独特的口头禅
- 不同教育背景=不同用词
- 性格决定说话方式

### 对白功能
1. **推进情节** - 传递新信息
2. **展现角色** - 体现性格
3. **制造冲突** - 观点碰撞
4. **暗示主题** - 金句台词

## 场景功能分类

### 信息场景
- 目的：传递必要信息
- 技巧：通过冲突中传递
- 避免：生硬的解释

### 情感场景
- 目的：建立情感连接
- 技巧：用细节打动人
- 节奏：可以放慢

### 动作场景
- 目的：视觉刺激
- 技巧：清晰的空间关系
- 节奏：加快

### 转折场景
- 目的：改变故事方向
- 技巧：铺垫+意外
- 节奏：先慢后快

## Prompt模板

### 场景生成
```
根据以下大纲写一个完整场景：

【场景信息】
- 位置：
- 时间：
- 出场角色：
- 场景功能：信息/情感/动作/转折

【需要完成】
- 情节点：
- 传递信息：
- 情感目标：

【风格】
- 对白风格：含蓄/直接
- 节奏：快/中/慢
- 氛围：

请按照标准剧本格式输出。
```

### 对白优化
```
优化以下对白，使其更有潜台词：

原对白：
{original_dialogue}

角色背景：
A：{character_a_background}
B：{character_b_background}

他们之间的关系：
{relationship}

这场戏的潜在冲突：
{underlying_conflict}

请重写，要求：
1. 不直接说出情感
2. 用行动和小细节暗示
3. 保持角色声音差异
```

## 检查清单

### 场景检查
- [ ] 场景有明确功能吗？
- [ ] 能删掉这场戏吗？
- [ ] 氛围建立了吗？
- [ ] 动作描写简洁吗？

### 对白检查
- [ ] 每句话都必要吗？
- [ ] 有潜台词吗？
- [ ] 角色声音有差异吗？
- [ ] 读出来自然吗？

### 整集检查
- [ ] 节奏有变化吗？
- [ ] 信息量合适吗？
- [ ] 情感有起伏吗？
- [ ] 结尾有钩子吗？
