## 📋 追问问题设计

### 第一层：创作定位
- 创作意图：[ ]商业成功 [ ]艺术突破 [ ]社会影响 [ ]文化传承
- 情感核心：希望观众体验到____的情感深度
- 市场空位：发现____类型存在____缺口

### 第二层：艺术追求
- 视觉风格愿景：类似____导演的____美学
- 叙事创新点：在____方面突破传统模式
- 文化独特性：融入____文化的____元素

### 第三层：商业考量
- 目标观众画像：年龄____，地域____，消费习惯____
- 竞品分析：区别于《____》的独特价值是____
- 衍生潜力：[ ]系列化 [ ]跨媒体 [ ]周边开发 [ ]沉浸体验

---

## ✅ 概念生成标准
- 必含要素：高概念核心、情感引擎、视觉抓手、市场定位
- 创新要求：至少在1个维度突破同类型作品
- 情感层次：包含至少3种情感体验的交织
- 文化考量：尊重文化真实性，避免刻板印象

---

## 📊 输出格式

### 1. 专业概念方案书（3套）
```
• 概念标题：《____》（含英文名）
• 一句话定位：____风格的____故事，探讨____主题
• 情感结构：从____到____的情感旅程，关键转折在____
• 视觉标识：核心视觉符号是____，色彩基调为____
• 创新亮点：在____方面实现突破，因为____
• 市场分析：填补了____空白，预估____市场接受度
```

### 2. 详细开发文档
- 完整概念阐述（800-1200字）
- 情感体验路线图
- 视觉风格参考集（3-5部参考作品）
- 风险评估与规避策略
