# 🎬 潜台词 Skill
> 来源：《Story》Robert McKee + 《编剧的艺术》Lajos Egri

---

## 📐 什么是Subtext（潜台词）

**定义**：台词背后真正要表达的意思，角色说的和想的不一样

### 为什么需要潜台词
| 原因 | 说明 |
|------|------|
| **真实感** | 现实中人们很少直说 |
| **深度** | 增加角色复杂度 |
| **张力** | 知道真相的观众更紧张 |
| **参与感** | 观众需要"读懂" |

---

## 🎯 潜台词类型

### 1. 情感型潜台词
```
说的："祝你们幸福。"
想的："我恨你抢走了他。"
```

### 2. 目的型潜台词
```
说的："你最近看起来累了。"
想的："辞职吧，我要你的位置。"
```

### 3. 自我保护型
```
说的："我没事。"
想的："我快崩溃了但不能让你知道。"
```

### 4. 社交型潜台词
```
说的："改天一起吃饭。"
想的："我们永远不会吃这顿饭。"
```

## 🔧 潜台词表现技巧

### 视觉表现
| 技巧 | 示例 |
|------|------|
| **表情与台词相反** | 微笑说"我很高兴"但眼中含泪 |
| **手部动作** | 说"我不紧张"但在扭手指 |
| **视线躲避** | 说"我没骗你"但不敢对视 |
| **身体语言** | 说"欢迎"但交叉双臂 |

### 听觉表现
| 技巧 | 示例 |
|------|------|
| **停顿** | "我...我没事" (停顿泄露犹豫) |
| **语调变化** | 音调上扬表示不确定 |
| **语速变化** | 说谎时说得太快或太慢 |
| **音量变化** | 突然降低音量 |

### 剪辑表现
| 技巧 | 示例 |
|------|------|
| **反应镜头** | 切到听者怀疑的表情 |
| **插入镜头** | 切到揭示真相的细节 |
| **时长控制** | 反应镜头停留更久 |

## 🎯 潜台词场景示例

### 示例1：分手场景
```
场景：男女主角在咖啡店

TEXT: "希望你找到更适合你的人。"
SUBTEXT: "我还爱你但我不够好。"

表现：
- 表情：努力微笑但眼眶泛红
- 动作：不停搅动咖啡
- 语调：声音略微颤抖
- 镜头：特写她的手在发抖
```

### 示例2：职场对话
```
场景：上司与下属

TEXT: "你的方案很有创意。"
SUBTEXT: "这方案太天真了，永远不会通过。"

表现：
- 表情：礼貌微笑不达眼底
- 动作：把方案放到一边
- 语调：过于热情反而虚假
- 镜头：切到下属失望的反应
```

## 🎬 AI番剧应用

### 潜台词的视觉表达
由于AI番剧无法有真人表演，需要通过：
1. **面部表情** - 选择正确的表情
2. **肢体语言** - 姿势传达真实情感
3. **环境暗示** - 场景设计暗示情感

### Prompt技巧
```
表面情感 + 暗示情感的细节

示例：
"girl smiling while saying goodbye, 
but her hands are clenched tightly,
bittersweet expression, hiding sadness"
```

### 对话字幕设计
```
[台词]: "没什么，只是想起一些事。"
[画面]: 角色微笑，但目光望向远方
[音效]: 轻微叹息声

→ 潜台词通过画面传达
```

## 💡 潜台词写作技巧

### Do's
- ✅ 让角色有不能直说的理由
- ✅ 通过行为暗示真实意图
- ✅ 让观众比角色先知道真相
- ✅ 在关键时刻才揭示真相

### Don'ts
- ❌ 角色直接说出心里话
- ❌ 潜台词太隐晦无人能懂
- ❌ 每句话都有潜台词（太累）
- ❌ 潜台词与剧情无关

---

## 📐 潜台词层级

```
Level 1：观众能猜到（大众向）
"我很好" → 明显不好（表情）

Level 2：需要思考（标准）
"改天再说" → 永远不会说（需理解语境）

Level 3：重看才懂（精品向）
伏笔式潜台词，结局揭示后才明白
```
