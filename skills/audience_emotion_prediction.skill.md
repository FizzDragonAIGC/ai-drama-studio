## 📐 情感预测模型

### 情绪反应类型
| 情绪 | 触发条件 | 观众表现 |
|------|----------|----------|
| 爽 | 主角打脸、逆袭 | "卧槽！太爽了！" |
| 燃 | 热血、突破极限 | 起鸡皮疙瘩、握拳 |
| 甜 | 浪漫、暧昧 | 嘴角上扬、脸红 |
| 虐 | 悲剧、分离 | 哭、难受 |
| 爆笑 | 意外、反差 | 笑出声 |
| 恐惧 | 未知、威胁 | 紧张、加速心跳 |
| 感动 | 真情、牺牲 | 热泪盈眶 |
| 惊讶 | 反转、揭示 | "我靠！没想到！" |

## 🎯 情绪曲线设计

### 理想情绪曲线
```
情绪强度
   │
   │         燃点      最终高潮
   │          ↗↘        ↗↘
   │      ↗      ↘    ↗    ↘
   │    ↗          ↘↗        ↘ 余韵
   │  ↗   小高潮                 ↘
   │ ↗                            ↘
 ──┼──────────────────────────────────→ 时间
   │ 开篇钩子                    结局
```

### 节奏把控
| 阶段 | 情绪设计 | 时长 |
|------|----------|------|
| 开篇 | 钩子、好奇 | 前30秒 |
| 铺垫 | 期待、好奇 | 20% |
| 发展 | 起伏、小高潮 | 50% |
| 高潮 | 最强情绪 | 20% |
| 收尾 | 余韵、期待下集 | 10% |

## 📊 观众反应预测表

### 场景→情绪映射
| 场景类型 | 预期情绪 | 强度 | 弹幕预测 |
|----------|----------|------|----------|
| 主角被欺负 | 愤怒/期待 | ★★★ | "快打他！" |
| 主角反杀 | 爽/燃 | ★★★★★ | "卧槽！舒服！" |
| 告白成功 | 甜/感动 | ★★★★ | "啊啊啊在一起！" |
| 分离/死亡 | 悲伤/震惊 | ★★★★★ | "哭死我了" |
| 搞笑场面 | 欢乐 | ★★★ | "哈哈哈哈" |
| 伏笔回收 | 惊讶/满足 | ★★★★ | "原来如此！" |
| 悬念时刻 | 紧张/期待 | ★★★★ | "然后呢？！" |

### 弹幕/评论预测
```
【预测模板】

场景：[描述场景]
预期情绪：[情绪类型]
情绪强度：★★★★☆

观众预期反应：
- 弹幕："[预测弹幕内容]"
- 表情：[观众可能的表情]
- 行为：[继续看/暂停/分享]

潜在风险：
- 如果[某条件]，可能导致[负面反应]
```

## 🎭 情绪设计技巧

### 爽感设计
```
公式：压抑 → 爆发
越压抑，爆发越爽

技巧：
1. 让主角先吃亏（观众憋着）
2. 让反派嚣张（观众更憋）
3. 关键时刻反杀（观众释放）
4. 反派表情要有反差（从嚣张到惊恐）
```

### 燃点设计
```
公式：绝境 + 信念 + 爆发

技巧：
1. 把主角逼到绝境
2. 闪回/信念强化
3. 配乐升调
4. 视觉爆发（变身/升级）
5. 最后一击
```

### 甜度设计
```
公式：铺垫 + 暧昧 + 确认

技巧：
1. 先建立好感
2. 制造暧昧瞬间（不小心碰到手）
3. 欲言又止
4. 终于确认（观众等不及了）
```

### 虐心设计
```
公式：美好 → 失去

技巧：
1. 先让观众爱上角色
2. 建立幸福时刻
3. 伏笔暗示
4. 悲剧发生
5. 回忆杀
```

## 📋 情绪检测清单

### 每场戏问自己
- [ ] 观众此刻应该感受什么？
- [ ] 有没有达到这个情绪？
- [ ] 情绪够不够强？
- [ ] 是否给了观众反应时间？
- [ ] 会不会太拖沓？
- [ ] 会不会太赶？

### 整集检测
- [ ] 有几个情绪高点？
- [ ] 开头有钩子吗？
- [ ] 中间有起伏吗？
- [ ] 结尾有悬念吗？
- [ ] 观众会想看下一集吗？

```
【观众情绪预测】

场景描述：
[详细描述这个场景]

目标情绪：
[希望观众感受什么]

请分析：
1. 当前设计能否达到目标情绪？(1-10分)
2. 观众可能的弹幕反应是什么？
3. 有什么潜在问题？
4. 如何增强情绪效果？
5. 修改建议是什么？
```
