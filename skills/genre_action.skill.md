## 📐 动作片核心元素

| 元素 | 特点 | AI Prompt |
|------|------|-----------|
| 🥋 **打斗** | 拳脚对决、武器搏斗 | fight scene, combat, martial arts |
| 💥 **爆炸** | 视觉冲击、火焰场面 | explosion, fire, debris flying |
| 🚗 **追逐** | 高速运动、紧张刺激 | car chase, pursuit, high speed |
| 🔫 **枪战** | 子弹交锋、战术配合 | gunfight, shootout, tactical |
| 🏃 **跑酷** | 攀爬跳跃、极限运动 | parkour, free running, acrobatic |

## 🎯 动作片视觉风格

### 镜头运动
| 技法 | 效果 | AI描述 |
|------|------|--------|
| 手持晃动 | 临场感强 | handheld camera, shaky cam |
| 跟拍长镜 | 连贯流畅 | tracking shot, one-take action |
| 快速剪辑 | 节奏紧凑 | fast cuts, rapid editing |
| 慢动作 | 强调关键 | slow motion, bullet time |
| 升格拍摄 | 细节放大 | high speed camera, detail shot |

### 动作场景构图
| 场景 | 构图特点 | AI描述 |
|------|----------|--------|
| 一对一 | 双人对峙 | two figures facing off, symmetrical |
| 群战 | 多角度混战 | melee, multiple combatants |
| Boss战 | 强弱对比 | size contrast, underdog vs giant |
| 逃亡 | 纵深透视 | deep perspective, running towards camera |

## 🔥 动作类型分解

### 格斗风格
| 风格 | 特点 | AI关键词 |
|------|------|----------|
| 拳击 | 直接有力 | boxing, punching, uppercut |
| 功夫 | 连招优美 | kung fu, flowing moves |
| 摔跤 | 力量对抗 | wrestling, grappling, throw |
| 街头斗殴 | 粗野真实 | street fight, brawl, raw |
| 剑术 | 利落致命 | sword fighting, blade clash |

### 爆破效果
| 类型 | 视觉效果 | AI描述 |
|------|----------|--------|
| 汽车爆炸 | 翻滚火焰 | car explosion, rolling fireball |
| 建筑崩塌 | 尘土飞扬 | building collapse, debris cloud |
| 手雷爆炸 | 冲击波 | grenade explosion, shockwave |
| 油罐爆炸 | 蘑菇云 | fuel tank explosion, mushroom cloud |

## 🎬 经典动作片结构

```
【三幕式动作片】

第一幕：建立
- 主角展示能力
- 初次小规模冲突
- 危机降临

第二幕：升级
- 多场动作场面递进
- 主角受挫
- 准备反击

第三幕：高潮
- 终极Boss战
- 最大规模动作场面
- 英雄胜利
```
