# 🎭 Skill: 潛台詞表演

> 來源：表演理論、導演學

## 什麼是潛台詞

**定義**：角色說的話背後真正的意思

```
台詞：「你來了。」
潛台詞A：「終於等到你了，我好擔心。」
潛台詞B：「你怎麼才來，我等得好煩。」
潛台詞C：「你不該來，這裡很危險。」
```

**同樣的台詞，不同的潛台詞 = 完全不同的表演**

## 潛台詞類型

### 1. 隱藏情感
- 說反話、壓抑真實感受
- 例：「我很好」（實際很痛苦）

### 2. 言外之意
- 暗示、旁敲側擊
- 例：「天氣不錯」（我想單獨和你說話）

### 3. 權力遊戲
- 控制、威脅、操縱
- 例：「請坐」（你逃不掉了）

### 4. 自我保護
- 逃避、轉移話題
- 例：「改天再說吧」（我不想面對）

## 潛台詞外化方式

### 眼神
| 潛台詞 | 眼神表現 |
|-------|---------|
| 說謊 | 躲避直視、眨眼頻繁 |
| 威脅 | 直視不移、瞳孔收縮 |
| 愛慕 | 柔和、停留時間長 |
| 恐懼 | 頻繁掃視、瞳孔放大 |

### 身體距離
| 潛台詞 | 距離變化 |
|-------|---------|
| 親近 | 主動靠近 |
| 排斥 | 後退、側身 |
| 控制 | 侵入私人空間 |
| 不確定 | 保持但想靠近 |

### 小動作
| 潛台詞 | 小動作 |
|-------|--------|
| 緊張 | 摸脖子、攥拳頭 |
| 說謊 | 摸鼻子、遮嘴 |
| 不耐煩 | 敲手指、看別處 |
| 掩飾 | 假笑、過度點頭 |

## 分鏡應用

### 標記方式
```
[鏡頭 #XX]
台詞：「我沒事。」
潛台詞：（我快要崩潰了）
表演指示：眼眶泛紅但強忍、聲音微顫、轉身避開對方視線
```

## AI Prompt 應用

```

[環境互動：通過環境表達內心]
```

