## 🔧 爆炸参数设计

```
【爆炸效果设计】

规模属性：
├─ 大小：小型→巨大
├─ 威力：轻微→毁灭性
├─ 范围：集中→广域
└─ 持续：瞬间→延续

视觉元素：
├─ 火球：颜色/大小
├─ 烟雾：颜色/密度
├─ 碎片：类型/数量
├─ 冲击波：可见度
└─ 闪光：强度/颜色

环境影响：
├─ 地面：坑洞/裂缝
├─ 周围：破坏程度
├─ 空气：热浪扭曲
└─ 声音：震耳欲聋
```

## 💡 爆炸表现技巧

### 时间分解
| 时刻 | 表现重点 |
|------|----------|
| 爆炸前 | 预兆、蓄力 |
| 爆炸瞬间 | 强闪光、白屏 |
| 爆炸中 | 火球膨胀 |
| 冲击波 | 空气扭曲 |
| 余波 | 烟雾、碎片 |

### 视觉增强
1. **镜头震动** - 冲击感
2. **闪白** - 强光效果
3. **慢动作** - 细节展示
4. **多角度** - 史诗感

### 环境反应
- 窗户破碎
- 地面震裂
- 物体飞散
- 人物被击飞
