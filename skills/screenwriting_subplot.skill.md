# 副线设计 Skill
> 来源：《故事》Robert McKee、《编剧的艺术》Lajos Egri
> 适用智能体：编剧助手、剧情设计师

## 核心理念

**"副线是主线的镜子"** —— 好的副线折射、对比、深化主线

副线（Subplot）不是填充，而是故事的立体化手段。

---

## 核心理论

### 副线的四大功能

| 功能 | 说明 | 示例 |
|------|------|------|
| 对照 | 与主线形成对比 | 主角追求名利，配角追求简单幸福 |
| 深化 | 强化主题表达 | 不同角色面对同一困境的不同选择 |
| 节奏 | 调节紧张节奏 | 紧张主线后插入轻松副线 |
| 补完 | 补充世界观/信息 | 通过配角视角展示更多世界 |

### 副线类型

```
┌─────────────────────────────────────────────┐
│  爱情副线 ── 最常见，软化主角形象           │
│  友情副线 ── 体现主角人际关系               │
│  成长副线 ── 配角的独立成长弧               │
│  镜像副线 ── 与主线形成镜像对照             │
│  悬念副线 ── 独立谜团，后与主线汇合         │
│  喜剧副线 ── 调节气氛，提供笑点             │
└─────────────────────────────────────────────┘
```

### 副线与主线的关系模式

**1. 平行模式**
```
主线：A ──────────────────────> 结局
副线：B ──────────────────────> 
     │        │        │
    主题     主题     主题共振
```

**2. 交汇模式**
```
主线：A ──────────\
                   > 汇合 ──> 高潮
副线：B ──────────/
```

**3. 因果模式**
```
副线：B ──> 结果 ──> 影响主线
主线：A ─────────────> 改变 ──> 结局
```

## 实操技巧

### 副线设计五步法

**Step 1：确定功能**
- 这条副线要实现什么？对照？深化？节奏？

**Step 2：选择载体**
- 由哪个角色承载？
- 最好是已存在的配角，而非新增角色

**Step 3：建立连接**
- 副线角色与主角的关系是什么？
- 副线事件如何影响主线？

**Step 4：设计弧光**
- 副线角色的起点和终点
- 变化要比主角弧光简单但清晰

**Step 5：规划节点**
- 副线在哪几集出现？
- 在第几集与主线交汇？

### 副线权重分配

**12集番剧建议**：
```
主线：60-70%
副线A（主要）：15-20%
副线B（次要）：10-15%
```

### 副线节奏安排

| 集数 | 副线处理 |
|------|----------|
| 1-2 | 副线人物登场，暗示其困境 |
| 3-4 | 副线问题显现 |
| 5-6 | 副线发展，与主线产生联系 |
| 7-8 | 副线危机 |
| 9-10 | 副线高潮/解决 |
| 11-12 | 副线服务于主线高潮 |

### 避免副线问题

| 问题 | 解决方案 |
|------|----------|
| 副线抢戏 | 压缩篇幅，收束到主角视角 |
| 副线无关 | 加强与主题的连接 |
| 副线中断 | 保持每2-3集出现一次 |
| 副线仓促 | 提前铺垫，给足发展空间 |

## Prompt模板

### 副线生成

```
请为以下主线设计3条副线：

主线故事：[你的主线描述]
主题：[核心主题]
主要角色：[列出主要角色]

要求：
1. 每条副线包含：
   - 载体角色
   - 核心冲突
   - 与主线的关系（对照/深化/节奏）
   - 关键节点（起/承/转/合）
2. 至少一条是对照式副线
3. 标注每条副线的权重建议
```

### 副线整合

```
检查以下主线与副线的整合度：

主线：[描述]
副线A：[描述]
副线B：[描述]

请分析：
1. 两条副线与主线主题的关联度
2. 副线之间是否有冲突或冗余
3. 建议的交汇节点
4. 权重分配建议
5. 可能的整合优化方案
```

### 镜像副线设计

```
为以下主角设计一个镜像副线：

主角：[描述]
主角的核心选择：[关键决定]
主题：[主题]

请设计：
1. 镜像角色：面对相同困境但做出相反选择
2. 镜像角色的背景
3. 两人的关系设定
4. 关键对照场景
5. 最终结局对比如何强化主题
```

## 检查清单

### 副线设计检查
- [ ] 每条副线都有明确功能？
- [ ] 副线角色有自己的弧光？
- [ ] 副线与主线有明确连接？
- [ ] 副线不喧宾夺主？
- [ ] 副线节奏分配合理？

### 副线整合检查
- [ ] 副线强化而非削弱主题？
- [ ] 副线与主线有交汇点？
- [ ] 副线收尾得当？
- [ ] 副线信息不冗余？

## 案例分析

### 案例：《钢之炼金术师》

**主线**：爱德华寻找贤者之石恢复弟弟身体

**副线分析**：

| 副线 | 载体 | 功能 | 与主题关系 |
|------|------|------|-----------|
| 休斯家庭线 | 休斯 | 深化 | 展示正常家庭vs兄弟的牺牲 |
| 斯卡复仇线 | 斯卡 | 对照 | 另一种应对悲剧的方式 |
| 军部阴谋线 | 马斯坦 | 悬念 | 揭示更大真相 |
| 温莉成长线 | 温莉 | 情感 | 等待与支持的力量 |

**镜像对照分析**：
- 爱德华vs斯卡
  - 相同：都失去了重要的人
  - 不同：爱德选择拯救，斯卡选择复仇
  - 主题证明：放下仇恨才能真正前进

**副线交汇设计**：
```
前期：各副线独立发展
中期：休斯事件牵连所有线
后期：所有角色汇聚对抗"父亲大人"
```

**经验提炼**：
1. 好的副线角色值得观众关心
2. 副线结局应服务主题
3. 多副线需要统一的收束点

## 高级技巧

### 副线的情感对位

在主线紧张时，副线可以：
- 提供喜剧缓冲
- 加剧观众担心（另一个角色也陷入危机）
- 提供希望（配角那边进展顺利）

### 副线的信息功能

通过副线揭示主线无法展示的信息：
- 反派视角
- 历史背景
- 世界观扩展

### 副线的情感投资

让观众投入副线角色：
- 给予独特魅力
- 创造共情时刻
- 但不要超过主角
