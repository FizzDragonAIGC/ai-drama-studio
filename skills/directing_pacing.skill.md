# 🎬 节奏把控 Skill
> 来源：《剪辑的语法》Roy Thompson + 《电影剪辑技巧》

---

## 📐 什么是Pacing（节奏）

**定义**：影片中事件发生和展开的速度感

### 节奏的构成要素
| 要素 | 影响 |
|------|------|
| **镜头时长** | 长镜头=慢，短镜头=快 |
| **剪辑频率** | 切换越多越紧张 |
| **动作速度** | 角色移动快慢 |
| **对白密度** | 台词多少 |
| **音乐节拍** | BGM的BPM |

## 🔧 节奏控制技巧

### 1. 镜头时长控制

| 情境 | 建议时长 | 效果 |
|------|----------|------|
| 建立镜头 | 3-5秒 | 让观众理解环境 |
| 对话中景 | 2-4秒 | 正常对话节奏 |
| 反应特写 | 1-2秒 | 捕捉表情 |
| 动作切换 | 0.5-1秒 | 紧张感 |
| 沉思镜头 | 5-10秒 | 内心世界 |

### 2. 剪辑节奏模式

```
稳定节奏：●──●──●──●──●──●
    用于：日常场景

加速节奏：●──●─●─●●●●●●
    用于：紧张升级

减速节奏：●●●●●─●──●───●────
    用于：高潮后平复

脉冲节奏：●●●──●●●──●●●──
    用于：动作场面
```

### 3. 呼吸点设计

在高速剪辑后，安排"呼吸点"：
- 远景镜头（3-5秒）
- 角色沉默时刻
- 环境空镜
- 反应镜头

## 🎯 节奏与类型片

| 类型 | 整体节奏 | 特点 |
|------|----------|------|

## 🎬 AI番剧应用

### 节奏规划表
```
| 场景 | 类型 | 节奏 | 平均镜头时长 |
|------|------|------|--------------|
| 01 | 日常 | 慢 | 3-4秒 |
| 02 | 对话 | 中 | 2-3秒 |
| 03 | 发现 | 加速 | 1-2秒 |
| 04 | 追逐 | 快 | 0.5-1秒 |
| 05 | 结束 | 减速 | 3-5秒 |
```

### Prompt中的节奏暗示
```
慢节奏: "contemplative, slow movement, peaceful"
中节奏: "casual, natural movement, everyday"
快节奏: "dynamic, quick movement, action-packed"
```

### 镜头数量估算
```
场景时长 ÷ 平均镜头时长 = 镜头数量

例：60秒紧张场景，平均1.5秒/镜头
60 ÷ 1.5 = 40个镜头

例：60秒平静场景，平均4秒/镜头
60 ÷ 4 = 15个镜头
```

## 💡 节奏常见问题

| 问题 | 原因 | 解决 |
|------|------|------|
| 感觉拖沓 | 镜头太长 | 缩短或切分 |
| 感觉仓促 | 没有呼吸点 | 加入空镜 |
| 感觉混乱 | 节奏无规律 | 建立模式 |
| 感觉平淡 | 节奏太均匀 | 增加变化 |
