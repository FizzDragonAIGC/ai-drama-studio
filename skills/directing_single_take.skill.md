# 🎬 长镜头 Skill
> 来源：《长镜头美学》André Bazin + 经典长镜头分析

---

## 📐 什么是长镜头（Single Take / Long Take）

**定义**：一个连续不剪切的镜头，时长远超普通镜头（通常>30秒）

### 长镜头的价值
| 价值 | 说明 |
|------|------|
| **真实感** | 连续时间 = 真实时间 |
| **沉浸感** | 观众跟随摄影机"在场" |
| **表演力** | 展示演员完整演技 |
| **技术炫技** | 展示调度能力 |

---

## 🎯 长镜头类型

### 1. 固定长镜头
```
┌─────────────────────────────────┐
│  📹 ────────────────→           │
│  (摄影机不动)                   │
│                                 │
│  人物在画面内活动              │
│  戏剧在镜头内完成              │
└─────────────────────────────────┘
```
**代表**: 小津安二郎

### 2. 移动长镜头
```
┌─────────────────────────────────┐
│  📹→ 📹→ 📹→ 📹→               │
│  (摄影机跟随)                   │
│                                 │
│  人物移动，镜头跟随            │
│  空间逐渐展开                   │
└─────────────────────────────────┘
```
**代表**: Goodfellas 后厨长镜头

### 3. 全片长镜头
```
整部电影看起来像一个镜头
实际由多个长镜头隐藏剪辑点连接
```
**代表**: 《1917》《Birdman》

## 🔧 经典长镜头分析

### 《Touch of Evil》开场（3分20秒）
| 要素 | 实现 |
|------|------|
| 开场 | 定时炸弹放入车内 |
| 跟随 | 升降机跟随汽车 |
| 引入 | 主角出场步行 |
| 悬念 | 炸弹倒计时 |
| 结束 | 爆炸！ |

### 《老男孩》走廊打斗（2分30秒）
| 要素 | 实现 |
|------|------|
| 形式 | 横向滚轴 |
| 动作 | 真实格斗 |
| 难度 | 无替身 |
| 风格 | 横版游戏感 |

## 📋 长镜头设计清单

### 前期准备
- [ ] 为什么需要长镜头？（不是为了炫技）
- [ ] 分解成多少个"段落"？
- [ ] 每个段落的焦点是什么？
- [ ] 技术难点在哪里？
- [ ] 需要多少排练时间？

### 技术考虑
- [ ] 焦点变化（跟焦员）
- [ ] 灯光变化（调光）
- [ ] 声音连续性（收音）
- [ ] 隐藏剪辑点（如需要）

## 🎯 长镜头的隐藏剪辑

### 常用技巧
| 技巧 | 实现 |
|------|------|
| **黑场过渡** | 经过黑暗区域时切 |
| **快速横移** | 高速移动时切 |
| **遮挡物** | 经过柱子/人物时切 |
| **向上移动** | 摇向天空时切 |
| **向下移动** | 摇向地面时切 |

### 示意图
```
镜头1 → 经过柱子 → 镜头2
         ↑
      隐藏剪辑点
```

## 🎬 AI番剧应用

### 长镜头的AI实现
由于AI视频通常较短，需要：
1. 生成多个连续片段
2. 使用隐藏剪辑连接
3. 保持场景一致性

### Prompt策略
```
"continuous tracking shot, following character through [场景],
camera movement: [运动方式], single take feel, no cuts,
cinematic, [风格]"
```

### 分段生成示例
```
片段1: "tracking shot entering building lobby, following woman in red dress"
片段2: "continuous shot, woman walking through crowd, camera following behind"
片段3: "following shot, woman approaching elevator, one take feel"
```

### 连贯性要点
- 保持相同的角色服装
- 保持相同的运动方向
- 保持相同的光线氛围
- 使用遮挡物作为过渡点

## 💡 何时使用长镜头

### 适合场景
| 场景类型 | 原因 |
|----------|------|
| 角色入场 | 建立人物和环境 |
| 对话高潮 | 连续情感不中断 |
| 动作场面 | 展示真实打斗 |
| 空间展示 | 建立地理关系 |
| 梦境/回忆 | 连续感觉 |

### 避免场景
| 场景类型 | 原因 |
|----------|------|
| 快节奏剪辑 | 与风格冲突 |
| 多线叙事 | 需要切换 |
| 紧张悬疑 | 需要特写 |
