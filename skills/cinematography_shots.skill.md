## 📐 基础镜头类型

### 按景别分类

| 景别 | 英文 | 缩写 | 画面范围 | 用途 |
|------|------|------|----------|------|
| **大远景** | Extreme Long Shot | ELS | 广阔环境 | 建立场景、史诗感 |
| **远景** | Long Shot | LS | 全身 + 环境 | 展示人物与环境关系 |
| **全景** | Full Shot | FS | 人物全身 | 展示动作 |
| **中景** | Medium Shot | MS | 腰部以上 | 对话、互动 |
| **中近景** | Medium Close-Up | MCU | 胸部以上 | 表情 + 手势 |
| **近景** | Close-Up | CU | 面部 | 情感表达 |
| **特写** | Extreme Close-Up | ECU | 眼睛/嘴/物品 | 强调细节 |

---

### 按角度分类

| 角度 | 英文 | 效果 |
|------|------|------|
| **平视** | Eye Level | 客观、中立 |
| **俯拍** | High Angle | 弱小、被压迫 |
| **仰拍** | Low Angle | 强大、威胁 |
| **鸟瞰** | Bird's Eye | 上帝视角 |
| **荷兰角** | Dutch Angle | 不安、混乱 |
| **主观** | POV | 代入感 |

---

### 按运动分类

| 运动 | 英文 | 效果 |
|------|------|------|
| **固定** | Static | 稳定、客观 |
| **横摇** | Pan | 展示环境、跟随动作 |
| **纵摇** | Tilt | 展示高度 |
| **推镜** | Dolly In | 接近、紧张、聚焦 |
| **拉镜** | Dolly Out | 疏离、揭示 |
| **跟拍** | Tracking | 跟随人物 |
| **斯坦尼康** | Steadicam | 流畅跟随 |
| **手持** | Handheld | 紧张、真实 |
| **升降** | Crane | 史诗、转场 |
| **变焦** | Zoom | 突然聚焦（慎用） |

## 🎬 100种大师镜头技巧

### 一、对话场景 (Dialogue)

#### 1. 过肩镜头 (Over-the-Shoulder)
```
┌─────────────────────────────────┐
│     A (背影)    │    B (正面)   │
│      肩膀       │     脸部      │
└─────────────────────────────────┘
```
**用途**：建立两人对话关系

#### 2. 正反打 (Shot/Reverse Shot)
交替切换A和B的单人镜头
**用途**：对话节奏、情感对比

#### 3. 双人镜头 (Two-Shot)
```
┌─────────────────────────────────┐
│      A       │       B         │
│              │                 │
└─────────────────────────────────┘
```
**用途**：展示关系、对峙

#### 4. 三人镜头 (Three-Shot)
**用途**：三角关系、权力动态

---

### 二、情感表达 (Emotion)

#### 5. 推近特写
从中景慢慢推进到特写
**效果**：逐渐揭示情感、紧张升级

#### 6. 反应镜头 (Reaction Shot)
先展示事件，再切到人物反应
**效果**：强调情感冲击

#### 7. 缓慢拉远
从特写慢慢拉远到远景
**效果**：孤独、绝望、疏离

#### 8. 静止凝视
人物静止，镜头不动
**效果**：内心活动、思考

---

### 三、动作场景 (Action)

#### 9. 跟拍
镜头跟随人物移动
**效果**：紧张、代入感

#### 10. 快速剪辑
多角度快速切换
**效果**：混乱、激烈

#### 11. 慢动作
关键时刻放慢
**效果**：强调重要瞬间

#### 12. 长镜头
一镜到底
**效果**：真实感、沉浸

---

### 四、揭示/悬念 (Reveal/Suspense)

#### 13. 先声夺人
先听到声音，再看到画面
**效果**：悬念、期待

#### 14. 迟到揭示
镜头缓慢揭示关键信息
**效果**：惊喜、震惊

#### 15. 镜头内揭示
人物移动或镜头移动揭示新元素
**效果**：不用剪辑的惊喜

#### 16. 聚焦转移
从前景移焦到后景（或相反）
**效果**：引导注意力

---

### 五、转场 (Transition)

#### 17. 匹配剪辑
两个相似形状/动作的镜头切换
**效果**：流畅过渡

#### 18. 跳切
同一场景跳过时间
**效果**：时间流逝

#### 19. 叠化
两个画面重叠过渡
**效果**：时间/空间转换

#### 20. 黑场
淡入黑/淡出黑
**效果**：段落结束

## 📐 180度法则

```
        摄像机可放置区域
    ┌─────────────────────────┐
    │                         │
    │   📹    📹    📹        │
    │                         │
────┼──────A═════════B────────┼──── 轴线
    │        ↑                │
    │    不能放摄像机         │
    └─────────────────────────┘
```

**法则**：摄像机必须在轴线的同一侧
**违反后果**：观众方向感混乱
**例外**：刻意制造混乱感

## 🔧 分镜设计流程

### 步骤1：确定场景目的
- 这场戏要表达什么？
- 观众应该感受到什么？

### 步骤2：确定关键时刻
- 场景的高潮是什么？
- 需要强调什么？

### 步骤3：选择镜头类型
| 目的 | 推荐镜头 |
|------|----------|
| 建立环境 | 大远景、远景 |
| 对话 | 中景、过肩、特写 |
| 动作 | 全景 + 特写 |
| 情感 | 特写、反应镜头 |

### 步骤4：设计镜头运动
- 静态还是运动？
- 运动方向？
- 速度？

### 步骤5：注意剪辑
- 景别变化（大跳跃好剪辑）
- 180度法则
- 视线方向
