# 对白设计技巧 Skill
> 来源：《对白》罗伯特·麦基
> 适用智能体：A03 对白编剧

## 核心理念

**对白 = 带着特定目的的言语行动**

对白不是闲聊，每句话都是角色为达成欲求而采取的手段。

---

## 对白三维度

1. **对别人说** — 角色间对话
2. **对自己说** — 内心独白、旁白
3. **对观众说** — 画外音、第四面墙

---

## 对白三层同心圆

```
┌─────────────────────────────────┐
│  外层：已说（文本）              │
│  ┌───────────────────────────┐ │
│  │  中层：未说（潜文本）       │ │
│  │  ┌─────────────────────┐ │ │
│  │  │  内层：不可说（潜意识）│ │ │
│  │  └─────────────────────┘ │ │
│  └───────────────────────────┘ │
└─────────────────────────────────┘
```

- **已说**：字面意思、选词措辞、语法口气
- **未说**：角色知道但选择不讲的心思
- **不可说**：角色自己都不自知的潜意识动机

---

## 有效对白六任务

1. ✅ 每个口语表达执行一个**内在行动**
2. ✅ 每个行动/反应节拍**推动场景至转折点**
3. ✅ 对白传达**信息和观点**
4. ✅ 独特口语风格**个性化角色**
5. ✅ 进展节拍**抓住注意力**
6. ✅ 语言**保持真实可信**

---

## 对白三功能

### 1. 解说功能
- 将背景、历史、角色信息供观众吸收
- **忌**：直接说出两人都已知道的事

### 2. 角色塑造功能
- 人物表象 vs 人物真相
- 目标：引发兴趣、取信说服、赋予个性

### 3. 行动功能
- 心理行动
- 肢体行动
- 口语行动

---

## 对白常见错误

| 错误类型 | 表现 | 解决方法 |
|---------|------|---------|
| 内容空洞 | 角色说两人都知道的事 | 让信息对一方是新的 |
| 情绪过激 | 小题大做，反应过度 | 检查潜文本是否支撑 |
| 知道太多 | 角色说了只有作者才知道的事 | 保持角色认知边界 |
| 借口当动机 | 角色说出真正动机 | 动机应藏于潜意识 |
| 陈词滥调 | 观众能猜到下句话 | 出乎意料但合情合理 |
| 笼统无奇 | 谁说都行的话 | 只有这个角色才会说 |
| 铺张卖弄 | 漂亮但不符人物 | 服从角色性格 |

---

## 对白写作原则

| 原则 | 说明 |
|------|------|
| 具体优于抽象 | 用具体名词量词 |
| 熟悉优于奇异 | 不脱离角色口出狂言 |
| 短优于长 | 冲突激烈时用短句 |
| 直言优于曲笔 | 曲笔≠潜台词 |
| 主动优于被动 | 被动句不是日常交流 |
| 长话短说 | 删除无意义枝节 |

---

## 潜台词设计方法

### 冰山技巧
- 文本只是冰山一角
- 大量内容在潜文本中
- 让观众"摸透角色心理"

### 潜台词示例

**浅白对白（❌）**：
> 王丽："我怀孕了，我不想再当贼了。"

**有潜台词的对白（✅）**：
> 王薄："你怎么了？"
> 王丽："没什么。"
> （过了一会）
> 王丽："我想这是最后一次了。"

## 对白与冲突层面

| 冲突层面 | 对白量 | 说明 |
|---------|--------|------|
| 自然面 | 最少 | 与环境的冲突 |
| 社会面 | 中等 | 与制度、规则的冲突 |
| 私人面 | 较多 | 人际关系冲突 |
| 内心面 | 最多 | 自我内心冲突 |

---

## 对白惯用语与角色塑造

同样的意思，不同表达塑造不同性格：

| 表达方式 | 角色A | 角色B |
|---------|-------|-------|
| 形容词 | "好漂亮" | "好壮观" |
| 副词 | "真快" | "真猛" |
| 语态 | "我来负责规划" | "事情由他们规划" |

## 对白创作流程

```
1. 设定触发事件
      ↓
2. 确定角色核心欲求
      ↓
3. 设计冲突层面
      ↓
4. 写出潜文本（角色真正想要什么）
      ↓
5. 最后才写文本（对白）
      ↓
6. 检查：删除能删的，暗示能暗示的
```

## AI番剧应用

### Prompt关键词
- 角色语气：formal/casual/sarcastic/warm
- 情绪状态：concealing/revealing/conflicted
- 说话风格：direct/indirect/metaphorical

### 台词设计模板
```
角色：[名字]
场景：[情境]
表面目的：[想要的]
真正目的：[需要的]
潜台词：[没说出口的]
对白：[实际台词]
```
