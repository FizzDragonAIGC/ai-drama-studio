# 动作描写 Skill
> 来源：好莱坞专业编剧手册、番剧分镜实践
> 适用智能体：编剧助手、分镜设计师

## 核心理念

**"动作线是剧本的骨架"** —— 精准、简洁、可视化

动作线（Action Lines）描述画面上发生的一切，是导演和动画师的蓝图。

---

## 核心理论

### 动作线的基本原则

```
可见 + 可听 = 可写
内心想法、情绪解释 = 不可写
```

**正确 vs 错误示例**：
- ❌ "小明感到悲伤，想起了过去的一切"
- ✅ "小明低下头，肩膀微微颤抖"

### 动作线的功能

| 功能 | 说明 | 示例 |
|------|------|------|
| 描述动作 | 角色做什么 | 她推开门 |
| 设置场景 | 环境是什么样 | 雨打在窗户上 |
| 暗示情绪 | 通过动作显示情感 | 他的拳头握紧 |
| 引导镜头 | 暗示摄影角度 | 阳光从背后照来 |

### 动作描写的层次

```
Level 1：基本动作
    "他走进房间"

Level 2：情感化动作  
    "他拖着脚步走进房间"

Level 3：电影化动作
    "他的影子先于身体进入——
     然后是他，像幽灵一样，无声滑入"
```

## 实操技巧

### 动作线写作法则

**1. 主动语态**
- ❌ "门被他推开"
- ✅ "他推开门"

**2. 具体动词**
- ❌ "他快速移动"
- ✅ "他冲/奔/窜/飞扑"

**3. 精准描述**
- ❌ "他带着武器"
- ✅ "他握着沾血的短刀"

**4. 简洁有力**
- ❌ "她非常缓慢地转过身来，用她那双充满泪水的眼睛看着他"
- ✅ "她转身。泪水模糊了视线"

### 番剧特有的动作描写

| 类型 | 描写要点 | 示例 |
|------|----------|------|
| 战斗动作 | 动作拆解，关键帧 | 剑光一闪——敌人分成两半 |
| 日常动作 | 细节观察 | 她整理裙摆，假装没看见 |
| 魔法/特效 | 视觉化描述 | 蓝色火焰从掌心蔓延成龙形 |
| 表情反应 | 微表情捕捉 | 他的眼神躲开0.5秒 |

### 动作节奏控制

**快节奏（动作戏）**：
```
剑出。
火光。
血溅三尺。
敌人倒下。
```

**慢节奏（情感戏）**：
```
她站在窗前。

雨一直下。

她的手，缓缓抚上玻璃——
指尖留下五道水痕。
```

### 空间与动作

```
【远景】城市在晨雾中苏醒。
【中景】小明走出公寓楼。
【近景】他看了眼手机。屏幕亮着：未接来电 47。
【特写】他的拇指，悬在"删除"按钮上方。
```

## Prompt模板

### 动作线生成

```
将以下场景描述转化为专业动作线：

场景描述：[你的文字描述]

要求：
1. 纯视觉/听觉描述，不写内心活动
2. 使用主动语态和具体动词
3. 每个段落3-4行，留白分段
4. 适当暗示镜头（远/中/近/特写）
5. 番剧风格，适合动画分镜
```

### 动作戏编写

```
请编写以下战斗场景的动作线：

战斗双方：[A角色 vs B角色]
场地环境：[战斗地点]
战斗目的：[为什么而战]
预期结果：[谁赢/平局/中断]

要求：
1. 分3-5个回合描写
2. 每个回合有攻防转换
3. 利用环境元素
4. 有节奏变化（快-慢-快）
5. 动作要可动画化
```

### 情感动作转化

```
将以下情感描述转化为可视化动作：

情感状态：[角色当前情感]
场景背景：[在哪里，什么情况]

请输出5种不同的动作表达方式，
从细微（Level 1）到强烈（Level 5），
不要使用情感词汇，只用动作和画面。
```

## 检查清单

### 动作线质量检查
- [ ] 全部是可见/可听的内容？
- [ ] 使用主动语态？
- [ ] 动词足够具体？
- [ ] 没有冗余描述？
- [ ] 节奏感合适？
- [ ] 动画师能据此画分镜？

### 番剧适配检查
- [ ] 关键动作有足够细节？
- [ ] 考虑了动画表现特点？
- [ ] 特效描述足够具象？
- [ ] 适合有限动画或全动画？

## 案例分析

### 案例：告白场景动作线

**原始描述**：
"小红很紧张地向小明告白，小明很惊讶但很开心"

**专业动作线改写**：

```
INT. 天台 - 黄昏

小红背对夕阳站着。风吹乱她的头发。

她的手指绞在一起——松开——又绞紧。

        小红
    那个……

她吸一口气。

        小红
    我喜欢你！

沉默。

小明愣住。眨眼。又眨眼。

嘴角开始上扬——

他移开视线，抬头看天。

        小明
    ……我也是。

小红的眼睛亮了。

夕阳把他们的影子拉得很长，
两个影子的手，几乎碰到一起。
```

**技巧分析**：
| 原文问题 | 改写方案 |
|----------|----------|
| "很紧张" | 手指绞紧、深呼吸 |
| "很惊讶" | 愣住、眨眼 |
| "很开心" | 嘴角上扬、移开视线 |
| 直接说情感 | 用影子做隐喻 |

## 动作动词库

### 移动类
| 情感色彩 | 动词选择 |
|----------|----------|
| 中性 | 走、跑、移动 |
| 疲惫 | 拖、挪、蹭 |
| 急切 | 冲、奔、飞扑 |
| 小心 | 挪、踱、蹑 |
| 优雅 | 滑、飘、旋 |

### 手部动作
| 情感 | 动作 |
|------|------|
| 紧张 | 握紧、绞、搓、抠 |
| 愤怒 | 拍、砸、甩、捶 |
| 温柔 | 抚、摸、轻触 |
| 犹豫 | 悬停、收回、抖 |

### 面部动作
| 情感 | 动作 |
|------|------|
| 惊讶 | 睁大眼、张嘴、后仰 |
| 悲伤 | 垂眼、低头、咬唇 |
| 愤怒 | 皱眉、咬牙、眯眼 |
| 喜悦 | 嘴角上扬、眼弯 |
