# 番剧叙事技巧 Skill
> 来源：《出发点》宫崎骏、日本动画制作理论
> 适用智能体：A21 番剧叙事专家

## 核心理念

**"创作动画就是在创造一个虚构的世界"** —— 宫崎骏

动画不是现实的复制，而是承载创作者梦想与情感的艺术表达。

---

## 宫崎骏创作哲学

### 五大核心信念

1. **角色意愿驱动**
   - 不是描述命运，而是描述意愿
   - 角色"想要"和"需要"的东西要分清

2. **自然敬畏**
   - 人与自然的关系是永恒主题
   - 风、水、树、飞行——自然元素的诗意表达

3. **技术警惕**
   - 技术是手段不是目的
   - 反对过度依赖CG

4. **儿童视角**
   - 为孩子创作，但不低估孩子
   - 相信观众的理解力

5. **文明反思**
   - 工业文明与自然的冲突
   - 战争与和平的思考

## 日本动画制作流程

### 职位体系

```
监督（Director）
    ↓ 统筹全片
系列构成（Series Composition）
    ↓ 剧本统筹
脚本（Screenplay）
    ↓ 单集剧本
分镜（Storyboard / 絵コンテ）
    ↓ 镜头设计
演出（Episode Director）
    ↓ 单集导演
作画监督（Animation Director）
    ↓ 作画质量把控
原画 → 动画 → 上色 → 摄影 → 后期
```

### 监督 vs 演出
- **监督**：统领全局，把控整体风格
- **演出**：负责单集，执行监督意图

## 番剧单集结构

### 标准TV动画（24分钟）

```
OP（片头曲）     ：1.5分钟
A Part          ：约10分钟
    - 开场钩子
    - 事件发展
    - 中场高潮/悬念
    ↓
Eye Catch（过场）：几秒
    ↓
B Part          ：约10分钟
    - 冲突升级
    - 高潮
    - 结尾钩子
ED（片尾曲）     ：1.5分钟
预告            ：30秒
```

### 单集情绪曲线

```
     高潮点
       /\
      /  \     B Part高潮
     /    \      /\
    /      \    /  \
   /   中场  \  /    \
  /    悬念   \/      \
 /                     \ 结尾钩子
开场钩子               ↓下集
```

## 动画演出技法

### 1. 时间控制
- **タメ（蓄力）**：动作前的停顿，增加张力
- **ツメ（收）**：动作后的余韵
- **间**：日式留白，情感空间

### 2. 镜头语言
- **寄り（靠近）**：特写，情感强调
- **引き（拉远）**：全景，环境交代
- **見上げ（仰视）**：威严、压迫
- **見下ろし（俯视）**：渺小、孤独

### 3. 动画特有表现
- **効果線（效果线）**：速度感、冲击感
- **漫符（漫画符号）**：汗滴、怒筋
- **SD化（Q版）**：喜剧效果
- **背景動画（背景动画）**：强调运动感

## 系列构成方法

### 单季12集结构

| 集数 | 功能 | 内容 |
|------|------|------|
| 1 | 开篇 | 世界观、主角、核心冲突 |
| 2-3 | 展开 | 配角登场、日常建立 |
| 4-5 | 发展 | 第一次危机 |
| 6 | 中场 | 转折点或日常回 |
| 7-9 | 升级 | 矛盾激化 |
| 10-11 | 高潮 | 最终对决 |
| 12 | 收尾 | 解决+新悬念 |

### 钩子设计

**集头钩子**：
- 冷开场（Pre-title）
- 悬念重现
- 冲击性画面

**集尾钩子**：
- 反转揭示
- 新威胁出现
- 情感高点中断

## AI番剧特有考量

### 一致性挑战
- 角色造型一致
- 服装道具一致
- 场景风格一致
- 光影氛围一致

### 动态感营造
- 动作前后帧设计
- 效果线/速度线
- 镜头运动暗示
- 表情变化序列

### 时间节奏
- 每镜头5-10秒
- 静态镜头需要"呼吸感"
- 对话镜头正反打设计
