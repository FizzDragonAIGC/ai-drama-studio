# ⚔️ 冲突设计 Skill
> 来源：Robert McKee《Story》/ Linda Seger《Making a Good Script Great》

---

## 📐 冲突三层级

### 第一层：内在冲突 (Inner Conflict)
角色与自己的斗争

**类型**：
| 冲突 | 例子 |
|------|------|
| 理智vs情感 | 知道该分手，但舍不得 |
| 责任vs欲望 | 要照顾家人，但想追梦 |
| 恐惧vs勇气 | 害怕，但必须行动 |
| 道德vs利益 | 应该揭发，但会失去一切 |
| 过去vs现在 | 心理创伤阻碍前进 |

### 第二层：人际冲突 (Personal Conflict)
角色与他人的斗争

**关系类型**：
- 主角 vs 反派
- 爱人之间
- 家人之间
- 朋友之间
- 同事/对手

### 第三层：外在冲突 (Extra-Personal Conflict)
角色与外部力量的斗争

**类型**：
- vs 社会/体制
- vs 自然/环境
- vs 时间
- vs 命运
- vs 超自然

---

## 🎯 冲突设计核心

### 黄金公式
```
角色 + 目标 + 障碍 = 冲突
```

### 有效冲突的标准
1. **具体化**：能看见、能感受
2. **持续性**：贯穿整个故事
3. **升级性**：越来越激烈
4. **相关性**：与主题相连
5. **有意义**：有真正的损失风险

## 📊 冲突升级模型

```
强度
 │
 │                              ┌──── 高潮
 │                         ┌────┘
 │                    ┌────┘
 │               ┌────┘
 │          ┌────┘
 │     ┌────┘
 │ ────┘
 │
 └───────────────────────────────────────→ 时间
       ↑     ↑      ↑       ↑      ↑
     小冲突 升级1  升级2   升级3   最终
```

### 升级方式
| 方式 | 说明 |
|------|------|
| 赌注升高 | 从个人→家庭→社会→世界 |
| 紧迫性增加 | 加入时间限制 |
| 力量失衡 | 敌人越来越强 |
| 资源减少 | 主角失去帮手/工具 |
| 情感加深 | 事情变得更个人化 |

## ⚡ 场景级冲突

每个场景都需要冲突：

### 场景冲突结构
```
角色A目标 ←──冲突──→ 角色B目标
         (或障碍)
```

### 场景冲突类型
| 类型 | 示例 |
|------|------|
| 直接对抗 | 两人争论 |
| 间接阻碍 | 门锁了，进不去 |
| 内心犹豫 | 要不要按下按钮 |
| 时间压力 | 炸弹倒计时 |
| 两难选择 | 救谁？ |

## 🎭 反派设计

### 好反派的标准
1. **有理由**：从他的角度，他是对的
2. **有能力**：能真正威胁主角
3. **有弱点**：最终能被击败
4. **有魅力**：观众想看他

### 反派类型
| 类型 | 例子 | 冲突性质 |
|------|------|----------|
| 镜像型 | 蝙蝠侠vs小丑 | 价值观对立 |
| 障碍型 | 老板/家长 | 阻挡目标 |
| 竞争型 | 情敌/对手 | 争夺同一目标 |
| 代表型 | 体制/社会 | 代表压迫力量 |

- [ ] 冲突持续升级？
- [ ] 与主题相连？

### 每个场景
- [ ] 有冲突存在？
- [ ] 冲突有结果？
- [ ] 推动故事前进？

### 主角
- [ ] 有明确目标？
- [ ] 面临真正障碍？
- [ ] 有失去的风险？
- [ ] 必须做出选择？

## 🔧 冲突设计练习

### 练习1：升级冲突
```
初始冲突：想辞职
↓ 升级1：老板威胁不给推荐信
↓ 升级2：发现公司违法行为
↓ 升级3：被威胁家人安全
↓ 最终：必须揭发vs保护家人
```

### 练习2：内外结合
```
外在冲突：必须在48小时内筹到钱
内在冲突：唯一方法是找帮助过他的骗子
人际冲突：妻子不同意这个方法
```

### 练习3：两难设计
```
选项A：说实话 → 后果：失去工作
选项B：撒谎 → 后果：背叛朋友
必须选择，不能逃避
```
