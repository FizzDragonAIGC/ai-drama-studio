## 📐 漫画的时间魔法

| 电影 | 漫画 |
|------|------|

## 🔲 格子（Panel）设计

### 格子形状与情感

| 形状 | 效果 | 使用场景 |
|------|------|----------|
| ▭ **横长格** | 稳定、广阔、叙事 | 远景、环境、平静场景 |
| ▯ **竖长格** | 紧张、压迫、速度 | 动作、坠落、紧迫 |
| ■ **方格** | 中立、平衡 | 对话、日常 |
| ◇ **斜格** | 动态、不安 | 打斗、混乱 |
| ○ **圆/异形** | 回忆、梦境、特殊 | 闪回、内心 |
| ∞ **出血格** | 冲击、史诗 | 高潮、震撼场面 |

### 格子大小与重要性

```
┌─────────────────────────────────────┐
│                                     │
│           大格 = 重要时刻            │  ← 高潮、转折
│                                     │
├─────────────────┬───────────────────┤
│                 │                   │
│    中格 = 关键   │    中格 = 关键     │  ← 主要动作
│                 │                   │
├────────┬────────┼────────┬──────────┤
│ 小格   │ 小格   │ 小格   │ 小格     │  ← 过渡、细节
└────────┴────────┴────────┴──────────┘
```

**法则**：格子越大，读者停留时间越长，越重要！

## 👁️ 视线引导（Reading Flow）

### 基本阅读顺序

**日漫**：右→左，上→下
```
┌───┬───┐
│ 2 │ 1 │
├───┼───┤
│ 4 │ 3 │
└───┴───┘
```

**美漫/国漫**：左→右，上→下
```
┌───┬───┐
│ 1 │ 2 │
├───┼───┤
│ 3 │ 4 │
└───┴───┘
```

### Z型/N型视线流

```
左→右阅读（Z型）          右→左阅读（N型）
┌─────────────┐           ┌─────────────┐
│  1 → → 2    │           │    1 ← ← 2  │
│      ↘      │           │      ↙      │
│  3 → → 4    │           │    3 ← ← 4  │
└─────────────┘           └─────────────┘
```

### 视线引导技巧

| 技巧 | 说明 |
|------|------|
| **角色视线** | 人物看向的方向引导读者 |
| **动作线** | 动作方向引导视线 |
| **对话气泡位置** | 气泡顺序引导阅读 |
| **留白** | 空白区域让视线休息 |
| **集中线** | 强制聚焦某点 |

## 📄 页面布局（Page Layout）

### 经典布局模式

**1. 稳定型（日常/对话）**
```
┌─────┬─────┐
│     │     │
├─────┼─────┤
│     │     │
├─────┼─────┤
│     │     │
└─────┴─────┘
```

**2. 冲击型（动作/高潮）**
```
┌─────────────┐
│             │
│   大出血格   │
│             │
├───┬───┬────┤
│   │   │    │
└───┴───┴────┘
```

**3. 对比型（转折）**
```
┌─────────────┐
│    上半页    │
│    (正常)   │
├─────────────┤
│    下半页    │
│    (反转)   │
└─────────────┘
```

**4. 碎片型（混乱/回忆）**
```
┌───┬─────┬──┐
│   │     │  │
├───┴──┬──┴──┤
│      │     │
├──┬───┴─────┤
│  │         │
└──┴─────────┘
```

## ⚡ 节奏控制（Pacing）

### 格子数量与节奏

| 格子数/页 | 节奏 | 使用场景 |
|----------|------|----------|
| 1-2格 | 极慢 | 震撼时刻、情感爆发 |
| 3-4格 | 慢 | 情感戏、重要对话 |
| 5-6格 | 中 | 正常叙事 |
| 7-9格 | 快 | 动作、紧张 |
| 10+格 | 极快 | 快速连续动作 |

### 节奏变化示例

```
页1: ████████ (6格) - 正常
页2: ████████ (6格) - 正常
页3: ██████████ (8格) - 加速
页4: ████████████ (10格) - 紧张！
页5: ██ (2格) - 爆发！高潮！
页6: ████ (4格) - 回落
```

## 🎭 漫画特有的分镜技巧

### 1. 同一格内的时间流动
```
┌─────────────────────────┐
│  人物A说话  →  人物B回应  │
│  (左)          (右)      │
└─────────────────────────┘
```
一格内展示对话的来回

### 2. 格间过渡类型（McCloud六种）

| 类型 | 说明 | 效果 |
|------|------|------|
| **时刻到时刻** | 微小时间变化 | 慢动作、细腻 |
| **动作到动作** | 同一主体不同动作 | 流畅动作 |
| **主体到主体** | 不同主体同一场景 | 对话、互动 |
| **场景到场景** | 不同地点/时间 | 转场 |
| **方面到方面** | 同一场景不同角度 | 氛围、环境 |
| **非线性** | 无逻辑关联 | 象征、意识流 |

### 3. 跨页大图（Double Page Spread）
```
←────────── 跨页 ──────────→
┌─────────────────────────────┐
│                             │
│       史诗级大场面           │
│                             │
└─────────────────────────────┘
```
**使用时机**：最震撼的瞬间！

## 📱 条漫适配（Webtoon/竖屏）

### 传统漫画 vs 条漫

| 维度 | 传统漫画 | 条漫 |
|------|----------|------|
| 方向 | 翻页（横向） | 滚动（纵向） |
| 单位 | 页 | 无限长条 |
| 节奏 | 翻页控制 | 滚动速度 |
| 格子 | 紧凑排列 | 竖向展开 |

### 条漫分镜技巧

**1. 竖向展开**
```
┌─────────┐
│  格1    │
│         │
├─────────┤
│  留白   │
├─────────┤
│  格2    │
│         │
├─────────┤
│  格3    │
└─────────┘
    ↓
  继续滚动
```

**2. 悬念设计（Scroll Stopper）**
在关键时刻让读者必须滚动才能看到结果

**3. 一屏一重点**
手机屏幕一屏只放一个核心信息
