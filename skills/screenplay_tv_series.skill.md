# 電視劇劇本寫作 - TV Series Writing

## 核心書籍方法論

### 1. Pamela Douglas《Writing the TV Drama Series》
**電視劇寫作聖經**

```
電視劇 vs 電影的本質區別：
- 電影 = 封閉結局 (Closed Ending)
- 電視劇 = 開放結局 (Open Ending)，角色持續發展

電視劇的核心 = 角色 > 情節
觀眾追劇是因為愛角色，不是愛情節
```

### 2. 劇集類型結構

#### 單元劇 (Procedural/Episodic)
```
每集獨立故事，可任意順序觀看
例：CSI、法醫秦明、名偵探柯南

結構：
- 開場案件/問題 (Teaser)
- 調查/處理過程 (Acts 1-4)
- 解決案件 (Resolution)
- 角色情感線貫穿 (B Story)

每集公式：
冷開場 → A故事（案件）+ B故事（人物）→ 結案 + 情感落點
```

#### 連續劇 (Serial/Serialized)
```
故事跨集發展，必須按順序觀看
例：Breaking Bad、權力的遊戲、甄嬛傳

結構：
- 季度大弧線 (Season Arc)
- 半季轉折 (Mid-Season Finale)
- 每集小鉤子 (Episode Hook)
- 季終高潮 (Season Finale)
```

#### 混合型 (Hybrid)
```
單集有完整故事 + 季度大線索
例：Mandalorian、黑鏡

最佳實踐：70%單元 + 30%連續
```

## 電視劇結構公式

### 一集的結構 (42-60分鐘劇)
```
Teaser (冷開場): 2-3分鐘
Act 1: 10-12分鐘 → 廣告
Act 2: 10-12分鐘 → 廣告  
Act 3: 10-12分鐘 → 廣告
Act 4: 10-12分鐘 → 廣告
Act 5 (Tag): 3-5分鐘

關鍵：每幕結尾必須是「鉤子」
```

### 串流平台結構 (無廣告)
```
Netflix/HBO/愛奇藝等：
- 不需要傳統廣告break
- 每幕結尾仍需要小高潮
- 集末必須有「追劇鉤子」

結構更靈活：
- 開場 + 3-4個大段落 + 結尾鉤子
- 時長45-70分鐘不等
```

## Writer's Room 工作流程

### 1. 季度規劃 (Season Break)
```
Showrunner主導，確定：
- 季度主題 (Season Theme)
- 角色弧線 (Character Arcs)
- 主要情節線 (Major Plot Lines)
- 季終目標 (Finale Goal)
```

### 2. 分集大綱 (Episode Outline)
```
每集確定：
- A Story (主線，占60%)
- B Story (副線，占30%)
- C Story (小線，占10%)
- Runner (貫穿多集的小細節)
```

### 3. Beat Sheet → Outline → Script
```
Beat Sheet: 場景列表，每場1-2句
Outline: 詳細場景描述，8-12頁
Script: 完整劇本，50-60頁
```

## 連續劇的季度設計

### 10集季度模板
```
Ep 1: 建立/引入（新觀眾友好）
Ep 2: 深化世界觀
Ep 3: 第一個小高潮
Ep 4: 人物深度挖掘
Ep 5: 中期轉折（Mid-Season）
Ep 6: 後果與調整
Ep 7: 副線高潮
Ep 8: 危機加劇
Ep 9: 最黑暗時刻
Ep 10: 季終高潮 + 下季鉤子
```

### 角色弧線設計
```
季度角色弧線 = 一部電影的角色發展
- 起點：角色的缺陷/困境
- 中點：認識到問題
- 終點：改變（或拒絕改變）

多季設計：每季一個主題
S1: 建立身份
S2: 考驗忠誠
S3: 面對過去
S4: 最終選擇
```

## 對白技巧

### 電視劇對白特點
```
比電影更「話多」：
- 預算限制，對話場景成本低
- 觀眾需要角色互動
- 對白承載更多信息

但仍要：
- 潛台詞 (Subtext)
- 角色獨特語言風格
- 推進劇情，不是閒聊
```

### 集末鉤子類型
```
1. 懸崖式 (Cliffhanger): 生死未卜
2. 揭示式 (Revelation): 驚人真相
3. 決定式 (Decision): 重大選擇
4. 到達式 (Arrival): 新角色/新地點
5. 情感式 (Emotional): 關係突破
```
