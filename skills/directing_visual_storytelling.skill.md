# 👁️ 视觉叙事 Skill
> 来源：《The Visual Story》Bruce Block / 《Film Directing Shot by Shot》Steven Katz

---

## 📐 视觉叙事核心

> "Show, don't tell" — 用画面讲故事，而非对白

### 三层叙事
| 层级 | 内容 | 例子 |
|------|------|------|
| 对白 | 角色说什么 | "我爱你" |
| 动作 | 角色做什么 | 默默收拾行李 |
| 视觉 | 画面暗示什么 | 框中框=囚禁 |

---

## 🎯 视觉叙事六要素

### 1. 空间 (Space)
```
深度：前景-中景-后景
维度：平面 vs 立体
开放 vs 封闭
```

| 空间 | 情感 |
|------|------|
| 开阔 | 自由、孤独 |
| 狭窄 | 压抑、紧张 |
| 深度 | 立体、真实 |
| 平面 | 疏离、图形化 |

### 2. 线条 (Line)
```
水平线：平静、稳定
垂直线：力量、庄严
对角线：动感、不安
曲线：柔和、有机
```

### 3. 形状 (Shape)
| 形状 | 含义 |
|------|------|
| 圆形 | 完整、柔和、女性 |
| 方形 | 稳定、可靠、男性 |
| 三角 | 动态、危险、方向 |
| 不规则 | 混乱、有机 |

### 4. 色调 (Tone)
```
亮调：希望、快乐、清白
暗调：绝望、危险、神秘
高对比：戏剧、冲突
低对比：平和、梦幻
```

### 5. 颜色 (Color)
（详见 color_emotion.skill.md）

### 6. 运动 (Movement)
```
快速运动：紧张、追逐
慢速运动：庄重、思考
静止：等待、张力
```

## 📊 对比创造意义

### 对比公式
```
A + 对比的B = 意义
```

### 常用对比
| 对比 | 含义 |
|------|------|
| 亮/暗 | 善/恶、希望/绝望 |
| 大/小 | 强/弱、重要/渺小 |
| 快/慢 | 紧张/舒缓 |
| 满/空 | 繁荣/萧条 |
| 热/冷 | 激情/理性 |

### 例子
```
场景1：温暖的家庭晚餐（暖色、中景、稳定）
场景2：冰冷的办公室（冷色、远景、静止）
对比 = 家庭vs工作的冲突
```

## 🎭 潜台词可视化

### 内心状态→画面元素
| 内心 | 画面 |
|------|------|
| 被困 | 框中框、窗户栏杆 |
| 孤独 | 大量负空间 |
| 压抑 | 低天花板、暗调 |
| 希望 | 光束、向上视角 |
| 失控 | 荷兰角、快速运动 |
| 决断 | 对称构图、稳定 |

## 📋 视觉主题设计

### 为角色设计视觉语言
```
角色A（主角）：
- 主色：蓝色（理性）
- 形状：方形（稳定）
- 光线：自然光
- 运动：稳定

角色B（反派）：
- 主色：红色（危险）
- 形状：三角（攻击性）
- 光线：强对比
- 运动：不可预测
```

### 为主题设计视觉主题
```
主题：束缚与自由

束缚的视觉：
- 框中框
- 垂直线条（栏杆）
- 暗调
- 封闭空间

自由的视觉：
- 开阔地平线
- 水平线
- 亮调
- 开放空间
```

## 🎬 视觉叙事案例

### 《公民凯恩》深焦摄影
- **技术**：全景深
- **含义**：揭示关系，多层信息
- **例子**：前景签合同，背景凯恩在玩

### 《美国丽人》塑料袋
- **画面**：风中飘舞的塑料袋
- **潜台词**：美在平凡中
- **视觉**：慢动作，柔和光线

### 《闪灵》对称构图
- **技术**：完美对称
- **含义**：不自然、压迫、疯狂
- **效果**：创造不安
