# ⏱️ 节奏设计 Skill
> 来源：《剪辑的语法》《电影剪辑技巧》

---

## 📐 节奏三层级

### 1. 宏观节奏（全集/全剧）
```
情节分布：
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
开篇钩子 → 发展 → 高潮 → 收尾
  10%      60%    20%   10%
```

### 2. 中观节奏（段落/场景）
```
每个场景的节奏：
进入 → 升级 → 高点 → 过渡
```

### 3. 微观节奏（镜头/剪辑）
```
镜头时长变化：
快 → 快 → 慢（强调）→ 快 → 停
```

---

## 🎯 节奏类型

### 快节奏
| 特点 | 方法 | 适用 |
|------|------|------|
| 短镜头 | <3秒/镜头 | 动作、紧张、追逐 |
| 信息密集 | 台词快、信息多 | 喜剧、悬疑 |
| 跳跃剪辑 | 跳切、快速切换 | 蒙太奇、回忆 |

### 慢节奏
| 特点 | 方法 | 适用 |
|------|------|------|
| 长镜头 | >10秒/镜头 | 情感、氛围 |
| 留白 | 沉默、空镜 | 悲伤、思考 |
| 缓慢运动 | 缓慢推进 | 庄重、沉思 |

### 变奏节奏
```
快 → 慢（强调）→ 快
例：战斗中突然慢镜头定格

慢 → 快（爆发）→ 慢
例：平静中突然爆发冲突
```

## 📊 类型片节奏参考

### 动作片
```
整体：快
对白场景：中速，短
动作场景：极快
决战前：慢（蓄势）
决战中：快→慢→快（最后一击慢镜头）
```

### 爱情片
```
整体：中慢
日常：慢，享受氛围
暧昧：慢，延长期待
确认：快→慢（定格在关键瞬间）
```

### 悬疑片
```
整体：中
日常：慢，建立不安
发现线索：中
追查：快
揭示：慢→快
```

### 喜剧片
```
整体：中快
铺垫：中
包袱：快
反应：慢（让观众笑）
```

## 🎭 镜头时长参考

| 镜头类型 | 建议时长 | 说明 |
|----------|----------|------|
| 大特写 | 1-3秒 | 强调细节 |
| 特写 | 2-4秒 | 情感反应 |
| 近景 | 3-5秒 | 对话 |
| 中景 | 4-8秒 | 动作、互动 |
| 全景 | 5-10秒 | 展示环境 |
| 远景 | 6-15秒 | 建立场景 |

### 短剧节奏
```
短视频/短剧（30秒-3分钟）：
- 平均镜头时长：1-2秒
- 信息密度：极高
- 每5秒一个钩子
- 没有"无用"镜头
```

### 传统剧节奏
```
传统剧集（30-60分钟）：
- 平均镜头时长：3-5秒
- 允许铺垫和过渡
- 有呼吸空间
- 情感场景可以长
```

## 📋 节奏检查清单

### 每场戏
- [ ] 有进入、发展、高点、过渡吗？
- [ ] 节奏和情绪匹配吗？
- [ ] 有没有太长的镜头？
- [ ] 有没有太赶的地方？
- [ ] 观众会觉得无聊吗？

### 整集
- [ ] 有没有明显的节奏变化？
- [ ] 高潮在对的位置吗？
- [ ] 开头够快吗？
- [ ] 结尾有余韵吗？
- [ ] 整体时长合适吗？

### 短剧特别检查
- [ ] 前3秒有钩子吗？
- [ ] 有没有"死镜头"？
- [ ] 节奏够快吗？
- [ ] 每5秒有变化吗？

## 🔧 节奏调整技巧

### 加快节奏
```
- 缩短镜头时长
- 删除过渡镜头
- 加快对白
- 使用跳切
- 压缩信息
```

### 放慢节奏
```
- 延长镜头时长
- 添加反应镜头
- 使用空镜头
- 添加沉默
- 慢动作
```

### 强调时刻
```
- 突然变慢
- 定格
- 放大情绪
- 配乐变化
- 沉默
```
