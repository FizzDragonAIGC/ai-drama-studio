# ✍️ 動作描寫技能

## 元信息
- **技能ID**: script_action_lines
- **所屬Agent**: 劇本Agent
- **參考書籍**: 《電影劇本寫作基礎》
- **難度**: ⭐⭐⭐

## 核心原則

動作描寫 = 能被攝影機拍到的東西
- 只寫看得見、聽得見的
- 不寫內心想法
- 簡潔有力

## 動作描寫規則

### 1. 現在時態
```
✅ 她走進房間。
✅ 他拿起電話。
❌ 她走進了房間。
❌ 他將會拿起電話。
```

### 2. 視覺化
```
✅ 她雙手顫抖，咖啡灑在桌上。
❌ 她感到緊張。

✅ 他轉過身，背對著她。
❌ 他不想面對她。
```

### 3. 簡潔
```
✅ 
他衝進房間，摔門。

❌
他快速地跑向房間，一把推開沉重的木門，
然後用力地把門摔上，發出巨大的聲響。
```

### 4. 分段
```
每個動作單元一段
避免大段落（超過4行）

✅
他打開抽屜。翻找。

找到一張舊照片。

他的手停住了。

❌
他打開抽屜，開始翻找。翻了一會兒後，他找到一張舊照片。他的手停住了，看著照片上的人。
```

## 角色首次出場

```
格式：角色名（年齡，特徵），動作。

小明（25，瘦弱，戴眼鏡）站在窗前，
望著外面的雨。

首次用全名+描述
之後只用名字
```

## 動作動詞選擇

### 強動詞 vs 弱動詞
```
❌ 弱：走、看、拿
✅ 強：闖、瞪、抓

❌ 他走向她。
✅ 他大步邁向她。

❌ 她看著他。
✅ 她盯著他。
```

### 動作詞彙表
| 情緒 | 弱動詞 | 強動詞 |
|------|--------|--------|
| 憤怒 | 走 | 衝、闖、踹 |
| 悲傷 | 坐 | 癱坐、跌坐 |
| 緊張 | 看 | 掃視、警覺地瞥 |
| 愛慕 | 看 | 凝視、含情脈脈 |

## 鏡頭暗示

```
不直接寫鏡頭術語，但暗示鏡頭

暗示特寫：
「她的手微微顫抖。」

暗示全景：
「小鎮全貌盡收眼底。」

暗示運鏡：
「他的視線從窗戶移向門口。」
```

## AI番劇動作描寫

### 增強格式
```
【動作】
小明推開門，環顧四周。
房間很暗，只有窗戶透進一縷月光。

【情緒提示】（給AI繪圖用）
主情緒：緊張、警覺
次情緒：好奇

【視覺重點】
- 門的方向
- 月光的角度
- 小明的側臉
```

## 輸出格式

```json
{
  "action_description": {
    "scene": 1,
    "actions": [
      {
        "text": "她推開門，走進房間。",
        "visual_focus": "門口逆光剪影",
        "emotion_hint": "緊張"
      }
    ]
  }
}
```
