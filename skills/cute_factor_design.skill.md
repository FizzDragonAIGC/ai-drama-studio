# 🥰 可爱设计 Skill
> 如何让角色/场面看起来超可爱

---

## 📐 可爱核心要素

### 可爱=什么
```
可爱 = 幼态特征 + 无害感 + 反差萌

幼态特征：大眼、小鼻、圆脸
无害感：弱小、需要保护
反差萌：意外的可爱行为
```

### 可爱三角
```
      可爱
       /\
      /  \
     /    \
    /      \
外表可爱──行为可爱
```

## 🎯 可爱类型

### 1. 天然呆可爱
```
特点：无意识的可爱
表现：
- 听不懂双关语
- 一本正经说傻话
- 不知道自己可爱
- 天真无邪

Prompt: innocent, airheaded, naturally cute
```

### 2. 元气可爱
```
特点：活力满满
表现：
- 蹦蹦跳跳
- 精力充沛
- 积极乐观
- 大声说话

Prompt: energetic, cheerful, bouncy, lively
```

### 3. 傲娇可爱
```
特点：口是心非
表现：
- 嘴上说不要
- 身体很诚实
- 脸红
- "才不是为了你！"

Prompt: tsundere, blushing, looking away
```

### 4. 呆萌可爱
```
特点：反应慢、软萌
表现：
- 反应迟钝
- 动作慢
- 软绵绵
- 需要照顾

Prompt: soft, slow, dopey, needs protection
```

### 5. 反差萌
```
特点：平时vs某个瞬间
表现：
- 高冷突然撒娇
- 强者露出弱点
- 大人变小孩

Prompt: gap moe, unexpected cuteness, contrast
```

## 📊 可爱视觉设计

### 面部设计
```
大眼睛：圆、亮、水汪汪
小鼻子：简单、不突出
小嘴巴：嘟嘴、微张
圆脸：婴儿肥

Prompt:
big round eyes, sparkling eyes,
small nose, small mouth,
round face, soft features,
cute expression
```

### 体型设计
```
身材比例：
- 头身比2-4（越小越萌）
- 手脚小
- 动作圆润
- 身体柔软

Q版变形：
- 2头身
- 简化四肢
- 夸张表情
```

### 动作设计
```
可爱动作：
- 歪头
- 比心
- 举手
- 蹦跳
- 扑向人
- 蹭蹭
- 嘟嘴
- 眨眼

Prompt:
tilted head, peace sign,
bouncing, glomp, nuzzling,
pouting, wink, kawaii pose
```

## 🎭 可爱场景

### 撒娇场景
```
元素：
- 拉袖子
- 眼睛湿润
- 软软的声音
- "求求了~"

Prompt:
acting spoiled, pulling sleeve,
puppy eyes, soft voice,
cute begging
```

### 吃东西场景
```
元素：
- 鼓起腮帮
- 幸福的表情
- 食物屑在嘴角
- 满足的叹息

Prompt:
eating cutely, puffed cheeks,
happy expression, food crumbs,
satisfied
```

### 困倦场景
```
元素：
- 眼睛半闭
- 揉眼睛
- 打哈欠
- 靠在人身上

Prompt:
sleepy, rubbing eyes,
yawning, drowsy,
leaning on someone
```

### 害羞场景
```
元素：
- 脸红
- 低头
- 手指戳戳
- 声音变小

Prompt:
shy, blushing, looking down,
fidgeting, embarrassed,
bashful
```

## 📋 可爱检查清单

### 设计检查
- [ ] 眼睛够大吗？
- [ ] 表情够软吗？
- [ ] 动作够萌吗？
- [ ] 有没有标志性萌点？

### 场景检查
- [ ] 有没有制造可爱机会？
- [ ] 有没有反差萌？
- [ ] 会让人想截图吗？
- [ ] 会让人想保护吗？
