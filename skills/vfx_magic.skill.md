## 🔧 魔法视觉设计

```
【魔法效果设计】

核心元素：
├─ 主色调：[选择颜色]
├─ 形态：光束/球体/环状/流动
├─ 纹理：光滑/粒子/符文
└─ 强度：微弱→强烈

动态效果：
├─ 蓄力阶段：聚集/旋转
├─ 释放阶段：爆发/射出
├─ 持续阶段：稳定/波动
└─ 消散阶段：淡化/碎裂

附加元素：
├─ 符文：有/无
├─ 魔法阵：简单/复杂
├─ 粒子：稀疏/密集
└─ 音效暗示：震动/发光
```

## 💡 魔法表现技巧

### 施法过程
| 阶段 | 视觉表现 |
|------|----------|
| 吟唱 | 微弱发光 |
| 蓄力 | 能量聚集 |
| 成型 | 形态显现 |
| 释放 | 光芒爆发 |
| 余波 | 粒子消散 |

### 层次感
1. **核心** - 最亮、最饱和
2. **中层** - 主要效果
3. **外围** - 淡化、粒子
4. **环境** - 反光、投影

### 细节增强
- 光线穿透
- 空气扭曲
- 衣物飘动
- 眼睛发光
