# 場景描述 (Scene Description)

## 概述
專業的場景描述方法，涵蓋空間、光線、氛圍、道具等所有視覺元素。

## 場景描述六要素

### 1. 空間 (Space)
- **類型**：室內/室外/半開放
- **大小**：具體尺寸感（狹窄、寬敞、壓迫）
- **結構**：建築特點、空間佈局
- **材質**：木、石、磚、金屬等主要材質

### 2. 光線 (Lighting)
- **光源**：自然光/人工光/混合
- **方向**：正面光/側光/逆光/頂光/底光
- **強度**：明亮/柔和/昏暗/幽暗
- **質感**：硬光（銳利陰影）/軟光（漸變陰影）
- **色溫**：暖光（橙黃）/冷光（藍白）/中性

### 3. 色調 (Color Palette)
- **主色調**：場景的主導顏色
- **輔助色**：點綴和對比色
- **情緒對應**：
  - 紅/橙 → 熱情、危險、緊張
  - 藍/紫 → 憂鬱、神秘、寧靜
  - 綠 → 自然、希望、詭異
  - 黃 → 溫暖、懷舊、警示
  - 黑白灰 → 壓抑、懸疑、記憶

### 4. 氛圍 (Atmosphere)
- **情緒**：緊張/溫馨/恐怖/神秘/歡樂/悲傷
- **氣候**：晴/雨/雪/霧/風
- **溫度感**：炎熱/溫暖/涼爽/寒冷
- **聲音環境**：喧囂/寧靜/自然聲/人聲

### 5. 道具 (Props)
- **大道具**：傢俱、車輛、建築元素
- **中道具**：器具、工具、陳設
- **小道具**：手持物品、桌上物品
- **關鍵道具**：對劇情有意義的物品

### 6. 動態元素 (Dynamic Elements)
- **自然動態**：風吹、水流、光影變化
- **人為動態**：人群、交通、活動
- **時間流逝**：光線變化、季節痕跡

## 常見場景模板

### 🏠 室內場景

#### 中式古宅
```
空間：四合院正廳，高挑的木樑結構，約50平方
材質：紅木樑柱、青磚地面、紙糊窗戶
光線：從天井灑入的自然光，形成明暗對比的光斑
色調：棕紅為主，金色點綴（牌匾、香爐）
氛圍：莊嚴肅穆，香火氣息
道具：
- 大：八仙桌、太師椅、屏風、祖宗牌位
- 中：香爐、花瓶、字畫、燭台
- 小：茶具、扇子、念珠、書籍
動態：香煙裊裊升起，窗紙偶爾被風吹動
```

#### 船艙
```
空間：大蕃舶下層船艙，低矮狹窄，約15平方
材質：厚重木板、麻繩、油布
光線：舷窗透入的有限自然光，油燈補光
色調：暗棕、黃褐、油燈的橙光
氛圍：壓抑潮濕，搖晃不穩
道具：
- 大：木桶、繩索、貨箱、吊床
- 中：油燈、水壺、航海圖、指南針
- 小：錢袋、匕首、乾糧、火摺子
動態：隨波浪輕微晃動，木板吱呀作響
```

#### 茅草棚/卜夢師的居所
```
空間：簡陋茅草棚，約10平方，低矮擁擠
材質：茅草頂、竹編牆、泥土地面
光線：燭光為主，搖曳不定，製造詭異光影
色調：暖黃燭光 + 粉紅寶石光 + 暗影
氛圍：神秘詭異，煙霧繚繞
道具：
- 大：草蓆床鋪、竹簾、小祭壇
- 中：香爐（不斷冒煙）、銅鈴、骨卜用具
- 小：粉紅寶石、銀鏈、符紙、蠟燭、草藥
動態：香煙繚繞，燭火搖曳，寶石時明時暗
```

### 🌳 室外場景

#### 市集/蕃坊街道
```
空間：繁華街道，兩側店鋪林立，約8米寬
材質：青石板路、木造店鋪、布幔招牌
光線：下午斜陽，金黃色調，逆光剪影
色調：金黃、土紅、各色布料點綴
氛圍：熱鬧喧囂，異域風情
道具：
- 大：攤位、騾馬、轎子、貨車
- 中：香料袋、瓷器、絲綢、水果籃
- 小：銅錢、秤、扇子、各國貨幣
動態：人流穿梭、叫賣聲、駝鈴聲
```

#### 碼頭
```
空間：開闊碼頭，遠處海面，近處繁忙
材質：石砌碼頭、木棧橋、繩纜
光線：清晨薄霧中的柔和散射光
色調：青灰、深藍、白帆點綴
氛圍：壯闘啟航、冒險即將開始
道具：
- 大：大蕃舶（三桅帆船）、起重吊臂、倉庫
- 中：貨箱、木桶、繩索、漁網
- 小：船票、羅盤、望遠鏡、乾糧袋
動態：海鷗盤旋、帆布鼓風、工人搬貨
```

#### 榕樹下
```
空間：村口巨大榕樹下，開闊陰涼
材質：古榕樹、氣根、青石板凳
光線：樹葉間漏下的斑駁光影
色調：深綠、青灰、月光藍（夜間）
氛圍：神秘、古老、智慧的象徵
道具：
- 大：百年老榕樹、石凳、小廟
- 中：竹杖、草蓆、茶具、棋盤
- 小：落葉、蟲鳴、螢火蟲（夜間）
動態：樹葉沙沙、光影搖曳、蟲鳴鳥叫
```

## 光線設計詳解

### 按時間分類
| 時間 | 光線特點 | 色溫 | 適合情緒 |
|------|---------|------|---------|
| 黎明 | 柔和散射，地平線泛紅 | 暖 | 希望、新生 |
| 上午 | 明亮清爽，側光 | 中性偏暖 | 活力、正常 |
| 正午 | 強烈頂光，陰影短 | 冷 | 緊張、暴露 |
| 下午 | 金黃斜射，長影 | 暖 | 懷舊、溫馨 |
| 黃昏 | 橙紅逆光，剪影 | 極暖 | 浪漫、離別 |
| 夜晚 | 月光/燈光，高對比 | 冷 | 神秘、危險 |

### 按光源分類
| 光源 | 特點 | 用法 |
|------|------|------|
| 燭光 | 搖曳、暖黃、範圍小 | 神秘、親密、不安 |
| 油燈 | 穩定、暖橙、有煙 | 古代室內標配 |
| 火把 | 動態強、橙紅、戲劇性 | 追逐、探險、危險 |
| 月光 | 冷藍、柔和、浪漫 | 夜戲、神秘、孤獨 |
| 星光 | 微弱、點狀、深邃 | 廣闘、渺小、哲思 |
