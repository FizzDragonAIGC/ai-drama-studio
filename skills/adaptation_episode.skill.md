# 📚 分集策略技能

## 元信息
- **技能ID**: adaptation_episode
- **所屬Agent**: 改編Agent
- **參考書籍**: 《電視劇編劇攻略》+ 《故事》
- **難度**: ⭐⭐⭐⭐

## 核心理念

分集 = 把長故事切成獨立又連貫的單元
- 每集是獨立的體驗（有開場有結尾）
- 同時是整體的一部分（推動主線）

## 分集決策矩陣

### 決定集數

| 原著長度 | 建議集數 | 理由 |
|----------|----------|------|
| 5萬字以下 | 4-6集 | 短番，不能稀釋 |
| 5-10萬字 | 8-12集 | 標準季度番 |
| 10-20萬字 | 12-16集 | 可能需要刪減 |
| 20萬字以上 | 24集/多季 | 分季處理 |

### 每集容量

```
24分鐘動畫：
- 10-15個場景
- 1-2個情感高點
- 1個主線推進
- 1個結尾鉤子

45分鐘真人劇：
- 20-25個場景
- 2-3個情感高點
- 主線+副線推進
- 更複雜的結構
```

## 分集類型

### 1. 建立集（Pilot）
```
功能：
- 建立世界觀
- 介紹核心人物
- 設置核心衝突
- 給出「承諾」（這部劇是什麼）

結構：
- 開場：抓住注意力
- 日常→打破日常
- 觸發事件
- 鉤子：讓觀眾想看下一集
```

### 2. 發展集
```
功能：
- 推進主線
- 深化人物
- 複雜化衝突
- 維持興趣

結構：
- 延續上集懸念
- 新的挑戰
- 部分解決+新問題
- 新的懸念
```

### 3. 轉折集（中點）
```
功能：
- 重大信息揭露
- 改變故事方向
- 提升賭注
- 標誌進入下半場

通常在：
全季的中間位置
```

### 4. 低谷集
```
功能：
- 主角最黑暗時刻
- 反思與成長
- 為高潮蓄力
- 情感深度

通常在：
倒數第2-3集
```

### 5. 高潮集
```
功能：
- 最大對決
- 解決核心衝突
- 角色弧光完成
- 主題證明

通常在：
倒數第2集
```

### 6. 結局集
```
功能：
- 收尾所有線索
- 展示新常態
- 情感落點
- 可能的開放結局/續集伏筆

結構：
- 高潮餘波
- 各線收束
- 最終意象
```

## 每集結構模板

### 四幕結構（24分鐘）
```
ACT 1（6分鐘）
├── 開場鉤子
└── 建立本集問題

ACT 2（8分鐘）
├── 嘗試解決
└── 遇到障礙

ACT 3（7分鐘）
├── 障礙升級
└── 小高潮/轉折

ACT 4（3分鐘）
├── 本集收尾
└── 下集懸念
```

## 懸念設計

### 每集必備懸念
```
結尾懸念類型：
- 危機懸念：「主角會死嗎？」
- 選擇懸念：「他會選誰？」
- 謎題懸念：「這是什麼意思？」
- 反轉懸念：「原來是這樣！接下來呢？」
- 情感懸念：「他們會和好嗎？」
```

### 懸念強度規劃
```
集1結尾：中等（吸引繼續）
中間集結尾：輕度-中等
中點結尾：強（大轉折）
倒數第2集結尾：最強（進入高潮）
最終集結尾：解決+開放
```

## 副線編織

### 副線類型
```
A線：主線（每集推進）
B線：主要副線（1-2集推進一次）
C線：次要副線（3-4集推進一次）
D線：伏筆線（偶爾出現，後期payoff）
```

### 多線交織
```
集1：A+B
集2：A+C
集3：A+B
集4：A+B+D（伏筆植入）
...
集10：A+B+C+D（線索匯合）
```
