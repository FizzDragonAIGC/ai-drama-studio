# 小说改编剧本 (Novel to Script Adaptation)

## 概述
将长篇小说（10万字+）改编为影视剧本的系统方法论。

## 核心理论

### 小说vs剧本的本质差异

| 维度 | 小说 | 剧本 |
|------|------|------|
| 叙事 | 可以深入内心 | 只能展示外在 |
| 时间 | 自由跳跃 | 需要紧凑 |
| 信息 | 可以解释背景 | 必须视觉化 |
| 节奏 | 读者自控 | 创作者控制 |
| 长度 | 10万字+ | 每集约1万字 |

### 改编三原则
1. **忠于精神，不必忠于文字** - 保留内核，调整形式
2. **视觉化优先** - 内心独白→外在行动
3. **戏剧性集中** - 删减支线，强化主线

## 改编流程（10万字小说）

### 第一阶段：拆解原著（1周）

**Step 1: 提取骨架**
- 列出所有主要事件
- 识别因果链条
- 标记高潮点

**Step 2: 角色谱系**
- 主角/配角/龙套分类
- 角色功能分析
- 可合并的角色

**Step 3: 主题提炼**
- 原著的核心表达
- 隐含的价值观
- 时代意义

### 第二阶段：分集大纲（2周）

**Step 4: 确定集数**
- 10万字 ≈ 12-16集（每集8-10分钟）
- 或 20-24集（每集5分钟短剧）

**Step 5: 切分节点**
- 每集一个小高潮
- 每3-4集一个大高潮
- 每集结尾留钩子

**Step 6: 大纲模板**
```
第X集：《标题》

【本集概要】50字

【情节点】
- 开场：
- 发展：
- 转折：
- 高潮：
- 钩子：

【出场角色】

【关键场景】

【情感目标】观众在这集要感受到___

【留悬念】观众会问___
```

### 第三阶段：剧本写作（4周）

**Step 7: 场景清单**
- 每集拆分为15-25个场景
- 标注场景功能（信息/情感/动作）

**Step 8: 对白写作**
- 潜台词优先
- 角色声音差异化
- 删除解释性对白

**Step 9: 动作描写**
- 只写能看到的
- 动词优先
- 避免形容词

## 常见改编难题

### 难题1：内心戏太多
**解决**：
- 内心独白 → 与他人对话
- 内心挣扎 → 外在行动选择
- 回忆 → 视觉化闪回

### 难题2：时间跨度大
**解决**：
- 字幕标注
- 角色外貌变化
- 季节/场景变化
- 删除非必要时间段

### 难题3：角色太多
**解决**：
- 功能相似的角色合并
- 删除对主线无影响的角色
- 龙套只保留功能

### 难题4：世界观复杂
**解决**：
- 分批释放信息
- 通过角色对话自然解释
- 视觉化世界观设定

## Prompt模板

### 小说分析
```
分析这部小说的改编潜力：

【原著信息】
书名：
字数：
类型：

【分析维度】
1. 核心故事线（一句话）：
2. 主要角色数量：
3. 时间跨度：
4. 场景数量：
5. 内心戏占比：
6. 对话占比：
7. 改编难度：简单/中等/困难
8. 建议集数：
```

### 分集大纲生成
```
将以下小说内容改编为第X集大纲：

【原著章节】
{chapter_content}

【前集回顾】
{previous_episode_summary}

【本集要求】
- 时长：X分钟
- 镜头数：X个
- 必须包含：{key_event}
- 结尾钩子：{cliffhanger}

请输出：
1. 本集标题
2. 50字概要
3. 5个关键情节点
4. 主要场景列表
5. 情感曲线
6. 结尾悬念
```

### 剧本写作
```
将以下大纲转化为分镜剧本：

【大纲】
{outline}

【风格要求】
- 视觉风格：
- 节奏：快/中/慢
- 情绪基调：

【输出格式】
每个场景包含：
- 场景编号
- 场景描述（环境/时间/天气）
- 动作描写
- 对白
- 镜头建议
```

## 检查清单

### 大纲阶段
- [ ] 核心主题是否保留？
- [ ] 每集是否有钩子？
- [ ] 角色是否精简？
- [ ] 时间线是否清晰？

### 剧本阶段
- [ ] 对白是否自然？
- [ ] 动作是否可拍？
- [ ] 节奏是否紧凑？
- [ ] 信息是否过载？
