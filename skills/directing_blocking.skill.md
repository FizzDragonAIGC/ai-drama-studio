# 🎭 场面调度 Skill
> 来源：《Directing Actors》Judith Weston / 《On Directing Film》David Mamet

---

## 📐 场面调度定义

> **Blocking**：演员在空间中的位置、移动和互动。
> **Staging**：如何安排场景元素来讲故事。

---

## 🎯 场面调度五要素

### 1. 位置 (Position)
角色在画面中的位置
```
┌─────────────────────┐
│  □            □     │  后景
│                     │
│      □     □        │  中景
│                     │
│  ○                  │  前景
└─────────────────────┘
    左     中     右
```

**位置含义**：
| 位置 | 含义 |
|------|------|
| 前景 | 重要/强势 |
| 背景 | 次要/弱势 |
| 中心 | 焦点 |
| 边缘 | 孤立/边缘化 |
| 高处 | 权力/控制 |
| 低处 | 弱势/服从 |

### 2. 距离 (Distance)
角色之间的距离

| 距离 | 含义 |
|------|------|
| 亲密 (<0.5m) | 爱情、冲突、威胁 |
| 个人 (0.5-1.2m) | 私人对话 |
| 社交 (1.2-3.6m) | 正式交谈 |
| 公众 (>3.6m) | 演讲、权力 |

### 3. 高度 (Height)
角色的高低关系

```
站立 → 坐着 → 蹲下 → 躺下
 ↓
权力递减
```

### 4. 角度 (Angle)
角色面对的方向

| 角度 | 含义 |
|------|------|
| 面对面 | 对抗/亲密 |
| 并肩 | 合作/友谊 |
| 背对背 | 冲突/分离 |
| 侧面 | 回避/思考 |
| 背对观众 | 神秘/退出 |

### 5. 移动 (Movement)
角色的动作轨迹

| 移动 | 含义 |
|------|------|
| 走向 | 接近/对抗 |
| 远离 | 逃避/退缩 |
| 绕行 | 试探/谨慎 |
| 穿越 | 打破/介入 |
| 固定 | 坚定/僵持 |

## 📊 三角构图

最稳定的多人构图：

```
        A
       /│\
      / │ \
     /  │  \
    B───┼───C
```

**使用**：
- 对话场景
- 权力关系
- 焦点转移

## 🎭 情感距离设计

### 情感曲线反映在距离
```
冷淡 ──────────────────→ 亲密
 │                        │
 │  远              近    │
 │  3m    2m    1m   0.5m │
 │                        │
```

### 关系变化通过距离
```
场景开始：两人距离3米（陌生）
     ↓
中间：逐渐靠近
     ↓
高潮：面对面或并肩（和解/冲突）
```

## 📋 场面调度检查清单

### 每个场景
- [ ] 角色位置有意义？
- [ ] 权力关系清晰？
- [ ] 情感距离恰当？
- [ ] 移动有理由？
- [ ] 观众知道看哪里？

### 避免
- ❌ 角色无故移动
- ❌ 忽略空间关系
- ❌ 平面站位
- ❌ 违反物理逻辑
- ❌ 混乱的视线

## 🎬 经典调度分析

### 《教父》开场
- 位置：教父在暗处，求助者在光中
- 距离：开始很远（权力距离）
- 移动：求助者逐渐走近
- 含义：权力的臣服

### 《好家伙》餐厅长镜头
- 移动：穿过后厨，穿过人群
- 含义：黑帮世界的深入
- 技巧：持续移动，不剪辑
