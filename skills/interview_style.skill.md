## 核心理念

風格 = 你做的所有選擇的總和
- 不是「你喜歡什麼風格」
- 而是「你會怎麼選擇」

## 風格三角定位

```
        [寫實]
         /\
        /  \
       /    \
      /      \
     /________\
[抽象]        [符號]
```

### 定位問題
```
「人物造型偏寫實還是卡通？」
「場景細節要多還是簡潔？」
「顏色是真實的還是表現性的？」
「用0-10分，0是《你的名字》，10是《瑞克和莫蒂》，你在哪？」
```

## 六大風格維度

### 1. 色彩風格
```
□ 高飽和 vs □ 低飽和
□ 暖色調 vs □ 冷色調  
□ 對比強烈 vs □ 柔和過渡
□ 統一色板 vs □ 多變色彩

參考：「像《愛樂之城》還是《銀翼殺手》？」
```

### 2. 線條風格
```
□ 粗線條 vs □ 細線條
□ 乾淨利落 vs □ 粗糙手繪
□ 幾何 vs □ 有機曲線
□ 有描邊 vs □ 無描邊

參考：「像《進擊的巨人》還是《你的名字》？」
```

### 3. 構圖風格
```
□ 對稱 vs □ 非對稱
□ 密集 vs □ 留白
□ 穩定 vs □ 動態
□ 規矩 vs □ 打破框架

參考：「像韋斯·安德森還是克里斯托弗·諾蘭？」
```

### 4. 動態風格
```
□ 流暢 vs □ 頓挫
□ 誇張 vs □ 克制
□ 快速 vs □ 緩慢
□ 寫實物理 vs □ 漫畫物理

參考：「像《鬼滅之刃》還是《昨日青空》？」
```

### 5. 敘事風格
```
□ 線性 vs □ 非線性
□ 旁白 vs □ 無旁白
□ 緊湊 vs □ 留白
□ 解釋型 vs □ 暗示型

參考：「像《盜夢空間》還是《千與千尋》？」
```

### 6. 情緒風格
```
□ 內斂 vs □ 外放
□ 寫實情感 vs □ 戲劇化
□ 苦中作樂 vs □ 正面對決
□ 留有餘韻 vs □ 情緒宣洩

參考：「像《情書》還是《你的婚禮》？」
```

## 快速定位法

### 二選一遊戲
```
「這部作品更像：」
- 夏天 or 冬天？
- 白天 or 晚上？
- 咖啡 or 茶？
- 搖滾 or 古典？
- 手寫信 or 短信？
```

### 參考作品矩陣
```
「請選出3部最接近你想要感覺的作品」
「這3部作品的共同點是什麼？」
「如果要結合它們的優點，是什麼？」
```

### 反面風格
```
「絕對不要像哪部作品？」
「那部作品哪裡讓你不舒服？」
「我們如何避免那種感覺？」
```

## 風格衝突檢測

| 維度 | 選擇A | 選擇B | 衝突嗎？ |
|------|-------|-------|----------|
| 敘事 | 快節奏 | 文藝感 | ⚠️ 需協調 |
| 視覺 | 寫實 | 誇張表情 | ⚠️ 需協調 |
| 情緒 | 克制 | 熱血 | ⚠️ 需協調 |

**發現衝突時**
```
「我注意到你想要XX但也想要YY，
 這兩個通常會衝突，
 我們要怎麼平衡？
 或者你更傾向哪一邊？」
```
