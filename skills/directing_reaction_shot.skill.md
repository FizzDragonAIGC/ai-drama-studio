# 🎬 反应镜头 Skill
> 来源：《剪辑的语法》Roy Thompson + 希区柯克剪辑理论

---

## 📐 什么是Reaction Shot（反应镜头）

**定义**：展示角色对某事件/台词/动作的反应的镜头

### 库里肖夫效应
```
同一张无表情的脸 + 不同画面 = 不同解读

脸 + 汤 = 饥饿
脸 + 棺材 = 悲伤
脸 + 美女 = 欲望

→ 反应镜头的力量：语境决定意义
```

---

## 🎯 反应镜头类型

### 1. 听者反应
```
A说话 → 切到B的反应
┌──────────┐    ┌──────────┐
│ A: "..." │ →  │   B: 😮   │
└──────────┘    └──────────┘
用途：展示台词的影响
```

### 2. 观察者反应
```
事件发生 → 切到旁观者反应
┌──────────┐    ┌──────────┐
│  💥事件   │ →  │   C: 😱   │
└──────────┘    └──────────┘
用途：通过他人反应强调事件
```

### 3. 群体反应
```
事件 → 切到人群反应
┌──────────┐    ┌──────────┐
│   事件   │ →  │ 😮😱😨👀  │
└──────────┘    └──────────┘
用途：强调事件的广泛影响
```

### 4. 延迟反应
```
事件 → 停顿 → 反应
让观众先消化 → 再看到角色反应
用途：强调震惊、需要处理时间
```

## 🔧 反应镜头用途

### 情感放大
| 原始事件 | + 反应镜头 | 效果 |
|----------|------------|------|
| 角色说"我爱你" | 对方惊讶特写 | 强调告白的重量 |
| 爆炸发生 | 路人尖叫 | 强调冲击力 |
| 真相揭露 | 角色崩溃 | 强调真相残酷 |

### 信息传递
```
观众不知道发生了什么时：
反应镜头告诉观众"应该怎么感受"

角色震惊 → 观众知道"这是大事"
角色微笑 → 观众知道"是好事"
```

### 悬念构建
```
希区柯克技巧：
先给观众看到危险 → 角色不知道
切到角色正常反应 → 紧张！

炸弹计时 → 角色聊天 → 炸弹特写 → 角色聊天
观众知道角色不知道的事 = 悬念
```

## 📋 反应镜头设计

### 时长选择
| 反应强度 | 建议时长 | 情况 |
|----------|----------|------|
| 轻微 | 0.5-1秒 | 日常反应 |
| 中等 | 1-2秒 | 一般情绪 |
| 强烈 | 2-4秒 | 重要时刻 |
| 延长 | 4秒+ | 关键情感高潮 |

### 景别选择
| 景别 | 效果 |
|------|------|
| 特写 | 微表情、强情感 |
| 近景 | 表情+部分肢体 |
| 中景 | 肢体语言、多人 |
| 全景 | 群体反应 |

### 反应顺序
```
单人反应：
事件 → 最重要角色反应

多人反应：
事件 → 主角反应 → 配角反应
或
事件 → 群体反应 → 主角反应（反衬）
```

## 🎯 反应镜头剪辑

### 剪辑时机
```
❌ 错误：A说完 → 立刻切B
   （太机械）

✓ 正确：A说到一半 → 切B反应 → 回A
   （自然、有对话感）
```

### L-Cut / J-Cut
```
L-Cut：
A画面 + A声音 → A画面 + B声音（声音先）
└─ 画面 ─┘└─ 画面 ─┘
└───── 声音 ─────┘

J-Cut：
A画面 + B声音 → B画面 + B声音（画面先）
└───── 声音 ─────┘
└─ 画面 ─┘└─ 画面 ─┘

用于反应镜头更自然
```

## 🎬 AI番剧应用

### 反应表情Prompt
| 反应类型 | Prompt关键词 |
|----------|--------------|
| 惊讶 | surprised, shocked, wide eyes, open mouth |
| 愤怒 | angry reaction, furious, clenched jaw |
| 悲伤 | sad reaction, teary, looking down |
| 恐惧 | scared, terrified, frozen in fear |
| 喜悦 | happy reaction, beaming smile, joyful |
| 困惑 | confused, puzzled, furrowed brow |

### 反应镜头场景模板
```
事件镜头：
"[事件描述], dramatic moment"

反应镜头：
"close-up [角色], [反应表情], 
reaction shot, [额外情绪细节]"
```

### 示例
```
事件：女主角告白

镜头1：
"girl confessing love, nervous expression, 
looking at camera, soft lighting"

镜头2（反应）：
"close-up boy's face, surprised expression,
wide eyes, slightly open mouth, reaction shot,
touched and moved"
```

## 💡 反应镜头检查

### Do's
- ✅ 在关键时刻使用
- ✅ 景别与情感强度匹配
- ✅ 给足够时长让观众读取
- ✅ 反应符合角色性格

### Don'ts
- ❌ 每句话都切反应（节奏拖沓）
- ❌ 反应与事件不匹配
- ❌ 反应太短看不清
- ❌ 反应过于夸张（除非喜剧）
