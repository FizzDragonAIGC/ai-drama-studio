## 📐 表情基础

### 面部肌肉组
```
眉毛区 → 思考、惊讶、愤怒
眼睛区 → 情感窗口、注视方向
鼻子区 → 厌恶、呼吸
嘴巴区 → 情绪、说话、表态
脸颊区 → 辅助表达
```

### 六大基本表情

| 表情 | 眉毛 | 眼睛 | 嘴巴 |
|------|------|------|------|
| 快乐 | 微抬 | 眯起/弯曲 | 上扬/露齿 |
| 悲伤 | 内收下垂 | 下垂/湿润 | 下垂/颤抖 |
| 愤怒 | 紧皱下压 | 瞪大/眯起 | 紧闭/咬牙 |
| 恐惧 | 上扬 | 睁大 | 张开/颤抖 |
| 惊讶 | 上扬 | 睁圆 | 张开O形 |
| 厌恶 | 单边皱起 | 眯起 | 撇嘴/皱鼻 |

## 🎭 表情设计原则

### 1. 清晰可读
远距离也能识别情绪

### 2. 夸张适度
```
写实风格：微妙夸张
动画风格：大幅夸张
喜剧风格：极致夸张
```

### 3. 一致性
保持角色表情风格统一

### 4. 非对称性
```
对称表情 → 假、僵硬
非对称表情 → 真、自然
```

### 5. 时机感
表情变化需要时间：
```
预备 → 主表情 → 延续/缓和
```

## 🔧 表情设计流程

### Step 1: 定义角色性格
```
外向？内向？
情绪化？冷静？
表情丰富？还是面瘫？
```

### Step 2: 设计基础表情
```
画出6大基本表情
确定这个角色的"版本"
```

### Step 3: 扩展表情库
```
每个基本表情扩展3-5级
添加复合表情
添加角色专属表情
```

### Step 4: 动态测试
```
想象表情转换过程
测试动画可行性
确保转换自然
```

## 📊 复合表情

### 情绪混合

| 混合 | 结果 | 例子 |
|------|------|------|
| 开心+悲伤 | 苦笑 | 强颜欢笑 |
| 愤怒+悲伤 | 悲愤 | 含泪愤怒 |
| 恐惧+愤怒 | 狂暴 | 绝望反击 |
| 惊讶+开心 | 惊喜 | 意外收获 |
| 厌恶+愤怒 | 鄙视 | 轻蔑看人 |
