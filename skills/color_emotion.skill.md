## 📐 色彩情感对照表

### 基础色彩情感

| 颜色 | 正面情感 | 负面情感 | 典型用法 |
|------|----------|----------|----------|
| 🔴 **红色** | 爱情、热情、力量 | 愤怒、危险、暴力 | 爱情片、动作片高潮 |
| 🟠 **橙色** | 温暖、活力、创意 | 警告、不安 | 日落、温馨场景 |
| 🟡 **黄色** | 快乐、希望、能量 | 疯狂、疾病、背叛 | 喜剧、阳光场景 |
| 🟢 **绿色** | 自然、成长、安全 | 嫉妒、毒药、腐败 | 自然、超自然 |
| 🔵 **蓝色** | 冷静、忧郁、信任 | 冷漠、悲伤、孤独 | 科幻、悲剧 |
| 🟣 **紫色** | 神秘、高贵、灵性 | 死亡、腐朽 | 魔法、梦境 |
| ⚪ **白色** | 纯洁、希望、空虚 | 死亡、冰冷、无菌 | 医院、天堂、极简 |
| ⚫ **黑色** | 优雅、力量、神秘 | 死亡、邪恶、未知 | 黑色电影、反派 |

## 📊 配色方案

### 1. 单色 (Monochromatic)
同一颜色的不同深浅
**效果**：统一、沉浸、风格化

### 2. 互补色 (Complementary)
色轮对面的颜色（红-绿、蓝-橙、黄-紫）
**效果**：对比强烈、冲突、戏剧性

### 3. 类似色 (Analogous)
色轮相邻的颜色
**效果**：和谐、自然

### 4. 三角色 (Triadic)
色轮等距的三个颜色
**效果**：活泼、平衡

## 🎯 场景色彩设计

### 情绪-色彩推荐

| 情绪 | 主色 | 辅色 | 光线 |
|------|------|------|------|
| 浪漫 | 粉/红 | 金/暖白 | 柔和、暖色 |
| 悲伤 | 蓝/灰 | 冷白 | 低对比、阴天 |
| 恐怖 | 黑/深红 | 绿 | 高对比、阴影 |
| 科幻 | 蓝/青 | 白 | 冷色、霓虹 |
| 史诗 | 金/暖色 | 深蓝 | 日出日落光 |
| 喜剧 | 明亮多彩 | - | 均匀、明亮 |
| 惊悚 | 绿/黄 | 黑 | 不稳定 |
