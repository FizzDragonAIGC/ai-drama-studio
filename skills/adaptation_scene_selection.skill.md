# 📚 場景篩選技能

## 元信息
- **技能ID**: adaptation_scene_selection
- **所屬Agent**: 改編Agent
- **參考書籍**: 《改編的藝術》Linda Seger
- **難度**: ⭐⭐⭐⭐

## 核心理念

場景篩選 = 做減法的藝術
- 小說100個場景 → 番劇可能只能容納30個
- 選擇的標準：對故事核心的貢獻度

## 場景評估矩陣

### 三維度評分

| 維度 | 1分 | 3分 | 5分 |
|------|-----|-----|-----|
| 劇情推動 | 無關主線 | 間接相關 | 直接推動 |
| 人物塑造 | 無新信息 | 補充細節 | 關鍵轉變 |
| 情感價值 | 平淡 | 有感覺 | 高潮點 |

**總分>12：必保留**
**總分9-12：條件保留**
**總分<9：考慮刪除**

## 場景分類

### A類：核心場景（必保留）
```
特徵：
- 推動劇情轉折
- 角色關鍵決定
- 情感高潮
- 標誌性場景（讀者都記得的）

數量控制：
占總場景30%
```

### B類：支撐場景（選擇保留）
```
特徵：
- 為A類場景做鋪墊
- 提供必要信息
- 增強人物深度
- 節奏調節

數量控制：
占總場景40%
選擇性保留60-70%
```

### C類：氛圍場景（合併或刪除）
```
特徵：
- 環境描寫
- 日常生活
- 重複性互動
- 單純過渡

處理方式：
- 合併多個為一個
- 用旁白/字幕帶過
- 直接刪除
```

## 場景保留決策樹

```
這個場景對主線劇情有推動嗎？
├── 是 → 保留
└── 否 → 
    這個場景對主角有塑造嗎？
    ├── 是 → 保留（可精簡）
    └── 否 → 
        這個場景有獨特視覺價值嗎？
        ├── 是 → 考慮保留
        └── 否 → 
            刪除這個場景會影響理解嗎？
            ├── 是 → 保留（精簡為1-2鏡）
            └── 否 → 刪除
```

## 場景合併技巧

### 1. 功能合併
```
原著：
- 場景A：主角得知信息X
- 場景B：主角得知信息Y

改編：
- 新場景：主角同時得知X和Y
```

### 2. 空間合併
```
原著：
- 咖啡廳對話
- 家中對話
- 辦公室對話（內容類似）

改編：
- 全部合併在一個地點
```

### 3. 角色合併
```
原著：
- 與角色A的互動
- 與角色B的互動（功能相似）

改編：
- 合併為一個角色
- 在一個場景完成兩個功能
```

## 必保場景清單

### 結構必備
```
□ 建立場景（世界觀/日常）
□ 觸發場景（打破日常）
□ 鎖定場景（確定方向）
□ 中點場景（重大轉折）
□ 低谷場景（最黑暗時刻）
□ 高潮場景（最終對決）
□ 結局場景（新的平衡）
```

### 情感必備
```
□ 主角脆弱時刻
□ 關係建立時刻
□ 關係破裂時刻
□ 和解/告別時刻
□ 成長證明時刻
```

## 刪除補償策略

### 刪除場景的信息如何補回
```
方法1：對話植入
「你還記得上次...」

方法2：視覺閃回
1-2秒的閃回畫面

方法3：旁白帶過
「那個夏天，他們一起...」

方法4：省略（觀眾可以腦補）
不是所有信息都必須呈現
```
