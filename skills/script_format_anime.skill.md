# 🇯🇵 番剧剧本格式 (Anime Script Format)
> 日式动画番剧剧本写作规范

---

## 📋 番剧基本结构

### 标准时长
| 类型 | 时长 | 实际内容 |
|------|------|----------|
| TV番剧 | 24分钟 | 22分钟（含OP/ED） |
| 短番 | 12分钟 | 10分钟 |
| 泡面番 | 3-5分钟 | 3分钟 |
| 剧场版 | 90-120分钟 | 全片 |

### 单集结构（24分钟）
```
┌────────────────────────────────────────────┐
│ 00:00-01:30  开场前情/钩子（Avante）        │
├────────────────────────────────────────────┤
│ 01:30-03:00  片头曲（OP）                   │
├────────────────────────────────────────────┤
│ 03:00-10:00  A Part（第一幕）              │
├────────────────────────────────────────────┤
│ 10:00-10:15  Eyecatch（过场画面）           │
├────────────────────────────────────────────┤
│ 10:15-20:00  B Part（第二幕）              │
├────────────────────────────────────────────┤
│ 20:00-21:30  片尾曲（ED）                   │
├────────────────────────────────────────────┤
│ 21:30-23:00  下集预告（Preview）            │
├────────────────────────────────────────────┤
│ 23:00-24:00  C Part（彩蛋/悬念）           │
└────────────────────────────────────────────┘
```

## 🎬 场景格式

### 标准写法
```
●シーン1　学校・教室　昼
（场景1 学校・教室 日）

描写：阳光从窗户照进来，黑板上还留着上节课的板书。

太郎（坐在窗边座位）
    今天的天气真好啊...

花子（走过来）
    在发什么呆？快上课了。

●シーン2　学校・走廊　同刻
（场景2 学校・走廊 同一时刻）
```

### AI番剧改良格式
```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
第X集 第Y场
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【场景】内景 - 地点 - 时间

【环境】
简短描述环境氛围，2-3句话。
注明光线、天气、背景音。

【动作】
角色动作描写。

角色名：对白内容
（括号内为语气/表情/小动作）

【画面建议】
镜头类型 + 构图 + 特效建议

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 🎭 番剧特有元素

### 表情符号化
| 元素 | 视觉表现 | 剧本写法 |
|------|----------|----------|
| 汗滴 | 额头大汗珠 | 【汗】或（冷汗） |
| 愤怒 | 井字纹 | 【怒】或（青筋暴起） |
| 惊讶 | 眼睛放大 | 【惊】或（瞳孔放大） |
| 脸红 | 腮红 | 【羞】或（双颊泛红） |
| 无语 | 三条线 | 【囧】或（无语状） |
| 发光 | 闪闪发亮 | 【闪】或（眼睛发光） |

### 番剧动作描写
```
太郎
    这不可能！！

【画面】太郎后退一步，背景崩裂成碎片，
       脚下出现冲击波纹，头发被气流吹起。
       
【特效】速度线 + 震撼波 + 碎片飞散
```

### 内心独白
```
太郎（内心OS）
    她为什么要这样说...
    难道她知道了什么？

【画面】太郎表情凝固，背景变暗，
       周围人物变成剪影。
```

## 📖 番剧节拍结构

### 12集番剧标准结构
| 集数 | 节拍 | 功能 |
|------|------|------|
| 1 | 开场 | 世界观建立，主角登场，激励事件 |
| 2 | 入门 | 主角进入新世界/接受使命 |
| 3 | 日常 | 展示新世界，建立关系 |
| 4 | 第一场考验 | 小BOSS战/初次挫折 |
| 5 | 训练/成长 | 主角变强，学习新技能 |
| 6 | 中点大战 | 假胜利，揭示更大威胁 |
| 7 | 敌人反击 | 坏人变强，盟友受伤 |
| 8 | 最低点 | 主角失败，陷入绝望 |
| 9 | 觉醒 | 主角顿悟，新力量觉醒 |
| 10 | 反攻 | 收集盟友，准备最终战 |
| 11 | 最终决战（上） | 高潮开始，激烈战斗 |
| 12 | 最终决战（下）+ 结局 | 胜利，世界恢复，伏笔 |

### 单集A/B Part结构
```
【A Part - 问题提出】
- 场景1-3：日常/平静
- 场景4：问题/威胁出现
- 场景5-6：初步应对
- Eyecatch前：小悬念

【B Part - 问题解决】
- 场景7-8：深入问题
- 场景9-10：危机升级
- 场景11：高潮/决战
- 场景12：解决/余韵
- C Part：下集伏笔
```

## 🎬 番剧常用技法

### 定番演出
| 名称 | 描述 | 使用时机 |
|------|------|----------|
| **变身** | 华丽变身过程 | 战斗前 |
| **必杀技** | 大招释放动画 | 高潮决战 |
| **回忆杀** | 插入回忆片段 | 情感高潮 |
| **立FLAG** | 死亡/分离预兆 | 悲剧前 |
| **热血BGM** | 配乐突然响起 | 逆转时刻 |
| **ED前悬念** | 留在关键时刻 | 每集结尾 |

### 战斗场景模板
```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
第8场 高潮决战
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【场景】外景 - 废墟战场 - 黄昏

【环境】
夕阳将整片废墟染成血红色。
风卷起尘土，远处有建筑崩塌的声音。

【画面】大远景，两个身影对峙。

主角（站定，喘息）
    这一次...我不会再逃了！

反派（冷笑）
    有意思。

【动作】
反派消失——

【特效】瞬移模糊 + 残影

下一秒出现在主角面前！

【画面】极速拉近，反派拳头逼近镜头。

主角瞳孔收缩。

【慢镜】0.5倍速

主角侧身闪避，同时出拳反击——

【特效】速度线 + 冲击波 + 地面碎裂

两人拳头相撞！

【画面】巨大冲击波向四周扩散。

【音效】BOOM!!!

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 🗣️ 番剧对白特点

### 角色口癖
| 类型 | 示例 |
|------|------|
| 结尾语气 | ～だぜ（阳光男）～わ（大小姐） |
| 口头禅 | "真是的～""切～""啊啦啊啦" |
| 自称 | 我/吾/本大爷/人家 |
| 敬称 | ～君/～さん/～前輩 |

### 番剧金句模板
- 热血宣言："我要成为__的男人！"
- 羁绊台词："我们是__！"
- 决战前："这一次，绝不后退！"
- 牺牲前："接下来，就拜托你了..."
- 逆转时："你以为这样就结束了吗？"

## 📋 下集预告格式

```
【下集预告】

【画面】快速剪辑下集精彩画面

旁白（神秘语气）
    命运的齿轮开始转动。
    当真相浮出水面，
    等待他们的将是...？

【画面】下集高潮镜头定格

太郎（画外音）
    下次的《XXXX》——
    
【字幕】第X集「标题」

太郎（画外音）
    敬请期待！
```
