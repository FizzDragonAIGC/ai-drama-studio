# 📺 时长驱动的智能体架构设计

## 核心理念
> 不同时长 = 不同叙事逻辑 = 不同编剧智能体

---

## 📊 三种时长模式

| 模式 | 时长/集 | 类型 | 核心吸引力 | 钩子设计 |
|------|---------|------|-----------|----------|
| **短剧** | 1-3分钟 | 竖屏短剧 | 即时反转、爽点密集 | 每30秒一个反转 |
| **番剧** | 5-8分钟 | AI番剧 | 情感共鸣、视觉冲击 | 每集一个情感高潮+悬念 |
| **剧集** | 10-15分钟+ | 电视剧/电影 | 人物深度、情节复杂 | 多线交织、角色弧光 |

---

## 🎬 短剧模式 (1-3分钟/集)

### 核心特征
- **节奏**：极快，平均3秒一个镜头切换
- **钩子**：前3秒必须抓住观众
- **反转**：每30-60秒一个小反转
- **情绪**：单一情绪放大（爽/甜/虐/搞笑）
- **结尾**：必须有悬念，逼着看下一集

### 需要的Skills
1. `short_drama_hook.skill.md` - 3秒钩子设计
2. `short_drama_pacing.skill.md` - 极速节奏控制
3. `short_drama_reversal.skill.md` - 反转技巧
4. `short_drama_cliffhanger.skill.md` - 悬念设计
5. `short_drama_algorithm.skill.md` - 完播率优化

### 剧本格式
```
【第X集】标题 (90秒)

[0-3秒] 钩子
画面：冲击性视觉
台词：一句话抓住注意力

[3-30秒] 建立冲突
快速切镜头，建立问题

[30-60秒] 升级
冲突加剧，观众紧张

[60-85秒] 反转
意想不到的转折

[85-90秒] 悬念
留下问题，必须看下一集
```

## 📺 番剧模式 (5-8分钟/集)

### 核心特征
- **节奏**：张弛有度，有呼吸感
- **情感**：每集一个完整情感弧线
- **视觉**：强调画面美感和氛围
- **角色**：有成长但不急于一集完成
- **结尾**：情感高潮 + 下集悬念

### 需要的Skills
1. `anime_episode_structure.skill.md` - A/B Part结构
2. `anime_emotional_arc.skill.md` - 单集情感曲线
3. `anime_visual_narrative.skill.md` - 视觉叙事
4. `anime_character_moment.skill.md` - 角色高光时刻
5. `anime_cliffhanger.skill.md` - 番剧式悬念

### 剧本格式
```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
第X集「标题」(8分钟)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【A Part】(0:00-4:00)

场景1: 开场钩子/前情回顾
场景2: 日常/建立
场景3: 问题出现
场景4: 初步应对

【Eyecatch】(4:00)

【B Part】(4:00-7:30)

场景5: 深入问题
场景6: 危机升级
场景7: 情感高潮
场景8: 解决/余韵

【下集预告】(7:30-8:00)
悬念揭示

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 🎭 剧集模式 (10-15分钟+/集)

### 核心特征
- **节奏**：有完整的三幕结构
- **人物**：深度挖掘，有内心戏
- **情节**：多线并进，相互交织
- **对白**：有潜台词，有层次
- **结尾**：可以是完整故事，也可以连续

### 需要的Skills
1. `tv_episode_structure.skill.md` - 三幕式单集结构
2. `tv_multiplot.skill.md` - 多线叙事
3. `tv_character_depth.skill.md` - 人物深度
4. `tv_dialogue_subtext.skill.md` - 潜台词对白
5. `tv_serialization.skill.md` - 连续剧设计

### 剧本格式
```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
第X集「标题」(15分钟)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【第一幕 - 建置】(0:00-4:00)
- 主线：开场，问题引入
- 副线A：角色B的困境
- 副线B：伏笔铺设

【第二幕 - 对抗】(4:00-11:00)
- 主线：冲突发展
- 副线A：与主线交织
- 中点：转折/揭示
- 压力升级

【第三幕 - 解决】(11:00-15:00)
- 高潮：主线爆发
- 副线收束或延续
- 结尾：情感落点/下集钩子

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 🔄 自动切换逻辑

```javascript
function selectScreenwriterByDuration(durationPerEpisode) {
    if (durationPerEpisode <= 3) {
        return 'script_shortform';  // 短剧编剧
    } else if (durationPerEpisode <= 8) {
        return 'script_anime';      // 番剧编剧
    } else {
        return 'script_tv';         // 电视剧编剧
    }
}
```
