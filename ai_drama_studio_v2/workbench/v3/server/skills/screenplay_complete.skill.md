## 一、對白寫作（最重要！）

### 🎯 Sorkin對白黃金法則

**1. 對白是衝突的武器**
```
❌ 平淡: "你想喝咖啡嗎？" "好啊。"
✅ 衝突: "你想喝咖啡嗎？" "你什麼時候開始關心我了？"
```

**2. 節奏是音樂**
- 快節奏對話（急促交鋒）
- 長短句交替（呼吸感）
- 打斷、搶話、沉默

**3. 角色聲音獨特化**
| 角色類型 | 說話特點 | 範例 |
|----------|----------|------|
| 知識分子 | 引用、長句、精準用詞 | "正如普魯斯特所說..." |
| 市井小民 | 俚語、短句、重複 | "我跟你說啊，真的，真的" |
| 權力者 | 命令句、停頓、低聲 | "你有三秒鐘。" |
| 焦慮者 | 碎碎念、自問自答 | "不對不對，等等，讓我想想..." |

### 🎭 潛台詞三層結構

```
表層：說出來的話
  ↓
意圖層：真正想要什麼
  ↓
需求層：內心深處的渴望/恐懼
```

**範例：**
```
對白："隨便，你高興就好。"
表層：同意對方
意圖：其實不同意，在表達不滿
需求：害怕被拋棄，不敢正面衝突
```

### ⚠️ 對白禁忌

| 禁忌 | 錯誤示範 | 正確做法 |
|------|----------|----------|
| 直說情感 | "我很傷心" | 用行動/沉默表現 |
| 信息灌輸 | "你知道，我們認識十年了..." | 讓信息自然流出 |
| 同質化 | 所有人說話一個調 | 每人有獨特語癖 |
| 解釋劇情 | "我要去找他報仇因為..." | 讓觀眾自己推理 |
| 空洞寒暄 | "最近怎麼樣？" "還好你呢" | 每句都推動情節 |

### 💡 對白增強技巧

**1. 答非所問**
```
"你愛過我嗎？"
"今天月亮很圓。"  ← 迴避=恐懼
```

**2. 打斷與搶話**
```
"我覺得我們應該——"
"不。"
"你連聽都不聽？"
"我聽了十年了。"
```

**3. 沉默與停頓**
```
他看著她。很久。
她低下頭。
他：「......好。」  ← 省略號=千言萬語
```

**4. 物件代替語言**
```
她把戒指放在桌上。
沒說一句話。
他看著戒指。
他：「......這是什麼意思？」
她：「你知道的。」
```

## 二、動作描寫（Show Don't Tell）

### 🎬 核心原則：每句話=一個鏡頭

```
❌ 錯誤：她感到非常害怕和絕望，不知道該怎麼辦。
✅ 正確：她的手在發抖。杯子裡的水漾起漣漪。她把杯子放下，握緊拳頭。
```

### 📝 動作行格式規範

**1. 現在時態 + 主動語態**
```
❌ 他曾經很害怕 / 他被嚇到了
✅ 他僵住。 / 他倒退一步。
```

**2. 三行原則**
每段動作描寫不超過3行，需要更多時分段：
```
他推開門。陽光刺入眼睛。

屋內一片狼藉。桌椅翻倒。牆上有抓痕。

他的目光落在角落——一個蜷縮的身影。
```

**3. 動詞選擇決定情緒**
| 情緒 | 弱動詞 | 強動詞 |
|------|--------|--------|
| 憤怒 | 看 | 瞪、怒視、盯 |
| 恐懼 | 走開 | 逃、躲、縮 |
| 悲傷 | 哭 | 啜泣、崩潰、顫抖 |
| 愛意 | 看 | 凝視、撫摸、靠近 |

### 🎭 情緒的肢體表達

**替代「他很害怕」：**
```
- 他的手在發抖
- 他咽了口唾沫
- 他不自覺地後退一步
- 他的呼吸急促起來
- 他攥緊了衣角
- 冷汗從額頭滲出
```

**替代「她很生氣」：**
```
- 她的下巴繃緊
- 她把筷子摔在桌上
- 她轉身，不看他
- 她的聲音變得冰冷
- 她的指甲掐進掌心
```

**替代「他很悲傷」：**
```
- 他的眼眶泛紅
- 他低下頭，不讓人看見
- 他的聲音開始發抖
- 他把照片翻過去
- 他站在窗前，很久沒動
```

### 🔍 細節暗示情感

| 物件/動作 | 暗示情感 |
|-----------|----------|
| 反覆整理領帶 | 緊張、不自信 |
| 不停看手機 | 焦慮、等待 |
| 撕碎紙巾 | 壓抑的憤怒 |
| 杯子裡的水沒喝 | 心不在焉、有心事 |
| 煙灰缸堆滿菸蒂 | 長時間的煎熬 |
| 咬嘴唇 | 猶豫、壓抑 |
| 眼神閃躲 | 說謊、心虛 |

## 三、場景描寫（視覺化寫作）

### 🌆 場景開頭：三要素

```
1. 時間感 - 什麼時候？光線如何？
2. 空間感 - 什麼地方？有什麼特徵？
3. 氛圍感 - 什麼情緒？暗示什麼？
```

**範例對比：**
```
❌ 平淡：客廳。白天。小明坐在沙發上。

✅ 有氛圍：
陽光從百葉窗的縫隙擠進來，在地板上畫出一道道光柵。
灰塵在光柱中飛舞。
小明蜷縮在沙發角落，電視開著，但他沒在看。
茶几上，三天前的外賣盒散發著腐臭味。
```

### 🎨 感官細節五層

| 感官 | 描寫內容 | 範例 |
|------|----------|------|
| **視覺** | 光影、色彩、物件 | "夕陽把房間染成橘紅色" |
| **聽覺** | 環境音、人聲、沉默 | "遠處傳來狗吠，然後歸於死寂" |
| **嗅覺** | 氣味、味道 | "空氣中瀰漫著潮濕和腐朽的味道" |
| **觸覺** | 溫度、質感 | "冰涼的金屬把手" |
| **氛圍** | 情緒暗示 | "一切看起來很平靜，過於平靜" |

### 🏠 場景類型與描寫重點

| 場景類型 | 重點 | 範例 |
|----------|------|------|
| 日常場景 | 細節暗示狀態 | 雜亂的房間=內心混亂 |
| 衝突場景 | 空間限制感 | 狹窄的電梯、關著的門 |
| 轉折場景 | 光影變化 | 從陰暗到明亮，或反之 |
| 情感場景 | 環境呼應情緒 | 下雨=悲傷，陽光=希望 |
| 高潮場景 | 動態與混亂 | 東西打翻、玻璃碎裂 |

### ✍️ 場景描寫模板

```
[時間/光線] + [空間概述] + [關鍵細節1] + [關鍵細節2] + [氛圍暗示]

範例：
深夜。月光從破損的屋頂漏進來。
廢棄工廠。生鏽的機器像沉睡的怪獸。
地上散落著玻璃碎片，踩上去嘎吱作響。
遠處，有什麼東西在動。
```

## 四、劇本格式

### 場景標題
```
場號. 場景名 + 時間 + 內/外
例：1. 咖啡館 日 內
```

### 完整場景範例
```
23. 老舊公寓樓道 夜 內

昏暗的走廊。一盞燈忽明忽暗，發出電流的滋滋聲。
牆皮剝落，露出斑駁的磚塊。

小雨站在3B門前。手裡攥著一把鑰匙。
她深吸一口氣。又吐出來。
手抬起。又放下。

樓下傳來嬰兒的哭聲。然後是一聲怒吼。
嬰兒哭得更響了。

小雨閉上眼睛。
她把鑰匙插進鎖孔。

        小雨
    （低聲，像是對自己說）
  三年了。

她轉動鑰匙。門開了。
屋內一片漆黑。
```

## 六、五大電影文化對白風格

### 🇨🇳 中國：含蓄內斂
```
特點：話中有話、借物言情
範例：
    母親
  飯菜涼了。

    兒子
  我吃過了。

    母親
  （收拾碗筷，很久）
  你爸走的時候，也是這麼說的。
```

### 🇯🇵 日本：留白與沉默
```
特點：「間」的運用、未完成的句子
範例：
    男
  其實我……

    女
  嗯。

  （兩人沉默。風吹過。）

    男
  ……沒什麼。

    女
  我知道。
```

### 🇺🇸 好萊塢：機智快節奏
```
特點：金句、互懟、信息密度高
範例：
    探員A
  你有十秒鐘解釋為什麼不該逮捕你。

    嫌疑人
  我只需要五秒：你來錯地方了。

    探員A
  還有五秒。

    嫌疑人
  剩下的五秒你需要跑。
```

### 🇪🇺 歐洲：哲學思辨
```
特點：長對白、自我質問、打破第四面牆
範例：
    男
  你相信命運嗎？

    女
  我相信選擇。

    男
  那如果所有選擇都通向同一個結局呢？

    女
  那至少過程是你自己的。
  （看向鏡頭）
  不是嗎？
```

## 七、質檢清單

### 對白檢查
✅ 每句對白有潛台詞嗎？
✅ 刪掉這句話影響場景嗎？（能刪就刪）
✅ 這句話推動情節/揭示人物了嗎？
✅ 不同角色說話風格有區別嗎？
✅ 有「答非所問」「打斷」「沉默」嗎？

### 動作檢查
✅ 用的是強動詞嗎？
✅ 情緒通過肢體表現了嗎？
✅ 有細節物件暗示嗎？
✅ 每段不超過3行嗎？

### 場景檢查
✅ 有時間/光線描寫嗎？
✅ 有空間細節嗎？
✅ 有氛圍暗示嗎？
✅ 環境呼應角色情緒嗎？
