## 一、台詞功能分類

| 功能 | 說明 | 範例 |
|------|------|------|
| 推進劇情 | 傳遞關鍵信息 | 「他們今晚就會動手。」 |
| 揭示角色 | 展現性格/背景 | 「我這輩子從沒求過人。」 |
| 製造衝突 | 對立觀點碰撞 | 「你根本不懂！」 |
| 建立關係 | 角色互動張力 | 「你什麼時候學會說謊的？」 |
| 埋設伏筆 | 看似無心的暗示 | 「這把鑰匙，你收好。」 |
| 製造笑料 | 喜劇效果 | 機智反諷、誤會對話 |
| 抒發情感 | 情緒宣洩 | 「我只是...太累了。」 |

## 二、台詞層次設計

### 表層 vs 潛台詞
```
表層（說的話）: 「今天天氣不錯。」
潛台詞（真實意圖）: 「我不想談這個話題。」
```

### 潛台詞技巧
| 技巧 | 說明 | 範例 |
|------|------|------|
| 迴避 | 不正面回答 | 「你餓不餓？我去做飯。」 |
| 轉移 | 話題轉向 | 「說起這個，你媽最近怎麼樣？」 |
| 沉默 | 不說比說更多 | （長久的沉默） |
| 反話 | 說相反的意思 | 「隨便你，我無所謂。」 |
| 雙關 | 一語雙關 | 「這盤棋，你想怎麼走？」 |

## 三、對白節奏

### 節奏類型
| 類型 | 特點 | 適用場景 |
|------|------|---------|
| 快速交鋒 | 短句、快接 | 爭吵、對峙 |
| 緩慢沉重 | 長停頓、慢語速 | 悲傷、告白 |
| 自然流暢 | 日常語感 | 生活場景 |
| 詩意獨白 | 文學性強 | 內心戲、旁白 |

### Sorkin式對白（快速機智）
```
A: 你知道這意味著什麼嗎？
B: 我知道。
A: 你不知道。
B: 我知道我不知道，但我知道你會告訴我。
A: 這意味著我們完了。
B: ...好吧，那我確實不知道。
```

### Tarantino式對白（閒聊中見張力）
```
A: 你知道法國麥當勞的四分之一磅漢堡叫什麼嗎？
B: 不知道。
A: 他們叫它皇家芝士堡。
B: 為什麼？
A: 因為他們用公制。
```

## 四、角色聲音區分

### 語言特徵
| 維度 | 選項 | 作用 |
|------|------|------|
| 用詞 | 文雅/粗俗/專業 | 教育背景 |
| 句式 | 長句/短句/破碎 | 性格特徵 |
| 口頭禪 | 習慣用語 | 識別度 |
| 語氣 | 直接/委婉/反諷 | 社交方式 |
| 方言 | 地域口音 | 身份背景 |

### 範例角色台詞設計
**教授（知識分子）**
> 「從認識論的角度來說，你的觀點存在根本性的謬誤。」

**街頭混混**
> 「少跟我廢話，你到底幹不幹？」

**老奶奶（樸實）**
> 「孩子啊，人這輩子，圖個啥呢？」

## 五、對白禁忌

### ❌ 避免
- **過度解釋**: 不要讓角色說出觀眾已經知道的
- **廣播對白**: 「作為你的哥哥，我必須告訴你...」
- **書面語**: 真實對話不會這麼完整通順
- **空洞感嘆**: 「天哪！」「不會吧！」

### ✅ 推薦
- **打斷**: 真實對話常被打斷
- **重疊**: 多人同時說話
- **口誤**: 說錯/改口
- **停頓**: ...（沉默的張力）
